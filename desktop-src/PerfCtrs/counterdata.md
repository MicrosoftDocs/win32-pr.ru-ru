# <a name="counterdata"></a>каунтердата

Таблица **каунтердата** содержит по одной строке для каждого счетчика, который собирается в определенный момент времени. Количество этих строк будет большим.

В таблице **каунтердата** определены следующие поля:

-   **Идентификатор GUID:** Идентификатор GUID для этого набора данных. Используйте этот ключ для объединения с таблицей [**дисплайтоид**](displaytoid.md) .
-   **Каунтерид:** Определяет счетчик. Используйте этот ключ для объединения с таблицей [**каунтердетаилс**](counterdetails.md) .
-   **Рекординдекс:** Пример индекса для указанного идентификатора счетчика и GUID коллекции. Значение увеличивается для каждого последующего образца в этом файле журнала.
-   **Каунтердатетиме:** Время запуска коллекции (в формате UTC).
-   **CounterValue:** Отформатированное значение счетчика. Это значение может быть нулевым для первой записи, если счетчику требуется два образца для расчета отображаемого значения.
-   **Фирствалуеа:** Объедините это 32-разрядное значение со значением **фирствалуеб** , чтобы создать элемент **Фирствалуе** [**\_ \_ счетчика PDH RAW**](/windows/desktop/api/Pdh/ns-pdh-pdh_raw_counter). **Фирствалуеа** содержит младшие биты.
-   **Фирствалуеб:** Объедините это 32-разрядное значение со значением **фирствалуеа** , чтобы создать элемент **Фирствалуе** [**\_ \_ счетчика PDH RAW**](/windows/desktop/api/Pdh/ns-pdh-pdh_raw_counter). **Фирствалуеб** содержит старшие биты.
-   **Секондвалуеа:** Объедините это 32-разрядное значение со значением **секондвалуеб** , чтобы создать элемент **SecondValue** [**\_ \_ счетчика PDH RAW**](/windows/desktop/api/Pdh/ns-pdh-pdh_raw_counter). **Секондвалуеа** содержит младшие биты.
-   **Секондвалуеб:** Объедините это 32-разрядное значение со значением **секондвалуеа** , чтобы создать элемент **SecondValue** [**\_ \_ счетчика PDH RAW**](/windows/desktop/api/Pdh/ns-pdh-pdh_raw_counter). **Секондвалуеб** содержит старшие биты.

Поля **GUID**, **каунтерид** и **рекординдекс** составляют первичный ключ для этой таблицы.
