# <a name="performance-counters-tools"></a>Средства счетчиков производительности

## <a name="data-collection-tools"></a>Средства сбора данных

Следующие средства полезны при использовании данных счетчика производительности Windows или управлении ими.

|Средство|Описание
|----|-----------
| [PerfMon](/windows-server/administration/windows-commands/perfmon) | Средство графического пользовательского интерфейса для сбора и просмотра данных счетчиков производительности. `perfmon.exe` Запускает MMC с оснасткой "системный монитор", которая предоставляет доступ к монитору производительности, наборам сборщиков данных и средствам отчетов.
| [типеперф](/windows-server/administration/windows-commands/typeperf) |Программа командной строки для сбора и печати данных счетчиков производительности. Можно использовать для вывода списка доступных каунтерсетс, вывода списка доступных счетчиков, вывода данных счетчиков на консоль или получения данных счетчиков в файл журнала (CSV, TDF, BLG).
| [Relog](/windows-server/administration/windows-commands/relog) |Программа командной строки для преобразования и слияния файлов журналов (CSV, TDF, BLG), захваченных через `typeperf.exe` или через API- `PDH.dll` интерфейсы.
| [LogMan](/windows-server/administration/windows-commands/logman) |Программа командной строки для управления наборами сборщиков данных.

## <a name="data-provider-tools"></a>Средства поставщика данных

Следующие средства полезны при публикации данных счетчика производительности Windows.

|Средство|Описание
|----|-----------
| [ктрпп](ctrpp.md) | Средство сборки командной строки из Windows SDK, которое проверяет и компилирует манифест поставщика v2 счетчиков производительности. Это средство создает `.h` заголовки и `.rc` сценарии ресурсов, необходимые для создания поставщика v2.
| [LodCtr](/windows-server/administration/windows-commands/lodctr) | Программа командной строки, используемая для установки поставщика в системе.
| [Lodctr](/windows-server/administration/windows-commands/unlodctr_1) | Программа командной строки, используемая для удаления поставщика из системы.
