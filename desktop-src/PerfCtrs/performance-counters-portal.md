# <a name="performance-counters"></a>Счетчики производительности

Счетчики производительности Windows предоставляют высокоуровневые уровни абстракции, обеспечивающие единообразный интерфейс для сбора различных типов системных данных, таких как ЦП, память и использование диска. Системные администраторы часто используют счетчики производительности для мониторинга систем на наличие проблем с производительностью или поведением. Разработчики программного обеспечения часто используют счетчики производительности для проверки использования ресурсов их программ.
