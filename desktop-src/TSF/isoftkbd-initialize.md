# <a name="isoftkbdinitialize-method"></a>Метод Исофткбд:: Initialize

Метод **исофткбд:: Initialize** инициализирует все необходимые поля для мягкой клавиатуры и создает стандартные раскладки клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Initialize();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                | Описание                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод успешно выполнен.<br/> |
