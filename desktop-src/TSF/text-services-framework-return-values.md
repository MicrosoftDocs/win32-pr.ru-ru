# <a name="text-services-framework-return-values"></a>Значения, возвращаемые платформой текстовых служб

Методы интерфейса платформы текстовых служб создают возвращаемые значения в диапазоне от 0x *HHHH* 0200 до 0x *HHHH* 0600, где *HHHH* — это шестнадцатеричное значение, которое зависит от кода серьезности и кода средства конкретного возвращаемого значения.

Возвращаемые значения группируются в следующие категории.

-   [Возвращаемые значения службы текстового ввода](text-service-return-values.md)
-   [Возвращаемые значения диспетчера](manager-return-values.md)
-   [Возвращаемые значения хранилища текста](text-store-return-values.md)
