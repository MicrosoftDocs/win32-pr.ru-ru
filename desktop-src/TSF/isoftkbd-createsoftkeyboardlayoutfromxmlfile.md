# <a name="isoftkbdcreatesoftkeyboardlayoutfromxmlfile-method"></a>Метод Исофткбд:: Креатесофткэйбоардлайаутфромксмлфиле

Метод **исофткбд:: креатесофткэйбоардлайаутфромксмлфиле** создает раскладку экранной клавиатуры из указанного XML-файла.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateSoftKeyboardLayoutFromXMLFile(
  [in]  WCHAR *lpszKeyboardDesFile,
  [in]  INT   szFileStrLen,
  [out] DWORD *pdwLayoutCookie
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпсзкэйбоарддесфиле* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, указывающую имя XML-файла.

</dd> <dt>

*сзфилестрлен* \[ окне\]
</dt> <dd>

Строка, завершающаяся нулем, указывающая длину XML-файла.

</dd> <dt>

*пдвлайауткукие* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает куки-файл раскладки клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>          |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один или несколько параметров недопустимы.<br/> |
