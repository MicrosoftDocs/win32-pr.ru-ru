# <a name="matrixtransformvectors-methods"></a>Методы Matrix. Трансформвекторс

В этом разделе перечислены методы Трансформвекторс класса [**Matrix**](/windows/win32/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) . Полный список методов для класса **Matrix** см. в разделе [методы матрицы](-gdiplus-class-matrix-methods.md).

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                 | Описание                                                                                                                                                                                                                                                                                                                                                                      |
|:-------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Трансформвекторс (Point \* , int)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformvectors(inoutpoint_inint))   | Метод [**Matrix:: трансформвекторс**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformvectors(inoutpoint_inint)) умножает каждый вектор в массиве на эту матрицу. Элементы смещения данной матрицы (третья строка) игнорируются. Каждый вектор рассматривается как матрица строк. Умножение выполняется с помощью матрицы строк слева и этой матрицы справа.<br/>  |
| [**Трансформвекторс (PointF \* , int)**](/previous-versions//ms535319(v=vs.85)) | Метод [**Matrix:: трансформвекторс**](/previous-versions//ms535319(v=vs.85)) умножает каждый вектор в массиве на эту матрицу.
