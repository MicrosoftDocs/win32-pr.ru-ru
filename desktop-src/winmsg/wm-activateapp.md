# <a name="wm_activateapp-message"></a>\_Сообщение АКТИВАТЕАПП WM

Посылается, когда будет активировано окно, принадлежащее другому приложению, чем активное окно. Сообщение отправляется в приложение, окно которого активируется, и в приложение, окно которого деактивируется.

Окно получает это сообщение через функцию [**WindowProc**](/previous-versions/windows/desktop/legacy/ms633573(v=vs.85)) .


```C++
#define WM_ACTIVATEAPP                  0x001C
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает, активируется ли окно или деактивируется. Этот параметр имеет **значение true** , если окно активируется; **значение false** , если окно деактивируется.

</dd> <dt>

*lParam* 
</dt> <dd>

Идентификатор потока. Если параметр *wParam* имеет **значение true**, то аргумент *lParam* является идентификатором потока, владеющего активным окном. Если параметр *wParam* имеет **значение false**, то параметр *lParam* является идентификатором потока, владеющего активным окном.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **lResult**

Если приложение обрабатывает это сообщение, оно должно вернуть ноль.
