# <a name="mrmcreateconfiginmemory-function"></a>Функция Мрмкреатеконфигинмемори

\[Некоторые сведения относятся к предварительно выпущенному продукту, который может быть значительно изменен перед коммерческой выпуском. Корпорация Майкрософт не предоставляет никаких гарантий, явных или подразумеваемых, относительно предоставленной здесь информации.\]

Создает новые инициализированные сведения о конфигурации PRI (как данные в памяти, а не в файле), определяющие указанные квалификаторы по умолчанию. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с тем же указателем, чтобы освободить эту память. Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HRESULT MrmCreateConfigInMemory(
  _In_     MrmPlatformVersion platformVersion,
  _In_opt_ PCWSTR             defaultQualifiers,
  _Out_    BYTE               **outputXmlData,
  _Out_    ULONG              *outputXmlSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*платформверсион* \[ окне\]
</dt> <dd>

Тип: **[ **мрмплатформверсион**](mrmplatformversion.md)**

Версия платформы (*targetOsVersion*), используемая для создания сведений о конфигурации.

</dd> <dt>

*дефаулткуалифиерс* \[ в необязательное\]
</dt> <dd>

Тип: **пквстр**

Список квалификаторов ресурсов по умолчанию. Например, L "язык-EN-US \_ Scale-100 \_ контраст — Standard"

</dd> <dt>

*аутпутксмлдата* \[ заполняет\]
</dt> <dd>

Тип: **Byte \* \***

Адрес указателя на байт. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с помощью указателя на Byte, чтобы освободить эту память.

</dd> <dt>

*аутпутксмлсизе* \[ заполняет\]
</dt> <dd>

Тип: **ulong \***

Адрес ULONG. В *аутпутксмлсизе* функция возвращает размер выделенной памяти, на которую указывает *аутпутксмлдата*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

\_ОК, если функция прошла успешно, в противном случае — другое значение. Чтобы определить успешность или сбой, используйте макросы Success () или FAILed () (определенные в файле Winerror. h).
