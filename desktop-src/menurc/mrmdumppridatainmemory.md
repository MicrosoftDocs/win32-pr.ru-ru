# <a name="mrmdumppridatainmemory-function"></a>Функция Мрмдумппридатаинмемори

\[Некоторые сведения относятся к предварительно выпущенному продукту, который может быть значительно изменен перед коммерческой выпуском. Корпорация Майкрософт не предоставляет никаких гарантий, явных или подразумеваемых, относительно предоставленной здесь информации.\]

Выводит сведения о PRI (в виде большого двоичного объекта в памяти, созданном предыдущим вызовом [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md)) к его эквиваленту в формате XML (как данные в памяти), чтобы сделать его более удобным для чтения. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с тем же указателем, чтобы освободить эту память. Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HRESULT MrmDumpPriDataInMemory(
  _In_     BYTE        *inputPriData,
  _In_     ULONG       inputPriSize,
  _In_opt_ BYTE        *schemaPriData,
  _In_     ULONG       schemaPriSize,
  _In_     MrmDumpType dumpType,
  _Out_    BYTE        **outputXmlData,
  _Out_    ULONG       *outputXmlSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*инпутпридата* \[ окне\]
</dt> <dd>

Тип: **Byte \***

Указатель на данные PRI, созданные при предыдущем вызове метода [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md).

</dd> <dt>

*инпутприсизе* \[ окне\]
</dt> <dd>

Тип: **ulong**

Размер данных, на которые указывает *инпутпридата*.

</dd> <dt>

*счемапридата* \[ в необязательное\]
</dt> <dd>

Тип: **Byte \***

Необязательный указатель на сведения о PRI (в виде большого двоичного объекта в памяти), представляющий данные схемы, созданные при предыдущем вызове метода [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md). Не освобождайте *счемапридата* до тех пор, пока не завершите использование индексатора ресурсов. См. также примечания.

</dd> <dt>

*счемаприсизе* \[ окне\]
</dt> <dd>

Тип: **ulong**

Размер данных, на которые указывает *счемапридата*.

</dd> <dt>

*думптипе* \[ окне\]
</dt> <dd>

Тип: **[ **мрмдумптипе**](mrmdumptype.md)**

Указывает, насколько подробным должен быть дамп XML, или следует ли создавать дамп схемы.

</dd> <dt>

*аутпутксмлдата* \[ заполняет\]
</dt> <dd>

Тип: **Byte \* \***

Адрес указателя на байт. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с помощью указателя на Byte, чтобы освободить эту память.

</dd> <dt>

*аутпутксмлсизе* \[ заполняет\]
</dt> <dd>

Тип: **ulong \***

Адрес ULONG. В *аутпутксмлсизе* функция возвращает размер выделенной памяти, на которую указывает *аутпутксмлдата*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

\_ОК, если функция прошла успешно, в противном случае — другое значение. Чтобы определить успешность или сбой, используйте макросы Success () или FAILed () (определенные в файле Winerror. h).

## <a name="remarks"></a>Remarks

Пакет ресурсов без схемы — это тот, который был создан с помощью аргумента [**мрмпаккагингоптионсомитсчемафромресаурцепаккс**](mrmpackagingoptions.md) , переданного в [**мрмкреатересаурцефиле**](mrmcreateresourcefile.md) или [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md) (или с параметром *omitSchemaFromResourcePacks* в файле конфигурации PRI). Чтобы создать дамп пакета ресурсов без схемы, передайте путь к основным данным пакета PRI в качестве аргумента для параметра *счемапридата* .
