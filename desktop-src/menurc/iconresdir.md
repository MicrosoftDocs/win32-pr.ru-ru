# <a name="iconresdir-structure"></a>Структура ИКОНРЕСДИР

Содержит измерения и формат цвета отдельного изображения значка в группе ресурсов. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  BYTE Width;
  BYTE Height;
  BYTE ColorCount;
  BYTE reserved;
} ICONRESDIR;
```



## <a name="members"></a>Члены

<dl> <dt>

**Width**
</dt> <dd>

Тип: **Byte**

</dd> <dd>

Ширина значка в пикселях. Допустимые значения: 16, 32 и 64.

</dd> <dt>

**Height**
</dt> <dd>

Тип: **Byte**

</dd> <dd>

Высота значка в пикселях. Допустимые значения: 16, 32 и 64.

</dd> <dt>

**колоркаунт**
</dt> <dd>

Тип: **Byte**

</dd> <dd>

Число цветов в значке. Допустимые значения: 2, 8 и 16.

</dd> <dt>

**процессу**
</dt> <dd>

Тип: **Byte**

</dd> <dd>

Процессу необходимо задать то же значение, что и для зарезервированного поля в заголовке файла значка.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Структура **иконресдир** передается в структуру [**ресдир**](resdir.md) , если структура **ресдир** описывает значок.
