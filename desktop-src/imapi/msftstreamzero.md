# <a name="msftstreamzero-object"></a>Объект Мсфтстреамзеро

Используйте этот объект, чтобы создать поток данных только для чтения, данные которого инициализируются нулями. Чтобы задать запрошенный размер потока, необходимо вызвать метод **SetSize** .

## <a name="members"></a>Элементы

Объект **мсфтстреамзеро** не определяет никаких членов.
