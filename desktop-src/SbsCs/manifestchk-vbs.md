# <a name="manifestchkvbs"></a>Manifestchk.vbs

файл VBScript Manifestchk.vbs — это средство проверки, предоставляемое в пакете SDK для Microsoft Windows, которое проверяет файлы манифеста приложения и сборки.

для выполнения этого примера требуется Windows сервер скриптов. дополнительные сведения о сервере сценариев Windows см. в разделе сервер скриптов Windows Windows SDK. Windows Сервер сценариев на самом деле состоит из двух узлов. CScript.exe — это версия, позволяющая запускать скрипты из командной строки. CScript.exe предоставляет параметры командной строки для установки свойств скрипта.

Формат командной строки следующий:

**Cscript//нолого manifestchk.vbs/s:** *\[ диск: \] \[ путь \] счемафиленаме* **/m:** *\[ диск: \] \[ путь \] манифестфиленаме* **\[ /q \] /t:** *параметр*

Флаги, определенные для Manifestchk.vbs, описаны в следующей таблице.



| Флаг | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| /s   | Указывает имя файла схемы манифеста для проверки манифестов. См. схему в [схеме файла манифеста](manifest-file-schema.md).                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| /m   | Указывает имя файла манифеста для проверки.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| /q   | Подавляет все выходные данные в консоли.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| /t   | Указывает тип файла манифеста. Допустимые значения: Проверка [схемы файла манифеста](manifest-file-schema.md) манифеста [сборки](assembly-manifests.md) или [манифеста приложения](application-manifests.md) .<br/> КОМПЬЮТЕР Проверка [схемы файла конфигурации издателя](publisher-configuration-file-schema.md) [файла конфигурации издателя](publisher-configuration-files.md)<br/> AC проверяет схему файла конфигурации приложения для [файла конфигурации приложения](application-configuration-files.md).<br/> |



 

Если флаг/q не указан, Manifestchk.vbs отображает подробные сведения о первой ошибке, обнаруженной в файле, и отображает сообщение о том, успешно ли прошел процесс проверки.

Эта программа проверяет наличие следующих средств:

-   Допустимая Командная строка.
-   MSXML установлена версия 3.
-   В манифесте используется XML-код правильного формата.
-   , Что манифест соглашается с предоставленной схемой. Обратите внимание, что Manifestchk.vbs проверяет файл манифеста на основе только того, что указано в указанной схеме. Пример схемы манифеста см. в разделе [Схема файла манифеста](manifest-file-schema.md).

Cscript.exe возвращает значение 0, если процесс проверки прошел успешно, и 1, если он не был успешным. При наличии ошибки в аргументе командной строки возвращается значение 2.
