# <a name="configuring-and-enabling-server-side-logging"></a><span data-ttu-id="71ab1-103">Настройка и включение ведения журнала на стороне сервера</span><span class="sxs-lookup"><span data-stu-id="71ab1-103">Configuring and Enabling Server Side Logging</span></span>

<span data-ttu-id="71ab1-104">Приложение включает ведение журнала в сеансе сервера или в группе URL-адресов перед отправкой ответа с помощью [**хттпсендхттпреспонсе**](/windows/desktop/api/Http/nf-http-httpsendhttpresponse).</span><span class="sxs-lookup"><span data-stu-id="71ab1-104">The application enables logging on the server session or the URL group before sending the response with [**HttpSendHttpResponse**](/windows/desktop/api/Http/nf-http-httpsendhttpresponse).</span></span> <span data-ttu-id="71ab1-105">В следующем примере показано, как настроить и включить ведение журнала на стороне сервера типа W3C.</span><span class="sxs-lookup"><span data-stu-id="71ab1-105">The following example shows how to configure and enable W3C type server side logging:</span></span>

1.  <span data-ttu-id="71ab1-106">Приложение инициализирует структуру [**\_ ведения журнала HTTP \_**](/windows/desktop/api/Http/ns-http-http_logging_info) с помощью **HttpLoggingTypeW3C** , указанной в элементе **Format** , и битовую маску констант [**\_ \_ поля журнала HTTP**](http-log-field--constants.md) в элементе **Fields** .</span><span class="sxs-lookup"><span data-stu-id="71ab1-106">The application initializes the [**HTTP\_LOGGING\_INFO**](/windows/desktop/api/Http/ns-http-http_logging_info) structure with **HttpLoggingTypeW3C** specified in the **Format** member, and a bitmask of the [**HTTP\_LOG\_FIELD**](http-log-field--constants.md) constants in the **Fields** member.</span></span>
2.  <span data-ttu-id="71ab1-107">Приложение вызывает [**хттпсетсерверсессионпроперти**](/windows/desktop/api/Http/nf-http-httpsetserversessionproperty) или [**хттпсетурлграуппроперти**](/windows/desktop/api/Http/nf-http-httpseturlgroupproperty) с **хттпсерверлоггингпроперти** , указанным в параметре *Property* , и указателем на [**структуру \_ \_ сведений о ведении журнала HTTP**](/windows/desktop/api/Http/ns-http-http_logging_info) в параметре *pPropertyInformation* .</span><span class="sxs-lookup"><span data-stu-id="71ab1-107">The application calls [**HttpSetServerSessionProperty**](/windows/desktop/api/Http/nf-http-httpsetserversessionproperty) or [**HttpSetUrlGroupProperty**](/windows/desktop/api/Http/nf-http-httpseturlgroupproperty) with **HttpServerLoggingProperty** specified in the *Property* parameter and a pointer to the [**HTTP\_LOGGING\_INFO**](/windows/desktop/api/Http/ns-http-http_logging_info) structure in the *pPropertyInformation* parameter.</span></span>

<span data-ttu-id="71ab1-108">Битовая маска констант [**\_ \_ поля журнала HTTP**](http-log-field--constants.md) содержит поля, которые могут быть зарегистрированы в файле журнала W3C.</span><span class="sxs-lookup"><span data-stu-id="71ab1-108">The bitmask of the [**HTTP\_LOG\_FIELD**](http-log-field--constants.md) constants contain the fields that may be logged in the W3C log file.</span></span> <span data-ttu-id="71ab1-109">Обратите внимание, что установка свойства **хттпсерверлоггингпроперти** в сеансе сервера или в группе URL-адресов не означает, что будут регистрироваться HTTP-ответы.</span><span class="sxs-lookup"><span data-stu-id="71ab1-109">Note that setting the **HttpServerLoggingProperty** property on a server session or a URL group does not mean that HTTP responses will be logged.</span></span> <span data-ttu-id="71ab1-110">Ведение журнала выполняется для каждого запроса, когда в вызове [**хттпсендхттпреспонсе**](/windows/desktop/api/Http/nf-http-httpsendhttpresponse) или [**хттпсендреспонсинтитибоди**](/windows/desktop/api/Http/nf-http-httpsendresponseentitybody)включен W3C.</span><span class="sxs-lookup"><span data-stu-id="71ab1-110">Logging is performed on a per request basis when W3C is enabled in the call to [**HttpSendHttpResponse**](/windows/desktop/api/Http/nf-http-httpsendhttpresponse) or [**HttpSendResponseEntityBody**](/windows/desktop/api/Http/nf-http-httpsendresponseentitybody).</span></span>

2.  <span data-ttu-id="71ab1-113">**Базовый член. Type** структуры [**\_ \_ \_ данных полей журнала HTTP**](/windows/desktop/api/Http/ns-http-http_log_fields_data) должен инициализироваться в **хттплогдататипефиелдс**.</span><span class="sxs-lookup"><span data-stu-id="71ab1-113">The **Base.Type** member of the [**HTTP\_LOG\_FIELDS\_DATA**](/windows/desktop/api/Http/ns-http-http_log_fields_data) structure should be initialized to **HttpLogDataTypeFields**.</span></span> <span data-ttu-id="71ab1-114">Поле **base.
