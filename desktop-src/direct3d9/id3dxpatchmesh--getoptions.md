# <a name="id3dxpatchmeshgetoptions-method"></a>Метод ID3DXPatchMesh:: Options

Возвращает тип исправления.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetOptions();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **DWORD**](../winprog/windows-data-types.md)**

Тип исправления.

## <a name="remarks"></a>Remarks

Дополнительные сведения о типах исправлений см. в разделе [**D3DXPATCHMESHTYPE**](./d3dxpatchmeshtype.md).
