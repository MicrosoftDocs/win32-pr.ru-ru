# <a name="id3dxkeyframedanimationsetgetnumtranslationkeys-method"></a>Метод ID3DXKeyframedAnimationSet:: Жетнумтранслатионкэйс

Возвращает число ключей перевода в указанной анимации по ключевым кадрам.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumTranslationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Анимация* \[ окне\]
</dt> <dd>

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Индекс анимации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Число ключей перевода в указанной анимации по ключевым кадрам.
