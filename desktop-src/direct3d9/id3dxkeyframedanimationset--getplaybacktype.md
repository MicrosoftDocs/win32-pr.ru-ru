# <a name="id3dxkeyframedanimationsetgetplaybacktype-method"></a>Метод ID3DXKeyframedAnimationSet:: Жетплайбакктипе

Возвращает тип цикла воспроизведения набора анимации.

## <a name="syntax"></a>Синтаксис


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **D3DXPLAYBACK \_ тип**](./d3dxplayback-type.md)**

Тип цикла воспроизведения набора анимации. См. раздел [**\_ тип D3DXPLAYBACK**](./d3dxplayback-type.md).
