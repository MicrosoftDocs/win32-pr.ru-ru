## <a name="animating-textures-by-translation-or-rotation-on-a-model"></a>Анимация текстур (по сдвигу или повороту) в модели

-   Определите координаты двухмерной текстуры в формате вершин.
    ```
    // Use a single texture, with 2D texture coordinates. This
    // bit-pattern should be expanded to include position, normal, 
    // and color information as needed.
    DWORD dwFVFTex = D3FVF_TEX1 | D3DFVF_TEXCOORDSIZE2(0);
    ```

    

-   Настройте средство программной прорисовки для использования координат двухмерной текстуры.
    ```
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   Определите и установите соответствующую матрицу преобразования координат текстуры.
    ```
    // M is a D3DMATRIX being set to translate texture
    // coordinates in the U and V directions.
    //      1   0  0  0
    //      0   1  0  0
    //      du dv  1  0 (du and dv change each frame)
    //      0   0  0  1

    D3DMATRIX M = D3DXMatrixIdentity(); // declared in d3dutil.h
    M._31 = du; 
    M._32 = dv; 
    ```

    

## <a name="creating-texture-coordinates-as-a-linear-function-of-a-models-camera-space-position"></a>Создание координат текстуры в виде линейной функции положения камеры в модели

-   Используйте флаг D3DTSS \_ тЦи \_ камераспацепоситион, чтобы указать системе передать расположение вершины в качестве входных данных для преобразования текстуры.
    ```
    // The input vertices have no texture coordinates, saving 
    // bandwidth. Three texture coordinates are generated by 
    // using vertex position in camera space (x, y, z).
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEPOSITION);
    ```

    

-   Попросите средство программной прорисовки заждать координаты двухмерной текстуры.
    ```
    // Two output coordinates are used.
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   Определите и установите матрицу, которая применяет линейную функцию.
    ```
    // Generate texture coordinates as linear functions 
    // so that:
    //      u = Ux*x + Uy*y + Uz*z + Uw 
    //      v = Vx*x + Vy*y + Vz*z + Vw
    // The matrix M for this case is:
    //      Ux  Vx  0  0 
    //      Uy  Vy  0  0 
    //      Uz  Vz  0  0 
    //      Uw  Vw  0  0 

    SetTransform(D3DTS_TEXTURE0, &M);
    ```

    

## <a name="performing-environment-mapping-with-a-cubic-environment-map"></a>Сопоставление среды с картой кубических сред

-   Используйте флаг D3DTSS \_ тЦи \_ камераспацерефлектионвектор, чтобы дать указание системе автоматически создавать координаты текстуры в виде векторов отражения для кубических сопоставлений.
    ```
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEREFLECTIONVECTOR);
    ```

    

-   Укажите, что средство программной прорисовки должно рассчитывать координаты текстуры с тремя элементами.
    ```
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT3);
    ```

    

## <a name="performing-projective-texturing"></a>Выполнение проецированного текстурирования

-   Используйте флаг D3DTSS \_ тЦи \_ камераспацепоситион, чтобы указать системе передать расположение вершины в качестве входных данных в матрицу преобразования текстуры.
    ```
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEPOSITION);
    ```

    

-   Создайте и примените матрицу текстурной проекции. Эта документация выходит за рамки данной документации, и она является темой нескольких отраслевых статей.
-   Укажите, что средство программной прорисовки должно запланировать запланированные координаты текстур с тремя элементами.
    ```
    // Two output coordinates are used.

    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTF_PROJECTED | D3DTTFF_COUNT3);
    ```
