## <a name="animating-textures-by-translation-or-rotation-on-a-model"></a><span data-ttu-id="23883-109">Анимация текстур (по сдвигу или повороту) в модели</span><span class="sxs-lookup"><span data-stu-id="23883-109">Animating textures (by translation or rotation) on a model</span></span>

-   <span data-ttu-id="23883-110">Определите координаты двухмерной текстуры в формате вершин.</span><span class="sxs-lookup"><span data-stu-id="23883-110">Define 2D texture coordinates in your vertex format.</span></span>
    ```
    // Use a single texture, with 2D texture coordinates. This
    // bit-pattern should be expanded to include position, normal, 
    // and color information as needed.
    DWORD dwFVFTex = D3FVF_TEX1 | D3DFVF_TEXCOORDSIZE2(0);
    ```

    

-   <span data-ttu-id="23883-111">Настройте средство программной прорисовки для использования координат двухмерной текстуры.</span><span class="sxs-lookup"><span data-stu-id="23883-111">Configure the rasterizer to use 2D texture coordinates.</span></span>
    ```
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   <span data-ttu-id="23883-112">Определите и установите соответствующую матрицу преобразования координат текстуры.</span><span class="sxs-lookup"><span data-stu-id="23883-112">Define and set an appropriate texture coordinate transformation matrix.</span></span>
    ```
    // M is a D3DMATRIX being set to translate texture
    // coordinates in the U and V directions.
    //      1   0  0  0
    //      0   1  0  0
    //      du dv  1  0 (du and dv change each frame)
    //      0   0  0  1

    D3DMATRIX M = D3DXMatrixIdentity(); // declared in d3dutil.h
    M._31 = du; 
    M._32 = dv; 
    ```

    

## <a name="creating-texture-coordinates-as-a-linear-function-of-a-models-camera-space-position"></a><span data-ttu-id="23883-113">Создание координат текстуры в виде линейной функции положения камеры в модели</span><span class="sxs-lookup"><span data-stu-id="23883-113">Creating texture coordinates as a linear function of a model's camera-space position</span></span>

-   <span data-ttu-id="23883-114">Используйте флаг D3DTSS \_ тЦи \_ камераспацепоситион, чтобы указать системе передать расположение вершины в качестве входных данных для преобразования текстуры.</span><span class="sxs-lookup"><span data-stu-id="23883-114">Use the D3DTSS\_TCI\_CAMERASPACEPOSITION flag to instruct the system to pass the vertex position, in camera space, as input to a texture transformation.</span></span>
    ```
    // The input vertices have no texture coordinates, saving 
    // bandwidth. Three texture coordinates are generated by 
    // using vertex position in camera space (x, y, z).
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEPOSITION);
    ```

    

-   <span data-ttu-id="23883-115">Попросите средство программной прорисовки заждать координаты двухмерной текстуры.</span><span class="sxs-lookup"><span data-stu-id="23883-115">Instruct the rasterizer to expect 2D texture coordinates.</span></span>
    ```
    // Two output coordinates are used.
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   <span data-ttu-id="23883-116">Определите и установите матрицу, которая применяет линейную функцию.</span><span class="sxs-lookup"><span data-stu-id="23883-116">Define and set a matrix that applies a linear function.</span></span>
    ```
    // Generate texture coordinates as linear functions 
    // so that:
    //      u = Ux*x + Uy*y + Uz*z + Uw 
    //      v = Vx*x + Vy*y + Vz*z + Vw
    // The matrix M for this case is:
    //      Ux  Vx  0  0 
    //      Uy  Vy  0  0 
    //      Uz  Vz  0  0 
    //      Uw  Vw  0  0 

    SetTransform(D3DTS_TEXTURE0, &M);
    ```
