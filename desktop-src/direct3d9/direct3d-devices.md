<span data-ttu-id="6f6d4-127">Созданное приложением устройство Direct3D должно соответствовать возможностям оборудования, на котором запущено приложение.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-127">The Direct3D device that an application creates must correspond to the capabilities of the hardware on which the application is running.</span></span> <span data-ttu-id="6f6d4-128">Direct3D предоставляет возможности отрисовки путем использования установленного в компьютере оборудования для работы с трехмерной графикой или эмуляции возможностей такого оборудования программными средствами.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-128">Direct3D provides rendering capabilities, either by accessing 3D hardware that is installed in the computer or by emulating the capabilities of 3D hardware in software.</span></span> <span data-ttu-id="6f6d4-129">Таким образом, Direct3D предоставляет устройства как для аппаратного доступа, так и для программной эмуляции.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-129">Therefore, Direct3D provides devices for both hardware access and software emulation.</span></span>

<span data-ttu-id="6f6d4-133">За исключением эталонного устройства, программные устройства не всегда поддерживают те же возможности, что и аппаратные устройства.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-133">With the exception of the reference device, software devices do not always support the same features as a hardware device.</span></span> <span data-ttu-id="6f6d4-134">Приложения всегда должны запрашивать возможности устройств, чтобы определить, какие возможности поддерживаются.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-134">Applications should always query for device capabilities to determine which features are supported.</span></span>

<span data-ttu-id="6f6d4-135">Поскольку поведение программных и эталонных устройств с Direct3D 9 идентично поведению устройства слоя HAL, код приложения, предназначенный для работы с устройством слоя HAL, будет совместим с программными или эталонными устройствами без изменений.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-135">Because the behavior of the software and reference devices provided with Direct3D 9 is identical to that of the hal device, application code authored to work with the hal device will work with the software or reference devices without modifications.</span></span> <span data-ttu-id="6f6d4-136">Обратите внимание, что, хотя поведение указанного программного обеспечения или ссылочного устройства идентично характеристикам устройства HAL, возможности устройства различаются, а конкретное программное устройство может реализовывать намного меньший набор возможностей.</span><span class="sxs-lookup"><span data-stu-id="6f6d4-136">Note that while the provided software or reference device behavior is identical to that of the hal device, the device capabilities do vary, and a particular software device may implement a much smaller set of capabilities.</span></span>
