# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Создает первую константу (в регистре register src2) для инструкции шейдера пикселей [синкос-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает первую константу, необходимую [синкос-PS](../direct3dhlsl/sincos---ps.md).
