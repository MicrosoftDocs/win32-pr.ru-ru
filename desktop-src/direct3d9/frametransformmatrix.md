# <a name="frametransformmatrix"></a>фраметрансформматрикс

Определяет локальное преобразование для кадра (и всех его дочерних объектов).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Где:

-   Фрамематрикс — шаблон Matrix4x4. См. [**Matrix4x4**](matrix4x4.md).
