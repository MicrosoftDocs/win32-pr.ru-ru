# <a name="use-of-commas-and-semicolons"></a>Использование запятых и точек с запятой

Использование запятых и точек с запятой может быть наиболее сложной проблемой синтаксиса в формате файла, и это использование очень строго. Запятые используются для разделения элементов массива; точка с запятой завершает каждый элемент данных.

Например, если шаблон определен следующим образом:


```
template mytemp {
DWORD myvar;
}
```



Затем экземпляр этого шаблона выглядит следующим образом:


```
mytemp dataTemp {
1;
}
```



Если шаблон, содержащий другой шаблон, определен следующим образом;


```
template mytemp {
DWORD myvar;
DWORD myvar2;
}
template container {
FLOAT aFloat;
mytemp aTemp;
}
```



Затем экземпляр этого шаблона выглядит следующим образом:


```
container dataContainer {
1.1;
2; 
3;;
}
```



Обратите внимание, что вторая строка, представляющая митемп внутри контейнера, содержит две точки с запятой в конце строки. Первая точка с запятой обозначает конец элемента данных, Атемп (внутри контейнера), а вторая точка с запятой обозначает конец контейнера.

Если массив определен следующим образом:


```
Template mytemp {

array DWORD myvar[3];

}
```



Затем экземпляр этого типа выглядит следующим образом:


```
mytemp aTemp {
1, 2, 3;
}
```



В примере массива не требуется, чтобы элементы данных были разделены точкой с запятой, так как они разделяются запятыми. Точка с запятой в конце отмечает конец массива.

Рассмотрим шаблон, содержащий массив элементов данных, определенных шаблоном.


```
template mytemp {
DWORD myvar;
DWORD myvar2;
}
template container {
DWORD count;
array mytemp tempArray[count];
}
```



Экземпляр этого типа будет выглядеть, как в следующем примере.


```
container aContainer {
3;
1;2;,3;4;,5;6;;
}
```
