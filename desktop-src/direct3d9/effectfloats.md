# <a name="effectfloats"></a>еффектфлоатс

Определяет устаревший результат с плавающей запятой. Этот шаблон был заменен шаблоном [**еффектпарамфлоатс**](effectparamfloats.md) .

``` syntax
template EffectFloats
{
    < F1CFE2B3-0DE3-4e28-AFA1-155A750A282D >
    DWORD nFloats;
    array float Floats[nFloats];
} 
```

Где:

-   Нфлоатс — количество элементов float.
-   Нфлоатс с плавающей запятой \[ \] — массив значений float.
