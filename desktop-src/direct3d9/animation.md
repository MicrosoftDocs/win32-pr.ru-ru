# <a name="animation-direct3d-9-graphics"></a>Анимация (графика Direct3D 9)

Содержит анимацию, ссылающуюся на предыдущий кадр. Он должен содержать одну ссылку на кадр и по крайней мере один набор шаблонов [**аниматионкэй**](animationkey.md) . Он также может содержать объект данных [**аниматионоптионс**](animationoptions.md) .

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Где:

-   \[ ... \] -Здесь можно использовать любой шаблон файла. x. Это делает архитектуру расширяемой.
