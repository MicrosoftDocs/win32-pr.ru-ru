# <a name="d3dxshcompressqualitytype-enumeration"></a>Перечисление D3DXSHCOMPRESSQUALITYTYPE

Параметр энергосферовой гармонии (SH).

## <a name="syntax"></a>Синтаксис


```C++
typedef enum D3DXSHCOMPRESSQUALITYTYPE { 
  D3DXSHCQUAL_FASTLOWQUALITY   = 1,
  D3DXSHCQUAL_SLOWHIGHQUALITY  = 2,
  D3DXSHCQUAL_FORCE_DWORD      = 0x7fffffff
} D3DXSHCOMPRESSQUALITYTYPE, *LPD3DXSHCOMPRESSQUALITYTYPE;
```



## <a name="constants"></a>Константы

<dl> <dt>

<span id="D3DXSHCQUAL_FASTLOWQUALITY"></span><span id="d3dxshcqual_fastlowquality"></span>**D3DXSHCQUAL \_ фастловкуалити**
</dt> <dd>

Сжатие данных имеет низкий уровень качества, но ускоряет сжатие.

</dd> <dt>

<span id="D3DXSHCQUAL_SLOWHIGHQUALITY"></span><span id="d3dxshcqual_slowhighquality"></span>**D3DXSHCQUAL \_ словхигхкуалити**
</dt> <dd>

Сжатие данных имеет высокое качество, но сжимается слишком долго.

</dd> <dt>

<span id="D3DXSHCQUAL_FORCE_DWORD"></span><span id="d3dxshcqual_force_dword"></span>**D3DXSHCQUAL \_ Force \_ DWORD**
</dt> <dd>

Заставляет это перечисление компилироваться до 32 бит в размере. Без этого значения некоторые компиляторы позволяли бы компилировать это перечисление до размера, отличного от 32 бит. Это значение не используется.

</dd> </dl>
