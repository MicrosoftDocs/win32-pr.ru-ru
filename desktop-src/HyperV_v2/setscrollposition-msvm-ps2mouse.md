# <a name="setscrollposition-method-of-the-msvm_ps2mouse-class"></a>Метод Сетскроллпоситион \_ класса Ps2Mouse мсвм

Настраивает z-координату элемента управления "колесо" для указывающего устройства. Значения, записанные в это свойство, всегда являются относительными смещениями координат.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetScrollPosition(
  [in] sint8 scrollPositionDelta
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*скроллпоситионделта* \[ окне\]
</dt> <dd>

Тип: **sint8**

Разность в положении прокрутки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Возвращаемое значение, равное нулю, указывает на успешное выполнение. Ненулевое значение указывает на ошибку изменения расположения прокрутки.
