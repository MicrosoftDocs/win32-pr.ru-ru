# <a name="storing-dynamic-data"></a>Сохранение платформа динамических данных

Большинство данных, хранящихся с помощью домен Active Directory Services, являются статическими. Например, объекты каталога, представляющие пользователей, группы, принтеры, сетевые службы и т. д., обычно имеют длительное время существования и редко изменяются.

В Windows Server 2003 и более поздних версиях Windows службы домен Active Directory обеспечивают поддержку следующих функций, позволяющих приложениям использовать каталог для хранения динамических данных.

-   [Динамические объекты](dynamic-objects.md)
-   [Динамические вспомогательные классы](dynamic-auxiliary-classes.md)
