# <a name="including-attributes-in-the-global-catalog"></a><span data-ttu-id="a40f9-105">Включение атрибутов в глобальный каталог</span><span class="sxs-lookup"><span data-stu-id="a40f9-105">Including Attributes in the Global Catalog</span></span>

<span data-ttu-id="a40f9-106">Глобальный каталог леса содержит частичную реплику каждого объекта в лесу.</span><span class="sxs-lookup"><span data-stu-id="a40f9-106">The global catalog of a forest includes a partial replica of every object in the forest.</span></span> <span data-ttu-id="a40f9-107">Для каждого объекта глобальный каталог включает только подмножество атрибутов каждого объекта.</span><span class="sxs-lookup"><span data-stu-id="a40f9-107">For each object, the global catalog includes only a subset of each object's attributes.</span></span> <span data-ttu-id="a40f9-108">Атрибут [**исмемберофпартиалаттрибутесет**](/windows/desktop/ADSchema/a-ismemberofpartialattributeset) объекта [**AttributeSchema**](/windows/desktop/ADSchema/c-attributeschema) имеет значение **true** , если атрибут реплицируется в глобальный каталог.</span><span class="sxs-lookup"><span data-stu-id="a40f9-108">The [**isMemberOfPartialAttributeSet**](/windows/desktop/ADSchema/a-ismemberofpartialattributeset) attribute of an [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) object is set to **TRUE** if the attribute is replicated to the global catalog.</span></span>

<span data-ttu-id="a40f9-109">Атрибуты со следующими характеристиками подходят для хранения в глобальном каталоге:</span><span class="sxs-lookup"><span data-stu-id="a40f9-109">Attributes with the following characteristics are appropriate for storage in the global catalog:</span></span>

-   <span data-ttu-id="a40f9-110">Атрибут является глобально интересным, так как атрибут необходим для поиска объектов, которые могут встречаться в любом месте леса, или потому, что доступ для чтения к атрибуту важен, даже если полный объект недоступен.</span><span class="sxs-lookup"><span data-stu-id="a40f9-110">The attribute is globally interesting, either because the attribute is required for locating objects that can occur anywhere in the forest, or because read access to the attribute is valuable even when the full object is not accessible.</span></span> <span data-ttu-id="a40f9-111">Примером первого типа является атрибут [**Location**](/windows/desktop/ADSchema/a-location) , который можно использовать для поиска объекта [**очереди печати**](/windows/desktop/ADSchema/c-printqueue) .</span><span class="sxs-lookup"><span data-stu-id="a40f9-111">An example of the first type is the [**location**](/windows/desktop/ADSchema/a-location) attribute, which can be used to find a [**printQueue**](/windows/desktop/ADSchema/c-printqueue) object.</span></span> <span data-ttu-id="a40f9-112">Примером второго типа является [**telephoneNumber**](/windows/desktop/ADSchema/a-telephonenumber), так как вы можете вызвать пользователя, даже если вы не можете получить доступ к полной реплике своего объекта [**User**](/windows/desktop/ADSchema/c-user) .</span><span class="sxs-lookup"><span data-stu-id="a40f9-112">An example of the second type is [**telephoneNumber**](/windows/desktop/ADSchema/a-telephonenumber), because you can call someone even if you cannot access a full replica of their [**user**](/windows/desktop/ADSchema/c-user) object.</span></span>
-   <span data-ttu-id="a40f9-113">Изменчивости атрибута очень низкий.</span><span class="sxs-lookup"><span data-stu-id="a40f9-113">The volatility of the attribute is very low.</span></span> <span data-ttu-id="a40f9-114">Это важно, поскольку если класс атрибута включен в глобальный каталог, изменения каждого значения этого класса атрибута в корпоративном лесу реплицируются на все серверы глобального каталога предприятия.</span><span class="sxs-lookup"><span data-stu-id="a40f9-114">This is important, because if an attribute class is included in the global catalog, changes to every value of that attribute class throughout the enterprise forest are replicated to all global catalog servers in the enterprise.</span></span>
-   <span data-ttu-id="a40f9-115">Размер значения атрибута невелик.</span><span class="sxs-lookup"><span data-stu-id="a40f9-115">The size of the attribute value is small.</span></span> <span data-ttu-id="a40f9-116">"Малый" — очень субъективная: При размещении атрибута в глобальном каталоге рассмотрите влияние репликации атрибута на все серверы глобального каталога в предприятии.</span><span class="sxs-lookup"><span data-stu-id="a40f9-116">"Small" is highly subjective: when placing an attribute in the global catalog, consider the impact of replicating the attribute to all global catalog servers in the enterprise.</span></span> <span data-ttu-id="a40f9-117">Чем меньше атрибут, тем меньше его влияние.</span><span class="sxs-lookup"><span data-stu-id="a40f9-117">The smaller the attribute, the lower the impact.</span></span> <span data-ttu-id="a40f9-118">Поскольку репликация происходит только при изменении атрибута, влияние репликации также меньше, чем изменчивости снижается, поэтому большой атрибут с очень низким изменчивости может иметь меньшее влияние, чем небольшой атрибут с высоким изменчивости.</span><span class="sxs-lookup"><span data-stu-id="a40f9-118">Because replication occurs only when the attribute changes, the impact of replication is also smaller as volatility decreases, so a large attribute with very low volatility may have a smaller overall impact than a small attribute with high volatility.</span></span>

<span data-ttu-id="a40f9-119">При принятии решения о необходимости размещения атрибута в глобальном каталоге Помните, что вы намерены повысить репликацию и увеличить объем дискового хранилища на серверах глобального каталога для, что может повысить производительность запросов.</span><span class="sxs-lookup"><span data-stu-id="a40f9-119">When deciding whether or not to place an attribute in the global catalog remember that you are trading increased replication and increased disk storage on global catalog servers for, potentially, faster query performance.</span></span> <span data-ttu-id="a40f9-120">Как правило, глобальный каталог используется для поиска интересующего объекта, что позволяет считывать выбранные атрибуты объекта.</span><span class="sxs-lookup"><span data-stu-id="a40f9-120">Typically, you would use the global catalog to search for an object of interest so you can read selected attributes of the object.</span></span> <span data-ttu-id="a40f9-121">Если нужные атрибуты реплицируются в глобальный каталог, их можно прочитать непосредственно из глобального каталога.</span><span class="sxs-lookup"><span data-stu-id="a40f9-121">If the attributes you are interested in are replicated to the global catalog, you can read them directly from the global catalog.</span></span> <span data-ttu-id="a40f9-122">Кроме того, для чтения атрибутов, которые не реплицируются в глобальный каталог, необходимо выполнить дополнительные действия для их извлечения.</span><span class="sxs-lookup"><span data-stu-id="a40f9-122">Alternately, to read attributes that are not replicated to the global catalog, you must perform additional steps to retrieve them.</span></span> <span data-ttu-id="a40f9-123">В этом случае после поиска в глобальном каталоге для поиска интересующего объекта необходимо прочитать различающееся имя объекта из глобального каталога, использовать DN для прямого связывания с полной репликой объекта, которая может находиться на другом сервере, и, наконец, считывать атрибуты, не являющиеся глобальным каталогом, из полной реплики объекта.</span><span class="sxs-lookup"><span data-stu-id="a40f9-123">In this case, after searching the global catalog to find the object of interest, you must read the object's distinguished name from the global catalog, <span data-ttu-id="a40f9-124">Часто запрашиваемые и упоминаемые атрибуты, такие как имя сотрудника и номер телефона, хорошо подходят для включения в глобальный каталог.</span><span class="sxs-lookup"><span data-stu-id="a40f9-124">Frequently queried and referenced attributes, such as employee name and phone number, are good to include in the global catalog.</span></span> <span data-ttu-id="a40f9-125">Нечасто упоминаемый атрибут, например "Дриверверсион" для принтеров, лучше всего выходит из глобального каталога.</span><span class="sxs-lookup"><span data-stu-id="a40f9-125">An infrequently referenced attribute such as "driverVersion" for printers is best left out of the global catalog.</span></span>
