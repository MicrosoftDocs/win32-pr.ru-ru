# <a name="partition-properties"></a>Свойства секции

Для секций доступно пять свойств. Ниже приведены эти свойства и их описания.

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Имя секции
</dt> <dd>

Текстовая строка, связанная с секцией.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>Идентификатор секции
</dt> <dd>

Идентификатор GUID, используемый системой для корреляции пользователя с секцией.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Nописание
</dt> <dd>

Текстовое описание секции.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Отключить удаление
</dt> <dd>

Механизм запрета удаления раздела пользователем.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Отключить изменения
</dt> <dd>

Механизм, позволяющий запретить пользователю вносить изменения в секцию.

</dd> </dl>

Эти свойства можно просмотреть, щелкнув правой кнопкой мыши имя секции в средстве администрирования "службы компонентов" и выбрав пункт " **Свойства**".

Дополнительные сведения о свойстве идентификатора секции см. [в разделе Регистрация и Активация компонентов в секциях](registering-and-activating-components-in-partitions.md).
