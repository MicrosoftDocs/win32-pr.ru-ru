# <a name="subscriptionsforcomponent-collection"></a>Коллекция Субскриптионсфоркомпонент

Содержит объект для каждой подписки для коллекции родительских [**компонентов**](components.md) .

Эта коллекция поддерживает методы [**Add**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-add) и [**Remove**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-remove) объекта [**комадминкаталогколлектион**](comadmincatalogcollection.md) .

## <a name="members"></a>Элементы

Коллекция **субскриптионсфоркомпонент** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.

## <a name="related-collections"></a>Связанные коллекции

Можно переходить от этой коллекции к любой из следующих коллекций:

-   [**ErrorInfo**](errorinfo.md)
-   [**PropertyInfo**](propertyinfo.md)
-   [**публишерпропертиес**](publisherproperties.md)
-   [**релатедколлектионинфо**](relatedcollectioninfo.md)
-   [**субскриберпропертиес**](subscriberproperties.md)

Вы можете переходить к этой коллекции из следующих коллекций:

-   [**Компоненты**](components.md)

## <a name="properties"></a>Свойства

Объект [**комадминкаталогобжект**](comadmincatalogobject.md) в коллекции поддерживает следующие свойства:

-   [Описание](#description)
-   [Enabled](#enabled)
-   [евенткласспартитионид](#eventclasspartitionid)
-   [евентклсид](#eventclsid)
-   [Параметром filterCriteria](#filtercriteria)
-   [Идентификатор](#subscriptionsforcomponent-collection)
-   [InterfaceID](#interfaceid)
-   [MachineName](#machinename)
-   [MethodName](#methodname)
-   [Name](#machinename)
-   [Изучение](#peruser)
-   [PublisherID](#publisherid)
-   [Поставлено в очередь](#queued)
-   [субскрибермоникер](#subscribermoniker)
-   [субскриберпартитионид](#subscriberpartitionid)
-   [UserName](#username)
