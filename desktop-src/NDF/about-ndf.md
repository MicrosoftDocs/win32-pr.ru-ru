# <a name="about-ndf"></a><span data-ttu-id="15754-103">О NDF</span><span class="sxs-lookup"><span data-stu-id="15754-103">About NDF</span></span>

<span data-ttu-id="15754-104">Инфраструктура диагностики сети (NDF) сокращает вовлеченность сетевых администраторов и пользователей компьютеров, обрабатывая распространенные сетевые проблемы по мере их возникновения.</span><span class="sxs-lookup"><span data-stu-id="15754-104">The Network Diagnostics Framework (NDF) reduces the involvement of network administrators and computer users by handling common network issues as they occur.</span></span> <span data-ttu-id="15754-105">Используя возможности диагностики и восстановления NDF, пользователям и администраторам не требуются дополнительные средства для решения некоторых относительно распространенных проблем.</span><span class="sxs-lookup"><span data-stu-id="15754-105">By using NDF's diagnostic and repair capabilities, users and administrators do not need additional tools in order to handle some relatively common problems.</span></span> <span data-ttu-id="15754-106">NDF поставляется в составе Windows Vista, Windows Server 2008 и более поздних версий.</span><span class="sxs-lookup"><span data-stu-id="15754-106">NDF ships as part of Windows Vista, Windows Server 2008, and later.</span></span> <span data-ttu-id="15754-107">Он доступен при загрузке системы (но не может работать в защищенном режиме).</span><span class="sxs-lookup"><span data-stu-id="15754-107">It is available whenever a system is booted (but cannot run in Safe Mode).</span></span>
