# <a name="raytcurrent"></a>райткуррент

Значение типа float, представляющее текущую конечную точку параметрической для луча.  

## <a name="syntax"></a>Синтаксис

```
float RayTCurrent();

```


## <a name="remarks"></a>Примечания

**Райткуррент** определяет текущую конечную точку луча в соответствии со следующей формулой: Origin + (направление * райткуррент).  *Источник* и *направление* могут находиться в мировом или объектном пространстве, что приводит к заполнению точки мира или объектного пространства.

**Райткуррент** инициализируется в вызове [**трацерай**](traceray-function.md) с использованием значения [**райдеск:: тмакс**](raydesc.md) , а затем обновляется во время выполнения запроса трассировки в отчетах о пересечениях (при любом попадании) и принимается.

В [шейдере пересечения](intersection-shader.md)он представляет расстояние до ближайшего пересечения, обнаруженного до сих пор.  Оно будет обновлено после () для значения сит, если оно было принято.

В [любом шейдере попадания](any-hit-shader.md)он представляет расстояние до переданного перекрестного сообщения.

В [ближайшем шейдере попадания](closest-hit-shader.md)он представляет расстояние до ближайшего принимаемого пересечения.

В [шейдере промахов](miss-shader.md)он равен *тмакс* , переданному в вызов **трацерай** .


Эту функцию можно вызывать из следующих типов шейдеров райтраЦинг:

* [**Шейдер любых попаданий**](any-hit-shader.md)
* [**Шейдер ближайшего попадания**](closest-hit-shader.md)
* [**Шейдер пересечения**](intersection-shader.md)
* [**Шейдер непопаданий**](miss-shader.md)
