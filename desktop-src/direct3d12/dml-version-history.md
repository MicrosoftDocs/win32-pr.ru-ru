## <a name="directml-version-versus-feature-level"></a><span data-ttu-id="2ca0b-160">Версия Директмл и уровень компонентов</span><span class="sxs-lookup"><span data-stu-id="2ca0b-160">DirectML version versus feature level</span></span>

<span data-ttu-id="2ca0b-161">Версия Директмл (например, 1.0.0 или 1.4.0) описывает определенный выпуск Директмл, включая соответствующий `DirectML.h` файл заголовка и `.lib` файл.</span><span class="sxs-lookup"><span data-stu-id="2ca0b-161">The DirectML version (for example, 1.0.0, or 1.4.0) describes a particular release of DirectML, including its associated `DirectML.h` header file and `.lib` file.</span></span>
