# <a name="about-the-dns-wmi-provider"></a>Сведения о поставщике WMI для DNS

Серверы DNS сопоставляют IP-адреса с именами компьютеров и именами компьютеров и IP-адресами. Таким образом, они предоставляют механизм для размещения сетевых ресурсов. Поставщик WMI для DNS позволяет приложениям взаимодействовать с DNS-серверами через унифицированную инфраструктуру управления инструментарий управления Windows (WMI) (WMI).
