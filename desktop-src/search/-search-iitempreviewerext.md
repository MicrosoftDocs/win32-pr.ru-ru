# <a name="iitempreviewerext-interface"></a>Интерфейс Иитемпревиеверекст

Предоставляет методы для предоставления параметров браузера. Интерфейс **иитемпревиеверекст** поддерживается только в Windows XP и windows Server 2003 и больше не должен использоваться.

## <a name="members"></a>Элементы

Интерфейс **иитемпревиеверекст** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Иитемпревиеверекст** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иитемпревиеверекст** содержит следующие методы.



| Метод                                                                               | Описание                                                                                  |
|:-------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------|
| [**жетлинкедконтент**](-search-iitempreviewerext-getlinkedcontent.md)               | Допускает расширение к расширенному содержимому для свойства. <br/>                            |
| [**жетрелатедпарт**](-search-iitempreviewerext-getrelatedpart.md)                   | Возвращает связанную часть тела для внедрения в выходной поток MHTML.<br/>              |
| [**процесстрансформкомманд**](-search-iitempreviewerext-processtransformcommand.md) | Разрешает обработку команды преобразования, обнаруженной в шаблоне предварительной версии.<br/> |
| [**сугжестбровсерполици**](-search-iitempreviewerext-suggestbrowserpolicy.md)       | Предлагает политику безопасности, применяемую к браузеру.<br/>                        |



 

## <a name="remarks"></a>Комментарии

Интерфейс **иитемпревиеверекст** поддерживается только в Windows XP и windows Server 2003 и больше не должен использоваться.

Для предварительного просмотра вложений с помощью обработчика протоколов стороннего производителя на компьютерах под управлением Windows XP или Windows Server 2003 может потребоваться использовать интерфейс **иитемпревиеверекст** и следующие API: интерфейсы [**исеарчпротоколуи**](-search-isearchprotocolui.md), [**Иитемпропертибаг**](iitempropertybag.md) и [**исеарчитем**](-search-isearchitem.md) , структура [**линкинфо**](-search-linkinfo.md) и [**перечисление**](-search-linktype.md) типов ссылок.

|-------------------------------------|------------------------------------------------------|

| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2003\]<br/> |
