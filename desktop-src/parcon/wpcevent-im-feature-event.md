# <a name="wpcevent_im_feature-event"></a>\_Событие компонента обмена мгновенными сообщениями впцевент \_

Событие "на пользователя", формируемое клиентом обмена мгновенными сообщениями, когда определенные функции используются в родительских элементах управления.


```C++
const EVENT_DESCRIPTOR WPCEVENT_IM_FEATURE = {0xb, 0x0, 0x10, 0x4, 0x16, 0xb, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения для обмена мгновенными сообщениями.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Строка версии для приложения.

</dd> <dt>

*AccountName* 
</dt> <dd>

Имя учетной записи мгновенного обмена сообщениями этого пользователя.

</dd> <dt>

*конвид* 
</dt> <dd>

Идентификатор для этого диалога.

</dd> <dt>

*Мультимедиа* 
</dt> <dd>

Значение перечисления [**\_ \_ функций обмена мгновенными сообщениями впкфлаг**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_im_feature) , которое указывает сведения о функциях, доступ к которым осуществляется во время взаимодействия с мгновенными сообщениями.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*реЦипкаунт* 
</dt> <dd>

Число пользователей с мгновенными сообщениями, которые получают функцию и удостоверения, определенные в поле получателя.

</dd> <dt>

*Recipient* 
</dt> <dd>

Строка с разделителями, которая содержит идентификаторы учетных записей мгновенных сообщений для всех пользователей, получивших эту функцию.

</dd> <dt>

*Отправитель* 
</dt> <dd>

Имя учетной записи для обмена мгновенными сообщениями для пользователя, который наследует эту функцию.

</dd> <dt>

*сендерип* 
</dt> <dd>

IP-адрес компьютера отправителя.

</dd> <dt>

*Data* 
</dt> <dd>

Описание данных в компоненте.

</dd> </dl>
