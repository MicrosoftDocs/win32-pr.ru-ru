# <a name="wpcevent_im_leave-event"></a>\_ \_ Событие выхода из IM впцевент

Событие "на пользователя", созданное клиентом обмена мгновенными сообщениями, когда пользователь покидает диалог в службе родительского контроля.


```C++
const EVENT_DESCRIPTOR WPCEVENT_IM_LEAVE = {0x9, 0x0, 0x10, 0x4, 0x16, 0x9, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Строка версии для приложения, создающего событие.

</dd> <dt>

*AccountName* 
</dt> <dd>

Строка удостоверения учетной записи для обмена мгновенными сообщениями для этого пользователя.

</dd> <dt>

*конвид* 
</dt> <dd>

Идентификатор для этого диалога.

</dd> <dt>

*леавингип* 
</dt> <dd>

Строка, содержащая IP-адрес компьютера, который выходит из диалога.

</dd> <dt>

*леавингусер* 
</dt> <dd>

Строка идентификатора учетной записи мгновенных сообщений для пользователя, который выходит из системы.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*MemberCount* 
</dt> <dd>

Число участников, которые находятся в диалоге и имеют удостоверения, определенные в поле члена.

</dd> <dt>

*Член* 
</dt> <dd>

Строка с разделителями, содержащая строки идентификаторов учетных записей мгновенных сообщений для всех текущих членов этого диалога.

</dd> <dt>

*Флаги* 
</dt> <dd>

Значение перечисления [**впкфлаг \_ IM \_ Leave**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_im_leave) , указывающее сведения о том, когда участник покидает взаимодействие с мгновенными сообщениями.

</dd> </dl>
