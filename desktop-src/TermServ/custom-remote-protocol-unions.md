# <a name="remote-desktop-protocol-provider-unions"></a>Объединения поставщиков протокол удаленного рабочего стола

API настраиваемого удаленного протокола поддерживает следующие объединения.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_параметр подключения \_ врдс**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_setting)
</dt> <dd>

Содержит различные уровни параметров для подключения к удаленному рабочему столу.

</dd> <dt>

[**\_параметр прослушивателя врдс \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_setting)
</dt> <dd>

Содержит различные уровни параметров прослушивателя для подключения к удаленному рабочему столу.

</dd> <dt>

[**\_параметр врдс**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_setting)
</dt> <dd>

Содержит различные уровни параметров, связанных с политикой, для подключения к удаленному рабочему столу.

</dd> <dt>

[**\_ \_ Отмена статистики кэша \_ ВТС**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wts_cache_stats_un)
</dt> <dd>

Содержит статистику кэша.

</dd> </dl>
