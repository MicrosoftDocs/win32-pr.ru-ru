# <a name="imsrdpclient5transportsettings-property"></a>Свойство IMsRdpClient5:: TransportSettings

Возвращает сведения о том, что было передано через скрипт в интерфейс [**имсрдпклиенттранспортсеттингс**](imsrdpclienttransportsettings.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_TransportSettings(
  [out] IMsRdpClientTransportSettings **ppXportSet
);
```



## <a name="property-value"></a>Значение свойства

Указатель интерфейса [**имсрдпклиенттранспортсеттингс**](imsrdpclienttransportsettings.md) .
