# <a name="imsrdpclient7advancedsettings8-property"></a>Свойство IMsRdpClient7:: AdvancedSettings8

Извлекает объект, который поддерживает интерфейс [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_AdvancedSettings8(
  [out, retval] IMsRdpClientAdvancedSettings7 **ppAdvSettings
);
```



## <a name="property-value"></a>Значение свойства

Адрес указателя интерфейса [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) , который получает объект.
