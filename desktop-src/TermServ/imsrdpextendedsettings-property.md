# <a name="imsrdpextendedsettingsproperty-property"></a>Имсрдпекстендедсеттингс: свойство фильтры:P

Содержит именованное свойство.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_Property(
  [in]          BSTR    bstrPropertyName,
  [in]          VARIANT *pValue
);

HRESULT get_Property(
  [in]          BSTR    bstrPropertyName,
  [out, retval] VARIANT *pValue
);
```



## <a name="property-value"></a>Значение свойства

Значение именованного свойства.

| Имя свойства | Тип данных | Доступ | Можно изменить после начала подключения | Описание |
|----------|-----------|--------|-----------------------------------------|-------------|
| коннектточилдсессион | **Логическое значение VT \_** | Чтение/запись | Да | Установка этого свойства в **значение true** приводит к тому, что клиентский элемент управления подключается к дочернему сеансу на локальном компьютере, а не на удаленном сервере. Если это свойство имеет значение **true**, подключиться к удаленному серверу невозможно, так как все соединения перенаправляются на localhost. Дополнительные сведения о дочерних сеансах см. в разделе [дочерние сеансы](child-sessions.md). |
| дисаблекредентиалсделегатион | **Логическое значение VT \_** | Чтение/запись | Нет | Если **значение — true**, учетные данные не отправляются на удаленный сервер. |
| енаблефрамебуфферредиректион | **Логическое значение VT \_** | Чтение/запись | Нет | **Значение true** показывает, что выполняется перенаправление буфера кадров. Для подключения замыкания на себя (один и тот же компьютер является клиентским и серверным) перенаправление буфера кадров позволяет совместно использовать память для буфера фрейма между сеансами. |
| енаблехардваремоде | **Логическое значение VT \_**  | Только запись | Нет | Если задано **значение true**, при помощи аппаратной декодирования осуществляется помощь оборудования. |
| игнорекурсорс | **Логическое значение VT \_** | Только запись | Нет | **Значение true** показывает, что курсоры, отправленные удаленным сервером, игнорируются. |
| мануалклипбоардсинценаблед | **Логическое значение VT \_** | Чтение/запись | Да | Установка этого свойства в **значение true** означает, что локальный и удаленный буфер обмена не будут автоматически синхронизироваться. Вместо этого необходимо использовать интерфейс [**имсрдпклипбоард**](imsrdpclipboard.md) для синхронизации форматов буфера обмена из локального буфера обмена с удаленным буфером обмена и удаленным буфером обмена с локальным буфером обмена. |
| зумлевел | ***VT \_ UI4** | Чтение/запись | Да | реализует функцию масштабирования с помощью элемента управления ActiveX RDP. Функция масштабирования доступна в **системном** меню RDP. Свойство **зумлевел** не действует в режиме RemoteApp и полноэкранном режиме. [**Имсрдпклиентадванцедсеттингс:: смартсизинг**](imsrdpclientadvancedsettings-smartsizing.md) и **зумлевел** являются взаимоисключающими. |
