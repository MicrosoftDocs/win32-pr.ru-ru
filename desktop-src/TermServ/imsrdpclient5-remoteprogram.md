# <a name="imsrdpclient5remoteprogram-property"></a>Свойство IMsRdpClient5:: Ремотепрограм

Извлекает объект, который поддерживает интерфейс [**итсремотепрограм**](itsremoteprogram.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_RemoteProgram(
  [out] ITSRemoteProgram **ppRemoteProgram
);
```



## <a name="property-value"></a>Значение свойства

Указатель интерфейса [**итсремотепрограм**](itsremoteprogram.md) .
