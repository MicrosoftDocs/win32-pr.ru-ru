# <a name="itsremoteprogramserverstartprogram-method"></a>Метод Итсремотепрограм:: Серверстартпрограм

Указывает программу RemoteApp для запуска в удаленном сеансе. Эта функция должна вызываться для подключенного сеанса (после получения уведомления о подключении к сеансу на клиенте). В сеансе можно запустить любое количество удаленных приложений RemoteApp. Время ожидания сеанса RemoteApp истечет, если в сеансе не будет запущено ни одной программы RemoteApp в течение времени ожидания, что составляет два минуты для Windows Server 2008.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ServerStartProgram(
  [in] BSTR         bstrExecutablePath,
  [in] BSTR         bstrFilePath,
  [in] BSTR         bstrWorkingDirectory,
  [in] VARIANT_BOOL vbExpandEnvVarInWorkingDirectoryOnServer,
  [in] BSTR         bstrArguments,
  [in] VARIANT_BOOL vbExpandEnvVarInArgumentsOnServer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстрексекутаблепас* \[ окне\]
</dt> <dd>

Путь к исполняемому файлу программы RemoteApp на сервере.

</dd> <dt>

*бстрфилепас* \[ окне\]
</dt> <dd>

Путь к файлу, открываемому на сервере с помощью сопоставления файлов, например "C: \\ \\ documents \\ \\MyReport.docx". При указании *бстрфилепас* не следует указывать параметр *бстрексекутаблепас* и наоборот. Следует указать только один из параметров.

</dd> <dt>

*бстрворкингдиректори* \[ окне\]
</dt> <dd>

Рабочий каталог на сервере для программы RemoteApp.

</dd> <dt>

*вбекспанденвваринворкингдиректорйонсервер* \[ окне\]
</dt> <dd>

Указывает, должен ли сервер расширять переменные среды в пути к рабочему каталогу. Присвойте этому параметру значение **\_ true** , если путь к рабочему каталогу содержит переменные среды, или значение **\_ false** , если путь к рабочему каталогу не содержит переменных среды.

</dd> <dt>

*бстраргументс* \[ окне\]
</dt> <dd>

Аргументы командной строки для программы RemoteApp, указанные в *бстрексекутаблепас*. Установите значение **null** , если *бстрексекутаблепас* не указан.

</dd> <dt>

*вбекспанденвваринаргументсонсервер* \[ окне\]
</dt> <dd>

Указывает, должен ли сервер расширять переменные среды в аргументах командной строки. Присвойте этому параметру значение **\_ true** , если аргументы содержат переменные среды, **или \_ значение false** , если аргументы не содержат переменных среды.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

При успешном выполнении возвращает значение **\_ ОК** .
