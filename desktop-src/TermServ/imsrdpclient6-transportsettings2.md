# <a name="imsrdpclient6transportsettings2-property"></a>Свойство IMsRdpClient6:: TransportSettings2

Извлекает интерфейс [**IMsRdpClientTransportSettings2**](imsrdpclienttransportsettings2.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_TransportSettings2(
  [out] IMsRdpClientTransportSettings2 **ppXportSet2
);
```



## <a name="property-value"></a>Значение свойства

Указатель интерфейса [**IMsRdpClientTransportSettings2**](imsrdpclienttransportsettings2.md) .
