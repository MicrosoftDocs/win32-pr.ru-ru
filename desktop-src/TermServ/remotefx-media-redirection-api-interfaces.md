# <a name="remotefx-media-redirection-api-interfaces"></a>Интерфейсы API перенаправления мультимедиа RemoteFX

API перенаправления мультимедиа RemoteFX поддерживает следующие интерфейсы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**ивтсбитмапрендерер**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderer)
</dt> <dd>

Используется динамически подключаемым модулем виртуального канала для отображения растровых изображений.

</dd> <dt>

[**ивтсбитмапрендереркаллбакк**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderercallback)
</dt> <dd>

Динамический подключаемый модуль виртуального канала реализует этот интерфейс, чтобы получать уведомления при изменении размера области отрисовки.

</dd> <dt>

[**ивтсбитмапрендерсервице**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderservice)
</dt> <dd>

Эта служба используется для создания визуального сопоставления на клиенте, соответствующем сопоставленному окну на сервере.

</dd> <dt>

[**ивтсплугинсервицепровидер**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtspluginserviceprovider)
</dt> <dd>

Предоставляет способ для динамических подключаемых модулей виртуальных каналов для запроса различных удаленный рабочий стол клиентских служб.

</dd> </dl>
