# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3: свойство Евицеколлектион:D

Извлекает коллекцию объектов устройств самонастраивающийся (PnP) для перенаправления.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>Значение свойства

Извлекает коллекцию объектов устройств PnP типа [**имсрдпдевице**](imsrdpdevice.md).
