# <a name="imsrdpclient5advancedsettings6-property"></a>Свойство IMsRdpClient5:: AdvancedSettings6

Извлекает интерфейс [**IMsRdpClientAdvancedSettings5**](imsrdpclientadvancedsettings5.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_AdvancedSettings6(
  [out] IMsRdpClientAdvancedSettings5 **ppAdvSettings
);
```



## <a name="property-value"></a>Значение свойства

Указатель интерфейса [**IMsRdpClientAdvancedSettings5**](imsrdpclientadvancedsettings-interface.md) .
