# <a name="remote-desktop-gateway-enumeration-types"></a>Типы перечислений шлюза удаленный рабочий стол

Интерфейс API шлюза удаленный рабочий стол поддерживает следующие типы перечислений.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**аааккаунтингдататипе**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaaccountingdatatype)
</dt> <dd>

Указывает тип события, по которому уведомляется метод [**итсгаккаунтинженгине::D оаккаунтинг**](/windows/desktop/api/TSGPolicyEngine/nf-tsgpolicyengine-itsgaccountingengine-doaccounting) .

</dd> <dt>

[**ааауссчемес**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaauthschemes)
</dt> <dd>

Указывает тип проверки подлинности, используемый для подключения к шлюзу удаленных рабочих столов.

</dd> <dt>

[**полициаттрибутетипе**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-policyattributetype)
</dt> <dd>

Задает параметры перенаправления, связанные с соединением.

</dd> </dl>
