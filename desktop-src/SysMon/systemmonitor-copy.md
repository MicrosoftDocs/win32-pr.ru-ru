# <a name="systemmonitorcopy-method"></a>Метод Системмонитор:: Copy

Копирует параметры свойств элемента управления, список счетчиков и данные счетчиков в буфер обмена в виде объекта HTML.

## <a name="syntax"></a>Синтаксис


```VB
Sub Copy()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

В буфере обмена может существовать только одна копия.
