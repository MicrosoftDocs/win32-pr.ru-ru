# <a name="systemmonitortimebarcolor-property"></a>Системмонитор. Тимебарколор, свойство

Возвращает или задает цвет временной шкалы. Временная шкала — это вертикальная линия, которая перемещается вдоль окна графа для указания прохода каждого интервала выборки в представлении графика.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Значение свойства

Цвет временной шкалы.
