# <a name="mci-device-capabilities"></a>Возможности устройства MCI

МЦивнд содержит следующие макросы, позволяющие запрашивать их возможности на устройствах MCI.



| Макрос                                      | Описание                                                                                                                                 |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**мЦивндканконфиг**](/windows/desktop/api/Vfw/nf-vfw-mciwndcanconfig) | Определяет, имеет ли устройство диалоговое окно конфигурации для поддержки нескольких конфигураций, например устройства МЦИАВИ.                   |
| [**мЦивндканежект**](/windows/desktop/api/Vfw/nf-vfw-mciwndcaneject)   | Определяет, имеет ли устройство функцию извлечения, управляемой программным обеспечением.                                                                       |
| [**мЦивндканплай**](/windows/desktop/api/Vfw/nf-vfw-mciwndcanplay)     | Определяет, может ли устройство воспроизводить существующее содержимое.                                                                                  |
| [**мЦивндканрекорд**](/windows/desktop/api/Vfw/nf-vfw-mciwndcanrecord) | Определяет, может ли устройство записывать.                                                                                                     |
| [**мЦивндкансаве**](/windows/desktop/api/Vfw/nf-vfw-mciwndcansave)     | Определяет, может ли устройство хранить данные.                                                                                                 |
| [**мЦивндканвиндов**](/windows/desktop/api/Vfw/nf-vfw-mciwndcanwindow) | Определяет, поддерживает ли устройство команды окна MCI (например, [**окно**](window.md), [**Размещение**](put.md) и [**место**](where.md)). |



 

Эти макросы возвращают **значение true** , если устройство поддерживает определенную возможность, или **false** в противном случае.
