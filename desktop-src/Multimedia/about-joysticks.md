<span data-ttu-id="b4672-111">Джойстик является вспомогательным устройством ввода для приложений, которые предоставляют альтернативы использованию клавиатуры и мыши.</span><span class="sxs-lookup"><span data-stu-id="b4672-111">The joystick is an ancillary input device for applications that provide alternatives to using the keyboard and mouse.</span></span> <span data-ttu-id="b4672-112">Джойстик предоставляет сведения о позиционировании в системе координат с абсолютным максимальным и минимальным значениями на каждой оси перемещения.</span><span class="sxs-lookup"><span data-stu-id="b4672-112">The joystick provides positional information within a coordinate system that has absolute maximum and minimum values in each axis of movement.</span></span>
