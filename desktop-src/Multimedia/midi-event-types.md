# <a name="midi-event-types"></a>Типы событий MIDI

Элемент **двевент** структуры [**МИДИЕВЕНТ**](/windows/win32/api/mmeapi/ns-mmeapi-midievent) описывает событие MIDI, которое необходимо выполнить. Короткие события полностью помещаются в этот элемент. Для длинных событий требуется одно или несколько значений даублеворд в дополнение к элементу **двевент** для хранения описаний событий.

Старший байт элемента **двевент** содержит сведения о том, является ли событие длинным или коротким и о том, создается ли обратный вызов вместе с событием. Кроме того, этот байт используется для описания типа события. Оставшиеся 24 бита элемента **двевент** используются либо для хранения параметров события (для коротких сообщений), либо для хранения длительности параметров события (для длинных сообщений). Чтобы извлечь сведения из члена **двевент** , используйте макросы [**Мевт \_ EVENTTYPE**](/windows/win32/api/mmeapi/nf-mmeapi-mevt_eventtype) и [**мевт \_ евентпарм**](/windows/win32/api/mmeapi/nf-mmeapi-mevt_eventparm) .

Описание предопределенных типов событий см. в справочном материале по структуре **мидиевент** .
