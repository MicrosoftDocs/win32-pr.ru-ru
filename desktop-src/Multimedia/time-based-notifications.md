# <a name="time-based-notifications"></a>Уведомления Time-Based

Вы можете уведомить операционную систему, чтобы отправлять сообщения джойстика в приложение через регулярные интервалы времени, установив для параметра *Фчанжед* [**жойсеткаптуре**](/windows/win32/api/joystickapi/nf-joystickapi-joysetcapture) **значение false** и указав длину интервала между последовательными сообщениями. Для этого назначьте параметру *упериод* значение между минимальной и максимальной частотой опроса для джойстика. Этот диапазон можно определить с помощью функции [**жойжетдевкапс**](/windows/win32/api/joystickapi/nf-joystickapi-joygetdevcaps) , которая заполняет элементы **впериодмин** и **впериодмакс** в структуре [**жойкапс**](/windows/win32/api/joystickapi/ns-joystickapi-joycaps) . Если значение *упериод* находится за пределами диапазона допустимых частот опроса для джойстика, драйвер джойстика использует минимальную или максимальную частоту опроса, в зависимости от значения *упериод* .

> [!Note]  
> Windows настраивает событие таймера при каждом вызове **жойсеткаптуре**.
