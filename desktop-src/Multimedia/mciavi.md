# <a name="mciavi"></a>мЦиави

AVI-файл может содержать более двух потоков — например, последовательность видео, звуковая дорожка на английском языке и французская звуковая дорожка. Приложение может использовать поток независимо от других потоков в файле.

Тип устройства **дигиталвидео** управляет видеофайлами. Список команд MCI, распознаваемых цифровыми устройствами, см. в разделе [набор команд цифрового видео](digital-video-command-set.md).

Драйвер МЦИАВИ воспроизводит видеопоследовательности и другие потоки данных под контролем команд MCI. Потоки данных могут содержать изображения, звук и палитры. Данные изображения могут состоять из изображений с цветовыми палитрами или со сведениями о истинном цвете.

Звук синхронизируется с видео в пределах одного сиртиес секунды. Однако если звуковое оборудование недоступно, драйвер воспроизводит только поток видео. Драйвер МЦИАВИ может удалять видеокадры, если это необходимо, для воспроизведения потока без прерывания звука.

Приложение может использовать службы классов окон МЦивнд вместо интерфейса команд MCI для управления любым драйвером MCI. Этот класс окон обрабатывает множество деталей управления окном, поддерживающим устройство MCI, и упрощает программирование, необходимое для отправки команд MCI. Приложение может напрямую использовать службы библиотеки МЦивнд для управления устройством MCI или МЦивнд отображать панель инструментов, полосу прокрутки и меню, позволяющие пользователю управлять устройством. Дополнительные сведения о классе окна МЦивнд см. в разделе [класс окна мЦивнд](mciwnd-window-class.md).
