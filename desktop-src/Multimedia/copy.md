# <a name="copy-command"></a>команда copy

Команда Copy копирует данные в буфер обмена. Устройство Digital-Video распознает эту команду.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("copy %s %s %s"), 
  lpszDeviceID, 
  lpszItem, 
  lpszFlags
); 
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lpszDeviceID"></span><span id="lpszdeviceid"></span><span id="LPSZDEVICEID"></span>*лпсздевицеид*
</dt> <dd>

Идентификатор устройства MCI. Этот идентификатор или псевдоним назначается при открытии устройства.

</dd> <dt>

<span id="lpszItem"></span><span id="lpszitem"></span><span id="LPSZITEM"></span>*лпсзитем*
</dt> <dd>

Один из следующих флагов, определяющих копируемый элемент.



| Значение                 | Значение                                                                                                                                                                                                                                   |
|-----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| в *прямоугольнике*        | Указывает часть каждого кадра, который будет скопирован. Если этот параметр опущен, по умолчанию используется весь фрейм.                                                                                                                             |
| *поток* аудиопотока | Указывает аудиопоток в рабочей области, затронутой командой. Если вы используете этот флаг, а также хотите скопировать видео, необходимо также использовать флаг "поток видео". (Если ни один из флагов не указан, копируются все потоки аудио и видео.) |
| от *расположения*       | Задает начало копируемого диапазона. Если этот параметр опущен, то значение по умолчанию равно текущему положению.                                                                                                                                         |
| в *Расположение*         | Задает конец копируемого диапазона. Скопированные аудио и видео данные являются исключительно этой позицией. Если этот параметр опущен, то значение по умолчанию — конец рабочей области.                                                                       |
| *поток* видео поток | Указывает поток видео в рабочей области, на который влияет команда. Если вы используете этот флаг, а также хотите скопировать аудио, необходимо также использовать флаг "Audio Stream". (Если ни один из флагов не указан, копируются все потоки аудио и видео.) |



 

</dd> <dt>

<span id="lpszFlags"></span><span id="lpszflags"></span><span id="LPSZFLAGS"></span>*лпсзфлагс*
</dt> <dd>

Может принимать значение "Wait", "notify", "Test" или их сочетание. Дополнительные сведения об этих флагах см. [в разделе Флаги ожидания, уведомления и проверки](the-wait-notify-and-test-flags.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.
