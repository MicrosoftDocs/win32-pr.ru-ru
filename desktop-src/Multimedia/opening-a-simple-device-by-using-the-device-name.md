# <a name="opening-a-simple-device-by-using-the-device-name"></a><span data-ttu-id="c47dc-104">Открытие простого устройства с помощью имени устройства</span><span class="sxs-lookup"><span data-stu-id="c47dc-104">Opening a Simple Device by Using the Device Name</span></span>

<span data-ttu-id="c47dc-105">В следующем примере показано, как открыть устройство воспроизведения компакт-дисков, указав имя устройства с помощью функции [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) .</span><span class="sxs-lookup"><span data-stu-id="c47dc-105">The following example opens a CD audio device by specifying the device name using the [**mciSendCommand**](/previous-versions//dd757160(v=vs.85)) function.</span></span>


```C++
UINT wDeviceID;
DWORD dwReturn;
MCI_OPEN_PARMS mciOpenParms;
 
// Opens a CD audio device by specifying the device name.

mciOpenParms.lpstrDeviceType = "cdaudio";

if (dwReturn = mciSendCommand(NULL, MCI_OPEN, MCI_OPEN_TYPE,
    (DWORD)(LPVOID) &mciOpenParms))
{
    
    // Error, unable to open device.
    
}

// The device opened successfully; get the device ID.
wDeviceID = mciOpenParms.wDeviceID;
```
