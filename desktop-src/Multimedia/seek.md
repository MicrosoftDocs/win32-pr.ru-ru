# <a name="seek-command"></a>Seek, команда

Команда Seek перемещается в указанную положение и останавливается. Эта команда распознает аудио компакт-диск, цифровое видео, устройство MIDI Sequencer, ВИДЕОМАГНИТОФОН, видеодиск и волна-аудиоустройства.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("seek %s %s %s"), 
  lpszDeviceID, 
  lpszSeekFlags, 
  lpszFlags
); 
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Комментарии

Перед выполнением команд, использующих значения позиций, необходимо задать требуемый формат времени с помощью команды [Set](set.md) .

Устройства Digital-Video поддерживают два режима поиска, которые можно изменить с помощью команды [Set](set.md) . Режим поиска в точном режиме приводит к перемещению команды Seek к указанному кадру. Режим поиска в режиме «в точности» приводит к тому, что команда Seek переходит к ближайшему ключевому кадру до указанного кадра.

Если при выполнении команды Seek воспроизводится устройство CD Audio, воспроизведение останавливается. Когда команда Seek выдается с устройства видеодиск, устройство выполняет поиск с использованием быстрых или быстрых обратных видео и звука.

Когда команда Seek выполняется с устройством аудио-аудио, поведение зависит от размера выборки. Если размер выборки составляет 16 бит или больше, поиск переходит к началу ближайшего примера, когда указанная положение не совпадает с началом образца.

## <a name="examples"></a>Примеры

Следующая команда выполняет поиск в начале файла мультимедиа, связанного с устройством "мисаунд".

``` syntax
seek mysound to start
```
