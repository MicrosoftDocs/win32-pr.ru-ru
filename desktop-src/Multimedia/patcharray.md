# <a name="patcharray"></a>патчаррай

ПАТЧАРРАЙ — это тип, используемый для определения массива исправлений MIDI.


```C++
typedef WORD PATCHARRAY[MIDIPATCHSIZE];
```



<dl> <dt>

**ПАТЧАРРАЙ \[ мидипатчсизе\]**
</dt> <dd>

Каждый элемент в массиве соответствует исправлению с каждым из 16 битов, представляющим один из 16 каналов MIDI. Биты задаются для каждого канала, который использует это конкретное исправление. Например, если номер исправления 0 используется физическими каналами MIDI 0 и 8, то элемент 0 массива должен иметь значение 0x0101.

</dd> </dl>
