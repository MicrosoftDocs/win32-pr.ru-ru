# <a name="window-command"></a>команда окна

Команда Window управляет окном просмотра. С помощью этой команды можно изменить параметры экрана окна или указать окно назначения, чтобы драйвер использовался вместо окна просмотра по умолчанию. Эта команда распознает цифровые видеоролики и устройства наложения видео.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("window %s %s %s"), 
  lpszDeviceID, 
  lpszWindowFlags, 
  lpszFlags
); 
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Remarks

Устройства наложения видео обычно создают и отображают окно при открытии. Если приложение предоставляет окно для драйвера, приложение несет ответственность за управление сообщениями, отправленными в окно.

Так как можно использовать команду [Status](status.md) для получения маркера в окне просмотра драйвера, можно также использовать стандартные функции диспетчера окон (например, [ShowWindow](/windows/win32/api/winuser/nf-winuser-showwindow)) для управления окном.

## <a name="examples"></a>Примеры

Следующая команда отображает и задает заголовок для окна воспроизведения "Movie".

``` syntax
window movie text "Welcome to the Movies" state show
```
