# <a name="mcierr-return-values"></a>Возвращаемые значения МЦИЕРР

Функции [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) и [**mciSendString**](/previous-versions//dd757161(v=vs.85)) возвращают нуль в случае успеха; в противном случае они возвращают значение **DWORD** , которое содержит одно из следующих значений ошибок в нижнем слове.

-   [Общие ошибки MCI](general-mci-errors.md)
-   [Ошибки mciSendString](mcisendstring-errors.md)
-   [Ошибки цифрового видео](digital-video-errors.md)
-   [Ошибки Sequencer](sequencer-errors.md)
-   [Волна-ошибки аудио](waveform-audio-errors.md)

Можно получить описание отдельных возвращаемых значений, передав возвращаемые значения в функцию [**мЦижетеррорстринг**](/previous-versions//dd757158(v=vs.85)) .
