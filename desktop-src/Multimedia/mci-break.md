# <a name="mci_break-command"></a>\_Команда MCI Break

Команда MCI \_ break задает ключ разрыва для устройства MCI. MCI поддерживает эту команду напрямую, а не передает ее на устройство. С помощью этой команды можно использовать любое приложение MCI.

Чтобы отправить эту команду, вызовите функцию [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) со следующими параметрами.


```C++
MCIERROR mciSendCommand(
  MCIDEVICEID wDeviceID, 
  MCI_BREAK, 
  DWORD dwFlags, 
  (DWORD) (LPMCI_BREAK_PARMS) lpBreak
);
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Remarks

Может потребоваться несколько раз нажать клавишу Break, чтобы прервать операцию ожидания. Нажатие клавиши break после завершения ожидания устройства отменяет возможность отправки перерыва в приложение. Если приложение содержит действие, определенное для кода виртуального ключа, оно может случайно ответить на прерывание. Например, приложение, использующее VK \_ Cancel для сочетания клавиш, может ответить на клавишу Ctrl + Break по умолчанию, если она нажата после отмены ожидания.

Следующие дополнительные флаги применяются ко всем устройствам:

<dl> <dt>

<span id="MCI_BREAK_HWND"></span><span id="mci_break_hwnd"></span>\_HWND разрыва MCI \_
</dt> <dd>

Элемент **хвндбреак** структуры, идентифицируемой *лпбреак* , содержит маркер окна, который должен быть текущим окном, чтобы включить обнаружение прерываний для этого устройства MCI. Обычно это главное окно приложения. Если этот параметр опущен, MCI не проверяет окно текущего окна.

</dd> <dt>

<span id="MCI_BREAK_KEY"></span><span id="mci_break_key"></span>\_ключ разрыва MCI \_
</dt> <dd>

Элемент **нвирткэй** структуры, определяемой параметром *лпбреак* , указывает код виртуального ключа, используемый для ключа разрыва. По умолчанию MCI назначает сочетание клавиш CTRL + BREAK в качестве клавиши разрыва. Этот флаг необходим, если \_ \_ не указан разрыв MCI.

</dd> <dt>

<span id="MCI_BREAK_OFF"></span><span id="mci_break_off"></span>\_разрыв MCI \_ отключен
</dt> <dd>

Отключает любой существующий ключ разрыва для указанного устройства.

</dd> </dl>
