# <a name="frame-windows-multimedia"></a>Рамка (мультимедиа Windows)

В окне отслеживания используются сообщения об ответном вызове кадров для уведомления приложения о доступности нового кадра видео. Окно Capture (захват) включает эти уведомления обратного вызова только в том случае, если скорость предварительной версии ненулевая, а запись потоковой передачи не выполняется.
