# <a name="mci_seek-command"></a>\_Команда MCI Seek

Команда MCI \_ Seek изменяет текущую положение в содержимом как можно быстрее. Во время поиска выходные данные видео и звука отключаются. После завершения поиска устройство будет остановлено. Эта команда распознает аудио компакт-диск, цифровое видео, устройство MIDI Sequencer, ВИДЕОМАГНИТОФОН, видеодиск и волна-аудиоустройства.

Чтобы отправить эту команду, вызовите функцию [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) со следующими параметрами.


```C++
MCIERROR mciSendCommand(
  MCIDEVICEID wDeviceID, 
  MCI_SEEK, 
  DWORD dwFlags, 
  (DWORD) (LPMCI_SEEK_PARMS) lpSeek
);
```
