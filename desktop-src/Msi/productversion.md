# <a name="productversion-property"></a>ProductVersion, свойство

Значение свойства **ProductVersion** — это версия продукта в строковом формате. Это свойство является обязательным.

Формат строки выглядит следующим образом:

<dl> ОсновнойНомерВерсии.ДополнительныйНомерВерсии.НомерПостроения  
</dl>

Первое поле является основной версией и имеет максимальное значение 255. Второе поле является дополнительным номером версии и имеет максимальное значение 255. Третье поле называется версией сборки или версией обновления и имеет максимальное значение 65 535.

## <a name="remarks"></a>Комментарии

По крайней мере одно из трех полей **ProductVersion** должно измениться для обновления с помощью [таблицы обновления](upgrade-table.md). Любое обновление, которое изменяет только код пакета, но оставляет **ProductVersion** и [**ProductCode**](productcode.md) без изменений, называется [небольшим обновлением](small-updates.md). Три поля версий предоставляются в первую очередь для удобства. Например, если вы хотите изменить **ProductVersion**, но не хотите изменять основную или дополнительную версии, можно изменить версию сборки.

Обратите внимание, что установщик Windows использует только первые три поля версии продукта. Если включить в версию продукта четвертое поле, установщик игнорирует четвертое поле.
