# <a name="moduleadvtuisequence-table"></a>Таблица Модулеадвтуисекуенце

Не создавать эту таблицу. Установщик Windows не использует [таблицу адвтуисекуенце](advtuisequence-table.md). Таблица Адвтуисекуенце не должна существовать или оставаться пустой в базе данных установки. Кроме того, не следует создавать таблицу Модулеадвтуисекуенце.
