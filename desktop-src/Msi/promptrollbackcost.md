# <a name="promptrollbackcost-property"></a>ПРОМПТРОЛЛБАКККОСТ, свойство

Свойство **промптроллбакккост** указывает действие, которое требуется установщику, если включены возможности [установки отката](rollback-installation.md) и недостаточно места на диске для завершения установки.

Возможные значения **промптроллбакккост** :



| Значение | Действие                                                              |
|-------|---------------------------------------------------------------------|
| С     | Отображение диалогового окна с вопросом, следует ли продолжить без отката. |
| D     | Отключите откат и продолжайте, не запрашивая пользователя.                  |
| C     | Завершается с запросом об ошибке нехватки места на диске.                       |



 

## <a name="remarks"></a>Remarks

Когда пользовательский интерфейс запускается на уровне Basic или без пользовательского интерфейса, установщик обрабатывает всю логику вне места на диске автоматически.

Когда пользовательский интерфейс выполняется на полном уровне, пользователю могут быть предоставлены дополнительные параметры, такие как запрос перед выполнением отката, отключение отката или продолжение без отката при заполнении диска. Разработчику пакета необходимо разработать последовательность диалоговых окон, чтобы предоставить пользователю соответствующие варианты. Доступные для этого элементы: [Енаблероллбакк таблице ControlEvent событие](enablerollback-controlevent.md), свойство [**Аутофдискспаце**](outofdiskspace.md) , свойство [**аутофнорбдискспаце**](outofnorbdiskspace.md) и свойство **промптроллбакккост** .
