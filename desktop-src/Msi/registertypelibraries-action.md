# <a name="registertypelibraries-action"></a>Действие Регистертипелибрариес

Действие Регистертипелибрариес регистрирует библиотеки типов в системе. Это действие применяется к каждому файлу, на который ссылается [Таблица typelib](typelib-table.md) , для которой запланирована установка.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Регистертипелибрариес должно следовать после действия [инсталлфилес](installfiles-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия                   |
|-------|----------------------------------------------|
| \[1\] | [Идентификатор GUID](guid.md) зарегистрированной библиотеки типов. |



 

## <a name="remarks"></a>Комментарии

Действие Регистертипелибрариес требует, чтобы язык библиотеки типов был правильно создан в поле Language таблицы TypeLib. Неправильная разработка поля Language может привести к тому, что установщик не сможет зарегистрировать библиотеку типов, допустимую иным образом.
