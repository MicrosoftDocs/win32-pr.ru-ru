# <a name="timeremaining-control-attribute"></a>Атрибут элемента управления TimeRemaining

Этот атрибут позволяет элементу управления "текст" отображать приблизительное количество минут и секунд, оставшихся при установке. Запись, передаваемая в элемент управления текстом, содержит одно целое число, представляющее количество оставшихся секунд. Элемент управления "текст" ищет в [таблице уитекст](uitext-table.md) строку с именем TimeRemaining и форматирует целое число в соответствующую строку, отображающую минуты и секунды.

## <a name="valid-controls"></a>Допустимые элементы управления

[Text](text-control.md)

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.
