# <a name="billboard-control"></a>Элемент управления "объявление"

Элемент управления "Реклама на стендах" отображает часто используемые элементы управления, которые добавляются и удаляются из диалогового окна Контролевентс. На объявление можно помещать только те элементы управления, которые не связаны со свойством, например [текст](text-control.md), [точечный рисунок](bitmap-control.md)или [значок](icon-control.md)или пользовательские элементы управления. Обычно элементы управления "объявление" отображают сообщения о ходе выполнения.

## <a name="control-attributes"></a>Атрибуты элемента управления

С элементом управления объявлением можно использовать следующие атрибуты. Чтобы изменить значение атрибута с помощью события, подпишите элемент управления на таблице ControlEvent событие в [таблице таблица eventmapping](eventmapping-table.md) и перечислите идентификатор атрибута в столбце Attribute. Введите идентификатор таблице ControlEvent событие в столбце событий.



| Идентификатор атрибута                                 | Шестнадцатеричный бит                  | Описание                                                                                                                                                                                                                                                                                                                  |
|------------------------------------------------------|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [биллбоарднаме](billboardname-control-attribute.md) |                                  | Имя текущего объявления. Введите идентификатор объявления в столбце объявления [таблицы ббконтрол](bbcontrol-table.md).<br/>                                                                                                                                                                            |
| [Позиция](position-control-attribute.md)           |                                  | Расположение элемента управления в диалоговом окне. Введите ширину элемента управления, высоту и координаты левого угла элемента управления в столбцах ширина, высота, X и Y [таблицы ббконтрол](bbcontrol-table.md). Используйте [единицы установщика](installer-units.md) для длины и расстояния.<br/>                                |
| [Visible](visible-control-attribute.md)             | 0x00000000 0x00000001<br/> | Скрытый элемент управления. Видимый элемент управления.<br/> Включите этот бит в битовое слово столбца Attributes в [таблице ббконтрол](bbcontrol-table.md) , чтобы сделать элемент управления видимым или скрытым при его создании.<br/> Скрытие или отображение элемента управления с помощью [таблицы таблица controlcondition](controlcondition-table.md).<br/> |
| [Sunken](sunken-control-attribute.md)               | 0x00000000 0x00000004<br/> | Отображает стиль оформления по умолчанию. ## <a name="remarks"></a>Комментарии

Этот элемент управления не имеет собственного окна.

Элементы управления объявлением, которые отображаются в полном пользовательском интерфейсе, перечислены в [таблице объявлений](billboard-table.md).

Элементы управления, расположенные на рекламном стенде, должны быть перечислены в [таблице ббконтрол](bbcontrol-table.md) , а не в [таблице элементов управления](control-table.md).
