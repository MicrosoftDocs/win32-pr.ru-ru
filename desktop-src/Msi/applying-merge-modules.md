# <a name="applying-merge-modules"></a>Применение модулей слияния

Модули слияния должны быть объединены в установочный пакет с помощью средства слияния. Лучшим вариантом является получение свободно распределенного инструмента слияния или приобретение одного из средств слияния, доступных от независимых поставщиков программного обеспечения. Например, можно использовать функциональные возможности, предоставляемые Mergemod.dll.

Модули слияния на нескольких языках могут быть открыты на разных языках, а затем объединены в целевой объект для создания языковой версии пакета. Дополнительные сведения можно найти в разделе

-   [Использование API для слияния модуля слияния с базой данных](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Открытие модуля слияния Multiple-Language на определенном языке](opening-a-multiple-language-merge-module-in-a-specific-language.md)
