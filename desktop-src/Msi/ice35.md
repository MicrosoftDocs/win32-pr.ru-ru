# <a name="ice35"></a>ICE35

ICE35 проверяет, что компоненты, содержащие сжатые файлы, хранящиеся в [CAB-файле](cabinet-files.md) , не настроены для запуска из источника. при использовании установщик Windows 2,0 или более поздней версии это ограничение было удалено.

ICE35 запрашивает столбец CAB-файла [таблицы Media](media-table.md) , чтобы определить, какие файлы сжаты и хранятся в CAB-файле. Он запрашивает [таблицу файлов](file-table.md) , чтобы определить, какие компоненты содержат эти файлы. Наконец, он проверяет [таблицу Component](component-table.md) , чтобы определить, заданы ли биты запуска от источника в столбце Attributes.

## <a name="result"></a>Результат

ICE35 отправляет сообщение об ошибке, если имеется сжатый файл, хранящийся в CAB-файле, принадлежащем компоненту с установленным битом Мсидбкомпонентаттрибутессаурцеонли в столбце Attributes [таблицы Component](component-table.md). при использовании установщик Windows 2,0 или более поздней версии это сообщение об ошибке меняется на предупреждение. в пакете, поддерживающем только установщик Windows 2,0 и более поздних версий, \_ свойство PID PAGECOUNT потока сводных данных имеет значение не менее 200.

ICE35 отправляет предупреждение, если имеется сжатый файл, хранящийся в CAB-файле, принадлежащем компоненту с установленным битом Мсидбкомпонентаттрибутесоптионал в столбце Attributes [таблицы Component](component-table.md). это предупреждающее сообщение было удалено с установщик Windows 2,0 и более поздних версий.

Если несколько файлов в компоненте находятся в CAB-файле, ICE35 сообщает об ошибках для каждого файла с установленным битом запуска из источника.

## <a name="example"></a>Пример

ICE35 сообщает о следующих ошибках и предупреждениях в примере с использованием более ранней версии, чем установщик Windows версии 2,0.



| ICE35, ошибка                                                                                                | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ошибка: Component3 компонента невозможно запустить только из источника, так как файл его члена "Файл3" сжат. | Существует сжатый файл, хранящийся в CAB-файле, и этот файл принадлежит компоненту с Саурцеонли битом, заданным в столбце Attributes [таблицы Component](component-table.md). Чтобы устранить эту ошибку, измените младшие 2 бита значения атрибутов Component2's на "00", то есть только локально или удалите "Файл4" из CAB-файла.<br/>                                                                                                                                                                         |
| Ошибка: Component3 компонента невозможно запустить только из источника, так как файл его члена "Файл3" сжат. | Существует сжатый файл, хранящийся в CAB-файле, и этот файл принадлежит компоненту с Саурцеонли битом, заданным в столбце Attributes [таблицы Component](component-table.md). Поскольку файлы в компоненте не должны исходить из одного носителя, ICE35 сообщает об ошибках для каждого файла в компоненте, который находится в CAB-файле.<br/> Чтобы устранить эту ошибку, измените младшие 2 бита значения атрибутов Component2's на "00", то есть только локально или удалите "Файл4" из CAB-файла.<br/> |
