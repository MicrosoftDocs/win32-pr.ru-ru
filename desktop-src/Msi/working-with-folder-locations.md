# <a name="working-with-folder-locations"></a>Работа с расположениями папок

Используйте следующие функции для поиска источника или целевого объекта указанной папки в [таблице Directory](directory-table.md):

-   Получите путь к указанной исходной папке, вызвав функцию [**мсижетсаурцепас**](/windows/desktop/api/Msiquery/nf-msiquery-msigetsourcepatha) .
-   Получите путь к указанной целевой папке, вызвав функцию [**мсижеттаржетпас**](/windows/desktop/api/Msiquery/nf-msiquery-msigettargetpatha) .
-   Измените целевое расположение папки с помощью функции [**мсисеттаржетпас**](/windows/desktop/api/Msiquery/nf-msiquery-msisettargetpatha) .
