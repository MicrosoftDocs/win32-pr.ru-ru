# <a name="ice58"></a>ICE58

ICE58 проверяет, что в [таблице мультимедиа](media-table.md) не более 80 строк.

## <a name="result"></a>Результат

предупреждения, о которых сообщает ICE58, приводят к сбою установки, если пакет не установлен с установщик Windows 2,0 или более поздней версии. начиная с установщик Windows 2,0, ограничение на более 80 записей таблицы мультимедиа удаляется. Если свойство [**сводки количества страниц**](page-count-summary.md) пакета больше или равно 150, предупреждение не выдается. пакеты схемы 200 или более поздней версии можно установить только установщик Windows 2,0 или более поздней версии.

## <a name="example"></a>Пример

ICE58 сообщает о следующих ошибках и предупреждениях для приведенного примера.

``` syntax
This package has 81 media entries. Packages are limited to 80 entries in the media table.
```

Чтобы устранить эту ошибку, исключите все неиспользуемые записи таблицы мультимедиа, объедините записи таблицы мультимедиа, которые ссылаются на один и тот же носитель, и переупакуйте приложение, чтобы уменьшить необходимый объем носителей.

[Таблица мультимедиа](media-table.md) (частичная)



| DiskId | ластсекуенце\_ |
|--------|----------------|
| 1      | 10             |
| 2      | 20             |
| ...    | ...            |
| 81     | 810            |
