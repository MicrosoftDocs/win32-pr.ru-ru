# <a name="msitranexe"></a>Msitran.exe

Msitran.exe использует [**мсидатабасеженератетрансформ**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasegeneratetransforma), [**мсикреатетрансформсуммаринфо**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa)и [**мсидатабасеапплитрансформ**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseapplytransforma) для создания или применения файла преобразования.

это средство доступно только в [компонентах Windows SDK для разработчиков установщик Windows](platform-sdk-components-for-windows-installer-developers.md).

## <a name="syntax"></a>Синтаксис

Для создания преобразования используйте следующий синтаксис.

**мситран-g** *{base DB} {ref DB} {имя файла преобразования} \[ {условия ошибок/условия проверки} \]*

Используйте следующий синтаксис для применения преобразования

**мситран-a** *{Преобразование} {Database} \[ {условия ошибки} \]*

## <a name="command-line-options"></a>Параметры командной строки

В Msitran.exe используются следующие параметры командной строки без учета регистра. Вместо тире можно использовать разделитель с косой чертой.



| Параметр | Описание            |
|--------|------------------------|
| -g     | Создание преобразования.  |
| -a     | Преобразование приложения. |



 

При применении преобразования могут быть подавлены следующие ошибки. Чтобы отключить ошибку, включите соответствующий символ в аргумент {error conditions}. Условия, заданные параметром-g, помещаются в сводные данные преобразования, но не используются при применении преобразования с параметром-a. Дополнительные сведения см. в разделе [**мсидатабасеапплитрансформ**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseapplytransforma).



| Параметр | Подавленная ошибка           |
|--------|----------------------------|
| а      | Добавить существующую строку.          |
| b      | Удаление несуществующей строки.   |
| с      | Добавить существующую таблицу.        |
| d      | Удаление несуществующей таблицы. |
| й      | Изменить существующую строку.       |
| f      | Изменить кодовую страницу.           |



 

Для указания, когда преобразование может быть применено к пакету, можно использовать следующие условия проверки. Эти условия могут быть указаны с помощью-g, но не от-a.



| Параметр | Условие проверки                                  |
|--------|-------------------------------------------------------|
| н      | Проверьте код обновления.                                   |
| l      | Проверить язык.                                       |
| p      | Проверка платформы.                                       |
| r      | Проверьте продукт.                                        |
| s      | Проверьте только основной номер версии.                             |
| t      | Проверьте только основной и дополнительный версии.                  |
| u      | Проверьте основные, дополнительные и обновленные версии.             |
| v      | Примененная версия базы данных < версия базовой базы данных.  |
| w      | Примененная версия базы данных <= базовая версия базы данных. |
| x      | Примененная версия базы данных = базовая версия базы данных.     |
| y      | Примененная версия базы данных >= базовая версия базы данных. |
| z      | Примененная версия базы данных > версия базовой базы данных.  |
