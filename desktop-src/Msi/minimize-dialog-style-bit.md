# <a name="minimize-dialog-style-bit"></a>Минимальный бит стиля диалогового окна

Если этот бит задан, то диалоговое окно можно будет сделать минимальным. Этот бит игнорируется для модальных диалоговых окон, которые не могут быть сведены к сведению.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **мсидбдиалогаттрибутесминимизе** |
