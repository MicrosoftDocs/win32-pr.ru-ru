# <a name="msipatchheaders-table"></a>Таблица Мсипатчхеадерс

Таблица Мсипатчхеадерс содержит потоки заголовков двоичного обновления, используемые для проверки исправлений.

Таблица Мсипатчхеадерс используется, если ключи длинных [таблиц файлов](file-table.md) приводят к сбою при создании потока заголовков исправлений в [таблице Patch](patch-table.md). это может быть вызвано ограничением имен потоков, описанным в разделе [ограничения по OLE для Потоки](ole-limitations-on-streams.md). В этом случае таблица Patch может ссылаться на таблицу Мсипатчхеадерс для создания потока заголовков исправлений.

Таблица Мсипатчхеадерс содержит следующие столбцы.



| Столбец    | Type                         | Ключ | Допускает значения NULL |
|-----------|------------------------------|-----|----------|
| стреамреф | [Идентификатор](identifier.md) | Д   | Нет        |
| Заголовок    | [Двоичный](binary.md)         | Нет   | Нет        |

## <a name="remarks"></a>Remarks

Эта таблица обрабатывается [действием патчфилес](patchfiles-action.md). Эта таблица обычно добавляется в пакет установки путем преобразования из пакета исправлений. Обычно он не создается непосредственно в установочном пакете.
