# <a name="msintsuitesmallbusiness-property"></a>Мсинтсуитесмаллбусинесс, свойство

в Windows 2000 и более поздних операционных системах установщик устанавливает для свойства **мсинтсуитесмаллбусинесс** значение 1, если установлен Microsoft Small Business Server. Установщик задает для этого свойства значение 1, только если в \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) установлен флаг смаллбусинесс набора ver Suite. В противном случае установщик не устанавливает это свойство.
