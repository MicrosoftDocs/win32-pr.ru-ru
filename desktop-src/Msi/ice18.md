# <a name="ice18"></a>ICE18

ICE18 проверяет, что все пустые каталоги, используемые в качестве пути к ключу для компонента, перечислены в [таблице креатефолдер](createfolder-table.md).

Если столбец ключевого пути в [таблице Component](component-table.md) имеет значение null, это означает, что каталог, указанный в столбце Directory, \_ является ключевым путем для этого компонента. Поскольку папки, созданные установщиком, удаляются, когда они становятся пустыми, эта папка должна быть указана в [таблице креатефолдер](createfolder-table.md) , чтобы установщик не пытался установить каждый раз.

Не следует делать каталог Системфолдер ключевым путем к компоненту. Так как эта папка имеется в каждой операционной системе, установщик всегда определяет путь к ключу независимо от наличия компонента. В этом случае путь к ключу должен быть файлом, записью реестра или источником данных ODBC.

При выполнении проверки ICE18 сначала проверяет, выполняются ли следующие условия:

-   Ключевой столбец [таблицы Component](component-table.md) содержит значение null.
-   В [таблице File](file-table.md)нет файлов, перечисленных для компонента.
-   Отсутствие файлов для компонента, указанного в [таблице ремовефиле](removefile-table.md) , и то, что значение в дирпроперти совпадает со \_ столбцом Directory [таблицы Component](component-table.md).
-   Отсутствие файлов для компонента, указанного в [таблице дупликатефиле](duplicatefile-table.md) , и то, что значение в DestFolder совпадает со \_ столбцом Directory [таблицы Component](component-table.md).
-   Отсутствие файлов для компонента, указанного в [таблице MoveFile](movefile-table.md) , и то, что значение в DestFolder совпадает со \_ столбцом Directory [таблицы Component](component-table.md).

Если все это верно, ICE18 проверяет следующее:

-   \_Столбец Component [таблицы креатефолдер](createfolder-table.md) имеет то же значение, что и столбец Component [таблицы Component](component-table.md).
-   \_Столбец Directory таблицы креатефолдер имеет то же значение, что и \_ столбец Directory [таблицы Component](component-table.md).

## <a name="result"></a>Результат

ICE18 отправляет сообщение об ошибке, если установочный пакет указывает каталог в качестве пути к разделу для компонента, который не указан в [таблице креатефолдер](createfolder-table.md).
