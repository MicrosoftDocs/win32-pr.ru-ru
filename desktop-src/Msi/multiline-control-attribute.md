# <a name="multiline-control-attribute"></a>Атрибут элемента управления MultiLine

Если этот бит установлен в [элементе управления](edit-control.md)"поле ввода", установщик создает многострочный элемент управления "поле ввода" с вертикальной полосой прокрутки.

Если этот бит не задан, он задает отображение обычного элемента управления "Правка".

## <a name="valid-controls"></a>Допустимые элементы управления

[Изменить элемент управления](edit-control.md).



| Decimal | Шестнадцатеричный | Константа                            |
|---------|-------------|-------------------------------------|
| 65536   | 0x00010000  | **мсидбконтролаттрибутесмултилине** |



 

## <a name="remarks"></a>Комментарии

Чтобы задать этот атрибут для элемента управления, включите многострочный бит в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
