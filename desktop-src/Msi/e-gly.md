# <a name="e-windows-installer"></a>E (установщик Windows)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) E [F](f-gly.md) [G](g-gly.md) р [. J K](i-gly.md) L [M](m-gly.md) N [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [U](u-gly.md) [V](v-gly.md) W X Y Z

<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**повышенными**
</dt> <dd>

Действия, выполняемые с системными привилегиями, называются повышенными.

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**этап выполнения**
</dt> <dd>

Когда Установщик выполняет сценарий действий установщика. в Microsoft Windows 2000 этот процесс выполняется службой установщика. В управляемом приложении скрипт выполняется с системными привилегиями. Дополнительные сведения см. в разделе [механизм установки](installation-mechanism.md).

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**сценарий выполнения**
</dt> <dd>

Действия установщика для установки. Сценарий выполнения создается на этапе [*получения*](a-gly.md) установки и выполняется на [*этапе выполнения*](/windows). Дополнительные сведения см. в разделе [механизм установки](installation-mechanism.md).

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**Внешние исходные файлы**
</dt> <dd>

Исходные файлы устанавливаемого приложения, которые хранятся вне [*файла.msi*](m-gly.md). Несколько внешних исходных файлов можно сжать и сохранить вместе в [*CAB-файле*](c-gly.md) , входящем в [*пакет*](p-gly.md).

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**внешний пользовательский интерфейс**
</dt> <dd>

Пользовательский интерфейс, предоставленный автором пакета установки. Он не использует внутренние возможности пользовательского интерфейса установщика. Дополнительные сведения см. [в разделе о пользовательском интерфейсе](about-the-user-interface.md).

</dd> </dl>
