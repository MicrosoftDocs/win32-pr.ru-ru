# <a name="previewing-the-user-interface"></a>Предварительный просмотр пользовательского интерфейса

Установщик сохраняет все сведения о [пользовательском интерфейсе](user-interface.md) в таблицах базы данных установки. Для упрощения разработки пользовательского интерфейса установщик также предоставляет режим предварительного просмотра для просмотра этих сведений. В следующей процедуре описывается включение режима предварительного просмотра пользовательского интерфейса и отображение текущего внешнего вида диалоговых окон и объявлений.

**Просмотр пользовательского интерфейса в режиме предварительного просмотра**

1.  Включите режим предварительного просмотра, вызвав функцию [**мсиенаблеуипревиев**](/windows/desktop/api/Msiquery/nf-msiquery-msienableuipreview) .
2.  Отображение определенных диалоговых окон путем вызова функции [**мсипревиевдиалог**](/windows/desktop/api/Msiquery/nf-msiquery-msipreviewdialoga) .
3.  Отображение определенных объявлений путем вызова функции [**мсипревиевбиллбоард**](/windows/desktop/api/Msiquery/nf-msiquery-msipreviewbillboarda) .
