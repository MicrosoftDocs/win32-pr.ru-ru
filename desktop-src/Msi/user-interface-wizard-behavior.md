# <a name="user-interface-wizard-behavior"></a>Поведение мастера пользовательского интерфейса

Для создания простого пользовательского интерфейса разработчики пакетов установки могут использовать структуру пользовательского интерфейса, которая соответствует поведению мастера интерфейса.

Поведение мастера термина "Пользовательский интерфейс" означает, что каждое диалоговое окно в последовательности содержит **следующую>>** кнопку, которую пользователь нажимает для перехода к следующему диалоговому окну после ввода данных или настройки сведений в текущем диалоговом окне. Если пользователь решает вернуться и изменить любые сведения, указанные в предыдущем диалоговом окне, каждое диалоговое окно содержит кнопку **<<назад** , которую пользователь щелкает для возврата. В конце последовательности мастера пользователь нажимает кнопку **Готово** , чтобы начать процесс установки.

Пользовательский интерфейс, описанный в примере ПОЛЬЗОВАТЕЛЬСКОГО интерфейса, соответствует поведению мастера пользовательского интерфейса. См. [Пример установки](an-installation-example.md).
