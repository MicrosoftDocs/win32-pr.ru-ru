# <a name="ice48"></a>ICE48

ICE48 проверяет наличие каталогов, которые жестко запрограммированы на локальные пути в [таблице свойств](property-table.md).

Не следует жестко кодировать пути к каталогам на локальных дисках, так как компьютеры отличаются в процессе установки локального диска. Такой подход может быть приемлемым при развертывании приложения на большом количестве компьютеров, где все соответствующие части дисков одинаковы.

## <a name="result"></a>Результат

ICE48 отправляет сообщение об ошибке, если имеется каталог с жестко запрограммированным локальным путем в таблице свойств.

## <a name="example"></a>Пример

ICE48 сообщит о следующем предупреждении для показанного примера.

``` syntax
Directory 'Dir1' appears to be hardcoded in the property 
    table to a local drive.
```

[Таблица каталогов](directory-table.md) (частичная)



| Каталог | \_Родительский каталог | дефаултдир |
|-----------|-------------------|------------|
| Dir1      |                   | SourceDir  |



 

[Таблица свойств](property-table.md) (частичная)



| Свойство | Значение      |
|----------|------------|
| Dir1     | d: \\ источник |
