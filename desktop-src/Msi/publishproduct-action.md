# <a name="publishproduct-action"></a>Действие Публишпродукт

Действие Публишпродукт управляет объявлением сведений о продукте в системе. Это действие публикует продукт, если продукт находится в режиме объявления или если какой-либо компонент устанавливается или переустанавливается.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Публишпродукт должно следовать после действия [публишфеатурес](publishfeatures-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия        |
|-------|-----------------------------------|
| \[1\] | Идентификатор объявленного продукта. |
