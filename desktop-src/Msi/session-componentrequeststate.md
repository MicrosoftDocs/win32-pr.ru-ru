# <a name="sessioncomponentrequeststate-property"></a>Свойство Session. Компонентрекуестстате

Свойство **компонентрекуестстате** объекта [**Session**](session-object.md) получает или запрашивает изменение в состоянии действия строки в [таблице Component](component-table.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.ComponentRequestState
```



## <a name="property-value"></a>Значение свойства

Обязательное строковое имя элемента компонента, первичный ключ таблицы Component.

## <a name="remarks"></a>Remarks



| Состояние выбора        | Значение | Описание                                                    |
|------------------------|-------|----------------------------------------------------------------|
| NULL                   | NULL  | Запрашивает, что действие для этого элемента не выполняется.                |
| мсиинсталлстатеабсент  | 2     | Элемент будет удален.                                         |
| мсиинсталлстателокал   | 3     | Элемент должен быть установлен локально.                               |
| мсиинсталлстатесаурце  | 4     | Элемент должен быть установлен и запущен с исходного носителя.         |
| мсиинсталлстатедефаулт | 5     | При установке элемент будет переустановлен в том же состоянии. |



 

Если свойство завершается неудачно, можно получить расширенные сведения об ошибке с помощью метода [**ластерроррекорд**](installer-lasterrorrecord.md) .
