# <a name="fatalerror-dialog"></a>Диалоговое окно FatalError

Это модальное диалоговое окно отображается в конце установки, если установка прекращена из-за неустранимой ошибки. Обычно это диалоговое окно содержит только некоторый текст и кнопку **ОК** . Имя этого диалогового окна должно появиться в [таблице админуисекуенце](adminuisequence-table.md) и [таблице инсталлуисекуенце](installuisequence-table.md) с-3 в качестве порядкового номера.
