# <a name="timedate"></a>Время и Дата

Тип данных "время/Дата" имеет время и дату, хранимую по отдельности, с использованием целых чисел без знака в качестве битовых полей, упакованных следующим образом.

## <a name="time"></a>Время

Время кодируется в виде 2-байтового целого числа без знака со следующими битовыми полями.



| Содержимое           | Bits        | Диапазон значений |
|--------------------|-------------|-------------|
| часы              | 0 1 2 3 4   | 0–23        |
| minutes            | 5 6 7 8 9 A | 0–59        |
| 2-секундные интервалы | B C D E F   | 0 – 29        |



 

## <a name="date"></a>Дата

Дата кодируется в виде 2-байтового целого числа без знака со следующими битовыми полями.



| Содержимое | Bits          | Диапазон значений              |
|----------|---------------|--------------------------|
| year     | 0 1 2 3 4 5 6 | 0 – 119 (относительно 1980) |
| month    | 7 8 9 A       | 1–12                     |
| day      | B C D E F     | 1–31                     |
