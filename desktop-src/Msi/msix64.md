# <a name="msix64-property"></a>Msix64, свойство

Свойство **Msix64** определяется только при запуске на процессоре x64. Значение задается установщиком для числового уровня процессора. Значения совпадают с полями **впроцессорлевел** в структуре [**системных \_ сведений**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) .

## <a name="remarks"></a>Remarks

**установщик Windows 2,0 и установщик Windows 3,0:** Не поддерживается. свойство **Msix64** доступно, начиная с установщик Windows 3,1.
