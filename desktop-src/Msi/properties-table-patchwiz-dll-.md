# <a name="properties-table-patchwizdll"></a>Таблица свойств (Patchwiz.dll)

В таблице свойств содержатся глобальные параметры для пакета исправлений. Таблица свойств является обязательной в базе данных создания исправлений (файл. PCP) и используется функцией [уикреатепатчпаккажеекс](uicreatepatchpackageex--patchwiz-dll-.md) .

Таблица свойств содержит следующие столбцы.



| Столбец | Type | Ключ | Допускает значения NULL |
|--------|------|-----|----------|
| Имя   | text | Да   | Нет        |
| Значение  | text |     | Да        |
