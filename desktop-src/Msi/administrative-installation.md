<span data-ttu-id="c8176-109">Администраторы могут запускать административную установку из командной строки с помощью [параметра командной строки](command-line-options.md)/a.</span><span class="sxs-lookup"><span data-stu-id="c8176-109">Administrators can run an administrative installation from the command line by using the /a [command line option](command-line-options.md).</span></span>

<span data-ttu-id="c8176-110">[Действие администратора](admin-action.md) — это действие верхнего уровня, используемое для запуска административной установки.</span><span class="sxs-lookup"><span data-stu-id="c8176-110">The [ADMIN action](admin-action.md) is the top-level action used to initiate an administrative installation.</span></span> <span data-ttu-id="c8176-111">При выполнении этого действия установщик вызывает действия в таблицах [админексекутесекуенце](adminexecutesequence-table.md) и [админуисекуенце](adminuisequence-table.md) для выполнения административной установки.</span><span class="sxs-lookup"><span data-stu-id="c8176-111">When this action is executed the installer calls the actions in the [AdminExecuteSequence](adminexecutesequence-table.md) and [AdminUISequence](adminuisequence-table.md) tables to perform the administrative installation.</span></span>

<span data-ttu-id="c8176-112">Свойство [**админпропертиес**](adminproperties.md) представляет собой разделенный точками с запятой список свойств, которые задаются во время установки администрирования.</span><span class="sxs-lookup"><span data-stu-id="c8176-112">The [**AdminProperties**](adminproperties.md) property is a semicolon delimited list of properties that are set at the time of an administration installation.</span></span> <span data-ttu-id="c8176-113">Установщик использует эти свойства во время установки приложения, выполняемого после администрирования, из административного образа.</span><span class="sxs-lookup"><span data-stu-id="c8176-113">The installer uses these properties during a post administration installation of the application from the administrative image.</span></span>

<span data-ttu-id="c8176-114">Пользователи, у которых нет постоянного доступа к сети, могут установить приложение из административного образа, а иногда в качестве источника резервного копирования должны полагаться на носители, например диски компакт-дисков.</span><span class="sxs-lookup"><span data-stu-id="c8176-114">Users who do not have continuous access to the network may install an application from an administrative image and then at times have to rely on media, such as CD-ROM disks, as their backup source.</span></span> <span data-ttu-id="c8176-115">В таких случаях длина имен файлов в административном образе и на носителе должны совпадать.</span><span class="sxs-lookup"><span data-stu-id="c8176-115">In these cases the length of the file names in the administrative image and on the media must match.</span></span> <span data-ttu-id="c8176-116">Оба файла должны использовать длинные имена файлов, или оба они должны использовать короткие имена файлов.</span><span class="sxs-lookup"><span data-stu-id="c8176-116">Both must use long file names or both must use short file names.</span></span> <span data-ttu-id="c8176-117">Например, компакт-диск, который поддерживает только короткие имена файлов, может предоставить как исходный носитель для установки административного образа, так и источник резервного копирования.</span><span class="sxs-lookup"><span data-stu-id="c8176-117">For example, a CD-ROM that only supports short file names could provide both the original media for installing the administrative image and a backup source.</span></span>

<span data-ttu-id="c8176-118">Если свойство [**шортфиленамес**](shortfilenames.md) задано во время административной установки, это свойство может потребоваться установить повторно с помощью пользователя, который впоследствии применяет исправление к административному образу.</span><span class="sxs-lookup"><span data-stu-id="c8176-118">If the [**SHORTFILENAMES**](shortfilenames.md) property is set during an administrative installation, this property may need to be set again by a user subsequently applying a patch to the administrative image.</span></span> <span data-ttu-id="c8176-119">При использовании установщик Windows для применения исправления установщик автоматически устанавливает свойство **шортфиленамес** , если в административном образе используются короткие имена файлов.</span><span class="sxs-lookup"><span data-stu-id="c8176-119">When using Windows Installer to apply the patch, the installer automatically sets the **SHORTFILENAMES** property if the administrative image uses short file names.</span></span>

<span data-ttu-id="c8176-120">Если администратор использует пакет со свойством " [**Сводка слов**](word-count-summary.md) " со значением 2 или 3 для выполнения административной установки, пользователи административного образа не могут автоматически переустановиться с исходного носителя.</span><span class="sxs-lookup"><span data-stu-id="c8176-120">If an administrator uses a package having a [**Word Count Summary**](word-count-summary.md) property of 2 or 3 to perform an administrative installation, users of the administrative image cannot automatically reinstall from the original media source.</span></span> <span data-ttu-id="c8176-121">Если административный образ становится недоступным, пользователи, установленные из административного образа, не смогут вернуться к исходному носителю.</span><span class="sxs-lookup"><span data-stu-id="c8176-121">If the administrative image becomes unavailable, users who have installed from the administrative image are prevented from reverting to the original media.</span></span>

<span data-ttu-id="c8176-122">Применение [*преобразований*](t-gly.md) во время создания административного образа не имеет действующего действия.</span><span class="sxs-lookup"><span data-stu-id="c8176-122">The application of [*transforms*](t-gly.md) during the creation of an administrative image has no valid effect.</span></span> <span data-ttu-id="c8176-123">Чтобы сделать настроенную версию продукта доступной Рабочей группе, примените преобразование во время установки продукта из административного образа.</span><span class="sxs-lookup"><span data-stu-id="c8176-123">To make a customized version of a product available to a work group, apply the transform during the installation of the product from the administrative image.</span></span>
