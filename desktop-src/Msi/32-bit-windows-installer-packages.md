# <a name="32-bit-windows-installer-packages"></a>32-разрядные пакеты установщик Windows

32-разрядный пакет состоит только из 32-разрядных [установщик Windows компонентов](windows-installer-components.md) и должен иметь значение "Intel", введенное в поле "платформа" Свойства " [**Сводка" шаблона**](template-summary.md) .

Дополнительные сведения см. [в разделе установщик Windows на 64-разрядных операционных системах](windows-installer-on-64-bit-operating-systems.md) и [64-разрядных установщик Windows пакетах](64-bit-windows-installer-packages.md).
