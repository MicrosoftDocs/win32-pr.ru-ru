# <a name="glgetmaterialfv-function"></a>Функция Глжетматериалфв

Функции **глжетматериалфв** и [**глжетматериалив**](glgetmaterialiv.md) возвращают параметры материала.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGetMaterialfv(
   GLenum  face,
   GLenum  pname,
   GLfloat *params
);
```

<dl> <dt>

*личным* 
</dt> <dd>

</dd> <dt>
