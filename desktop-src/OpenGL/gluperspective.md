# <a name="gluperspective-function"></a>Функция Глуперспективе

Функция **глуперспективе** настраивает матрицу перспективной проекции.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluPerspective(
   GLdouble fovy,
   GLdouble aspect,
   GLdouble zNear,
   GLdouble zFar
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*фови* 
</dt> <dd>

Поле угла представления (в градусах) в направлении по оси y.

</dd> <dt>

*aspect* 
</dt> <dd>

Пропорции, определяющие поле представления в направлении по оси x. Пропорции — это отношение *x* (ширина) к *y* (высота).

</dd> <dt>

*знеар* 
</dt> <dd>

Расстояние от средства просмотра до ближайшей плоскости обрезки (всегда положительное).

</dd> <dt>

*зфар* 
</dt> <dd>

Расстояние от средства просмотра до дальней плоскости (всегда положительное).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глуперспективе** задает Просмотр фрустум в мировой системе координат. Как правило, пропорции в **глуперспективе** должны соответствовать пропорциям связанного окна просмотра. Например, *аспект* = 2,0 означает, что угол окна просмотра в два раза шире, чем в *x* , как и в *y*. Если окно просмотра в два раза шире, чем высота, оно отображает изображение без искажений.

Матрица, созданная функцией **глуперспективе** , умножается на текущую матрицу, точно так же, как если бы [**глмултматрикс**](glmultmatrix.md) вызывалась с созданной матрицей. Чтобы загрузить матрицу перспективы на текущий стек матриц, перед вызовом **глуперспективе** с вызовом [**гллоадидентити**](glloadidentity.md).
