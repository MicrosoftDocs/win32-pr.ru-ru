# <a name="u-v"></a>U V

<dl> <dt>

<span id="opengl_vertex"></span><span id="OPENGL_VERTEX"></span>**положений**
</dt> <dd>

Точка в трехмерном пространстве.

</dd> <dt>

<span id="opengl_vertices"></span><span id="OPENGL_VERTICES"></span>**вершины**
</dt> <dd>

Предпочтительное множественное число вершин. См. раздел *вершина*.

</dd> <dt>

<span id="opengl_viewpoint"></span><span id="OPENGL_VIEWPOINT"></span>**окна**
</dt> <dd>

Источник глаза или система координат обрезки в зависимости от контекста. Например, при обсуждении освещения точка зрения является источником системы координат глаза. При обсуждении проекции точка зрения является источником обрезки системы координат. В типичной матрице проекции точки координат и координат глаз находятся в одном расположении.

</dd> <dt>

<span id="opengl_view_volume"></span><span id="OPENGL_VIEW_VOLUME"></span>**Просмотреть том**
</dt> <dd>

Том в координатах обрезки, координаты которого соответствуют трем условиям

-   w ≤ x ≤ w
-   w ≤ y ≤ w
-   w ≤ z ≤ w

</dd> </dl>
