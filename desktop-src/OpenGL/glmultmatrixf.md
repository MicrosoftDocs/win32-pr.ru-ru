# <a name="glmultmatrixf-function"></a>Функция Глмултматриксф

Функции [**глмултматриксд**](glmultmatrixd.md) и **глмултматриксф** умножают текущую матрицу на произвольную матрицу.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glMultMatrixf(
   const GLfloat *m
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*m* 
</dt> <dd>

Указатель на матрицу 4x4, хранящийся в основном порядке столбцов, как 16 последовательных значений.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

С помощью функции [**глжетеррор**](glgeterror.md) можно получить следующий код ошибки.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глмултматрикс** Умножает текущую матрицу на единицу, указанную в *m*. То есть, если M — это текущая матрица, а T — матрица, передаваемая в **глмултматрикс**, то m заменяется на m T.

Текущая матрица — матрица проекции, моделвиев матрица или Текстурная матрица, определяемая текущим режимом матрицы (см. [**глматриксмоде**](glmatrixmode.md)).

Параметр *m* указывает на матрицу 4x4 значений с плавающей запятой одиночной точности или двойной точности, хранящихся в основном порядке столбцов. То есть Матрица сохраняется, как показано на следующем рисунке.

![! [Диаграмма, показывающая матрицу 4x4, на которую указывает параметр m.]](images/multi01.png)

Следующие функции извлекают сведения, относящиеся к **глмултматрикс**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

**глжет** с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

**глжет** с аргументом \_ матрица проекции главной книги \_

**глжет** с аргументом \_ \_ Матрица текстуры GL
