# <a name="glistexture-function"></a>Функция Глистекстуре

Функция **глистекстуре** определяет, соответствует ли имя текстуре.

## <a name="syntax"></a>Синтаксис


```C++
GLboolean WINAPI glIsTexture(
   GLuint texture
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*поддержки* 
</dt> <dd>

Значение типа, которое является именем текстуры.

</dd> </dl>

## <a name="error-codes"></a>Коды ошибок

С помощью функции [**глжетеррор**](glgeterror.md) можно получить следующий код ошибки.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Если параметр *текстуры* в настоящее время является именем текстуры, функция **глистекстуре** возвращает \_ значение true. Функция **глистекстуре** возвращает \_ значение false, если *текстура* равна нулю. Он также возвращает значение GL \_ false, если это ненулевое значение, которое в настоящее время не является именем текстуры, или если возникает ошибка.

В списки вывода нельзя включать вызовы **глистекстуре** .

> [!Note]  
> Функция **глистекстуре** доступна только в OpenGL версии 1,1 или более поздней.
