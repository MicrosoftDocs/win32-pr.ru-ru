# <a name="glx-rendering-context-code-sample"></a><span data-ttu-id="4c1c5-108">Пример кода контекста подготовки к просмотру ГЛКС</span><span class="sxs-lookup"><span data-stu-id="4c1c5-108">GLX Rendering Context Code Sample</span></span>

<span data-ttu-id="4c1c5-109">В следующем образце кода показано, как программа X Window System OpenGL использует функции контекста отрисовки ГЛКС.</span><span class="sxs-lookup"><span data-stu-id="4c1c5-109">The following code sample shows how an X Window System OpenGL program uses GLX rendering context functions.</span></span>


```C++
Display *dpy;             /* display variable */ 
XVisualInfo *vi;          /* visual variable */ 
Window win;               /* window variable */ 
GLXDrawable drawable;     /* drawable variable */ 
GLXContext cx, cxTemp;    /* rendering context variables */ 
 
/* Code to open a display and get a visual. */ 
        
/* Create a GLX context. */ 
cx = glXCreateContext(dpy, vi, 0, GL_FALSE); 
if (!cx) { 
    fprintf(stderr, "Cannot create context.\n"); 
    exit(-1); 
} 
        
        .    
/* Connect the context to the window. */ 
glXMakeCurrent(dpy, win, cx); 
        
        .    
/* When it's time to destroy the rendering context. . . */ 
cx = glXGetCurrentContext; 
glXDestroyContext(dpy, cx);
```
