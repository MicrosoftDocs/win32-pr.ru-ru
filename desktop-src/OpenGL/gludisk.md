# <a name="gludisk-function"></a>Функция Глудиск

Функция **глудиск** рисует диск.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluDisk(
   GLUquadric *qobj,
   GLdouble   innerRadius,
   GLdouble   outerRadius,
   GLint      slices,
   GLint      loops
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кобж* 
</dt> <dd>

Объект куадрик (созданный с помощью [**глуневкуадрик**](glunewquadric.md)).

</dd> <dt>

*иннеррадиус* 
</dt> <dd>

Внутренний радиус диска (может равняться нулю).

</dd> <dt>

*аутеррадиус* 
</dt> <dd>

Внешний радиус диска.

</dd> <dt>

*срезы* 
</dt> <dd>

Число подразделений вокруг оси z.

</dd> <dt>

*бираются* 
</dt> <dd>

Количество концентрических колец относительно источника, в который делится диск.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глудиск** визуализирует диск на плоскости *z* = 0. Диск имеет радиус *аутеррадиус* и содержит концентрические циклическое отверстие с радиусом *иннеррадиус*. Если *иннеррадиус* имеет значение 0, отверстие не создается. Диск разделяется вокруг оси z на срезы (например, срезы пиццы), а также о оси z в кольцах (как указано *срезами* и *циклами* соответственно).

В отношении ориентации положительная *z*-сторона диска считается *внешней* (см. [**глукуадрикориентатион**](gluquadricorientation.md)). Это означает, что если для ориентации задано значение GLU \_ снаружи, то все создаваемые нормали указываются вдоль положительной оси z.

Если текстурирование включено (с [**глукуадриктекстуре**](gluquadrictexture.md)), координаты текстуры создаются линейно, так, где *r*  =  *аутеррадиус*, значение в (*r*, 0, 0) равно (1, 0,5); в (0, *r*, 0) оно равно (0,5, 1); в (-*r*, 0, 0) он равен (0, 0,5), а в (0,-*r*, 0) — (0,5, 0).
