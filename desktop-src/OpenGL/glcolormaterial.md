# <a name="glcolormaterial-function"></a>Функция Глколорматериал

Функция **глколорматериал** заставляет цвет материала контролировать текущий цвет.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glColorMaterial(
   GLenum face,
   GLenum mode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*личным* 
</dt> <dd>

Указывает, следует ли отслеживанию текущего цвета на внешнем, обратном или обоих параметрах переднего и заднего плана. Допустимые значения: GL \_ Front, GL \_ назад и GL \_ Front \_ и \_ Back. Значение по умолчанию — GL \_ Front \_ и \_ Back.

</dd> <dt>

*mode* 
</dt> <dd>

Указывает, какой из нескольких параметров материала позволяет контролировать текущий цвет. Допустимые значения: «Эмиссия GL», «наружная высветка», «Диффузия GL», « \_ \_ \_ \_ освещение» и « \_ \_ \_ диффузия». Значение по умолчанию — « \_ внешняя \_ » и « \_ диффузная».

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глколорматериал** указывает, какие параметры материала отписывают текущий цвет. При включении \_ цветового \_ материала GL для каждого материала или материалов, заданных *лицом*, параметр материала или параметры, заданные *режимом* , записывают текущий цвет в любое время. Включать и отключать \_ цветовые \_ материалы GL можно с помощью функций [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md), которые вызываются с использованием \_ цветового материала GL в \_ качестве аргумента. По умолчанию \_ цветовой \_ материал GL отключен.

С помощью **глколорматериал** можно изменить подмножество параметров материала для каждой вершины, используя только функцию [**глколор**](glcolor-functions.md) , не вызывая [**глматериал**](glmaterial-functions.md). Если вы собираетесь указать только такое подмножество параметров для каждой вершины, лучше сделать это с помощью **глколорматериал** , чем с помощью **глматериал**.

Следующие функции извлекают сведения, относящиеся к **глколорматериал**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ " \_ параметр цветового материала GL" \_

**глжет** с аргументом « \_ ЦВЕТовой \_ материал GL» \_

[**глисенаблед**](glisenabled.md) с аргументом " \_ ЦВЕТовой материал GL" \_
