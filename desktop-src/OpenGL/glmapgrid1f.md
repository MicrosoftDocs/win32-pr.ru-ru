# <a name="glmapgrid1f-function"></a>Функция glMapGrid1f

Определяет одномерный сетку.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glMapGrid1f(
   GLint   un,
   GLfloat u1,
   GLfloat u2
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*понижен* 
</dt> <dd>

Число секций в диапазоне диапазонов сетки \[ : U1, U2 \] . Это значение должно быть положительным.

</dd> <dt>

*U1* 
</dt> <dd>

Значение, используемое в качестве сопоставления для значения домена целочисленной сетки i = 0.

</dd> <dt>

*U2* 
</dt> <dd>

Значение, используемое в качестве сопоставления для значения домена целочисленной сетки i = un.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функции **глмапгрид** и [глевалмеш](glevalmesh-functions.md) используются совместно для эффективного создания и вычисления ряда значений домена сопоставлений с равными пробелами. Функция Глевалмеш выполняет шаг с целым доменом одномерной или двухмерной сетки, диапазон которого представляет собой домен карт оценки, заданных в [**glMap1**](glmap1.md) и [**glMap2**](glmap2.md).

Функции [**glMapGrid1**](glmapgrid1d.md) и [**glMapGrid2**](glmapgrid2d.md) определяют сопоставления линейной сетки между координатами сетки i (или i и j) целого числа с координатами карты вычисления числа с плавающей запятой u (или a и v). Сведения о том, как оцениваются координаты, см. в разделе [**glMap1**](glmap1.md) и [**glMap2**](glmap2.md) .

Функция [**glMapGrid1**](glmapgrid1d.md) задает одно линейное сопоставление таким, *что координата* целочисленной сетки 0 сопоставляется только с U1, а целочисленная сетка целых чисел отменяет сопоставление ровно с *U2*. Все остальные координаты целочисленной сетки *я* сопоставил таким образом, что:

*u = i (U2 U1)/Ун + U1*

Функция [**glMapGrid2**](glmapgrid2d.md) указывает два таких линейных сопоставления. Одна карта сопоставляет целочисленную сетку *i = 0* ровно с *U1*, а целочисленная сетка — координата *i =* в точности до *U2*. Вторая сопоставляет сетку целых чисел *j = 0* ровно с *v1*, а целочисленная сетка с координатами « *j = VN» —* только *v2*. Другие координаты целочисленных сеток i и j сопоставлены таким, что

*u = i (U2 U1)/Ун + U1*

*v = j (V2 v1)/VN + v1*

Сопоставления, заданные параметром [**глмапгрид**](glmapgrid1d.md) , используются одинаково в [глевалмеш](glevalmesh-functions.md) и [**глевалпоинт**](glevalpoint.md).
