# <a name="glpolygonoffset-function"></a>Функция Глполигоноффсет

Функция **глполигоноффсет** задает масштаб и единицы, используемые OpenGL для вычисления значений глубины.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPolygonOffset(
   GLfloat factor,
   GLfloat units
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*многофакторной* 
</dt> <dd>

Задает коэффициент масштабирования, используемый для создания смещения глубины переменной для каждого многоугольника. Начальное значение равно нулю.

</dd> <dt>

*Комплект* 
</dt> <dd>

Задает значение, умноженное на значение, зависящее от реализации, для создания смещения глубины константы. Начальное значение равно 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Если \_ включено смещение многоугольников GL \_ , то значение глубины каждого фрагмента будет смещено после интерполяции из значений глубины соответствующих вершин. Значение смещения равно *множителю* \* ? z + r \* , где? z — это измерение изменения глубины относительно области экрана многоугольника, а r — наименьшее значение, которое гарантированно создает разрешаемое смещение для данной реализации. Смещение добавляется перед выполнением теста глубины и перед записью значения в буфер глубины.

Функция **глполигоноффсет** полезна для визуализации изображений в скрытом тексте, для применения декалс к поверхностям и для отрисовки сплошных с выделенными краями.

Функция **глполигоноффсет** не влияет на координаты глубины, помещенные в буфер обратной связи. Он также не влияет на выбор.

Следующие функции извлекают сведения, относящиеся к **глполигоноффсет**:

-   [**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ " \_ \_ коэффициент смещения" в ГК
-   [**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ " \_ \_ единицы смещения" в GL
-   [**глисенаблед**](glisenabled.md) с аргументом GL, \_ \_ смещение смещения многоугольника \_
-   [**глисенаблед**](glisenabled.md) с аргументом \_ \_ СМЕЩ. смещение \_ строки
-   [**глисенаблед**](glisenabled.md) с аргументом в качестве \_ \_ точки смещения для многоугольника GL \_

> [!Note]  
> Функция **глполигоноффсет** доступна только в OpenGl версии 1,1 или более поздней.
