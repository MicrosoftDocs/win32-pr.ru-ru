# <a name="g-opengl"></a>G (OpenGL)

<dl> <dt>

<span id="opengl_gamma_correction"></span><span id="OPENGL_GAMMA_CORRECTION"></span>**гамма-коррекция**
</dt> <dd>

Функция, применяемая к цветам, хранящимся в буфере кадров, чтобы исправить нелинейный отклик глаза (и иногда монитор) на линейные изменения значений интенсивности цветов.

</dd> <dt>

<span id="opengl_geometric_model"></span><span id="OPENGL_GEOMETRIC_MODEL"></span>**геометрическая модель**
</dt> <dd>

Вершины координат объекта и параметры, описывающие объект. Обратите внимание, что OpenGL не определяет синтаксис для геометрических моделей, а является синтаксисом и семантикой для отрисовки геометрических моделей.

</dd> <dt>

<span id="opengl_geometric_object"></span><span id="OPENGL_GEOMETRIC_OBJECT"></span>**геометрический объект**
</dt> <dd>

Геометрическая модель.

</dd> <dt>

<span id="opengl_geometric_primitive"></span><span id="OPENGL_GEOMETRIC_PRIMITIVE"></span>**геометрический примитив**
</dt> <dd>

Точка, линия или многоугольник.

</dd> <dt>

<span id="opengl_gouraud_shading"></span><span id="OPENGL_GOURAUD_SHADING"></span>**Заливка по методу Гуро**
</dt> <dd>

Плавная интерполяция цветов в многоугольнике или сегменте линии. Цвета присваиваются в вершинах и линейно интерполируются по примитиву для создания относительно гладких вариаций в цвете. Также называется плавной заливкой.

</dd> <dt>

<span id="opengl_group"></span><span id="OPENGL_GROUP"></span>**сгруппировать**
</dt> <dd>

Группа из одного, двух, трех или четырех элементов, представляющих каждый пиксел изображения в памяти клиента. Таким образом, в контексте образа памяти клиента группа и пиксель имеют одно и то же действие.

</dd> </dl>
