# <a name="glpolygonstipple-function"></a>Функция Глполигонстиппле

Функция **глполигонстиппле** задает шаблон стипплинг многоугольника.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPolygonStipple(
   const GLubyte *mask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*mask* 
</dt> <dd>

Указатель на шаблон размером 32x32 стиппле, который будет распакован из памяти таким же образом, как [**глдравпикселс**](gldrawpixels.md) распаковать Пиксели.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

## <a name="remarks"></a>Remarks

Функция **глполигонстиппле** задает шаблон стипплинг многоугольника. Многоугольник стипплинг, как и Line стипплинг (см. [**гллинестиппле**](gllinestipple.md)), маскирует определенные фрагменты, созданные с помощью растрирования, создавая шаблон. Стипплинг не зависит от сглаживания многоугольников.

Параметр *Mask* является указателем на шаблон размером 32x32 стиппле, который хранится в памяти, так же, как данные пикселей, передаваемые в **глдравпикселс** с *высотой* и *шириной* , равной 32, *Формат* пикселей для \_ индекса цвета GL \_ и *тип* данных \_ точечного рисунка GL. Таким образом, шаблон стиппле представляется в виде массива размером 32 x 1-разрядных индексов цвета, упакованных в байты без знака. Параметры функции [**глпикселсторе**](glpixelstore-functions.md) , такие как GL \_ RePack \_ reswap \_ bytes и GL \_ RePack \_ ЛСБ, в \_ первую очередь влияют на сборку битов в шаблон стиппле. Однако операции перемещения пикселей (сдвиг, смещение и схема пикселей) не применяются к изображению стиппле.

Многоугольник стипплинг включен и отключен с [**гленабле**](glenable.md) и **глдисабле** с использованием аргумента GL \_ Polygon \_ стиппле. Если этот параметр включен, растровый фрагмент многоугольника с координатами окна *x*<sub>w</sub> и *y*<sub>w</sub> отправляется на следующий этап OpenGL только в том случае, если *бит (**x*<sub>w</sub> <sub>Mod 32</sub> ) в строке шаблона стиппле имеет значение One. Если параметр Polygon стипплинг отключен, это так, как если бы шаблон стиппле был все.

Следующие функции извлекают сведения, относящиеся к **глполигонстиппле**:

[**глисенаблед**](glisenabled.md) с аргументом GL \_ Polygon \_ стиппле
