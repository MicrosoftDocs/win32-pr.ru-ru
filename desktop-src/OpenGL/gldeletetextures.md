# <a name="gldeletetextures-function"></a>Функция Глделететекстурес

Функция **глделететекстурес** Удаляет именованные текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glDeleteTextures(
         GLsizei n,
   const GLuint  *textures
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*n* 
</dt> <dd>

Число удаляемых текстур.

</dd> <dt>

*текстур* 
</dt> <dd>

Массив текстур для удаления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недопустимое \_ значение GL**</dt> </dl>     | *n* было отрицательным значением.<br/>                                                                                                  |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глделететекстурес** удаляет из *n* текстур, названных элементами *текстур* массива. После удаления текстуры она не имеет содержимого или размерности, а ее имя может быть свободно для повторного использования (например, с помощью **глжентекстурес**). Функция **глделететекстурес** игнорирует нули и имена, которые не соответствуют существующим текстурам.

Если удаляемая в данный момент текстура удаляется, привязка возвращается к нулю (текстура по умолчанию).

В списки вывода нельзя включать вызовы **глделететекстурес** .

> [!Note]  
> Функция **глделететекстурес** доступна только в OpenGL версии 1,1 или более поздней.

 

Следующая функция получает сведения, связанные с **глделететекстурес**:

-   [**глистекстуре**](glistexture.md)
