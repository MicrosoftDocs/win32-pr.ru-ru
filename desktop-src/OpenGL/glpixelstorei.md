# <a name="glpixelstorei-function"></a>Функция Глпикселстореи

Задает режимы хранения в пикселях.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPixelStorei(
   GLenum pname,
   GLint  param
);
```

## <a name="remarks"></a>Remarks

Функция **глпикселсторе** устанавливает режимы хранения пикселов, влияющие на работу последующих [**глдравпикселс**](gldrawpixels.md) и [**глреадпикселс**](glreadpixels.md) , а также распаковку шаблонов многоугольников стиппле (см. [**глполигонстиппле**](glpolygonstipple.md)), точечные рисунки (см. [**Глбитмап**](glbitmap.md)) и узоры текстур (см. [**glTexImage1D**](glteximage1d.md), [**glTexImage2D**](glteximage2d.md), [**glTexSubImage1D**](gltexsubimage1d.md)и [**glTexSubImage2D**](gltexsubimage2d.md)).

В следующей таблице приводится тип, начальное значение и диапазон допустимых значений для каждого параметра хранилища, который можно задать с помощью **глпикселсторе**.



| Pname                    | Тип    | Начальное значение | Допустимый диапазон значений   |
|--------------------------|---------|---------------|---------------|
| \_ \_ байты подкачки пакета GL \_    | Логическое | false         | true или false |
| \_ \_ байты подкачки пакета GL \_    | Логическое | false         | true или false |
| \_ \_ Длина строки пакета \_ GL    | целое число | 0             | \[0,?)        |
| \_ \_ пропуск строк в пакете GL \_     | целое число | 0             | \[0,?)        |
| \_ \_ пропуск ПИКСЕЛЕЙ пакета \_ GL   | целое число | 0             | \[0,?)        |
| \_Выравнивание пакета \_ GL      | Целое число | 4             | 1, 2, 4 или 8 |
| \_распаковать \_ \_ байты подкачки GL  | Логическое | false         | true или false |

| \_Длина строки РАСпаковать "GL" \_ \_  | целое число | 0             | \[0,?)        |

Функцию [**глпикселстореф**](glpixelstoref.md) можно использовать для задания любого параметра хранилища пикселей. Если параметр имеет тип Boolean, а параметр *param* имеет значение 0,0, то значение параметра равно false. в противном случае — значение true. Если *pname* — целочисленный параметр типа *, параметр округляется* до ближайшего целого числа.

Аналогично, функция **глпикселстореи** может также использоваться для задания любого из параметров хранилища пикселей. Для логических параметров задано значение false, если *параметр param* равен 0, в противном случае — значение true. Параметр *param* преобразуется в плавающую точку перед тем, как он будет назначен параметрам с реальными значениями.

Режимы хранения в пикселях, действующие при помещении [**глдравпикселс**](gldrawpixels.md), [**глреадпикселс**](glreadpixels.md), [**glTexImage1D**](glteximage1d.md), [**glTexImage2D**](glteximage2d.md), [**глбитмап**](glbitmap.md)или [**глполигонстиппле**](glpolygonstipple.md) , помещаются в список отображений, контролируя интерпретацию данных в памяти. Режим хранения в пикселях, действующий при выполнении списка экранов, не имеет значения.
