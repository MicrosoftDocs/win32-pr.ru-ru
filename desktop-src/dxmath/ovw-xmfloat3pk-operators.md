# <a name="xmfloat3pk-operators"></a>Операторы XMFLOAT3PK

Структура [**XMFLOAT3PK**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                               | Описание                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMFLOAT3PK` экземпляра в упакованном формате.<br/> |
| [**Оператор =**](xmfloat3pk-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMFLOAT3PK` .<br/>                                       |
