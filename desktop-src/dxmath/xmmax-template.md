# <a name="xmmax-template"></a>Шаблон Ксммакс

Сравнивает два экземпляра типа числовых данных или два экземпляра объекта, которые поддерживают перегрузку <, и возвращает больше одного из двух экземпляров. Тип данных аргументов и возвращаемое значение одинаковы.

## <a name="syntax"></a>Синтаксис

``` syntax
template<class T> T XMMax(
  [in]  T a,
  [in]  T b
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="a"></span><span id="A"></span>*конкретного*
</dt> <dd>

\[в \] задает первый из двух объектов.

</dd> <dt>

<span id="b"></span><span id="B"></span>*&*
</dt> <dd>

\[в \] задает два из двух объектов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает большее из двух входных объектов.

## <a name="remarks"></a>Remarks

`XMMax` является шаблоном, а тип T указывается при создании экземпляра шаблона.

> [!Note]  
> `XMMax`Шаблон является новым для директксмас и недоступен для кснамас 2. x. `XMMax` доступен в виде макроса в Кснамас 2. x.

 

> [!Note]  
> В идеале используйте std:: Max вместо `XMMax` . чтобы избежать конфликтов с Windows заголовками с std:: max, необходимо \# определить номинмакс перед включением заголовков Windows.
