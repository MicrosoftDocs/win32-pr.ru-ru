# <a name="running-32-bit-applications"></a><span data-ttu-id="ecc33-108">Выполнение 32-разрядных приложений</span><span class="sxs-lookup"><span data-stu-id="ecc33-108">Running 32-bit Applications</span></span>

<span data-ttu-id="ecc33-109">WOW64 — это эмулятор x86, позволяющий легко работать с 32-разрядными приложениями на базе Windows в 64-разрядной версии Windows.</span><span class="sxs-lookup"><span data-stu-id="ecc33-109">WOW64 is the x86 emulator that allows 32-bit Windows-based applications to run seamlessly on 64-bit Windows.</span></span> <span data-ttu-id="ecc33-110">Это позволяет легко работать с 32-разрядными (x86) приложениями Windows в 64-разрядных (x64) окнах, а также в случае с 32-разрядными (x86) и 32-разрядными (ARM) приложениями Windows для беспрепятственного выполнения в 64-разрядных (ARM64) окнах.</span><span class="sxs-lookup"><span data-stu-id="ecc33-110">This allows for 32-bit (x86) Windows applications to run seamlessly in 64-bit (x64) Windows, as well as for 32-bit (x86) and 32-bit (ARM) Windows applications to run seamlessly in 64-bit (ARM64) Windows.</span></span> <span data-ttu-id="ecc33-111">WOW64 поставляется с операционной системой и не обязательно должен быть явно включен.</span><span class="sxs-lookup"><span data-stu-id="ecc33-111">WOW64 is provided with the operating system and does not have to be explicitly enabled.</span></span> <span data-ttu-id="ecc33-112">Дополнительные сведения см. в разделе [сведения о реализации WOW64](wow64-implementation-details.md).</span><span class="sxs-lookup"><span data-stu-id="ecc33-112">For more information, see [WOW64 Implementation Details](wow64-implementation-details.md).</span></span>

<span data-ttu-id="ecc33-113">Система изолирует 32-разрядные приложения от 64-разрядных приложений, что включает в себя предотвращение конфликтов файлов и реестра.</span><span class="sxs-lookup"><span data-stu-id="ecc33-113">The system isolates 32-bit applications from 64-bit applications, which includes preventing file and registry collisions.</span></span> <span data-ttu-id="ecc33-114">Поддерживаются консоль, графический пользовательский интерфейс и приложения службы.</span><span class="sxs-lookup"><span data-stu-id="ecc33-114">Console, GUI, and service applications are supported.</span></span> <span data-ttu-id="ecc33-115">Система обеспечивает взаимодействие на границе 32/64 для таких сценариев, как вырезание и вставка и COM.</span><span class="sxs-lookup"><span data-stu-id="ecc33-115">The system provides interoperability across the 32/64 boundary for scenarios such as cut and paste and COM.</span></span> <span data-ttu-id="ecc33-116">Однако 32-разрядные процессы не могут загружать 64-разрядные библиотеки DLL для выполнения, а 64-разрядные процессы не могут загружать 32-разрядные библиотеки DLL для выполнения.</span><span class="sxs-lookup"><span data-stu-id="ecc33-116">However, 32-bit processes cannot load 64-bit DLLs for execution, and 64-bit processes cannot load 32-bit DLLs for execution.</span></span> <span data-ttu-id="ecc33-117">Это ограничение не применяется к DLL, загруженным в виде файлов данных или файлов ресурсов образа. Дополнительные сведения см. в разделе [**LoadLibraryEx**](/windows/desktop/api/libloaderapi/nf-libloaderapi-loadlibraryexa).</span><span class="sxs-lookup"><span data-stu-id="ecc33-117">This restriction does not apply to DLLs loaded as data files or image resource files; for more information, see [**LoadLibraryEx**](/windows/desktop/api/libloaderapi/nf-libloaderapi-loadlibraryexa).</span></span>

<span data-ttu-id="ecc33-118">32-разрядное приложение может определить, выполняется ли оно в эмуляторе WOW64, вызвав функцию [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process) (используйте [**IsWow64Process2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process2) при нацеливании на Windows 10).</span><span class="sxs-lookup"><span data-stu-id="ecc33-118">A 32-bit application can detect whether it is running under WOW64 by calling the [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process) function (use [**IsWow64Process2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process2) if targeting Windows 10).</span></span> <span data-ttu-id="ecc33-119">Приложение может получить дополнительные сведения о процессоре с помощью функции [**жетнативесистеминфо**](/windows/desktop/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo) .</span><span class="sxs-lookup"><span data-stu-id="ecc33-119">The application can obtain additional information about the processor by using the [**GetNativeSystemInfo**](/windows/desktop/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo) function.</span></span>

<span data-ttu-id="ecc33-120">Обратите внимание, что 64-разрядная версия Windows не поддерживает выполнение 16-разрядных приложений на основе Windows.</span><span class="sxs-lookup"><span data-stu-id="ecc33-120">Note that 64-bit Windows does not support running 16-bit Windows-based applications.</span></span> <span data-ttu-id="ecc33-121">Основная причина заключается в том, что дескрипторы имеют 32 значащих бит на 64-разрядной версии Windows.</span><span class="sxs-lookup"><span data-stu-id="ecc33-121">The primary reason is that handles have 32 significant bits on 64-bit Windows.</span></span> <span data-ttu-id="ecc33-122">Поэтому дескрипторы не могут быть усечены и переданы в 16-разрядные приложения без потери данных.</span><span class="sxs-lookup"><span data-stu-id="ecc33-122">Therefore, handles cannot be truncated and passed to 16-bit applications without loss of data.</span></span> <span data-ttu-id="ecc33-123">Попытки запуска 16-разрядных приложений завершаются следующей ошибкой: **Ошибка \_ неправильный \_ \_ Формат exe**.</span><span class="sxs-lookup"><span data-stu-id="ecc33-123">Attempts to launch 16-bit applications fail with the following error: **ERROR\_BAD\_EXE\_FORMAT**.</span></span>
