# <a name="newdevh"></a>Невдев. h

В этом разделе содержатся справочные разделы по заголовку Невдев. h.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**диинсталлдевице**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | Функция **диинсталлдевице** устанавливает указанный драйвер, предварительно установленный в [хранилище драйверов](/windows-hardware/drivers/install/driver-store) на указанном устройстве, присутствующем в системе.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**диинсталлдривер**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | Функция **диинсталлдривер** предварительно устанавливает драйвер в [хранилище драйверов](/windows-hardware/drivers/install/driver-store) , а затем устанавливает драйвер на устройствах, поддерживаемых драйвером.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**дироллбаккдривер**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | Функция **дироллбаккдривер** выполняет откат драйвера, установленного на указанном устройстве.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**дишовупдатедевице**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | Функция **дишовупдатедевице** отображает мастер обновления оборудования для указанного устройства.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**диунинсталлдевице**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | Функция [**диунинсталлдевице**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) удаляет устройство и отменяет его узел устройства ([девноде](/windows-hardware/drivers/)) из системы. Это отличается от использования [**сетупдикаллклассинсталлер**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) с кодом [**\_ удаления DIF**](/windows-hardware/drivers/install/dif-remove) , так как он пытается удалить узел устройства в дополнение к дочерним девнодес, которые имеются во время вызова.<br/> до Windows 8 все дочерние устройства, отсутствующие во время вызова, не будут удалены. однако начиная с Windows 8 все дочерние устройства, отсутствующие во время вызова, будут удалены.<br/> |
| [**упдатедриверфорплугандплайдевицес**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | При наличии INF-файла и [идентификатора оборудования](/windows-hardware/drivers/install/hardware-ids)функция **упдатедриверфорплугандплайдевицес** устанавливает обновленные драйверы для устройств, соответствующих идентификатору оборудования. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
