# <a name="nap-datatypes"></a>Типы типов защиты доступа к сети

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Для API защиты доступа к сети (NAP) доступны следующие типы типов:


```C++
typedef FILETIME ProbationTime;
typedef UINT32 ProtocolMaxSize;
typedef UINT32 NapComponentId;
typedef NapComponentId SystemHealthEntityId;
typedef NapComponentId EnforcementEntityId;
typedef UINT16 SystemHealthEntityCount;
typedef UINT16 EnforcementEntityCount;
typedef CountedString StringCorrelationId;
typedef GUID ConnectionId;
typedef UINT8 Percentage;
typedef UINT32 MessageId;
```



<dl> <dt>

**пробатионтиме**
</dt> <dd>

Структура [fileTime](/windows/win32/api/minwinbase/ns-minwinbase-filetime) , которая содержит время, относящееся к надзору на клиентском компьютере.

</dd> <dt>

**протоколмакссизе**
</dt> <dd>

Значение, указывающее диапазон возможных значений для максимального размера пакета [**SoH**](/windows/win32/api/naptypes/ns-naptypes-soh) в байтах, определенного диапазоном ([**минпротоколмакссизе, макспротоколмакссизе**](nap-type-constants.md)).

</dd> <dt>

**напкомпонентид**
</dt> <dd>

Уникальный 4-байтовый идентификатор, используемый SHA, SHV и клиенты принудительного применения для самостоятельной идентификации. Первые три байта — это код SMI поставщика, назначенный IETF, а последний байт идентифицирует сам компонент.

</dd> <dt>

**системхеалсентитид**
</dt> <dd>

Значение **напкомпонентид** , используемое для распознавания пар SHA/SHV.

</dd> <dt>

**енфорцементентитид**
</dt> <dd>

Значение **напкомпонентид** , используемое для распознавания клиентов принудительного применения.

</dd> <dt>

**системхеалсентитикаунт**
</dt> <dd>

Значение, указывающее число зарегистрированных SHA в системе защиты доступа к сети в диапазоне 0 (ноль) до [**макссистемхеалсентитикаунт**](nap-type-constants.md).

</dd> <dt>

**енфорцементентитикаунт**
</dt> <dd>

Значение, указывающее число клиентов принудительного применения в системе защиты доступа к сети в диапазоне 0 (ноль) до [**максенфорцеркаунт**](nap-type-constants.md).

</dd> <dt>

**стрингкоррелатионид**
</dt> <dd>

[**Каунтедстринг**](/windows/win32/api/naptypes/ns-naptypes-countedstring) версия структуры [**correlationId**](/windows/win32/api/naptypes/ns-naptypes-correlationid) , используемая для связывания [**сохрекуестс**](/windows/win32/api/naptypes/ns-naptypes-soh) с **сохреспонсес**.

</dd> <dt>

**ConnectionId**
</dt> <dd>

Уникальный глобальный уникальный идентификатор (GUID), используемый для идентификации подключений NAP, обслуживаемых клиентами принудительного применения.

</dd> <dt>

**Percentage**
</dt> <dd>

Значение, содержащее процент от 0 (нуля) до 100 исправления, которое завершено

</dd> <dt>

**MessageId**
</dt> <dd>

Уникальное значение, используемое для распознавания системных сообщений NAP.

</dd> </dl>
