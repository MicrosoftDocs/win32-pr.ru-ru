# <a name="inapsystemhealthvalidator-interface"></a>Интерфейс Инапсистемхеалсвалидатор

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

**Инапсистемхеалсвалидатор** предоставляет методы, которые должны быть реализованы РАЗРАБОТЧИКами SHV, чтобы обеспечить взаимодействие системы защиты доступа к сети с SHV.

## <a name="members"></a>Элементы

Интерфейс **инапсистемхеалсвалидатор** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапсистемхеалсвалидатор** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапсистемхеалсвалидатор** содержит следующие методы.



| Метод                                                                                   | Описание                                                                                                                               |
|:-----------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Инапсистемхеалсвалидатор:: Validate**](inapsystemhealthvalidator-validate-method.md) | Система защиты доступа к сети вызывает этот определенный приложением метод для проверки [**сохрекуест**](/windows/win32/api/naptypes/ns-naptypes-soh) , полученного от клиента. <br/> |
