# <a name="inapcertrelyingparty-interface"></a>Интерфейс Инапцертрелингпарти

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапцертрелингпарти** предоставляет методы, которые проверяющие стороны сертификатов должны использовать для взаимодействия с напажент.

## <a name="members"></a>Элементы

Интерфейс **инапцертрелингпарти** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапцертрелингпарти** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапцертрелингпарти** содержит следующие методы.



| Метод                                                                                                        | Описание                                                                     |
|:--------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------|
| [**Инапцертрелингпарти:: Жетсубскрибедрелингпартиес**](inapcertrelyingparty-getsubscribedrelyingparties.md) | Возвращает список проверяющих сторон, на которые оформлена подписка.<br/>                 |
| [**Инапцертрелингпарти:: Субскрибецертбиграуп**](inapcertrelyingparty-subscribecertbygroup.md)               | Подписывается на уже зарегистрированный сервер сертификатов работоспособности (HCS).<br/> |
| [**Инапцертрелингпарти:: Унсубскрибецертбиграуп**](inapcertrelyingparty-unsubscribecertbygroup.md)           | Отменяет Подписывание сервера HCS.<br/>                                     |
