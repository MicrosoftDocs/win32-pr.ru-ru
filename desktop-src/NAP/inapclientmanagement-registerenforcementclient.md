# <a name="inapclientmanagementregisterenforcementclient-method"></a>Метод Инапклиентманажемент:: Регистеренфорцементклиент

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **регистеренфорцементклиент** регистрирует клиент принудительного применения в системе защиты доступа к сети.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT RegisterEnforcementClient(
  [in] const NapComponentRegistrationInfo *enforcer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*принудительное применение* \[ окне\]
</dt> <dd>

Указатель на структуру данных [**напкомпонентрегистратионинфо**](/windows/win32/api/naptypes/ns-naptypes-napcomponentregistrationinfo) , содержащую сведения о регистрации, связанные с клиентом принудительного применения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает код состояния HRESULT, включая, помимо прочего, один из следующих.



| Код возврата                                                                                            | Описание                                                                       |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                   | Операция выполнена успешно.<br/>                                                  |
| <dl> <dt>**E \_ ACCESSDENIED**</dt> </dl>         | Ошибка разрешений, доступ запрещен.<br/>                                      |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>          | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/>                |
| <dl> <dt>**\_ \_ конфликтующий идентификатор NAP \_ E**</dt> </dl> | Агент принудительного применения, использующий данный идентификатор, уже зарегистрирован.<br/> |
