# <a name="inapenforcementclientcallback-interface"></a>Интерфейс Инапенфорцементклиенткаллбакк

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапенфорцементклиенткаллбакк** предоставляет методы обратного вызова, которые должны быть реализованы клиентами, чтобы позволить напажент взаимодействовать с ними.

## <a name="members"></a>Элементы

Интерфейс **инапенфорцементклиенткаллбакк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапенфорцементклиенткаллбакк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапенфорцементклиенткаллбакк** содержит следующие методы.



| Метод                                                                                                         | Описание                                                                      |
|:---------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**Инапенфорцементклиенткаллбакк:: соединений**](inapenforcementclientcallback-getconnections-method.md)   | Используется Напажент для получения соединений.<br/>                         |
| [**Инапенфорцементклиенткаллбакк:: Нотифисохчанже**](inapenforcementclientcallback-notifysohchange-method.md) | Используется Напажент для информирования клиента принудительного применения об изменениях состояния работоспособности.<br/> |
