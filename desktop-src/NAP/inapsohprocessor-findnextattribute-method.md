# <a name="inapsohprocessorfindnextattribute-method"></a>Метод Инапсохпроцессор:: Финднекстаттрибуте

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсохпроцессор:: финднекстаттрибуте** находит расположение (индекс) следующего атрибута типа, указанного в *сохаттрибутетипе*.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindNextAttribute(
  [in]  UINT16           fromLocation,
  [in]  SoHAttributeType type,
  [out] UINT16           *attributeLocation
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*фромлокатион* \[ окне\]
</dt> <dd>

Начальное расположение (индекс) в пакете состояния работоспособности (SoH) для начала поиска атрибутов. Это значение должно находиться в диапазоне от 0 до (**нуматтриб** -1), где **нуматтриб** извлекается с помощью [**инапсохпроцессор:: жетнумберофаттрибутес**](inapsohprocessor-getnumberofattributes-method.md).

> [!Note]  
> Пакет SoH использует индексы атрибутов на основе 0.

 

</dd> <dt>

*тип* \[ окне\]
</dt> <dd>

Структура [**сохаттрибутетипе**](sohattributetype-enum.md) , содержащая тип атрибута для размещения.

</dd> <dt>

*аттрибутелокатион* \[ заполняет\]
</dt> <dd>

Указатель, содержащий расположение (индекс) в пакете SoH первого атрибута типа *сохаттрибутетипе* из индекса *фромлокатион*.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Метод **финднекстаттрибуте** ищет атрибуты типа *сохаттрибутетипе* из индекса, заданного параметром *фромлокатион* , и выше, пока не будет найдено соответствие. Если совпадений не найдено, **возвращается \_ файл ошибки \_ не \_ найден** .
