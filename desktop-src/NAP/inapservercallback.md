# <a name="inapservercallback-interface"></a>Интерфейс Инапсерверкаллбакк

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

**Инапсерверкаллбакк** предоставляет метод, который SHV использует для сигнализации завершения асинхронного запроса.

## <a name="members"></a>Элементы

Интерфейс **инапсерверкаллбакк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапсерверкаллбакк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапсерверкаллбакк** содержит следующие методы.



| Метод                                                                         | Описание                                                        |
|:-------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| [**Инапсерверкаллбакк:: OnComplete**](inapservercallback-oncomplete-method.md) | Используется SHV для сигнализации о завершении асинхронного запроса.<br/> |
