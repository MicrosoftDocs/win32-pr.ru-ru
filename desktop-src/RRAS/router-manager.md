# <a name="router-manager"></a>Диспетчер маршрутизаторов

Диспетчер маршрутизаторов — это компонент маршрутизатора Windows NT/Windows 2000, который запускает различные компоненты маршрутизации и управляет ими. К этим компонентам относятся протоколы маршрутизации, диспетчер таблиц маршрутизации и Диспетчер групп многоадресной рассылки.
