# <a name="other-connection-information"></a>Другие сведения о соединении

Члены структуры [**расдиалпарамс**](/previous-versions/windows/desktop/legacy/aa377238(v=vs.85)) также могут задавать следующие сведения о соединении.

-   Номер телефона для переопределения номера в записи телефонной книги.
-   Номер телефона [обратного вызова](callback-connections.md) , который удаленный сервер может вызвать для установления соединения.
-   Имя домена удаленной сети, в котором будет выполняться проверка подлинности.

Для номера обратного вызова и домена члены [**расдиалпарамс**](/previous-versions/windows/desktop/legacy/aa377238(v=vs.85)) могут указать, что служба RAS должна использовать информацию в записи телефонной книги, или предоставить сведения, переопределяющие данные телефонной книги.

Клиент RAS может использовать параметр *лпрасдиалекстенсионс* функции [**rasdial**](/windows/desktop/api/Ras/nf-ras-rasdiala) для управления тем, использует ли RAS префикс номера телефона или суффикс, указанный в записи телефонной книги.
