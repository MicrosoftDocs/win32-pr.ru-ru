# <a name="b-rras"></a>B (RRAS)

<dl> <dt>

<span id="_rras_best_route_gly"></span><span id="_RRAS_BEST_ROUTE_GLY"></span>**лучший маршрут**
</dt> <dd>

Лучшим маршрутом является оптимальный маршрут к назначению сети на основе указанных критериев. Эта концепция основана на том факте, что имеется определенный «затратный», вовлеченный в создание маршрута по сети.

Наилучший маршрут, который нужно выполнить, — это самая низкая стоимость, основанная на заданных условиях. Этот критерий может включать число перекрестных сетей, тип перекрестной сети (например, Открытый или частный), а также денежный лимит или пропускную способность.

</dd> <dt>

<span id="_rras_border_gateway_protocol_bgp__gly"></span><span id="_RRAS_BORDER_GATEWAY_PROTOCOL_BGP__GLY"></span>**Протокол BGP**
</dt> <dd>

BGP является [*протоколом внешнего шлюза*](e-gly.md). BGP пересылает сведения между двумя [*автономными системами*](a-gly.md). Он определен в [rfc 1267](routing-protocols-request-for-comments.md) и [RFC 1268](routing-protocols-request-for-comments.md).

</dd> <dt>

<span id="_mpr_brute_force_attack"></span><span id="_MPR_BRUTE_FORCE_ATTACK"></span>**Подбор — принудительная атака**
</dt> <dd>

Попытка нарушить безопасность шифра, пытаясь выполнить все возможные попытки.

</dd> </dl>
