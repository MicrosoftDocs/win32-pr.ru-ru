# <a name="ui_pkey_globaltextcolor"></a>UI \_ PKEY \_ глобалтекстколор

Определяет свойство UI \_ PKey \_ глобалтекстколор.

```
propertyDescription
   name = UI_PKEY_GlobalTextColor
   shellPKey = UI_PKEY_GlobalTextColor
   formatID = 00002002-7363-696e-8441798acf5aebb7
   propID = 2002
   typeInfo
      type = VT_UINT
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ глобалтекстколор используется приложением для запроса цвета глобального текста для настроенной ленты.

Введено в Windows 8 **:** изменение значения по умолчанию [UI \_ PKEY \_ глобалбаккграундколор](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md) в Windows 8 может потребовать корректировки пользовательского интерфейса \_ PKEY \_ Глобалтекстколор в лентах, предназначенных для Windows 7.
