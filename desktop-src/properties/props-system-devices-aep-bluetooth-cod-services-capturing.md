Bluetooth класс записи службы устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.Capturing
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_Capturing
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```
