# <a name="systemphotophotometricinterpretationtext"></a>System. photo. ФотометриЦинтерпретатионтекст

Понятная пользователю форма [System. photo. фотометриЦинтерпретатион](./props-system-photo-photometricinterpretation.md). Не предназначен для программного анализа.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.PhotometricInterpretationText
   shellPKey = PKEY_Photo_PhotometricInterpretationText
   formatID = 821437D6-9EAB-4765-A589-3B1CBBD22A61
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```
