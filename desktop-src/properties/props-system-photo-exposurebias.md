# <a name="systemphotoexposurebias"></a>System. photo. Експосуребиас

Величина смещения экспозиции, используемая в фотографии, считанная с камеры. Это свойство вычисляется из [System. photo. експосуребиаснумератор](./props-system-photo-exposurebiasnumerator.md) и [System. photo. експосуребиасденоминатор](./props-system-photo-exposurebiasdenominator.md). Единицей является значение ВЕРШИНЕ. Обычно он находится в диапазоне от-99,99 до 99,99.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureBias
   shellPKey = PKEY_Photo_ExposureBias
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37380
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
