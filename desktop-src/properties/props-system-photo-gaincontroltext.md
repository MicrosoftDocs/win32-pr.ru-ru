# <a name="systemphotogaincontroltext"></a>System. photo. Гаинконтролтекст

Понятная пользователю форма [System. photo. гаинконтрол](./props-system-photo-gaincontrol.md). Не предназначен для программного анализа.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.GainControlText
   shellPKey = PKEY_Photo_GainControlText
   formatID = C06238B2-0BF9-4279-A723-25856715CB9D
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
