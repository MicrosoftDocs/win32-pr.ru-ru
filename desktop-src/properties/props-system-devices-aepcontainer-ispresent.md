# <a name="systemdevicesaepcontainerispresent"></a>System.Devices.AepContainer.IsPresent

Указывает, имеется ли в данный момент одна из конечных точек сопоставления устройств

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AepContainer.IsPresent
   shellPKey = PKEY_Devices_AepContainer_IsPresent
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
