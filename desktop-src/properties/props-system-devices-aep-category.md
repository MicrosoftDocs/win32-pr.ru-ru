# <a name="systemdevicesaepcategory"></a>System.Devices.Aep.Category

Категории, в которые входит устройство. Например, принтер, Камера и т. д.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.Aep.Category
   shellPKey = PKEY_Devices_Aep_Category
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
