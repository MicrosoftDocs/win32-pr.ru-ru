# <a name="systemgpsdestlongitudedenominator"></a>System. GPS. Дестлонгитудеденоминатор

Знаменатель объекта PKEY \_ GPS \_ дестлонгитуде.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestLongitudeDenominator
   shellPKey = PKEY_GPS_DestLongitudeDenominator
   formatID = 425D69E5-48AD-4900-8D80-6EB6B8D0AC86
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
