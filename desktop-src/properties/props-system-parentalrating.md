# <a name="systemparentalrating"></a>System. Паренталратинг

Уровень контроля родителей хранится в формате, обычно определяемом организацией с именем в [System. паренталратингсорганизатион](./props-system-parentalratingsorganization.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.

Эта строка отображается непосредственно пользователю, поэтому следует использовать понятные для пользователя слова. Например, "R-Restricted" (с ограничениями по уровням) в системе оценок Motion (МПАА).
