# <a name="systemdevicesdnssdinstancename"></a>System. Devices. Днссд. InstanceName

Часть экземпляра DNS-SD имя экземпляра службы. (например, "MyService" в "MyService. \_ HTTP. \_ TCP. local ")

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.Dnssd.InstanceName
   shellPKey = PKEY_Devices_Dnssd_InstanceName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
