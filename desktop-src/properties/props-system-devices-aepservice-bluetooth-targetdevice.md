# <a name="systemdevicesaepservicebluetoothtargetdevice"></a>System. Devices. Аепсервице. Bluetooth. целевое устройство

Родительское устройство Bluetooth для запроса. Требуется для некэшированных запросов.

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.TargetDevice
   shellPKey = PKEY_Devices_AepService_Bluetooth_TargetDevice
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
