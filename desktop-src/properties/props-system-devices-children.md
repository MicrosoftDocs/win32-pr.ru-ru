# <a name="systemdeviceschildren"></a>System. Devices. Children

Идентификаторы экземпляров устройств, дочерние для этого устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Children
   shellPKey = PKEY_Devices_Children
   formatID = 4340A6C5-93FA-4706-972C-7B648008A5A7
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
