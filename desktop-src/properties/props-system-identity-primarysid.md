# <a name="systemidentityprimarysid"></a>System. Identity. Примарисид

Основной идентификатор безопасности удостоверения.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.PrimarySid
   shellPKey = PKEY_Identity_PrimarySid
   formatID = 2B1B801E-C0C1-4987-9EC5-72FA89814787
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
