# <a name="systemdevicesaepcontainerdialprotocolinstalledapplications"></a>System. Devices. Аепконтаинер. Диалпротокол. Инсталледаппликатионс

Список приложений, поддерживающих протокол DIAL, в контейнере конечной точки сопоставления устройств

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AepContainer.DialProtocol.InstalledApplications
   shellPKey = PKEY_Devices_AepContainer_DialProtocol_InstalledApplications
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
