# <a name="systemlayoutpatterncontentviewmodeforsearch"></a>System. Лайаутпаттерн. Контентвиевмодефорсеарч

Определяет шаблон макета, который должен применяться в режиме просмотра содержимого для этого элемента в контексте поиска.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.LayoutPattern.ContentViewModeForSearch
   shellPKey = PKEY_LayoutPattern_ContentViewModeForSearch
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 501
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Alpha
            value = alpha
            text
            defineToken = LAYOUTPATTERN_CVMFS_ALPHA
         enum
            name = Beta
            value = beta
            text
            defineToken = LAYOUTPATTERN_CVMFS_BETA
         enum
            name = Gamma
            value = gamma
            text
            defineToken = LAYOUTPATTERN_CVMFS_GAMMA
         enum
            name = Delta
            value = delta
            text
            defineToken = LAYOUTPATTERN_CVMFS_DELTA
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
