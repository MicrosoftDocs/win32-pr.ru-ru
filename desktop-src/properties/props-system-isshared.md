# <a name="systemisshared"></a>System. Совместное использование

Указывает, является ли элемент общим. При этом проверяются только неунаследованные списки ACL.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
