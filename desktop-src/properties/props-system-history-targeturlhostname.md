# <a name="systemhistorytargeturlhostname"></a>System. History. Таржетурлхостнаме

Пометьте веб-зону как значение перечисления УРЛЗОНЕ.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.History.TargetUrlHostName
   shellPKey = PKEY_History_TargetUrlHostName
   formatID = 1CE0D6BC-536C-4600-B0DD-7E0C66B350D5
   propID = 9
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
