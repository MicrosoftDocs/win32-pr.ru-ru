# <a name="common-constants"></a>Общие константы

Одноранговая инфраструктура использует следующие общие константы.



| Константа                                                      | Описание                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**Флаги однорангового \_ соединения \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Указывает типы подключений, которые может иметь одноранговый узел.      |
| [**состояние однорангового \_ подключения \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Указывает состояние однорангового прямого или соседнего подключения. |
| [**\_ \_ тип изменения одноранговой записи \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Указывает изменения, которые могут возникать в записи.             |
| [**Флаги одноранговой \_ записи \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Задает набор флагов для поведения одноранговой записи.           |
