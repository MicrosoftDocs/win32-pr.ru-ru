# <a name="people-near-me-apis"></a>API-интерфейсы "Соседние пользователи"

Все приложения, поддерживающие [соседние пользователи](about-people-near-me.md) (PNM), могут использовать следующие API-интерфейсы Win32:

### <a name="contacts-api"></a>API контактов

API контактов используется для доступа к контактным данным пользователя, выполнившего вход в систему, получения сведений о ранее сохраненных контактах или для хранения контактной информации и цифрового сертификата нового контакта.

### <a name="people-near-me-api"></a>API "Соседние пользователи"

API PNM используется для поиска ближайших пользователей, их объявленных интересов, любых произвольных объектов, которые они выбирают для публикации, и объявленных приложений.

### <a name="publication-api"></a>API публикации

API публикации используется для взаимодействия с удаленными контактами. Уровень [сигнализации однорангового узла](windows-peer-components-for-people-near-me.md) , используемый диспетчером публикаций, также используется модулем PNM для установления соединений.

### <a name="invitation-api"></a>API приглашения

API приглашения используется приложением для совместной работы, чтобы пригласить определенные узлы в действие совместной работы.

### <a name="application-registration-api"></a>API регистрации приложений

API регистрации приложений используется установщиком приложения для совместной работы для регистрации приложения. Приложение, поддерживающее PNM, будет запрашивать пользователя в процессе установки, чтобы определить, должно ли приложение быть доступно пользователям в подсети.
