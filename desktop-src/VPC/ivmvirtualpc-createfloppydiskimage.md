# <a name="ivmvirtualpccreatefloppydiskimage-method"></a>Метод Ивмвиртуалпк:: Креатефлоппидискимаже

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Создает файл образа гибкого диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateFloppyDiskImage(
  [in] BSTR                  imagePath,
  [in] VMFloppyDiskImageType imageType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*imagePath* \[ окне\]
</dt> <dd>

Полный путь к новому файлу образа диска. Содержащаяся папка будет создана, если она не существует.

</dd> <dt>

*imageType* \[ окне\]
</dt> <dd>

Тип создаваемого образа гибкого диска. Поддерживаются только **вмфлоппидискимаже \_ ловденсити** (1) и **вмфлоппидискимаже \_ хигхденсити** (2). См. [**вмфлоппидискимажетипе**](vmfloppydiskimagetype.md).

</dd> </dl>
