# <a name="locale_idefault-constants"></a>\_Константы идефаулт языкового стандарта \*

В этом разделе определяются \_ константы ИДЕФАУЛТ локали \* , используемые NLS.



| Значение                          | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|--------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| идефаултансикодепаже ЛОКАЛи \_   | Кодовая страница ANSI, используемая языковым стандартом для приложений, которые не поддерживают Юникод. Максимально допустимое число символов для этой строки равно шести, включая завершающий нуль-символ. Если кодовая страница ANSI недоступна, для локали можно использовать только Юникод. В этом случае значением является CP \_ ACP (0). Такой языковой стандарт не может быть установлен в качестве национальной настройки системы. Приложения, не поддерживающие Юникод, работают неправильно с языками, помеченными как "только Юникод". Список ANSI и других кодовых страниц см. в разделе [идентификаторы кодовых страниц](code-page-identifiers.md). |
| идефаулткодепаже ЛОКАЛи \_       | Кодовая страница изготовителя оборудования (OEM), связанная со страной или регионом. Кодовая страница OEM используется для преобразования из приложений в текстовом режиме, основанных на MS-DOS. Если языковой стандарт не использует кодовую страницу OEM, это значение равно CP \_ оемкп (1). Максимально допустимое число символов для этой строки равно шести, включая завершающий нуль-символ. Список кодовой страницы OEM и других страниц см. в разделе [идентификаторы кодовых страниц](code-page-identifiers.md).                                                                                                               |
| идефаулткаунтри ЛОКАЛи \_        | Является устаревшей. Не используйте. Это значение было предоставлено таким образом, чтобы частично заданные языковые стандарты были выполнены со значениями по умолчанию. Частично указанные национальные настройки теперь не рекомендуются.                                                                                                                                                                                                                                                                                                                                                                                               |
| идефаултебкдиккодепаже ЛОКАЛи \_ | **Windows 2000:** По умолчанию расширенная двоичная кодовая страница с десятичным кодированием (EBCDIC), связанная с языковым стандартом. Максимально допустимое число символов для этой строки равно шести, включая завершающий нуль-символ. Список, посвященный EBCDIC и другим кодовым страницам, см. в разделе [идентификаторы кодовых страниц](code-page-identifiers.md).                                                                                                                                                                                                                                     |
| идефаултлангуаже ЛОКАЛи \_       | Является устаревшей. Не используйте. Это значение было предоставлено таким образом, чтобы частично заданные языковые стандарты были выполнены со значениями по умолчанию. Частично указанные национальные настройки теперь не рекомендуются.                                                                                                                                                                                                                                                                                                                                                                                               |
| идефаултмаккодепаже ЛОКАЛи \_    | Кодовая страница Macintosh по умолчанию, связанная с языковым стандартом. Максимально допустимое число символов для этой строки равно шести, включая завершающий нуль-символ. Если языковой стандарт не использует кодовую страницу Macintosh, это значение равно CP \_ маккп (2). Список компьютеров Макинтош (MAC) и других кодовых страниц см. в разделе [идентификаторы кодовых страниц](code-page-identifiers.md).                                                                                                                                                                                                              |
