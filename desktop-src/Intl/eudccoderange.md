# <a name="eudccoderange"></a>еудккодеранже

Раздел реестра Еудккодеранже определяет диапазоны кодов [EUDC](end-user-defined-characters.md) для различных кодовых страниц (наборов символов). Он используется только инструментами, которые создают Еудкс и не являются непосредственной проблемой для пользователей EUDC. Этот раздел реестра находится в следующем расположении реестра:

\_ \_ \\ \\ CurrentControlSet \\ управления \\ \\ кодовой страницей NLS \\ в системе hKey локального компьютера

Формат будет следующим:

Еудккодеранже кодовая страница = Фромто \[ , фромто\]

где:



|          |                                                                                                                                                                                                          |
|----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CodePage | Одна из строк "932" (японский), "936" (упрощенный китайский), "949" (корейский), "950" (китайский (традиционное письмо)) или "Unicode" (Юникод). Другие значения не поддерживаются.                                     |
| фромто   | Строковое значение, состоящее из пары из 4-значных шестнадцатеричных значений, разделенных дефисом (-). Можно указать до четырех значений Фромто, но каждое из них должно быть отделено от предыдущего значения запятой (,). |



 

Ниже приведены правильные значения для записи реестра.


```C++
932=F040-F9FC
936=A140-A7A0,AAA1-AFFE,F8A1-FEFE
949=C9A1-C9FE,FEA1-FEFE
950=8140-8DFE,8E40-A0FE,C6A1-C8FE,FA40-FEFE
Unicode=E000-F8FF
```
