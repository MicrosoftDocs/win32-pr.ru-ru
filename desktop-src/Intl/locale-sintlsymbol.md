# <a name="locale_sintlsymbol"></a>синтлсимбол ЛОКАЛи \_

Три символа международного денежного символа, указанного в стандарте ISO 4217, за которым следует символ, разделяющий эту строку из суммы. Максимально допустимое число символов для этой строки равно девяти, включая завершающий нуль-символ.
