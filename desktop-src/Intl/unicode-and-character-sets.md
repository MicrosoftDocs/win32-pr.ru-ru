# <a name="unicode-and-character-sets"></a><span data-ttu-id="f5cac-103">Юникод и кодировки</span><span class="sxs-lookup"><span data-stu-id="f5cac-103">Unicode and Character Sets</span></span>

<span data-ttu-id="f5cac-104">Microsoft Windows обеспечивает поддержку множества различных письменных языков международного рынка в [Юникоде](unicode.md) и традиционных наборах символов.</span><span class="sxs-lookup"><span data-stu-id="f5cac-104">Microsoft Windows provides support for the many different written languages of the international marketplace through [Unicode](unicode.md) and traditional character sets.</span></span>

<span data-ttu-id="f5cac-105">Юникод — это стандартный стандарт кодировки символов, который предоставляет уникальный номер для представления каждого символа, используемого в современных вычислениях, включая технические символы и специальные символы, используемые при публикации.</span><span class="sxs-lookup"><span data-stu-id="f5cac-105">Unicode is a worldwide character encoding standard that provides a unique number to represent each character used in modern computing, including technical symbols and special characters used in publishing.</span></span> <span data-ttu-id="f5cac-106">Юникод требуется для современных стандартов, таких как XML и ECMAScript (JavaScript), и является официальным механизмом реализации ISO/IEC 10646.</span><span class="sxs-lookup"><span data-stu-id="f5cac-106">Unicode is required by modern standards, such as XML and ECMAScript (JavaScript), and is the official mechanism for implementing ISO/IEC 10646.</span></span> <span data-ttu-id="f5cac-107">Поддерживается многими операционными системами, всеми современными браузерами и многими другими продуктами.</span><span class="sxs-lookup"><span data-stu-id="f5cac-107">It is supported by many operating systems, all modern browsers, and many other products.</span></span> <span data-ttu-id="f5cac-108">Новые приложения Windows должны использовать Юникод, чтобы избежать несоответствий между различными кодовыми страницами и упростить локализацию.</span><span class="sxs-lookup"><span data-stu-id="f5cac-108">New Windows applications should use Unicode to avoid the inconsistencies of varied code pages and to aid in simplifying localization.</span></span>

<span data-ttu-id="f5cac-109">Традиционные наборы символов — это предыдущие стандарты кодирования символов, такие как [кодовые страницы](code-pages.md) Windows, использующие 8-битные значения кода или сочетания 8-разрядных значений для представления символов, используемых в определенном языке или географическом регионе.</span><span class="sxs-lookup"><span data-stu-id="f5cac-109">Traditional character sets are the previous character encoding standards — such as Windows [code pages](code-pages.md) that use 8-bit code values or combinations of 8-bit values to represent the characters used in a specific language or geographical region.</span></span>

<span data-ttu-id="f5cac-110">В этом обзоре описываются функции наборов символов и объясняется, как использовать их в приложениях.</span><span class="sxs-lookup"><span data-stu-id="f5cac-110">This overview describes the character set functions and explains how to use them in your applications.</span></span>
