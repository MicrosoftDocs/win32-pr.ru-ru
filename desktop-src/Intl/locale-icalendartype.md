# <a name="locale_icalendartype"></a>икалендартипе ЛОКАЛи \_

Текущий тип календаря. Возможные значения определяются в [сведениях о типе календаря](calendar-type-information.md). Тип календаря может представлять только календарь, доступный для соответствующего языкового стандарта. Чтобы получить все календари, доступные для языкового стандарта, приложение может использовать следующие функции NLS:

-   [енумкалендаринфо](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoa)
-   [енумкалендаринфоекс](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexa)
-   [енумкалендаринфоексекс](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexex)
