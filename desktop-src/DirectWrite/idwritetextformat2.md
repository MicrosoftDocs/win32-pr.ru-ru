# <a name="idwritetextformat2-interface"></a>Интерфейс IDWriteTextFormat2

Описывает свойства шрифта и абзаца, используемые для форматирования текста, а также сведения о языковых стандартах.

## <a name="members"></a>Элементы

Интерфейс **IDWriteTextFormat2** наследует от [**IDWriteTextFormat1**](idwritetextformat1.md). **IDWriteTextFormat2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IDWriteTextFormat2** содержит следующие методы.



| Метод                                                      | Описание                                                                      |
|:------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**жетлинеспаЦинг**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritetextformat2-getlinespacing) | Возвращает набор корректировок междустрочного интервала для абзаца из нескольких строк. <br/> |
| [**сетлинеспаЦинг**](idwritetextformat2-setlinespacing.md) | Задать Междустрочный зазор.<br/>                                                     |
