# <a name="enabledcom"></a>енабледком

Управляет глобальной политикой активации и вызова компьютера. Только администраторы и система имеют полный доступ к этой части реестра. Все остальные пользователи имеют доступ только для чтения.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole
   EnableDCOM = value
```

## <a name="remarks"></a>Комментарии

Это значение **reg \_ SZ** .



| Значение    | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| N (или n) | Удаленные клиенты не могут запускать серверы или подключаться к объектам на этом компьютере. Локальные клиенты не могут получить доступ к удаленным серверам DCOM; весь трафик DCOM блокируется. Локальный запуск кода класса и подключение к объектам разрешены для каждого класса в соответствии со значением и разрешениями на доступ к значению реестра [**лаунчпермиссион**](launchpermission.md) класса и глобальному значению реестра [**дефаултлаунчпермиссион**](defaultlaunchpermission.md) . |
| Y (или y) | Запуск серверов и подключение к объектам с помощью удаленных клиентов разрешено для каждого класса в соответствии со значением и разрешениями на доступ к значению реестра [**лаунчпермиссион**](launchpermission.md) класса и глобальному значению реестра [**дефаултлаунчпермиссион**](defaultlaunchpermission.md) .                                                                                                                                                    |
