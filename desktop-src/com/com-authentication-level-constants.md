# <a name="authentication-level-constants"></a>Константы уровня проверки подлинности

Эти значения указывают уровень проверки подлинности, указывающий объем проверки подлинности, предоставляемой для защиты целостности данных. Каждый уровень включает защиту, обеспечиваемую предыдущими уровнями.



| Константа/значение                                                                                                                                                                                                                                                                 | Описание                                                                                                                                                                                                    |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="RPC_C_AUTHN_LEVEL_DEFAULT"></span><span id="rpc_c_authn_level_default"></span><dl> <dt>**RPC \_ \_AUTHN \_ уровень \_ по умолчанию**</dt> <dt>0</dt> </dl>                    | Указывает DCOM выбрать уровень проверки подлинности с помощью обычного алгоритма согласования безопасности общего доступа. Дополнительные сведения см. в разделе [Безопасность — общий согласование](security-blanket-negotiation.md). <br/> |
| <span id="RPC_C_AUTHN_LEVEL_NONE"></span><span id="rpc_c_authn_level_none"></span><dl> <dt>**RPC \_ \_ \_ Уровень \_ AUTHN**</dt> No <dt>1</dt> </dl>                             | Не выполняет проверку подлинности.<br/>                                                                                                                                                                         |
| <span id="RPC_C_AUTHN_LEVEL_CONNECT"></span><span id="rpc_c_authn_level_connect"></span><dl> <dt>**RPC \_ \_AUTHN на \_ уровне \_ подключения**</dt> <dt>2</dt> </dl>                    | Проверяет подлинность учетных данных клиента только в том случае, если клиент устанавливает связь с сервером. Для транспорта датаграмм всегда используется RPC на \_ \_ уровне AUTHN \_ . <br/>                        |
| <span id="RPC_C_AUTHN_LEVEL_CALL"></span><span id="rpc_c_authn_level_call"></span><dl> <dt>**RPC \_ \_ \_ \_ Вызов уровня 3 в AUTHN**</dt> <dt></dt> </dl>                             | Проверка подлинности выполняется только в начале каждого удаленного вызова процедуры, когда сервер получает запрос. Для транспорта датаграмм используется маркер RPC \_ C \_ \_ уровня AUTHN \_ .<br/>                                  |
| <span id="RPC_C_AUTHN_LEVEL_PKT"></span><span id="rpc_c_authn_level_pkt"></span><dl> <dt>**RPC \_ \_AUTHN ( \_ \_ PKT) уровня**</dt> 3 <dt></dt> </dl>                                | Проверяет, что все полученные данные получены от ожидаемого клиента.<br/>                                                                                                                                   |
| <span id="RPC_C_AUTHN_LEVEL_PKT_INTEGRITY"></span><span id="rpc_c_authn_level_pkt_integrity"></span><dl> <dt>**RPC \_ \_ \_ \_ \_ Целостность маркера C AUTHN Level**</dt> <dt>5</dt> </dl> | Проверка подлинности и проверка того, что ни один из данных, передаваемых между клиентом и сервером, не был изменен.<br/>                                                                                           |
| <span id="RPC_C_AUTHN_LEVEL_PKT_PRIVACY"></span><span id="rpc_c_authn_level_pkt_privacy"></span><dl> <dt>**RPC \_ AUTHN на языке C, \_ \_ PKT, \_ \_ Конфиденциальность**</dt> <dt>6</dt> </dl>       | Проверяет подлинность всех предыдущих уровней и шифрует значение аргумента каждого удаленного вызова процедуры.<br/>                                                                                                    |
