# <a name="interfaces-controls-and-property-pages"></a>Интерфейсы (элементы управления и страницы свойств)

Для создания стандартных COM-объектов и страниц свойств используются следующие интерфейсы.



| Интерфейс                                             | Описание                                                                                                                                                                                                  |
|-------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IFont**](/windows/desktop/api/OCIdl/nn-ocidl-ifont)                                | Предоставляет оболочку для объекта шрифта Windows.                                                                                                                                                             |
| [**IFontDisp**](/windows/win32/api/ocidl/nn-ocidl-ifontdisp)                        | Предоставляет свойства объекта Font через автоматизацию. Он предоставляет подмножество методов [**IFont**](/windows/desktop/api/OCIdl/nn-ocidl-ifont) .                                                                                           |
| [**Метод интерфейса IOleControl**](/windows/desktop/api/OCIdl/nn-ocidl-iolecontrol)                    | Предоставляет функции для поддержки назначенных клавиш, внешних свойств и событий в управляющих объектах.                                                                                                  |
| [**IOleControlSite**](/windows/desktop/api/OCIdl/nn-ocidl-iolecontrolsite)            | Предоставляет методы, позволяющие объекту сайта управлять каждым внедренным элементом управления в контейнере.                                                                                                           |
| [**иперпропертибровсинг**](/windows/desktop/api/OCIdl/nn-ocidl-iperpropertybrowsing)  | Извлекает сведения на страницах свойств, предоставляемых объектом.                                                                                                                                        |
| [**ипиктуре**](/windows/desktop/api/OCIdl/nn-ocidl-ipicture)                          | Управляет объектом изображения и его свойствами. Объекты изображений обеспечивают абстракцию, не зависящую от языка, для точечных рисунков, значков и метафайлов.                                                                       |
| [**ипиктуредисп**](/windows/win32/api/ocidl/nn-ocidl-ipicturedisp)                  | Предоставляет свойства объекта изображения через автоматизацию. Он предоставляет подмножество функциональных возможностей, доступных через методы [**ипиктуре**](/windows/desktop/api/OCIdl/nn-ocidl-ipicture) .                                                |
| [**Интерфейс IPointerInactive**](/windows/desktop/api/OCIdl/nn-ocidl-ipointerinactive)          | Позволяет объекту оставаться неактивным в большую часть времени, но по-прежнему участвует в взаимодействии с мышью, включая перетаскивание.                                                                         |
| [**IPrint**](/windows/desktop/api/DocObj/nn-docobj-iprint)                              | Позволяет создавать составные документы в общих и активных документах в частности для поддержки программной печати.                                                                                                   |
| [**ипропертинотифисинк**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink)    | Реализуется объектом-приемником для получения уведомлений об изменениях свойств от объекта, поддерживающего [**ипропертинотифисинк**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink) в качестве исходящего интерфейса.                       |
| [**ипропертипаже**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertypage)                | Предоставляет основные возможности объекта страницы свойств, который управляет определенной страницей в таблице свойств.                                                                                                 |
| [**IPropertyPage2**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertypage2)              | Расширение, [**ипропертипаже**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertypage) для поддержки первоначального выбора свойства на странице.                                                                                                 |
| [**ипропертипажесите**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertypagesite)        | Предоставляет основные функции для объекта сайта страницы свойств.                                                                                                                                                  |
| [**икуиккактивате**](/windows/desktop/api/OCIdl/nn-ocidl-iquickactivate)              | Включает элементы управления и контейнеры, чтобы избежать узких мест производительности при загрузке элементов управления. В нем объединяются время загрузки или подтверждения во время инициализации между элементом управления и его контейнером в один вызов. |
| [**исимплефрамесите**](/windows/desktop/api/OCIdl/nn-ocidl-isimpleframesite)          | Предоставляет простые элементы управления Frame, которые служат простыми контейнерами для других вложенных элементов управления.                                                                                                                      |
| [**испеЦифипропертипаже**](/windows/desktop/api/OCIdl/nn-ocidl-ispecifypropertypages) | Указывает, что объект поддерживает страницы свойств.                                                                                                                                                            |
