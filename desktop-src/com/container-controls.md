# <a name="container-controls"></a>Контейнерные элементы управления

как описано выше, контейнерные элементы управления являются ActiveX элементами управления, которые визуально содержат другие элементы управления. архитектура элементов управления ActiveX определяет интерфейс [**исимплефрамесите**](/windows/desktop/api/OCIdl/nn-ocidl-isimpleframesite) для включения элементов управления контейнера. Контейнеры также могут поддерживать элементы управления контейнера без поддержки **исимплефрамесите**, хотя поведение не гарантируется. По этой причине для элементов управления Симплефрамесите существует категория компонента, где требуются все функциональные возможности этого интерфейса.

для поддержки элементов управления контейнера без реализации [**исимплефрамесите**](/windows/desktop/api/OCIdl/nn-ocidl-isimpleframesite)контейнер элемента управления ActiveX должен:

-   Активируйте все элементы управления все время.
-   Переподчиненные вложенные элементы управления в hWnd содержащего его элемента управления.
-   Оставить родительский элемент контейнерного элемента управления.
