# <a name="midl-compiler-options"></a>Параметры компилятора MIDL

Можно использовать следующие параметры командной строки для переопределения некоторых параметров по умолчанию компилятора MIDL и выбора оптимальных вариантов оптимизации для приложения. Полный список параметров командной строки MIDL см. в [справочнике по языку midl Command-Line](/windows/desktop/Midl/midl-command-line-reference).



| Параметр командной строки                                       | Описание                                                                                                                                                                                                                                      |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**/акф**](/windows/desktop/Midl/-acf)<br/>                          | Используйте для предоставления явного имени файла ACF. Этот параметр также позволяет использовать различные имена интерфейсов в файлах IDL и ACF.<br/>                                                                                                       |
| [**/dlldata**](/windows/desktop/Midl/-dlldata)<br/>                  | Указывает имя файла для созданного в DLL-файле данных библиотеки DLL. Имя файла по умолчанию — DLLDATA. c.<br/>                                                                                                                              |
| [**/env**](/windows/desktop/Midl/-env)<br/>                          | Направляет MIDL для создания заглушек или библиотеки типов для целевой среды.<br/>                                                                                                                                                            |
| [**/Header**](/windows/desktop/Midl/-header), [ **/h**](/windows/desktop/Midl/-h)<br/> | Указывает имя файла заголовка интерфейса. По умолчанию используется имя IDL-файла с расширением h.<br/>                                                                                                                       |
| [**/IID**](/windows/desktop/Midl/-iid)<br/>                          | Указывает имя файла идентификатора интерфейса, переопределяющего имя файла идентификатора интерфейса по умолчанию для COM-интерфейса.<br/>                                                                                                              |
| [**/LCID**](/windows/desktop/Midl/-lcid)<br/>                        | Обеспечивает полную поддержку DBCS, что позволяет использовать международные символы во входных файлах, именах файлов и путях к каталогам.<br/>                                                                                                          |
| [**/но в \_ формате \_ OPT**](/windows/desktop/Midl/-no-format-opt)<br/>    | По умолчанию, чтобы сократить размер кода, MIDL удаляет дублирующиеся дескрипторы. Этот параметр отключает оптимизацию этого поведения.<br/>                                                                                                               |
| [**/Oi**](/windows/desktop/Midl/-oi), **/ОИК**, **/OIF**<br/>        | Направляет MIDL для использования полностью интерпретируемого метода маршалирования. Параметры/ОИК и/Oicf обеспечивают дополнительные улучшения производительности.<br/>                                                                                                   |
| [**/out**](/windows/desktop/Midl/-out)<br/>                          | Указывает каталог, в который компилятор MIDL записывает выходные файлы. Выходной каталог можно указать с помощью буквы диска, абсолютного пути или и того, и другого. По умолчанию MIDL записывает файлы в текущий каталог.<br/> |
| [**/Proxy**](/windows/desktop/Midl/-proxy)<br/>                      | Определяет имя прокси-файла интерфейса для COM-интерфейса. По умолчанию используется имя IDL-файла и « \_ p. c».<br/>                                                                                                            |
| [**/TLB**](/windows/desktop/Midl/-tlb)<br/>                          | Задает имя файла библиотеки типов. По умолчанию используется имя IDL-файла с расширением TLB.<br/>                                                                                                                         |
