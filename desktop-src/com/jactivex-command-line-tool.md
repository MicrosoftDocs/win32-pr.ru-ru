# <a name="jactivex-command-line-tool"></a>Средство Command-Line Жактивекс

Приложение командной строки Жактивекс является компонентом Visual J++ 6,0. Он считывает библиотеку типов элемента управления Microsoft ActiveX и создает соответствующие исходные файлы Java. Затем можно скомпилировать эти исходные файлы и импортировать их в приложение Java.

Чтобы создать исходные файлы Java для COM-объекта, в командной строке выполните следующую команду:

 *имя файла* жактивекс

где *filename* — имя файла, содержащего библиотеку типов. Этот файл может иметь любое из следующих расширений имен файлов:. tlb,. OLB,. ocx,. dll или. exe.

Дополнительные сведения о необязательных параметрах, которые можно задать в Жактивекс, см. в документации по Visual J++ или введите следующую командную строку:

**жактивекс/?**

> [!WARNING]
> Не используйте компиляторы Visual J++ до версии 1.02.3920, чтобы скомпилировать файлы, созданные JActiveX.exe. Это обусловлено тем, что созданные исходные файлы должны быть скомпилированы с помощью @com-aware компилятора. Только компиляторы Visual J++ более поздней версии, чем версия 1.02.3920 @com-aware .
