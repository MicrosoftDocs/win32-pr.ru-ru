<span data-ttu-id="c024c-104">При вставке ссылки на объект в составной документ исходные данные или *источник связи* остаются в любом месте, где он был изначально создан, обычно в другом документе.</span><span class="sxs-lookup"><span data-stu-id="c024c-104">When a link to an object is inserted in a compound document, the source data, or *link source*, continues to reside wherever it was initially created, usually in another document.</span></span> <span data-ttu-id="c024c-105">Составной документ содержит только ссылку или *ссылку* на фактические данные, хранящиеся в источнике ссылки, а также сведения о том, как предоставить эти данные пользователю.</span><span class="sxs-lookup"><span data-stu-id="c024c-105">The compound document contains only a reference, or *link*, to the actual data stored at the link source, along with information about how to present that data to the user.</span></span>

<span data-ttu-id="c024c-106">При активации ссылки запускается серверное приложение источника связи, которое требуется пользователю для редактирования или иным образом манипулировать данными связи.</span><span class="sxs-lookup"><span data-stu-id="c024c-106">Activating a link runs the link source's server application, which the user requires in order to edit or otherwise manipulate the link data.</span></span> <span data-ttu-id="c024c-107">Связывание позволяет уменьшить размер составного документа.</span><span class="sxs-lookup"><span data-stu-id="c024c-107">Linking keeps the size of a compound document small.</span></span> <span data-ttu-id="c024c-108">Он также полезен, если источник данных поддерживается другим пользователем и должен совместно использоваться несколькими пользователями.</span><span class="sxs-lookup"><span data-stu-id="c024c-108">It is also useful when the data source is maintained by someone else and must be shared among many users.</span></span> <span data-ttu-id="c024c-109">Если пользователь, обслуживающий источник связи, изменяет данные, это изменение автоматически обновляется во всех документах, содержащих ссылку на эти данные.</span><span class="sxs-lookup"><span data-stu-id="c024c-109">If the person maintaining the link source changes the data, the change is automatically updated in all documents containing a link to that data.</span></span> <span data-ttu-id="c024c-110">Помимо создания простых ссылок, пользователи могут вкладывать ссылки и объединять связанные и внедренные объекты для создания сложных документов.</span><span class="sxs-lookup"><span data-stu-id="c024c-110">In addition to creating simple links, users can nest links and combine linked and embedded objects to create complex documents.</span></span>
