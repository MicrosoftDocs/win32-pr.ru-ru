# <a name="runningtaskstate-property"></a>Руннингтаск. State, свойство

Для создания скриптов возвращает идентификатор для состояния выполняющейся задачи.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
RunningTask.State As String
```



## <a name="property-value"></a>Значение свойства

Идентификатор состояния выполняемой задачи.



| Значение                                                                                                                                                                                                                                   | Значение                                                                                                                                           |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TASK_STATE_UNKNOWN"></span><span id="task_state_unknown"></span><dl> <dt>**Задача \_ \_Неизвестное состояние**</dt> <dt>0</dt> </dl>    | Состояние задачи неизвестно.<br/>                                                                                                      |
| <span id="TASK_STATE_DISABLED"></span><span id="task_state_disabled"></span><dl> <dt>**Задача \_ СОСТОЯНИЕ " \_ отключено**</dt> <dt>1</dt> " </dl> | Задача зарегистрирована, но отключена и ни одна из экземпляров задачи не поставлена в очередь или запущена. Задача не может быть выполнена, пока она не будет включена.<br/> |
| <span id="TASK_STATE_QUEUED"></span><span id="task_state_queued"></span><dl> <dt>**Задача \_ СОСТОЯНИЕ \_ поставлено в очередь**</dt> <dt>2</dt> </dl>       | Экземпляры задачи поставлены в очередь.<br/>                                                                                                      |
| <span id="TASK_STATE_READY"></span><span id="task_state_ready"></span><dl> <dt>**Задача \_ СОСТОЯНИЕ \_ готовности**</dt> <dt>3</dt> </dl>          | Задача готова к выполнению, но экземпляры не поставлены в очередь или запущены.<br/>                                                              |
| <span id="TASK_STATE_RUNNING"></span><span id="task_state_running"></span><dl> <dt>**Задача \_ СОСТОЯНИЕ \_ выполнения**</dt> <dt>4</dt> </dl>    | Выполняется один или несколько экземпляров задачи.<br/>                                                                                         |



 

## <a name="remarks"></a>Комментарии

Метод [**руннингтаск. Refresh**](runningtask-refresh.md) вызывается перед возвратом значения свойства.
