# <a name="idletrigger-object"></a>Объект Идлетригжер

Объект скрипта, представляющий триггер, который запускает задачу при возникновении условия простоя. Сведения об условиях простоя см. в разделе [условия простоя задачи](task-idle-conditions.md).

## <a name="remarks"></a>Remarks

Триггер простоя активирует действие задачи, только если компьютер переходит в состояние простоя после начальной границы триггера.

При чтении или записи XML-кода для задачи триггер простоя указывается с помощью элемента [**идлетригжер**](taskschedulerschema-idletrigger-triggergroup-element.md) схемы планировщик задач.

Если задача запускается триггером Idle, свойство [**идлесеттингс. ваиттимеаут**](idlesettings-waittimeout.md) игнорируется.

Если исходный экземпляр задачи с триггером Idle все еще выполняется, задача запускается только один раз без повторений, даже если в свойстве [**повторения**](trigger-repetition.md) определено несколько повторений. Такое поведение не происходит, если задача останавливается сама по себе.
