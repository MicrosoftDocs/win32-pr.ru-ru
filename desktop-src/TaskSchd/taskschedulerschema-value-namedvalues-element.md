# <a name="value-namedvalues-element"></a>Элемент value (Намедвалуес)

Содержит значение, связанное с именем в паре «имя-значение».

``` syntax
<xs:element name="Value"
    type="namedValue"
    minOccurs="1"
    maxOccurs="32"
 />
```

Элемент **value** определяется сложным типом [**намедвалуес**](taskschedulerschema-namedvalues-complextype.md) .

## <a name="parent-element"></a>Родительский элемент



| Элемент                                                                                              | Унаследован от                                                       | Описание                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Валуекуериес (Евенттригжертипе)**](taskschedulerschema-valuequeries-eventtriggertype-element.md) | [**намедвалуес**](taskschedulerschema-namedvalues-complextype.md) | Задает коллекцию именованных запросов XPath. Каждый запрос в коллекции применяется к последнему соответствующему XML-событию, возвращенному из запроса подписки, указанного в элементе [**Subscription**](taskschedulerschema-subscription-eventtriggertype-element.md) . Имя запроса можно использовать в качестве переменной в сообщении для действия ShowMessage.<br/> |
