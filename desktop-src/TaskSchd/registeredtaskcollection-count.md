# <a name="registeredtaskcollectioncount-property"></a>Регистередтаскколлектион. Count, свойство

Для сценариев возвращает количество зарегистрированных задач в коллекции.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTaskCollection.Count As long
```



## <a name="property-value"></a>Значение свойства

Число зарегистрированных задач в коллекции.
