# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: нулевые остановки коллекции ограничителей градиента

Для создания ID2D1GradientStopCollection требуется по крайней мере одна точка градиента.






 

## <a name="possible-causes"></a>Возможные причины

Приложение с именем ID2D1RenderTarget:: Креатеградиентстопколлектион, но указал число ограничителей градиента, равное 0.

## <a name="possible-fixes"></a>Возможные исправления

При создании ID2D1GradientStopCollection приложение должно указать по крайней мере одну точку градиента с его цветом и положением. Пример см. в разделе ID2D1RenderTarget:: Креатеградиентстопколлектион.
