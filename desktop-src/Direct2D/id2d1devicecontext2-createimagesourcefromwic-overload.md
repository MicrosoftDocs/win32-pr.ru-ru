# <a name="id2d1devicecontext2createimagesourcefromwic-methods"></a>Методы ID2D1DeviceContext2:: Креатеимажесаурцефромвик

Создает исходный объект изображения из источника битовой карты WIC, заполняя всю память в пикселях в источнике изображения. Образ загружается и сохраняется с минимальным объемом памяти.
