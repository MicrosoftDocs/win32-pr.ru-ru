# <a name="d1141-print-requires-srgb-color-space"></a>D1141: для печати требуется цветовое пространство sRGB

Direct2Dная печать: настраиваемое цветовое пространство не поддерживается. Вместо этого используйте sRGB.






 

## <a name="possible-causes"></a>Возможные причины

При создании элемента управления печатью Direct2D приложение использует \_ \_ свойства элемента управления печатью D2D1 \_ с атрибутом цветового пространства. Direct2D поддерживает [**\_ \_ \_ пользовательское цветовое пространство D2D1**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_color_space) для некоторых операций, таких как создание контекста цвета с помощью данных ICC. Однако при создании элемента управления печатью Direct2D поддерживает только встроенные цветовые пространства, например sRGB или scRGB.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить это сообщение, укажите нестандартное цветовое пространство, например sRGB или scRGB, при создании элемента управления печатью. Сообщение не указывает на ошибку, но указывает, что Direct2D будет игнорировать НАСТРАИВАЕМый описатель и создать элемент управления печатью с помощью цветового пространства sRGB.
