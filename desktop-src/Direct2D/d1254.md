# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: операция с одним потоковым доменом

Вызов [**ID2D1Multithread:: Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) или [**ID2D1Multithread:: Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) был сделан в однопотоковом домене фабрики.

Это может произойти, если приложение случайно создало фабрику D2D с [**\_ типом фабрики D2D1 с \_ \_ несколькими \_ потоками**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type). В этом случае вызов будет проигнорирован.
