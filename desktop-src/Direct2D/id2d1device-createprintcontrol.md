# <a name="id2d1devicecreateprintcontrol-methods"></a>Методы ID2D1Device:: Креатепринтконтрол

Создает объект [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) , который преобразует [Direct2D](/windows/desktop/direct2d/direct2d-portal) примитивы, хранящиеся в [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) , в фиксированное представление страницы. Затем подсистема печати использует примитивы.
