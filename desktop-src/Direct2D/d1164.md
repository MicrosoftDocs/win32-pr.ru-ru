# <a name="d1164-invalid-map-option"></a>D1164: недопустимый параметр Map

\_Параметры D2D1 \_ Map \_ None не являются допустимым параметром для сопоставлений






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ \_ параметром D2D1 Map Options \_ None.

## <a name="possible-fixes"></a>Возможные исправления

При вызове ID2D1Bitmap1:: Map используйте допустимый параметр Map. \_ \_ Описание параметров Map см. в разделе Параметры Map D2D1. Определите, требуется ли приложению доступ на чтение или запись.
