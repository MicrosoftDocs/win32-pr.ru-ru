<span data-ttu-id="e8030-105">Частный пул датчиков — это набор биометрических блоков, зарезервированных для монопольного использования клиентским приложением.</span><span class="sxs-lookup"><span data-stu-id="e8030-105">A private sensor pool is a collection of biometric units reserved for exclusive use by a client application.</span></span> <span data-ttu-id="e8030-106">Частные пулы поддерживают собственные методы проверки подлинности и позволяют клиентскому приложению получать доступ к биометрической единице с помощью определяемых поставщиком команд управления.</span><span class="sxs-lookup"><span data-stu-id="e8030-106">Private pools support proprietary authentication methods and enable a client application to access a biometric unit by using vendor-specified control commands.</span></span> <span data-ttu-id="e8030-107">Биометрические единицы в частном пуле датчиков:</span><span class="sxs-lookup"><span data-stu-id="e8030-107">Biometric units in a private sensor pool:</span></span>

-   <span data-ttu-id="e8030-108">Доступны только для клиентского приложения, создавшего пул.</span><span class="sxs-lookup"><span data-stu-id="e8030-108">Are available only to the client application that created the pool.</span></span>
-   <span data-ttu-id="e8030-109">Отправляйте уведомления о событиях, формируемые в результате выполнения биометрических операций непосредственно в приложение, не обращаясь к текущему фокусу окна.</span><span class="sxs-lookup"><span data-stu-id="e8030-109">Send event notices generated by the completion of biometric operations directly to the application without regard to current window focus.</span></span>

-   <span data-ttu-id="e8030-111">Совместное использование одной выбранной базы данных шаблона приложения.</span><span class="sxs-lookup"><span data-stu-id="e8030-111">Share a single, application selected template database.</span></span>

-   <span data-ttu-id="e8030-117">Собирает биометрические примеры, а не просто сопоставляет образцы с идентификаторами безопасности.</span><span class="sxs-lookup"><span data-stu-id="e8030-117">Collects biometric samples rather than simply maps samples to SIDs.</span></span>
-   <span data-ttu-id="e8030-118">Переводит оборудование биометрического модуля в режим обслуживания для обновления встроенного по.</span><span class="sxs-lookup"><span data-stu-id="e8030-118">Places the biometric unit hardware into maintenance mode to update the firmware.</span></span>
-   <span data-ttu-id="e8030-119">Отправляет определяемые поставщиком команды управления на оборудование или программное обеспечение биометрического модуля.</span><span class="sxs-lookup"><span data-stu-id="e8030-119">Sends vendor-defined control commands to the biometric unit hardware or software.</span></span>
-   <span data-ttu-id="e8030-120">Пытается настроить биометрический модуль с подключением хранилища для работы в расширенном режиме, но модуль не может выполнить необходимые операции хэширования.</span><span class="sxs-lookup"><span data-stu-id="e8030-120">Attempts to configure a biometric unit with onboard storage to operate in advanced mode but the unit cannot perform the required hashing operations.</span></span>
