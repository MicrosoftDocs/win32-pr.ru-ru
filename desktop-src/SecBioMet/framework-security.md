# <a name="framework-security"></a>Безопасность платформы

биометрическая служба Windows позволяет клиентским приложениям без привилегий выполнять биометрические операции, не предоставляя им прямой доступ к биометрических примерам или оборудованию. Служба создает границу безопасности между клиентами и компонентами биометрической метрики, как показано на следующем рисунке.

![граница безопасности между клиентами и компонентами биометрических компонентов](images/bioframework-security.png)
