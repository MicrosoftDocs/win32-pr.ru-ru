<span data-ttu-id="99af3-105">Пул датчиков системы — это набор биометрических единиц, которые предоставляют доступ к службам проверки подлинности Windows.</span><span class="sxs-lookup"><span data-stu-id="99af3-105">The system sensor pool is a collection of sharable biometric units that provide access to Windows authentication services.</span></span> <span data-ttu-id="99af3-106">Этот пул используется Winlogon, UAC и любым другим клиентом, связывающим идентификатор безопасности с конкретным биометрической шаблоном.</span><span class="sxs-lookup"><span data-stu-id="99af3-106">This pool is used by Winlogon, UAC, and any other client that associates a SID with a specific biometric template.</span></span> <span data-ttu-id="99af3-107">Биометрические единицы в системном пуле:</span><span class="sxs-lookup"><span data-stu-id="99af3-107">Biometric units in the system pool:</span></span>

-   <span data-ttu-id="99af3-108">Может совместно использоваться несколькими клиентскими приложениями.</span><span class="sxs-lookup"><span data-stu-id="99af3-108">Can be shared by multiple client applications.</span></span>
-   <span data-ttu-id="99af3-109">Отправка уведомлений о событиях, созданных при завершении биометрических операций только в приложении, которое имеет фокус текущего окна.</span><span class="sxs-lookup"><span data-stu-id="99af3-109">Send event notices generated by the completion of biometric operations only to the application that has current window focus.</span></span>

-   <span data-ttu-id="99af3-112">Зависит от надежного хранилища шаблонов, предоставляемого биометрической службой Windows.</span><span class="sxs-lookup"><span data-stu-id="99af3-112">Depend on trustworthy template storage provided by the Windows Biometric Service.</span></span>

-   <span data-ttu-id="99af3-115">Настроен для использования в расширенном режиме, но не имеет хранилища шаблонов.</span><span class="sxs-lookup"><span data-stu-id="99af3-115">Configured to operate in advanced mode but has no onboard template storage.</span></span> <span data-ttu-id="99af3-116">То есть он должен использовать адаптер хранилища и хранилище шаблонов, предоставляемое корпорацией Майкрософт.</span><span class="sxs-lookup"><span data-stu-id="99af3-116">That is, it must use the storage adapter and template store supplied by Microsoft.</span></span>
-   <span data-ttu-id="99af3-117">Настроен для использования в расширенном режиме, содержит встроенное хранилище шаблонов и может создавать необходимые хэши.</span><span class="sxs-lookup"><span data-stu-id="99af3-117">Configured to operate in advanced mode, contains onboard template storage, and can generate the required hashes.</span></span>
