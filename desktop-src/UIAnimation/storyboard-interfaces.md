# <a name="storyboard-interfaces"></a>Интерфейсы раскадровки

В этом разделе содержатся справочные спецификации для интерфейсов диспетчера анимации Windows, которые поддерживают раскадровки.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                                 | Описание                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**иуианиматионсторибоард**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Определяет раскадровку, которая содержит группу переходов, синхронизируемых относительно друг друга.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Определяет раскадровку, которая содержит группу переходов, синхронизируемых относительно друг друга.<br/> |
| [**иуианиматионсторибоардевенсандлер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Определяет методы обработки событий состояния и обновления для раскадровки.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Определяет методы для обработки событий раскадровки. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Определяет метод обработки событий итерации цикла раскадровки.<br/>                                            |
| [**иуианиматионприоритикомпарисон**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Определяет метод для сравнения приоритетов, который диспетчер анимации использует для разрешения конфликтов планирования.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Определяет метод, который разрешает конфликты планирования с помощью сравнения приоритетов.<br/>                           |
