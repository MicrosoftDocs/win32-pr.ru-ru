# <a name="iagentcharacterexthink"></a>Иажентчарактерекс:: подумать

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

``` syntax
HRESULT Think(
   BSTR bszText,    // text to think
   long * pdwReqID  // address of a request ID
);
```

Отображает подсказку относительного слова символа с указанным текстом.

-   Возвращает значение S \_ , чтобы указать, что операция выполнена успешно.

<dl> <dt>

<span id="bszText"></span><span id="bsztext"></span><span id="BSZTEXT"></span>*бсзтекст*
</dt> <dd>

Текст, отображаемый в фигурных выносках символа.

</dd> <dt>

<span id="pdwReqID"></span><span id="pdwreqid"></span><span id="PDWREQID"></span>*пдврекид*
</dt> <dd>

Адрес переменной, которая получает идентификатор запроса на **обдумывание** .

</dd> </dl>

Как и метод [**иажентчарактер:: говорите**](iagentcharacter--speak.md) , метод **обработки** является запросом в очереди, который отображает текст в выноске, за исключением того, что мысли отображаются в специальной фигурной подсказке. Всплывающее подсказка поддерживает только тег элемента управления речью в закладке (**\\ МРК**) и игнорирует все другие теги элементов управления речевыми функциями. В отличие от **иажентчарактер:: говорите**, метод **обработки** не изменяет состояние анимации символа.

Параметры [**иажентбаллун**](/windows/desktop/lwef/iagentballoon) также применяются к стилю оформления всплывающего элемента. Например, свойство [**включения**](enabled-property.md) всплывающего окна должно также иметь **значение true** для отображаемого текста.

Автоматическое разбиение слов в подсказке для Microsoft Agent разбивает слова на пробелы (например, пробелы и знаки табуляции). Однако он может нарушить работу слова, чтобы вместить всплывающее сообщение. В таких языках, как японский, китайский и тайский, если пробелы не используются для разбиения слов на слова, вставьте символ нулевой ширины (0x200B) в Юникоде между символами, чтобы определить логические разрывы слов.

> [!Note]  
> Задайте идентификатор языка символа (с помощью [**иажентчарактерекс:: сетлангуажеид**](iagentcharacterex--setlanguageid.md) перед использованием метода [**Иажентчарактер:: говорите**](iagentcharacter--speak.md) , чтобы обеспечить отображение текста в выноске слова.
