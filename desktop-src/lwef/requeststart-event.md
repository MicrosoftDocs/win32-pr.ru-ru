### <a name="remarks"></a><span data-ttu-id="ae136-113">Комментарии</span><span class="sxs-lookup"><span data-stu-id="ae136-113">Remarks</span></span>

<span data-ttu-id="ae136-114">Событие возвращает объект [**запроса**](/windows/desktop/lwef/the-request-object) .</span><span class="sxs-lookup"><span data-stu-id="ae136-114">The event returns a [**Request**](/windows/desktop/lwef/the-request-object) object.</span></span> <span data-ttu-id="ae136-115">Так как запросы обрабатываются асинхронно, можно использовать это событие, чтобы определить, когда сервер начинает обработку запроса (например, метод [**Get**](get-method.md), [**Play**](play-method.md)или [**говорите**](speak-method.md) ), и таким образом синхронизирует его с другими действиями, создаваемыми приложением.</span><span class="sxs-lookup"><span data-stu-id="ae136-115">Because requests are processed asynchronously, you can use this event to determine when the server begins processing a request (such as a [**Get**](get-method.md), [**Play**](play-method.md), or [**Speak**](speak-method.md) method) and thereby synchronize this with other actions generated by your application.</span></span> <span data-ttu-id="ae136-116">Событие отправляется только клиенту, который создал ссылку на объект **запроса** и только в том случае, если вы определили глобальную переменную для ссылки на запрос:</span><span class="sxs-lookup"><span data-stu-id="ae136-116">The event is sent only to the client that created the reference to the **Request** object and only if you defined a global variable for the request reference:</span></span>


```
   Dim MyRequest 
   Dim Genie 

   Sub window_Onload
   
   Agent1.Characters.Load "Genie", _
      "https://agent.microsoft.com/characters/v2/genie/genie.acf"   

   Set Genie = Agent1.Characters("Genie")

   ' This syntax will generate RequestStart and RequestComplete events.
   Set MyRequest = Genie.Get("state", "Showing")

   ' This syntax will not generate RequestStart and RequestComplete events.
   Genie.Get ("state", "Hiding")

   End Sub

   Sub Agent1_RequestStart(ByVal Request)

   If Request = MyRequest Then
      Status = "Loading the Showing animation"

   End Sub
```



<span data-ttu-id="ae136-117">[**Состояние**](status-property.md) возвращает значение 4 (запрос выполняется) для возвращенного объекта [**запроса**](/windows/desktop/lwef/the-request-object) .</span><span class="sxs-lookup"><span data-stu-id="ae136-117">The [**Status**](status-property.md) returns 4 (request in progress) for the [**Request**](/windows/desktop/lwef/the-request-object) object returned.</span></span>

<span data-ttu-id="ae136-118">Поскольку объекты [**запроса**](/windows/desktop/lwef/the-request-object) анимации не назначаются до тех пор, пока сервер не обработает запрос, убедитесь, что объект **запроса** существует, прежде чем пытаться его оценить.</span><span class="sxs-lookup"><span data-stu-id="ae136-118">Because animation [**Request**](/windows/desktop/lwef/the-request-object) objects don't get assigned until the server processes the request, make sure that the **Request** object exists before you attempt to evaluate it.</span></span> <span data-ttu-id="ae136-119">Например, в Visual Basic, если вы используете условную проверку на предмет завершения определенного запроса, можно использовать ключевое слово **Nothing** :</span><span class="sxs-lookup"><span data-stu-id="ae136-119">For example, in Visual Basic, if you use a conditional to test whether a specific request was completed, you can use the **Nothing** keyword:</span></span>

   End Sub
