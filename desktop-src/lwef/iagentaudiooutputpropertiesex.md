# <a name="iagentaudiooutputpropertiesex"></a>иажентаудиуутпутпропертиесекс

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Иажентаудиуутпутпропертиесекс является производным от интерфейса Иажентаудиуутпутпропертиесекс. Он включает все методы Иажентаудиуутпутпропертиесекс и предоставляет доступ к дополнительным функциям.

**Методы в порядке таблицы Vtable**



| Методы Иажентаудиуутпутпропертиесекс                         | Описание                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Возвращает состояние канала вывода звука. |
