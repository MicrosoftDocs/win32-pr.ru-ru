# <a name="the-word-balloon"></a>Всплывающее сообщение

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

В дополнение к речевым выходным данным интерфейс Microsoft Agent поддерживает текстовые субтитры в виде текстовых выходных данных в текстовых выносках со словами-стилями. Слова отображаются во всплывающем окне по мере их обречия. При завершении речевого вывода всплывающее сообщение скрывается. Окно Параметры расширенных символов содержит параметры для отключения отображения всплывающего окна, а также атрибутов элемента управления, связанных с его внешним видом.
