# <a name="mpscancontrol-function"></a>Функция Мпсканконтрол

Разрешает Управление сканированием, которое было асинхронно инициировано через [**мпсканстарт**](mpscanstart.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WINAPI MpScanControl(
  _In_ MPHANDLE  hScanHandle,
  _In_ MPCONTROL ScanControl
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсканхандле* \[ окне\]
</dt> <dd>

Тип: **мфандле**

Обработчик операции асинхронной проверки. Этот маркер возвращается функцией [**мпсканстарт**](mpscanstart.md) . Этот параметр также можно задать в качестве маркера интерфейса диспетчера защиты от вредоносных программ, возвращаемого функцией [**мпманажеропен**](mpmanageropen.md) для управления инициированной системой проверки. в этом случае вызывающая сторона должна иметь права администратора.

</dd> <dt>

*Сканконтрол* \[ окне\]
</dt> <dd>

Тип: **файл mpcontrol**

Задает параметр контроля проверки. Этот параметр должен быть одним из следующих параметров управления:



| Значение                                                                                                                                                                  | Значение                                      |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------|
| <span id="MPCONTROL_ABORT"></span><span id="mpcontrol_abort"></span><dl> <dt>**прервать файл MPCONTROL \_**</dt> </dl>    | Прервать операцию сканирования.<br/>         |
| <span id="MPCONTROL_PAUSE"></span><span id="mpcontrol_pause"></span><dl> <dt>**Приостановка файл MPCONTROL \_**</dt> </dl>    | Приостановка операции сканирования.<br/>         |
| <span id="MPCONTROL_RESUME"></span><span id="mpcontrol_resume"></span><dl> <dt>**возобновление файл MPCONTROL \_**</dt> </dl> | Возобновить приостановленную операцию просмотра.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если функция выполнена успешно, возвращается значение **S \_**.

Если функция завершается ошибкой, возвращаемое значение является неудачным кодом **HRESULT** . Вызывающий объект может использовать функцию [**мперрормессажеформат**](mperrormessageformat.md) для получения общего описания сообщения об ошибке.
