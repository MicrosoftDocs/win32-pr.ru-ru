# <a name="initializesecuritycontext-negotiate-function"></a><span data-ttu-id="f0252-103">Функция InitializeSecurityContext (Negotiate)</span><span class="sxs-lookup"><span data-stu-id="f0252-103">InitializeSecurityContext (Negotiate) function</span></span>

<span data-ttu-id="f0252-104">Функция **InitializeSecurityContext (Negotiate)** инициирует на стороне клиента исходящий [*контекст безопасности*](../secgloss/s-gly.md) из маркера учетных данных.</span><span class="sxs-lookup"><span data-stu-id="f0252-104">The **InitializeSecurityContext (Negotiate)** function initiates the client side, outbound [*security context*](../secgloss/s-gly.md) from a credential handle.</span></span> <span data-ttu-id="f0252-105">Функция используется для создания [*контекста безопасности*](../secgloss/s-gly.md) между клиентским приложением и удаленным одноранговым узлом.</span><span class="sxs-lookup"><span data-stu-id="f0252-105">The function is used to build a [*security context*](../secgloss/s-gly.md) between the client application and a remote peer.</span></span> <span data-ttu-id="f0252-106">**InitializeSecurityContext (Negotiate)** возвращает маркер, который клиент должен передать удаленному одноранговому узлу, который, в свою очередь, отправляется в локальную реализацию безопасности через вызов [**AcceptSecurityContext (Negotiate)**](acceptsecuritycontext--negotiate.md) .</span><span class="sxs-lookup"><span data-stu-id="f0252-106">**InitializeSecurityContext (Negotiate)** returns a token that the client must pass to the remote peer, which the peer in turn submits to the local security implementation through the [**AcceptSecurityContext (Negotiate)**](acceptsecuritycontext--negotiate.md) call.</span></span> <span data-ttu-id="f0252-107">Создаваемый токен должен считаться непрозрачным для всех вызывающих объектов.</span><span class="sxs-lookup"><span data-stu-id="f0252-107">The token generated should be considered opaque by all callers.</span></span>

<span data-ttu-id="f0252-108">Как правило, функция **InitializeSecurityContext (Negotiate)** вызывается в цикле до тех пор, пока не будет установлен достаточный [*контекст безопасности*](../secgloss/s-gly.md) .</span><span class="sxs-lookup"><span data-stu-id="f0252-108">Typically, the **InitializeSecurityContext (Negotiate)** function is called in a loop until a sufficient [*security context*](../secgloss/s-gly.md) is established.</span></span>

```C++
  _In_opt_    PCtxtHandle    phContext,

);
```
