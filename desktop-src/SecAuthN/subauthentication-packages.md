# <a name="subauthentication-packages"></a>Пакеты для проверки подлинности

[*Пакеты проверки подлинности*](../secgloss/a-gly.md) , предоставляемые при настройке поддержки Windows с помощью [*пакетов проверки подлинности*](../secgloss/s-gly.md). Пакет проверки подлинности — это библиотека DLL, которая дополняет или заменяет условия проверки подлинности и проверки, используемые основным пакетом проверки подлинности. Например, конкретный сервер может предоставить пакет для проверки подлинности, который проверяет пароль пользователя с помощью другого алгоритма или определяет ограничения рабочей станции в другом формате.

Дополнительные сведения о пакетах проверки подлинности см. в разделе Пример подсистемы авторизации, установленный вместе с пакетом SDK для платформы.
