# <a name="iscardtypeconv-interface"></a>Интерфейс Искардтипеконв

\[Интерфейс **искардтипеконв** доступен для использования в операционных системах, указанных в разделе требования. Он недоступен для использования в Windows Server 2003 с пакетом обновления 1 (SP1) и более поздней версии, Windows Vista, Windows Server 2008 и последующих версиях операционной системы. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Интерфейс **искардтипеконв** предоставляет методы, необходимые для поддержки пользователей других интерфейсов COM [*смарт-карты*](../secgloss/s-gly.md) . Этот интерфейс предоставляется только для обратной совместимости и больше не должен использоваться.

## <a name="members"></a>Элементы

Интерфейс **искардтипеконв** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Искардтипеконв** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **искардтипеконв** содержит следующие методы.



| Метод                                                                              | Описание                                                                                                                             |
|:------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------|
| [**конвертбитеаррайтобитебуффер**](iscardtypeconv-convertbytearraytobytebuffer.md) | Преобразует стандартный массив байтов C/C++ в универсальный буфер байтов (объект **IStream** ).<br/>                                   |
| [**конвертбитебуффертобитеаррай**](iscardtypeconv-convertbytebuffertobytearray.md) | Преобразует массив байтов, сопоставленный в универсальный буфер байтов (объект **IStream** ), в типичный массив байтов C/C++.<br/>          |
| [**конвертбитебуффертосафеаррай**](iscardtypeconv-convertbytebuffertosafearray.md) | Преобразует массив байтов, сопоставленный в универсальный буфер байтов (объект **IStream** ), в массив SafeArray неподписанного char (Byte).<br/> |
| [**конвертсафеаррайтобитебуффер**](iscardtypeconv-convertsafearraytobytebuffer.md) | Преобразует массив байтов, определенный как SAFEARRAY, в универсальный буфер байтов (объект **IStream** ).<br/>                          |
| [**креатебитеаррай**](iscardtypeconv-createbytearray.md)                           | Создает массив байтов.<br/>                                                                                                   |
| [**креатебитебуффер**](iscardtypeconv-createbytebuffer.md)                         | Создает универсальный буфер байтов, сопоставленный с объектом **IStream** ([**ибитебуффер**](ibytebuffer.md)).<br/>                  |
| [**креатесафеаррай**](iscardtypeconv-createsafearray.md)                           | Создает SAFEARRAY автоматизации для неподписанных символов (в байтах).<br/>                                                              |
| [**фриистреаммемориптр**](iscardtypeconv-freeistreammemoryptr.md)                 | Освобождает указатель памяти на блок памяти ХГЛОБАЛ, управляемый интерфейсом **IStream** com.<br/>                                  |
| [**жетатистреаммемори**](iscardtypeconv-getatistreammemory.md)                     | Получает указатель на байт для блока памяти ХГЛОБАЛ, управляемого интерфейсом **IStream** com.<br/>                        |
| [**сизеофистреам**](iscardtypeconv-sizeofistream.md)                               | Определяет размер (число байтов) интерфейса **IStream** com.<br/>                                                       |
