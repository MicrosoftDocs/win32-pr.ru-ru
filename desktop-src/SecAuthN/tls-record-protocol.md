# <a name="tls-record-protocol"></a>Протокол записи TLS

Протокол [*записи протокола TLS*](../secgloss/t-gly.md) обеспечивает защиту данных приложения с помощью ключей, созданных во время [подтверждения](tls-handshake-protocol.md). Протокол записи отвечает за защиту данных приложения и проверку его [*целостности*](../secgloss/i-gly.md) и происхождения. Он управляет следующими параметрами:

-   Разделение исходящих сообщений на управляемые блоки и сборка входящих сообщений.
-   Сжатие исходящих блоков и распаковка входящих блоков (необязательно).
-   Применение [*код проверки подлинности сообщения*](../secgloss/m-gly.md) (Mac) к исходящим сообщениям и проверка входящих сообщений с помощью Mac.
-   Шифрование исходящих сообщений и расшифровка входящих сообщений.

По завершении протокола записи исходящие зашифрованные данные передаются на уровень протокола TCP для транспорта.
