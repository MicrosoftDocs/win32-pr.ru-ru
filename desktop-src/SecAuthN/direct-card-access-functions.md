# <a name="direct-card-access-functions"></a>Функции прямого доступа к карте

С помощью подсистемы [*смарт-карт*](/windows/desktop/SecGloss/s-gly) можно взаимодействовать с картами, которые могут не соответствовать спецификациям ISO 7816. Для этого можно управлять атрибутами взаимодействия между приложением и картой, предоставляя прямую, низкоуровневые манипуляции с [*модулем чтения*](/windows/desktop/SecGloss/r-gly).

Чтобы использовать эти функции, необходимо предоставить идентификатор для рассматриваемого атрибута. Допустимые идентификаторы и значения атрибутов см. в таблице 3-1 в статье *требования к устройствам с интерфейсом PC-Connected*.



| Раздел                                    | Описание                           |
|------------------------------------------|---------------------------------------|
| [**скардконтрол**](/windows/desktop/api/Winscard/nf-winscard-scardcontrol)     | Обеспечить прямой контроль над модулем чтения. |
| [**скарджетаттриб**](/windows/desktop/api/Winscard/nf-winscard-scardgetattrib) | Получение атрибутов модуля чтения.                |
| [**скардсетаттриб**](/windows/desktop/api/Winscard/nf-winscard-scardsetattrib) | Задание атрибута чтения.                 |
