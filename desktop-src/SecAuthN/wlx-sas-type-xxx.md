# <a name="wlx_sas_type_xxx"></a>\_тип SAS \_ влкс \_ xxx

\[\_ \_ константа типа SAS влкс \_ XXX больше не доступна для использования в Windows Server 2008 и Windows Vista.\]

Значения **\_ типа влкс \_ SAS \_ xxx** определяют конкретный тип SAS.

> [!Note]  
> библиотеки dll GINA не учитываются в Windows Vista.

 

Все значения от нуля до 127 зарезервированы для типов, определенных корпорацией Майкрософт. Значения свыше 127 доступны для определяемых пользователем типов. Следующие типы передаются в функции, имеющие параметр *двсастипе* .



| Константа                                                                                                                                                                                                         | Описание                                                                                                                                              |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WLX_SAS_TYPE_CTRL_ALT_DEL"></span><span id="wlx_sas_type_ctrl_alt_del"></span><dl> <dt>**ВЛКС \_ SAS \_ Type \_ CTRL \_ Alt \_ Del**</dt> </dl>            | Указывает, что пользователь ввел стандартную комбинацию клавиш CTRL + ALT + DEL.<br/>                                                                                     |
| <span id="WLX_SAS_TYPE_SC_INSERT"></span><span id="wlx_sas_type_sc_insert"></span><dl> <dt>**ВЛКС \_ SAS \_ Type \_ SC \_ INSERT**</dt> </dl>                      | Указывает, что [*смарт-карта*](../secgloss/s-gly.md) вставлена в совместимое устройство.<br/> |
| <span id="WLX_SAS_TYPE_SC_REMOVE"></span><span id="wlx_sas_type_sc_remove"></span><dl> <dt>**\_тип SAS \_ влкс \_ SC \_ Remove**</dt> </dl>                      | Указывает, что смарт-карта была удалена из совместимого устройства.<br/>                                                                        |
| <span id="WLX_SAS_TYPE_SCRNSVR_ACTIVITY"></span><span id="wlx_sas_type_scrnsvr_activity"></span><dl> <dt>**\_ \_ \_ действие СКРНСВР типа SAS \_ влкс**</dt> </dl> | Указывает, что при активной заставке с помощью клавиатуры или мыши произошли действия.<br/>                                                    |
| <span id="WLX_SAS_TYPE_SCRNSVR_TIMEOUT"></span><span id="wlx_sas_type_scrnsvr_timeout"></span><dl> <dt>**ВЛКС \_ SAS \_ типа \_ скрнсвр \_ timeout**</dt> </dl>    | Указывает, что активация заставки произошел из-за неактивности клавиатуры и мыши.<br/>                                                     |
| <span id="WLX_SAS_TYPE_TIMEOUT"></span><span id="wlx_sas_type_timeout"></span><dl> <dt>**\_ \_ время ожидания типа SAS влкс \_**</dt> </dl>                             | Указывает, что в течение указанного периода ожидания не было получено ни одного пользовательского ввода.<br/>                                                               |
| <span id="WLX_SAS_TYPE_USER_LOGOFF"></span><span id="wlx_sas_type_user_logoff"></span><dl> <dt>**\_тип SAS \_ влкс \_ \_ Выход пользователя**</dt> </dl>                | Показывает, что пользователь вышел из системы.<br/>                                                                                            |
