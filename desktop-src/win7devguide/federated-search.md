# <a name="federated-search"></a>Федеративный поиск

Windows 7 поддерживает поиск документов за пределами компьютера пользователя. разработчики и специалисты по ит могут включать поисковые системы, репозитории документов, веб-приложения и собственные хранилища данных в Windows 7 без необходимости писать и развертывать клиентский код. это позволяет конечным пользователям выполнять поиск в корпоративной интрасети или в интернете так же легко, как и с помощью привычного интерфейса Windows.

поддержка Windows 7, совместимая *OpenSearch* может быть добавлена разработчиками и специалистами по ит в любое существующее веб-приложение с возможностью поиска путем добавления выходных данных *RSS* или *ATOM* , как определено в общедоступной [OpenSearch](http://www.opensearch.org/) стандарт. файлы описания *OpenSearch* могут быть созданы разработчиками и опытными пользователями для определения параметров подключения, позволяющих настольному клиенту создавать соединитель поиска для службы.

SharePoint сервер поиска также может запрашивать эти совместимые *OpenSearch* службы. это обеспечивает также доступ к службе поиска из веб-интерфейса центра поиска *SharePoint* .
