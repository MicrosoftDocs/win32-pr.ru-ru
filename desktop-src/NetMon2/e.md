# <a name="e-network-monitor"></a>E (сетевой монитор)

<dl> <dt>

<span id="_netmon_erp_gly"></span><span id="_NETMON_ERP_GLY"></span>**РОЗНИЧ**
</dt> <dd>

См. страницу справочника по событиям.

</dd> <dt>

<span id="_netmon_event_gly"></span><span id="_NETMON_EVENT_GLY"></span>**журнале**
</dt> <dd>

Значительное действие, выполняемое в операционной системе или в приложении, например сетевой монитор. Событие может уведомлять пользователей, записывать записи в журнал или выполнять оба действия. В сетевой монитор эксперт может сообщать о событиях, отображаемых в окне средства просмотра событий.

</dd> <dt>

<span id="_netmon_event_reference_page_gly"></span><span id="_NETMON_EVENT_REFERENCE_PAGE_GLY"></span>**страница ссылки на события (ERP)**
</dt> <dd>

HTML-документ, который отображается в окне просмотра событий сетевой монитор и связан с определенным событием эксперта.

</dd> <dt>

<span id="_netmon_expert_gly"></span><span id="_NETMON_EXPERT_GLY"></span>**рубрик**
</dt> <dd>

Библиотека DLL, управляющая кадрами, собранными в файле записи. Эксперт может предоставить сводку или отчет на основе захваченных данных. Например, отчет по верхнему краю пользователя.

</dd> </dl>
