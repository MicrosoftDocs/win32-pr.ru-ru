# <a name="nmeventdata-structure"></a>Структура НМЕВЕНТДАТА

Структура **нмевентдата** содержит сведения о состоянии события, которое передается сетевой монитор для вставки строки в средство просмотра экспертов.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  BYTE         Version;
  DWORD        EventIdent;
  DWORD        Flags;
  DWORD        Severity;
  BYTE         NumColumns;
  LPSTR        szSourceName;
  LPSTR        szEventName;
  LPSTR        szDescription;
  LPSTR        szMachine;
  JTYPE        Justification;
  LPSTR        szUrl;
  SYSTEMTIME   SysTime;
  NMCOLUMNINFO Column[];
} NMEVENTDATA, *PNMEVENTDATA;
```
