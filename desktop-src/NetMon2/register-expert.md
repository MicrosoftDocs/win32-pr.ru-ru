# <a name="register-expert-callback-function"></a>Регистрация функции обратного вызова эксперта

Эксперт должен реализовать функцию **регистрации** экспертов. Сетевой монитор вызывает функцию **регистрации** экспертов для получения сведений о эксперте.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI Register(
  _Inout_ PEXPERTENUMINFO pExpertInfo
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пекспертинфо* \[ в, out\]
</dt> <dd>

Указатель на структуру [**експертенуминфо**](expertenuminfo.md) , которая сетевой монитор выделяется. Эксперт заполняет структуру всеми запрошенными сведениями.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращается значение **true**, а функция возвращает запрошенную информацию.

Если функция завершается неудачно, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Элемент **версии** структуры [**експертенуминфо**](expertenuminfo.md) должен быть равен нулю.
