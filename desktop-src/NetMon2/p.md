# <a name="p-network-monitor"></a>P (сетевой монитор)

<dl> <dt>

<span id="_netmon_packet_gly"></span><span id="_NETMON_PACKET_GLY"></span>**пакетов**
</dt> <dd>

См. раздел [*Frame*](f.md).

</dd> <dt>

<span id="_netmon_parser_gly"></span><span id="_NETMON_PARSER_GLY"></span>**средство синтаксического анализа**
</dt> <dd>

Компонент, который обнаруживает конкретный протокол в отложенной записи. Каждое средство синтаксического анализа может обнаружить один протокол. Однако одна библиотека DLL средства синтаксического анализа может содержать несколько синтаксических анализаторов. Также называется средством синтаксического анализа протокола.

</dd> <dt>

<span id="_netmon_parser.ini_gly"></span><span id="_NETMON_PARSER.INI_GLY"></span>**parser.ini**
</dt> <dd>

Файл инициализации сетевой монитор, содержащий список всех установленных средств синтаксического анализа. Для каждого средства синтаксического анализа существует строка комментария, описывающая средство синтаксического анализа, следующий набор анализатора и файл справки, связанный с анализатором. INI-файл средства синтаксического анализа обновляется, когда сетевой монитор вызывает **парсераутоинсталлинфо** или если DLL-библиотека анализатора вызывает **креатехандоффтабле**.

</dd> <dt>

<span id="_netmon_perfmon_gly"></span><span id="_NETMON_PERFMON_GLY"></span>**Производительности**
</dt> <dd>

Программное средство, позволяющее просматривать переменные, связанные с производительностью сети, которые указывают на активность процесса.

</dd> <dt>

<span id="_netmon_promiscuous_mode_gly"></span><span id="_NETMON_PROMISCUOUS_MODE_GLY"></span>**неизбирательный режим**
</dt> <dd>

Состояние, в котором плата сетевого адаптера копирует все кадры, передаваемые по сети, независимо от адреса назначения в локальном буфере. Этот режим позволяет сетевой монитор записывать и отображать весь сетевой трафик.

</dd> <dt>

<span id="_netmon_property_gly"></span><span id="_NETMON_PROPERTY_GLY"></span>**свойства**
</dt> <dd>

Элемент протокола в средстве синтаксического анализа, описывающий поле данных в кадре, отправляемом с помощью этого протокола.

</dd> <dt>

<span id="_netmon_property_database_gly"></span><span id="_NETMON_PROPERTY_DATABASE_GLY"></span>**база данных свойств**
</dt> <dd>

Внутренняя база данных, определяющая все свойства, поддерживаемые конкретным протоколом. База данных свойств содержит структуру **PROPERTYINFO** для каждого свойства, определяемого анализатором. Сетевой монитор использует сведения в базе данных при присоединении определения свойства к экземпляру свойства, а также при форматировании данных свойства, отображаемых в области сведений пользовательского интерфейса сетевой монитор.

</dd> <dt>

<span id="_netmon_protocol_level_gly"></span><span id="_NETMON_PROTOCOL_LEVEL_GLY"></span>**уровень протокола**
</dt> <dd>

Логическое группирование свойств протокола.

</dd> </dl>
