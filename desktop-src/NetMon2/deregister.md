# <a name="deregister-callback-function"></a>Отменить регистрацию функции обратного вызова

Функция " **отменить регистрацию** экспорта" освобождает ресурсы, используемые для создания [*базы данных свойств*](p.md)протокола. Библиотека DLL средства синтаксического анализа должна реализовывать **отмену регистрации**.

## <a name="syntax"></a>Синтаксис


```C++
VOID Deregister(
  _In_ HPROTOCOL hProtocol
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпротокол* \[ окне\]
</dt> <dd>

Маркер протокола для определенной базы данных.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет.

## <a name="remarks"></a>Remarks

Сетевой монитор вызывает метод **Unregister** после передачи всех сведений о записи в библиотеку DLL средства синтаксического анализа. Библиотека DLL средства синтаксического анализа должна реализовать функцию **дерегистрации** для каждого протокола, поддерживаемого средством синтаксического анализа.

При реализации **отмены регистрации** библиотека DLL средства синтаксического анализа должна вызывать функцию [дестройпропертидатабасе](destroypropertydatabase.md) для освобождения ресурсов [*базы данных свойств*](p.md) .
