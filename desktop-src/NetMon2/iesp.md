# <a name="iesp-interface"></a>Интерфейс ИЕСП

Интерфейс **ИЕСП** предоставляет методы для подключения НПП к сети, записи сетевого трафика в файл записи, получения статистики и отключения НПП от сети. Этот интерфейс используется в основном при необходимости сбора [*статистики сетевого диалога*](c.md) в собственном формате файла ESP.

## <a name="members"></a>Элементы

Интерфейс **ИЕСП** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИЕСП** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ИЕСП** содержит следующие методы.



| Метод                                          | Описание                                                                                                                        |
|:------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Configure**](iesp-configure.md)             | Отправка сведений о конфигурации для записи<br/>                                                                         |
| [**Подключение**](iesp-connect.md)                 | Подключает НПП к сети.<br/>                                                                                        |
| [**Отключение**](iesp-disconnect.md)           | Отключает НПП от сети.<br/>                                                                                   |
| [**жетконтролстате**](iesp-getcontrolstate.md) | Возвращает состояние [*записи*](c.md), которое указывает, запущена или приостановлена запись.<br/> |
| [**Пауза**](iesp-pause.md)                     | Временно останавливает текущую запись.<br/>                                                                                  |
| [**куеристатионс**](iesp-querystations.md)     | Извлекает список всех компьютеров, использующих сетевой монитор для сбора данных в подсети.<br/>                           |
| [**QueryStatus**](iesp-querystatus.md)         | Получает состояние НПП.<br/>                                                                                        |
| [**Возобновить**](iesp-resume.md)                   | Возобновляет приостановленную запись.<br/>                                                                                               |
| [**Начать**](iesp-start.md)                     | Запускает запись и создает файл записи.<br/>                                                                        |
| [**Stop**](iesp-stop.md)                       | Останавливает текущую запись.<br/>                                                                                              |
