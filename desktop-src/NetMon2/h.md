# <a name="h-network-monitor"></a>H (сетевой монитор)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**заданный набор**
</dt> <dd>

Набор протоколов, который следует за протоколом. Средство синтаксического анализа использует набираемый набор протокола, если средство синтаксического анализа может обнаружить следующий протокол из данных в записи. См. также раздел [ *Set*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**Таблица передачи**
</dt> <dd>

Таблица, содержащая набираемый набор средств синтаксического анализа. Можно использовать таблицу передачи для доступа к сведениям о передаче данных, не обращаясь к INI-файлу, определенному средством синтаксического анализа. Сведения о переданном наборе хранятся в INI-файле каждого средства синтаксического анализа. См. также « *переданный набор* »

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**панель шестнадцатеричного представления**
</dt> <dd>

Панель в окне средства просмотра кадров сетевой монитор, показывающая шестнадцатеричные байты, передаваемые в кадрах, которые включены в запись. По умолчанию в нижней части окна отображается панель Шестнадцатеричный.

</dd> </dl>
