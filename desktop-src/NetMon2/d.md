# <a name="d-network-monitor"></a>D (сетевой монитор)

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**Отложенная запись**
</dt> <dd>

Собранный сетевой трафик, хранящийся в файле записи. Поставщик сетевых пакетов (НПП) выполняет отложенную запись. Анализаторы и эксперты используют отложенные записи. См. также [*запись*](c.md)и [*запись в режиме реального времени*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**адрес назначения**
</dt> <dd>

Адрес компьютера, на который отправляются данные.

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**область сведений**
</dt> <dd>

Панель в окне фрейма сетевой монитор, в которой отображаются подробные сведения о свойствах протокола. По умолчанию панель отображается в центре окна.

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**фильтр экрана**
</dt> <dd>

Фильтр, используемый для ограничения объема захваченных данных, отображаемых в окне средства просмотра кадров. Фильтр просмотра является постфиксным стеком объектов, которые могут быть оператором, значением или свойством.

</dd> </dl>
