# <a name="sslgetciphersuiteprfhashalgorithm-function"></a>Функция СслжетЦиферсуитепрфхашалгорисм

Функция **сслжетЦиферсуитепрфхашалгорисм** возвращает идентификатор алгоритма шифрования API: следующего поколения (CNG) [*алгоритма хэширования*](/windows/desktop/SecGloss/h-gly) , который используется для [*функции псевдо-Random*](/windows/desktop/SecGloss/p-gly) (PRF) [*для протокола ввода*](/windows/desktop/SecGloss/t-gly) , набора шифров и типа ключа.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslGetCipherSuitePRFHashAlgorithm(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _In_  DWORD              dwProtocol,
  _In_  DWORD              dwCipherSuite,
  _In_  DWORD              dwKeyType,
  _Out_ WCHAR              szPRFHash[NCRYPT_SSL_MAX_NAME_SIZE],
  _In_  DWORD              dwFlags
);
```

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Маркер экземпляра поставщика протокола [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL).

</dd> <dt>

*двпротокол* \[ окне\]
</dt> <dd>

Одно из значений [**идентификатора протокола поставщика SSL CNG**](https://msdn.microsoft.com/library/Hh971257(v=VS.85).aspx) .

</dd> <dt>

*двЦиферсуите* \[ окне\]
</dt> <dd>

Один из значений [**идентификатора комплекта шифров поставщика CNG SSL**](https://msdn.microsoft.com/library/Hh971253(v=VS.85).aspx) .

</dd> <dt>

*двкэйтипе* \[ окне\]
</dt> <dd>

Одно из значений [**идентификатора типа ключа поставщика SSL CNG**](https://msdn.microsoft.com/library/Hh971256(v=VS.85).aspx) . Для типов ключей, не являющихся [*шифрованием на основе эллиптических кривых*](/windows/desktop/SecGloss/e-gly) (ECC), установите этот параметр в ноль.

## <a name="remarks"></a>Комментарии

Эта функция **сслжетЦиферсуитепрфхашалгорисм** вызывается для запросов TLS 1,2 или более поздних версий для запроса алгоритма хэширования, который будет использоваться в протоколе TLS PRF.
