# <a name="cryptography-api-next-generation"></a>API шифрования: следующее поколение

## <a name="purpose"></a>Назначение

API шифрования: следующее поколение (CNG) — это долгосрочная замена интерфейса [*CryptoAPI*](../secgloss/c-gly.md). CNG разработан для расширения на множестве уровней и шифрования, независимо от поведения.

## <a name="developer-audience"></a>Аудитория разработчиков

CNG предназначен для использования разработчиками приложений, которые позволяют пользователям создавать и обмениваться документами и другими данными в безопасной среде, особенно на незащищенных носителях, таких как Интернет. Разработчики должны быть знакомы с языками программирования C и C++ и средой программирования на основе Windows. Хотя это и не является обязательным, рекомендуется понимать вопросы, связанные с шифрованием или безопасностью.

При разработке поставщика алгоритма шифрования CNG или поставщика хранилища ключей необходимо загрузить [пакет средств разработки поставщика служб шифрования](https://www.microsoft.com/download/details.aspx?id=30688) от корпорации Майкрософт.

## <a name="run-time-requirements"></a>Требования к среде выполнения

CNG поддерживается, начиная с Windows Server 2008 и Windows Vista. Сведения о требованиях времени выполнения для определенного программного элемента см. в разделе "требования" на странице справочника по этому элементу.
