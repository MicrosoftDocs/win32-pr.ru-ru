# <a name="sslencryptpacket-function"></a>Функция Ссленкриптпаккет

Функция **ссленкриптпаккет** шифрует пакет с одним [*протоколом SSL*](/windows/desktop/SecGloss/s-gly) (SSL).

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslEncryptPacket(
  _In_    NCRYPT_PROV_HANDLE hSslProvider,
  _Inout_ NCRYPT_KEY_HANDLE  hKey,
  _In_    PBYTE              *pbInput,
  _In_    DWORD              cbInput,
  _Out_   PBYTE              pbOutput,
  _In_    DWORD              cbOutput,
  _Out_   DWORD              *pcbResult,
  _In_    ULONGLONG          SequenceNumber,
  _In_    DWORD              dwContentType,
  _In_    DWORD              dwFlags
);
```
