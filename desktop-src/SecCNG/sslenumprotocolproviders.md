# <a name="sslenumprotocolproviders-function"></a>Функция Ссленумпротоколпровидерс

Функция **ссленумпротоколпровидерс** возвращает массив [*SSL*](/windows/desktop/SecGloss/s-gly) установленных поставщиков протокола SSL.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslEnumProtocolProviders(
  _Out_ DWORD              *pdwProviderCount,
  _Out_ NCryptProviderName **ppProviderList,
  _In_  DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пдвпровидеркаунт* \[ заполняет\]
</dt> <dd>

Указатель на значение **DWORD** для получения числа возвращаемых поставщиков протоколов.

</dd> <dt>

*пппровидерлист* \[ заполняет\]
</dt> <dd>

Указатель на буфер, который получает массив структур [**нкриптпровидернаме**](/windows/desktop/api/Ncrypt/ns-ncrypt-ncryptprovidername) .

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.



| Возвращаемый код и значение                                                                                                                                                       | Описание                                                                  |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> Не <dt>**превышать \_ Неправильные \_ Флаги**</dt> <dt>0x80090009L</dt> </dl>         | Параметр *dwFlags* не равен нулю.<br/>                              |
| <dl> Не <dt>**превышать \_ НЕТ \_**</dt> <dt>0x8009000EL</dt> памяти </dl>         | Недостаточно памяти для выделения необходимых буферов.<br/>     |
| <dl> Не <dt>**превышать \_ Недопустимый \_ параметр**</dt> <dt>0x80090027L</dt> </dl> | Параметр *пдвпровидеркаунт* или *Пппровидерлист* имеет **значение NULL**.<br/> |



 

## <a name="remarks"></a>Remarks

После завершения использования массива структур [**нкриптпровидернаме**](/windows/desktop/api/Ncrypt/ns-ncrypt-ncryptprovidername) вызовите функцию [**сслфрибуффер**](sslfreebuffer.md) , чтобы освободить массив.
