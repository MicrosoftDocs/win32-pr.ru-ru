# <a name="sslcomputefinishedhash-function"></a>Функция Сслкомпутефинишедхаш

Функция **сслкомпутефинишедхаш** выполняет вычисление [*хэша*](/windows/desktop/SecGloss/h-gly) , отправленного в сообщении SSL о завершении подтверждения [*протокола*](/windows/desktop/SecGloss/s-gly) SSL. Дополнительные сведения о последовательности подтверждения SSL см. [в разделе Описание подтверждения SSL (SSL)](https://support.microsoft.com/kb/257591).

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslComputeFinishedHash(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _In_  NCRYPT_KEY_HANDLE  hMasterKey,
  _In_  NCRYPT_HASH_HANDLE hHandshakeHash,
  _Out_ PBYTE              pbOutput,
  _In_  DWORD              cbOutput,
  _In_  DWORD              dwFlags
);
```

## <a name="remarks"></a>Remarks

Функция **сслкомпутефинишедхаш** является одной из трех функций, используемых для создания хэша, используемого во время подтверждения SSL.

1.  Для получения обработчика хэша вызывается функция [**сслкреатехандшакехаш**](sslcreatehandshakehash.md) .
2.  Функция [**сслхашхандшаке**](sslhashhandshake.md) вызывается любое количество раз с обработчиком хэша для добавления данных в хэш.
3.  Функция **сслкомпутефинишедхаш** вызывается с хэш-маркером для получения дайджеста хэшированных данных.

Хэш-значение вычисляются путем хэширования главного секрета с использованием хэша всех отправленных или полученных сообщений предыдущего подтверждения.

Значение *кбаутпут* определяет длину хэш-данных. Если используется [*протокол tls*](/windows/desktop/SecGloss/t-gly) 1,0, это значение всегда должно быть равно 12 (байт). Дополнительные сведения см. в описании [протокола TLS версии 1,0](https://www.ietf.org/rfc/rfc2246.txt).
