# <a name="default-queue-callback-routine-functions"></a>Функции подпрограммы обратного вызова очереди по умолчанию

Следующие функции используются с подпрограммыми обратного вызова очереди по умолчанию.



| Функция                                                                   | Описание                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**сетупдефаулткуеуекаллбакк**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)             | Подпрограммы обратного вызова очереди по умолчанию.                                                                                                                                                    |
| [**сетупинитдефаулткуеуекаллбакк**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback)     | Инициализирует сведения о контексте, необходимые для подпрограммы обратного вызова очереди по умолчанию.                                                                                                          |
| [**сетупинитдефаулткуеуекаллбаккекс**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex) | Инициализирует сведения о контексте, необходимые для подпрограммы обратного вызова очереди по умолчанию, и указывает альтернативное окно для вывода сообщений о ходе выполнения.                                           |
| [**сетуптермдефаулткуеуекаллбакк**](/windows/desktop/api/Setupapi/nf-setupapi-setuptermdefaultqueuecallback)     | Высвобождает ресурсы, выделенные [**сетупинитдефаулткуеуекаллбакк**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback) и [**сетупинитдефаулткуеуекаллбаккекс**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex). |
