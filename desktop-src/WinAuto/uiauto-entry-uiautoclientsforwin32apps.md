# <a name="ui-automation-clients"></a>Клиенты автоматизации пользовательского интерфейса

В этом разделе описываются интерфейсы и функции для неуправляемых клиентов автоматизации пользовательского интерфейса на основе COM.

> [!Note]  
> Клиентские интерфейсы автоматизации пользовательского интерфейса полностью заменяют функциональные возможности, ранее предоставляемые функциями шаблона элемента управления и функциями узла автоматизации пользовательского интерфейса, которые являются устаревшими и не должны использоваться.
