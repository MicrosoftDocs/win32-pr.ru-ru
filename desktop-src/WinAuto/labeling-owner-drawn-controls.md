# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="1c434-103">Добавление меток к элементам управления Owner-Drawn</span><span class="sxs-lookup"><span data-stu-id="1c434-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="1c434-104">Разработчики приложений должны определить метку, также известную как текст заголовка, для всех элементов управления, рисуемых владельцем, даже если метка не отображается на экране.</span><span class="sxs-lookup"><span data-stu-id="1c434-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="1c434-105">Метка должна располагаться непосредственно перед элементом управления, которому она помечена, и должен находиться в следующем элементе в последовательности табуляции.</span><span class="sxs-lookup"><span data-stu-id="1c434-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="1c434-106">Например, если вы определяете метку при создании рисуемой владельцем кнопки, в которой вместо обычного текста заголовка отображается изображение, клиентские приложения получают метку и используют ее для определения элемента управления.</span><span class="sxs-lookup"><span data-stu-id="1c434-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
