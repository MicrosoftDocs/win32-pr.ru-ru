# <a name="exposing-owner-drawn-combo-box-items"></a><span data-ttu-id="49495-103">Предоставление Owner-Drawn элементов поля со списком</span><span class="sxs-lookup"><span data-stu-id="49495-103">Exposing Owner-Drawn Combo Box Items</span></span>

<span data-ttu-id="49495-104">Разработчикам приложений не нужно реализовывать [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) , чтобы предоставить элементы в поле со списком, рисуемом владельцем, которое имеет стиль **CBS \_ Хасстрингс** , так как Microsoft Active Accessibility предоставляет элементы в полях со списком с этим стилем.</span><span class="sxs-lookup"><span data-stu-id="49495-104">Application developers do not need to implement [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) to expose the items in an owner-drawn combo box that has the style **CBS\_HASSTRINGS** because Microsoft Active Accessibility exposes the items in combo boxes with this style.</span></span> <span data-ttu-id="49495-105">Элементы в поле со списком, рисуемом владельцем, с использованием стиля **CBS \_ хасстрингс** отображаются в виде текста.</span><span class="sxs-lookup"><span data-stu-id="49495-105">The items in an owner-drawn combo box with the **CBS\_HASSTRINGS** style are displayed as text.</span></span> <span data-ttu-id="49495-106">Однако этот стиль также используется с полями со списком, рисуемыми владельцем, которые не отображают текст, чтобы элементы поля со списком предоставлялись корпорацией Майкрософт Active Accessibility.</span><span class="sxs-lookup"><span data-stu-id="49495-106">However, this style is also used with owner-drawn combo boxes that do not display text so that the combo box items are exposed by Microsoft Active Accessibility.</span></span>

<span data-ttu-id="49495-107">Чтобы разрешить Microsoft Active Accessibility предоставлять элементы в поле со списком, рисуемом владельцем, которое не отображает текст:</span><span class="sxs-lookup"><span data-stu-id="49495-107">To allow Microsoft Active Accessibility to expose the items in an owner-drawn combo box that does not display text:</span></span>

-   <span data-ttu-id="49495-108">Используйте стиль **\_ хасстрингс CBS** при создании поля со списком.</span><span class="sxs-lookup"><span data-stu-id="49495-108">Use the **CBS\_HASSTRINGS** style when creating the combo box.</span></span>
-   <span data-ttu-id="49495-109">Создайте текстовый аналог, названный или описывающий каждый элемент в поле со списком.</span><span class="sxs-lookup"><span data-stu-id="49495-109">Create a textual counterpart that names or describes each item in the combo box.</span></span>
-   <span data-ttu-id="49495-110">При добавлении элементов в поле со списком, рисуемое владельцем, используйте \_ сообщение ADDSTRING CB, чтобы добавить текст, который будет предоставляться Microsoft Active Accessibility.</span><span class="sxs-lookup"><span data-stu-id="49495-110">When adding items to the owner-drawn combo box, use the CB\_ADDSTRING message to add the text that you want Microsoft Active Accessibility to expose.</span></span> <span data-ttu-id="49495-111">Этот текст не отображается, поэтому он не должен быть частью рисуемых владельцем данных.</span><span class="sxs-lookup"><span data-stu-id="49495-111">This text is not displayed, so it must not be part of the owner-drawn data.</span></span> <span data-ttu-id="49495-112">Добавление рисуемых владельцем данных элемента с помощью \_ сообщения СЕТИТЕМДАТА CB.</span><span class="sxs-lookup"><span data-stu-id="49495-112">Add the owner-drawn item data using the CB\_SETITEMDATA message.</span></span>

<span data-ttu-id="49495-113">При использовании приведенного выше метода обратите внимание на следующее.</span><span class="sxs-lookup"><span data-stu-id="49495-113">When using the above method, note the following:</span></span>

-   <span data-ttu-id="49495-114">При использовании стиля **\_ сортировки CBS** поле со списком сортируется с использованием переданного строк, а не процедуры обратного вызова [WM \_ компареитем](../controls/wm-compareitem.md) .</span><span class="sxs-lookup"><span data-stu-id="49495-114">If you use the **CBS\_SORT** style, the combo box is sorted using the supplied strings and not the [WM\_COMPAREITEM](../controls/wm-compareitem.md) callback procedure.</span></span>
-   <span data-ttu-id="49495-115">С помощью рисуемых владельцем переменных полей со списком, созданных с помощью **\_ овнердраввариаблеа CBS**, используйте глобальную переменную или другой механизм, чтобы отследить, когда **итемдата** элемент [меасуреитемструкт](/windows/win32/api/winuser/ns-winuser-measureitemstruct) является допустимым.</span><span class="sxs-lookup"><span data-stu-id="49495-115">With owner-drawn variable combo boxes created with the style **CBS\_OWNERDRAWVARIABLE**, use a global variable or some other mechanism to keep track of when the **itemData** member of the [MEASUREITEMSTRUCT](/windows/win32/api/winuser/ns-winuser-measureitemstruct) is valid.</span></span> <span data-ttu-id="49495-116">Глобальная переменная является обязательной, так как система отправляет сообщение [WM \_ меасуреитем](../controls/wm-measureitem.md) сразу после добавления строки, но перед присоединением данных элемента, и на данном этапе элемент **итемдата** является недопустимым.</span><span class="sxs-lookup"><span data-stu-id="49495-116">The global variable is required because the system sends the [WM\_MEASUREITEM](../controls/wm-measureitem.md) message as soon as the string is added but before the item data is attached, and at this point the **itemData** member is not valid.</span></span>
-   <span data-ttu-id="49495-117">Чтобы изменить строку для элемента в поле со списком с помощью стиля **CBS \_ хасстрингс** , удалите элемент с сообщением [ \_ делетестринг CB](../controls/cb-deletestring.md) и добавьте новую строку с сообщением [ \_ ADDSTRING CB](../controls/cb-addstring.md) .</span><span class="sxs-lookup"><span data-stu-id="49495-117">To change the string for an item in a combo box with the **CBS\_HASSTRINGS** style, delete the item with the [CB\_DELETESTRING](../controls/cb-deletestring.md) message and add the new string with the [CB\_ADDSTRING](../controls/cb-addstring.md) message.</span></span>
