# <a name="methodexceptionoccured"></a>месодексцептионоккуред

## <a name="text"></a>Текст

Возникло исключение \\ " {0} \\ "

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Обнаружено непредвиденное исключение или значение HRESULT. Некоторые исключения и значения HRESULT являются допустимыми, но в этом случае результаты не ожидаются. Эта ошибка должна быть исправлена.
