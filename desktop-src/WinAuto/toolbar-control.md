# <a name="toolbar-control-msaa-ui-element-reference"></a>Элемент управления Toolbar (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описывается **Управление объектами панели инструментов** для ссылки на элемент ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание объектов **элементов управления панели инструментов** в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Элемент управления ToolBar содержит кнопки, выполняющие команды меню и обычно содержащиеся в окне под строкой меню.

Имя класса окна для элемента управления ToolBar — ТУЛБАРКЛАССНАМЕ, которое определено как "ToolbarWindow32" в Коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элемент управления ToolBar поддерживает следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :



| Метод                                                                    | Комментарии                                                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | Сама панель инструментов поддерживает метод [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) . Для кнопок на панели инструментов **аккдодефаултактион** вызывает [**сообщение**](/windows/desktop/api/winuser/nf-winuser-postmessagea) i с сообщением [**BM \_ Click**](/windows/desktop/Controls/bm-click) , чтобы нажать указанную кнопку. |
| [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 |                                                                                                                                                                                                                                                                                                                          |
| [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                                                                          |
| [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                                                                          |
| [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                                                                          |
