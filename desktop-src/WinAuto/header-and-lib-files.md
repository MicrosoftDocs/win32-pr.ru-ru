# <a name="header-and-lib-files"></a>Файлы заголовков и lib

Следующие файлы заголовков и библиотек поддерживают разработку приложений Microsoft Active Accessibility.



| Файл         | Описание                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser. h    | Файл заголовка C и C++ для использования элементов управления Windows                                                                                                                             |
| Олеакк. h     | Файлы заголовков C и C++, содержащие определения, необходимые для клиентов и серверов Microsoft Active Accessibility                                                                    |
| Winver. h     | C и заголовочный файл C++ для получения сведений о версии                                                                                                                                |
| Олеакк. lib   | Импорт библиотеки для Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Библиотека динамической компоновки, которая предоставляет время выполнения Microsoft Active Accessibility, управляет запросами от клиентов Microsoft Active Accessibility и общих элементов пользовательского интерфейса с учетными записями-посредниками. |
| Oleaccrc.dll | Многоязыковая библиотека динамических ссылок на ресурсы, используемая Oleacc.dll                                                                                                                |
