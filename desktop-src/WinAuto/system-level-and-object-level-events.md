# <a name="system-level-and-object-level-events"></a><span data-ttu-id="e174c-103">События System-Level и Object-Level</span><span class="sxs-lookup"><span data-stu-id="e174c-103">System-Level and Object-Level Events</span></span>

<span data-ttu-id="e174c-104">Microsoft Active Accessibility использует три класса Виневентс: *уровень системы*, *уровень объектов* и *консоль*.</span><span class="sxs-lookup"><span data-stu-id="e174c-104">Microsoft Active Accessibility uses three classes of WinEvents: *system level*, *object level*, and *console*.</span></span> <span data-ttu-id="e174c-105">Каждый из них имеет одно из следующих значений [константы события](event-constants.md) :</span><span class="sxs-lookup"><span data-stu-id="e174c-105">Each has one of the following corresponding [event constant](event-constants.md) values:</span></span>

-   <span data-ttu-id="e174c-106">Константы событий, начинающиеся с системы событий, \_ обозначают события системного уровня.</span><span class="sxs-lookup"><span data-stu-id="e174c-106">Event constants that begin with EVENT\_SYSTEM identify system-level events.</span></span> <span data-ttu-id="e174c-107">Эти события описывают ситуации, затрагивающие все приложения в системе.</span><span class="sxs-lookup"><span data-stu-id="e174c-107">These events describe situations affecting all applications in the system.</span></span>
-   <span data-ttu-id="e174c-108">Константы событий, начинающиеся с объекта события, \_ обозначают события уровня объекта.</span><span class="sxs-lookup"><span data-stu-id="e174c-108">Event constants that begin with EVENT\_OBJECT identify object-level events.</span></span> <span data-ttu-id="e174c-109">Эти события относятся к ситуациям, относящимся к объектам в рамках одного приложения.</span><span class="sxs-lookup"><span data-stu-id="e174c-109">These events pertain to situations specific to objects within one application.</span></span>
-   <span data-ttu-id="e174c-110">Константы событий, которые начинаются с консоли событий, \_ обозначают события уровня консоли.</span><span class="sxs-lookup"><span data-stu-id="e174c-110">Event constants that begin with EVENT\_CONSOLE identify console-level events.</span></span> <span data-ttu-id="e174c-111">Эти события указывают на изменения в окнах консоли.</span><span class="sxs-lookup"><span data-stu-id="e174c-111">These events indicate changes in console windows.</span></span>

<span data-ttu-id="e174c-112">Как системные, так и классы уровня объектов событий создаются операционной системой и серверными приложениями.</span><span class="sxs-lookup"><span data-stu-id="e174c-112">Both the system- and object-level classes of events are generated by the operating system and server applications.</span></span> <span data-ttu-id="e174c-113">Операционная система создает события уровня системы и объекта для следующих сценариев:</span><span class="sxs-lookup"><span data-stu-id="e174c-113">The operating system generates system-level and object-level events for the following scenarios:</span></span>

-   <span data-ttu-id="e174c-114">Уведомления на уровне системы об изменениях фокуса</span><span class="sxs-lookup"><span data-stu-id="e174c-114">System-wide notifications about focus changes</span></span>
-   <span data-ttu-id="e174c-115">Изменения активации</span><span class="sxs-lookup"><span data-stu-id="e174c-115">Activation changes</span></span>
-   <span data-ttu-id="e174c-116">События, касающиеся предоставляемых системой объектов, таких как общие элементы управления</span><span class="sxs-lookup"><span data-stu-id="e174c-116">Events regarding system-provided objects, such as common controls</span></span>

<span data-ttu-id="e174c-117">Серверные приложения создают события системного уровня для пользовательских объектов, которые реплицируют системные объекты, такие как настраиваемые меню и полосы прокрутки.</span><span class="sxs-lookup"><span data-stu-id="e174c-117">Server applications generate system-level events for custom objects that replicate system objects, such as custom menus and scroll bars.</span></span>

<span data-ttu-id="e174c-118">Серверные приложения обычно создают события на уровне объектов для внесения изменений в доступные объекты, такие как создание, уничтожение и выбор объектов.</span><span class="sxs-lookup"><span data-stu-id="e174c-118">Server applications typically generate object-level events for changes to the accessible objects they contain, such as object creation, destruction, and selection.</span></span>

<span data-ttu-id="e174c-119">Несмотря на то, что система создает события уровня объекта для объектов [**Window**](window.md) , серверы также должны отправить события уровня объекта для каждого доступного объекта, содержащегося в окне.</span><span class="sxs-lookup"><span data-stu-id="e174c-119">Although the system generates object-level events for [**window**](window.md) objects, servers must also send object-level events for every accessible object contained in a window.</span></span> <span data-ttu-id="e174c-120">Например, если серверное приложение регистрирует определяемый приложением класс окна для создания пользовательского элемента управления, система создает события уровня объекта для окна, содержащего пользовательский элемент управления. сервер создает события уровня объекта для доступного объекта, который предоставляет сведения об элементе управления.</span><span class="sxs-lookup"><span data-stu-id="e174c-120">For example, if a server application registers an application-defined window class to create a custom control, the system generates object-level events for the window that contains the custom control; the server generates object-level events for the accessible object that provides information about the control.</span></span>

<span data-ttu-id="e174c-121">Серверы создают только события уровня объекта для пользовательских элементов управления, для которых они реализуют интерфейс [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .</span><span class="sxs-lookup"><span data-stu-id="e174c-121">Servers only generate object-level events for the custom controls for which they implement the [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) interface.</span></span> <span data-ttu-id="e174c-122">Дополнительные сведения см. в разделе [пользовательские элементы пользовательского интерфейса](custom-user-interface-elements.md).</span><span class="sxs-lookup"><span data-stu-id="e174c-122">For more information, see [Custom User Interface Elements](custom-user-interface-elements.md).</span></span>
