# <a name="supported-platforms"></a>Поддерживаемые платформы

microsoft Active Accessibility поддерживает большинство платформ майкрософт и входит в состав Windows операционной системы, начиная с Windows XP. Если вы не знакомы с используемой версией Microsoft Active Accessibility, Узнайте, [какая версия Active Accessibility установлена в настоящее время?](which-version-of-active-accessibility-is-currently-installed.md)

Если у вас нет последней версии Microsoft Active Accessibility, используйте для обновления [набор распространяемых пакетов microsoft Active Accessibility 2,0 (RDK)](https://www.microsoft.com/downloads/details.aspx?familyid=9b14f6e1-888a-4f1d-b1a1-da08ee4077df&displaylang=en) . Этот установщик содержит все основные системные файлы, необходимые для внедрения технологии Microsoft Active Accessibility в клиентские и серверные приложения. он поддерживает Windows 98, Windows NT 4,0 с пакетом обновления 6 (SP6) и более поздних версий, Windows 2000, Windows Millennium Edition (Windows Me), Windows XP и Windows Server 2003.

Чтобы установить Microsoft Active Accessibility RDK, обратитесь в [Центр разработчиков специальных возможностей Майкрософт на сайте MSDN](/windows/apps/accessibility).
