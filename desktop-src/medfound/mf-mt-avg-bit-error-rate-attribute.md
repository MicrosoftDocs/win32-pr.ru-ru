# <a name="mf_mt_avg_bit_error_rate-attribute"></a>\_ \_ \_ Атрибут частоты среднего разряда \_ ошибок \_ MF MT

Частота ошибок данных в битах в секунду для типа видеоклипа.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Этот атрибут соответствует элементу **двбитерроррате** в структурах [**видеоинфохеадер**](/previous-versions/windows/desktop/api/amvideo/ns-amvideo-videoinfoheader) и [**VIDEOINFOHEADER2**](/previous-versions/windows/desktop/api/dvdmedia/ns-dvdmedia-videoinfoheader2) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
