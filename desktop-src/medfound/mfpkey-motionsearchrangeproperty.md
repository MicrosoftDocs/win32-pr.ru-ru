# <a name="mfpkey_motionsearchrange-property"></a>МФПКЭЙ \_ мотионсеарчранже, свойство

Указывает диапазон, используемый при поиске движения.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкмотионсеарчранже

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Для этого свойства можно задать одно из следующих значений. H обозначает горизонтальный диапазон, а V означает вертикальный диапазон. Все диапазоны задаются в пикселях.



| Значение | Диапазон                                |
|-------|--------------------------------------|
| 0     | + 63.75/-64.0» H, + 31.75/-32.0 V       |
| 1     | +127.75/-128.0 H, +63.75/-64.0 V     |
| 2     | + 511.75/-512.0 H, + 127.75/-128.0 V   |
| 3     | + 1023.75/-1024.0 H, + 255.75/-256.0 V |
| -1    | Макроблокк — адаптивный (автоматический)      |



 

Это свойство управляет размером области кадров, которую кодек будет искать элемент, который может иметь продвижение между кадрами. Более крупное окно поиска поможет быстро найти более быстрое движение, но потребует больше времени ЦП. Требования к ЦП задаются примерно вдвое на каждом уровне.

В автоматическом режиме (-1) будет динамически выбран наиболее эффективный режим.
