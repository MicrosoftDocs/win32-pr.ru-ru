# <a name="mfpkey_wmaaecma_featr_noise_fill-property"></a>МФПКЭЙ \_ вмааекма \_ Феатр \_ , \_ свойство заливки шума

Указывает, выполняется ли заполнение сигнала DSP для записи речи.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="default-value"></a>Значение по умолчанию

ВАРИАНТ \_ true

## <a name="applies-to"></a>Применение

-   [DSP для записи речи](voicecapturedmo.md)

## <a name="remarks"></a>Комментарии

При заполнении шума в части сигнала добавляется небольшой шум, при котором обрезка по центру удалила остаточные выводы. Это приводит к повышению удобства работы пользователя, чем при отсутствии пропусков в сигнале.

Это свойство может иметь следующие значения.



| Значение          | Описание            |
|----------------|------------------------|
| ВАРИАНТ \_ true  | Включить заполнение шума.  |
| ВАРИАНТ \_ false | Отключить заполнение шума. |



 

Значением этого свойства по умолчанию является вариант \_ true (включено). Перед установкой этого свойства необходимо присвоить свойству [ \_ \_ \_ режима компонента мфпкэй вмааекма](mfpkey-wmaaecma-feature-modeproperty.md) значение Variant \_ true.

DSP использует это свойство только в том случае, если включена обработка AEC.
