# <a name="mfpkey_motionsearchlevel-property"></a>МФПКЭЙ \_ мотионсеарчлевел, свойство

Указывает, как сведения о цвете используются в операциях поиска движения.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкмотионсеарчлевел

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Для этого свойства можно задать одно из следующих значений.



| Значение | Сведения об используемом видео                           |
|-------|--------------------------------------------------|
| 0     | Только яркость.                                       |
| 1     | Яркости с ближайшим целым числом чрома.                |
| 2     | Яркость с истинной цветностью.                           |
| -1    | Макроблокк — адаптивный с true чрома.            |
| -2    | Макроблокк — адаптивный с ближайшим целым числом чрома. |



 

По умолчанию кодек выполняет поиск движения только в канале яркости. Включение информации чрома в оценку движения может значительно повысить качество закодированного видео. Поиск движения с помощью яркости и true чрома обеспечит лучшее качество видео, но с наибольшими затратами на производительность. Два динамических режима (-1 и-2) и яркости с ближайшим целочисленным режимом чрома обеспечивают разумные компромиссы между качеством и производительностью.
