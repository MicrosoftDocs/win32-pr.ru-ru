# <a name="mf_mt_aac_payload_type-attribute"></a>\_ \_ \_ Атрибут типа полезных данных MF MT AAC \_

Указывает тип полезных данных для потока расширенного кодирования звука (AAC).

## <a name="data-type"></a>Тип данных

**UINT32**

Возможны следующие значения.



| Значение                                                                        | Значение                                                                                                                           |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Поток содержит только элементы необработанных \_ \_ блоков данных.<br/>                                                                    |
| <dl> <dt>1</dt> </dl> | Поток транспорта аудиоданных (ADTS). Поток содержит \_ последовательность ADTs, определенную MPEG-2.<br/>                       |
| <dl> <dt>2</dt> </dl> | Формат обмена звуковых данных (АДИФ). Поток содержит \_ последовательность Адиф, определенную MPEG-2.<br/>                     |
| <dl> <dt>3</dt> </dl> | Поток содержит потоковый транспорт MPEG-4 с уровнем синхронизации (УРОВНЯМИ гарантии) и уровнем мультиплексирования (ЛАТМ).<br/> |



 

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Применение

[**имфмедиатипе**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmediatype)

## <a name="remarks"></a>Комментарии

\_ \_ \_ Тип полезных данных AAC MF MT \_ не является обязательным. Если этот атрибут не указан, используется значение по умолчанию 0, указывающее, что поток содержит только элементы необработанных \_ \_ блоков данных.

Применяется только к **мфаудиоформат \_ AAC**.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
