# <a name="mfp_pkey_streamrenderingresults-property"></a>МФП \_ PKEY \_ стреамрендерингресултс, свойство

> [!IMPORTANT]
> Не рекомендуется. Этот API может быть удален из будущих выпусков Windows. Приложения должны использовать [сеанс мультимедиа](media-session.md) для воспроизведения.

 

Указывает, какие потоки были успешно подключены к приемнику мультимедиа.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Массив значений **типа DWORD** (**Каул**)

VT \_ вектор \| VT \_ UI4

**каул**



## <a name="remarks"></a>Комментарии

Это свойство может быть отправлено с событием **МФП \_ \_ типа события \_ MEDIAITEM \_ Set** .

Значением свойства является массив **HRESULT** s. Записи массива соответствуют потокам на текущем элементе мультимедиа. Каждая запись в массиве указывает, был ли соответствующий поток подключен к приемнику мультимедиа следующим образом:

-   Если поток подключен к приемнику мультимедиа, значение равно **S \_**.
-   Если поток не выбран, значение равно **\_ false**.
-   Если при попытке подключения к потоку произошла ошибка, значением будет код ошибки.

Если по крайней мере один поток был успешно подключен, воспроизведение возможно. Например, у пользователя может быть кодек, необходимый для воспроизведения аудиопотока, но не воспроизведение видеопотока.
