# <a name="lan_profile-schema-elements"></a>\_Элементы схемы профиля локальной сети

Профиль проводной сети содержит следующие элементы схемы. Все именованные элементы находятся в пространстве имен `https://www.microsoft.com/networking/LAN/profile/v1` .

В следующем списке показаны определенные элементы в порядке, в котором элементы отображаются в профиле. Упорядочение элементов выполняется принудительно. В этом списке не отображаются все возможные элементы, которые могут отображаться в профиле, так как элементы можно добавлять в **xs: любые** точки вставки.

-   [**ланпрофиле**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (Ланпрофиле)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**безопасность (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**Онексенфорцед (безопасность)**](lan-profileschema-onexenforced-security-element.md)
            -   [**Онексенаблед (безопасность)**](lan-profileschema-onexenabled-security-element.md)
