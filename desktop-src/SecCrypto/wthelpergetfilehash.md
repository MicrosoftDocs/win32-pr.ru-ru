# <a name="wthelpergetfilehash-function"></a>Функция Вселпержетфилехаш

\[Функция **вселпержетфилехаш** доступна для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен.\]

Функция **вселпержетфилехаш** проверяет подпись подписанного файла и получает хэш-значение и идентификатор алгоритма для файла.

> [!Note]  
> Эта функция не объявлена в опубликованном файле заголовка. Чтобы использовать эту функцию, объявите ее в точно указанном формате. Эта функция также не имеет связанной библиотеки импорта. Для динамической привязки к Wintrust.dll необходимо использовать функции [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) и [**GetProcAddress**](/windows/win32/api/libloaderapi/nf-libloaderapi-getprocaddress) .

 

## <a name="syntax"></a>Синтаксис


```C++
LONG WINAPI WTHelperGetFileHash(
  _In_        LPCWSTR pwszFilename,
  _In_        DWORD   dwFlags,
  _Inout_opt_ PVOID   pvReserved,
  _Out_opt_   BYTE    *pbFileHash,
  _Inout_opt_ DWORD   *pcbFileHash,
  _Out_opt_   ALG_ID  *pHashAlgid
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвсзфиленаме* \[ окне\]
</dt> <dd>

Указатель на строку в Юникоде, заканчивающуюся нулем и содержащую путь и имя файла, для которого нужно получить хэш.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр не используется и должен быть равен нулю.

</dd> <dt>

*пвресервед* \[ в, out, необязательно\]
</dt> <dd>

Этот параметр не используется и должен иметь **значение NULL**.

</dd> <dt>

*пбфилехаш* \[ out, необязательно\]
</dt> <dd>

Указатель на буфер для получения хэш-значения для файла. Параметр *пкбфилехаш* содержит размер этого буфера.

</dd> <dt>

*пкбфилехаш* \[ в, out, необязательно\]
</dt> <dd>

Указатель на переменную **типа DWORD** , которая во входных данных содержит размер (в байтах) буфера *пбфилехаш* , а выходные данные — размер (в байтах) хэш-значения.

Чтобы получить необходимый размер хэш-значения, передайте **значение NULL** для параметра *пбфилехаш* . Эта функция поместит требуемый размер (в байтах) хэш-значения в этом расположении.

Если параметр *пбфилехаш* имеет значение, отличное от **null**, и размер не достаточен для получения хэш-значения, эта функция поместит требуемый размер (в байтах) в этом расположении и возвратит **\_ Дополнительные \_ данные об ошибке**.

</dd> <dt>

*фашалгид* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную [**\_ идентификатора ALG**](alg-id.md) для получения идентификатора алгоритма, используемого для создания хэш-значения. Этот параметр может иметь **значение NULL** , если эти сведения не требуются.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает код состояния, который указывает на успешное или неуспешное завершение функции.

Возможные коды возврата включают, но не ограничиваются следующим.



| Код возврата                                                                                               | Описание                                                                                                                                           |
|-----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**Ошибка при \_ успешном выполнении**</dt> </dl>             | Файл подписан, и подпись проверена.<br/>                                                                                        |
| <dl> <dt>**Ошибка \_ дополнительных \_ данных**</dt> </dl>          | Параметр *пбфилехаш* не равен **null**, а размер, указанный в параметре *пкбфилехаш* , недостаточно велик для получения хэша.<br/> |
| <dl> <dt>**Ошибка \_ не \_ хватает \_ памяти**</dt> </dl> | Ошибка при выделении памяти.<br/>                                                                                                      |
| <dl> <dt>**ДОВЕРЯТЬ \_ E с \_ неверным \_ дайджестом**</dt> </dl>      | Подпись файла не проверена.<br/>                                                                                                |
| <dl> <dt>**ДОВЕРЯТЬ \_ электронной \_ подписи**</dt> </dl>      | Файл не подписан или имеет недопустимую подпись.<br/>                                                                              |
