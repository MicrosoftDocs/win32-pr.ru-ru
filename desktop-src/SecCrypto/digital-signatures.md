-   <span data-ttu-id="f4d6f-127">Длина хэш-значения определяется типом используемого алгоритма и его длина не зависит от размера сообщения.</span><span class="sxs-lookup"><span data-stu-id="f4d6f-127">The length of the hash value is determined by the type of algorithm used, and its length does not vary with the size of the message.</span></span> <span data-ttu-id="f4d6f-128">Наиболее распространенная длина хэш-значения — 128 или 160 бит.</span><span class="sxs-lookup"><span data-stu-id="f4d6f-128">The most common hash value lengths are either 128 or 160 bits.</span></span>
