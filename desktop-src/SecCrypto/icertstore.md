# <a name="icertstore-interface"></a>Интерфейс Ицертсторе

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP.\]

Интерфейс **ицертсторе** предоставляет доступ к контексту объекта CAPICOM- [**хранилища**](store.md) . Этот контекст позволяет использовать хранилище CAPICOM Certificate в других производных классах CryptoAPI.

## <a name="when-to-use"></a>Назначение

Используйте этот интерфейс, если необходимо использовать объект CAPICOM- [**Store**](store.md) в другом производном от CryptoAPI.

## <a name="members"></a>Элементы

Интерфейс **ицертсторе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ицертсторе** содержит следующие методы.



| Метод                                        | Описание                                                                                                         |
|:----------------------------------------------|:--------------------------------------------------------------------------------------------------------------------|
| [**CloseHandle**](icertstore-closehandle.md) | Закрывает обработчик ХЦЕРТСТОРЕ, полученный через свойство [**сторехандле**](icertstore-storehandle.md) .<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ицертсторе** имеет следующие свойства.



| Свойство                                                     | Тип доступа           | Описание                                                                                                         |
|:-------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------|
| [**сторехандле**](icertstore-storehandle.md)<br/>     | Чтение/запись<br/> | Задает или получает маркер ХЦЕРТСТОРЕ хранилища сертификатов.<br/>                                          |
| [**StoreLocation**](icertstore-storelocation.md)<br/> | Чтение/запись<br/> | Задает или получает [**\_ \_ Расположение CAPICOM-хранилища**](capicom-store-location.md) для хранилища сертификатов.<br/> |
