# <a name="chain-object"></a>Объект цепочки

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP. Вместо этого используйте [**класс X509Chain**](/dotnet/api/system.security.cryptography.x509certificates.x509chain?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

Объект **цепочки** представляет цепочку доверия сертификатов.

Этот объект предоставляет свойства и методы для создания цепочки доверия сертификатов для проверки действительности сертификатов. Цепочка строится с использованием значения свойства [**цертификатестатус. чеккфлаг**](certificatestatus-checkflag.md) и параметров политики объекта [**цертификатестатус**](certificatestatus.md) .

Объект **цепочки** предоставляет следующие интерфейсы:

-   **IChain2**: представлено в CAPICOM 2,0.
-   **Ичаин**: представлено в CAPICOM 1,0.

## <a name="when-to-use"></a>Назначение

Объект **цепочки** используется для выполнения следующих задач:

-   Создайте цепочку доверия сертификатов.
-   Получите идентификаторы объектов для всех политик сертификатов и приложений, допустимых для цепочки.
-   Проверьте состояние сертификатов в цепочке.
-   Получение расширенных сведений об ошибках.
-   Получение коллекции сертификатов в цепочке.
