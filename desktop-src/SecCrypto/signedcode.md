Объект **сигнедкоде** предоставляет функциональные возможности для подписывания исполняемых файлов с помощью цифровой подписи Authenticode.

## <a name="when-to-use"></a>Назначение

Объект **сигнедкоде** используется для выполнения следующих задач:

-   Подписывать исполняемые файлы.
-   Исполняемые файлы меток времени.
-   Определить, является ли подпись исполняемого файла допустимой.
-   Задайте или получите путь к исполняемому файлу.
-   Получите подписывающий и Time Стампер исполняемого файла.
-   Получение коллекции сертификатов для исполняемого файла.
-   Получите описание или URL-адрес для описания исполняемого файла.

## <a name="remarks"></a>Remarks

Объект **сигнедкоде** может быть создан и незащищен для создания скриптов. ProgID для объекта **сигнедкоде** — CAPICOM. Сигнедкоде. 1.

Исполняемый файл должен иметь тип, который может быть подписан с помощью технологии Authenticode, например файлы с расширением .cab, Cat, .exe, .dll, vbs или OCX.
