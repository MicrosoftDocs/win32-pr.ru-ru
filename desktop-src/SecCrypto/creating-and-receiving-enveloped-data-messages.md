# <a name="creating-and-receiving-enveloped-data-messages"></a>Создание и получение сообщений с запечатанными данными

Сообщение, упакованное в оболочку, является сообщением, зашифрованным для набора получателей. В процессе енвелопмент создается ключ шифрования сеанса, и сообщение шифруется с помощью этого ключа. Затем сам ключ шифрования шифруется отдельно для каждого получателя с помощью открытых ключей из каждого назначенного сертификата получателя. Сообщение с оболочкой состоит из зашифрованного сообщения, сертификатов предполагаемых получателей и набора зашифрованных ключей, по одному для каждого получателя. Созданное сообщение находится в формате [*PKCS \# 7*](../secgloss/p-gly.md)/КМС.
