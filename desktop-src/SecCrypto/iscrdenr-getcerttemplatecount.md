# <a name="iscrdenrgetcerttemplatecount-method"></a>Метод Искрденр:: Жетцерттемплатекаунт

Метод **жетцерттемплатекаунт** извлекает количество шаблонов сертификатов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT getCertTemplateCount(
  [in]  DWORD     dwFlags,
  [out] LONG *pdwCertTemplateCount
);
```


```VB

SCrdEnr.getCertTemplateCount( _
  ByVal dwFlags, _
  ByRef pdwCertTemplateCount _
)
```





## <a name="parameters"></a>Параметры

<dl> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Значение, определяющее, предназначен ли шаблон для сертификатов пользователя или компьютера. Если это значение равно бросить \_ зарегистрировать \_ \_ шаблон сертификата пользователя \_ (определяется как 1), то возвращаемое количество будет использоваться для шаблонов сертификатов пользователей. Если это значение равно бросить \_ зарегистрировать \_ \_ шаблон сертификата компьютера \_ (определяется как 2), то возвращаемое количество будет использоваться для шаблонов сертификатов компьютера.

</dd> <dt>

*пдвцерттемплатекаунт* \[ заполняет\]
</dt> <dd>

Указатель на значение **типа Long** , возвращающее число шаблонов сертификатов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

### <a name="c"></a>C++

Если метод завершается успешно, метод возвращает значение S \_ ОК.

Если метод завершается с ошибкой, возвращается значение **HRESULT** , указывающее на ошибку. Список распространенных кодов ошибок см. в разделе [Общие значения HRESULT](common-hresult-values.md).

### <a name="vb"></a>VB

Значение **типа Long** , представляющее количество шаблонов сертификатов.
