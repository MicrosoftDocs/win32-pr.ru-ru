# <a name="oidsclear-method"></a>Метод OID. Clear

\[Метод **clear** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс оидколлектион**](/dotnet/api/system.security.cryptography.oidcollection?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Метод **clear** удаляет все объекты [**OID**](oid.md) из коллекции.

## <a name="syntax"></a>Синтаксис


```VB
OIDs.Clear()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
