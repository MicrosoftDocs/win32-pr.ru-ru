# <a name="ichaincontext-interface"></a>Интерфейс Ичаинконтекст

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP.\]

Интерфейс **ичаинконтекст** предоставляет доступ к контексту объекта CAPICOM [**chain**](chain.md) . Этот контекст позволяет использовать цепочку доверия CAPICOM Certificate в других производных классах CryptoAPI.

## <a name="when-to-use"></a>Назначение

Этот интерфейс следует использовать, если необходимо использовать объект- [**цепочку**](chain.md) CAPICOM в другом производном интерфейсе CryptoAPI.

## <a name="members"></a>Элементы

Интерфейс **ичаинконтекст** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ичаинконтекст** содержит следующие методы.



| Метод                                           | Описание                                                                                                                    |
|:-------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------|
| [**фриконтекст**](ichaincontext-freecontext.md) | Освобождает \_ контекст цепочки пкцерт, \_ полученный через свойство [**чаинконтекст**](ichaincontext-chaincontext.md) .<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ичаинконтекст** имеет следующие свойства.



| Свойство                                                      | Тип доступа           | Описание                                                               |
|:--------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------|
| [**чаинконтекст**](ichaincontext-chaincontext.md)<br/> | Чтение/запись<br/> | Задает или получает \_ контекст цепочки пкцерт \_ сертификата.<br/> |
