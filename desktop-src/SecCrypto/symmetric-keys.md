# <a name="symmetric-keys"></a>симметричные ключи;

[*Симметричный ключ*](../secgloss/s-gly.md) — это один ключ, который используется с операциями со схемой [*симметричного шифрования*](../secgloss/s-gly.md) . Например, такой ключ может использоваться в алгоритме симметричного шифрования как для шифрования, так и для расшифровки.

Наиболее известные симметричные алгоритмы работают быстрее, чем наиболее известные [*алгоритмы открытого ключа*](../secgloss/p-gly.md). Поэтому они являются предпочтительными при обработке больших объемов данных. Некоторые из наиболее распространенных симметричных алгоритмов — [*RC4*](../secgloss/r-gly.md), Triple DES ([*стандарт шифрования данных*](../secgloss/d-gly.md)) и [*AES*](../secgloss/a-gly.md) (AES).

Так как большая часть действий, включающих симметричные ключи, относится к хранению секретов, важно, чтобы количество людей, имеющих определенный симметричный ключ, было как можно меньше. Рекомендуется, чтобы только один или два человека обладали одним симметричным ключом.
