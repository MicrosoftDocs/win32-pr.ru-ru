# <a name="symmetric-keys"></a><span data-ttu-id="ad9a8-104">симметричные ключи;</span><span class="sxs-lookup"><span data-stu-id="ad9a8-104">Symmetric Keys</span></span>

<span data-ttu-id="ad9a8-105">[*Симметричный ключ*](../secgloss/s-gly.md) — это один ключ, который используется с операциями со схемой [*симметричного шифрования*](../secgloss/s-gly.md) .</span><span class="sxs-lookup"><span data-stu-id="ad9a8-105">A [*Symmetric key*](../secgloss/s-gly.md) is a single key that is used with the operations of a [*symmetric encryption*](../secgloss/s-gly.md) scheme.</span></span> <span data-ttu-id="ad9a8-106">Например, такой ключ может использоваться в алгоритме симметричного шифрования как для шифрования, так и для расшифровки.</span><span class="sxs-lookup"><span data-stu-id="ad9a8-106">For example, such a key may be used in a symmetric encryption algorithm for both encryption and decryption.</span></span>

<span data-ttu-id="ad9a8-107">Наиболее известные симметричные алгоритмы работают быстрее, чем наиболее известные [*алгоритмы открытого ключа*](../secgloss/p-gly.md).</span><span class="sxs-lookup"><span data-stu-id="ad9a8-107">The best-known symmetric algorithms are faster than the best-known [*public key algorithms*](../secgloss/p-gly.md).</span></span> <span data-ttu-id="ad9a8-108">Поэтому они являются предпочтительными при обработке больших объемов данных.</span><span class="sxs-lookup"><span data-stu-id="ad9a8-108">Thus, they are preferred when processing large amounts of data.</span></span> <span data-ttu-id="ad9a8-109">Некоторые из наиболее распространенных симметричных алгоритмов — [*RC4*](../secgloss/r-gly.md), Triple DES ([*стандарт шифрования данных*](../secgloss/d-gly.md)) и [*AES*](../secgloss/a-gly.md) (AES).</span><span class="sxs-lookup"><span data-stu-id="ad9a8-109">Some of the more common symmetric algorithms are [*RC4*](../secgloss/r-gly.md), Triple DES ([*Data Encryption Standard*](../secgloss/d-gly.md)), and [*Advanced Encryption Standard*](../secgloss/a-gly.md) (AES).</span></span>

<span data-ttu-id="ad9a8-110">Так как большая часть действий, включающих симметричные ключи, относится к хранению секретов, важно, чтобы количество людей, имеющих определенный симметричный ключ, было как можно меньше.</span><span class="sxs-lookup"><span data-stu-id="ad9a8-110">Because much of the activity involving symmetric keys relates to keeping them secret, it is important to keep the number of people who possess a particular symmetric key as small as possible.</span></span> <span data-ttu-id="ad9a8-111">Рекомендуется, чтобы только один или два человека обладали одним симметричным ключом.</span><span class="sxs-lookup"><span data-stu-id="ad9a8-111">We recommend that only one or two people possess any one symmetric key.</span></span>
