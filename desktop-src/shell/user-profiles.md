# <a name="user-profiles"></a>Профили пользователей

Сведения о конфигурации, которые могут храниться для каждого пользователя, сохраняются в [профилях пользователей](about-user-profiles.md). Эти сведения включают расположение рабочего стола, группы личных программ, программные элементы в личных группах программ, цвета экрана, заставки, сетевые подключения, подключения к принтерам, параметры мыши, размер и расположение окна, а также т. д. Когда пользователь входит в систему, система загружает профиль пользователя и настраивает среду в соответствии со сведениями в профиле.

Этот раздел содержит следующие темы.

-   [О профилях пользователей](about-user-profiles.md)
-   [Справочник по профилям пользователей](user-profiles-reference.md)
