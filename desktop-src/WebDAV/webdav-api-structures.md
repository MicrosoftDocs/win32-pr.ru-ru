# <a name="webdav-api-structures"></a>Структуры API WebDAV

В API WebDAV используются следующие структуры.



| Структура                                                   | Описание                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_BLOB- \_ объект проверки обратного вызова DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Эта структура используется для хранения [*большого двоичного объекта*](/windows/desktop/SecGloss/b-gly) проверки подлинности, полученного функцией обратного вызова [*даваускаллбакк*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
| [**\_обратный вызов \_ DAV \_ УНП auth**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Эта структура используется для хранения сведений об имени пользователя и пароле, которые были получены функцией обратного вызова [*даваускаллбакк*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) .                                               |
| [**\_cred обратного вызова DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | Функция обратного вызова [*даваускаллбакк*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) использует эту структуру для хранения учетных данных пользователя.                                                                               |
