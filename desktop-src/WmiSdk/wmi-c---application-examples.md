# <a name="wmi-c-application-examples"></a>Примеры приложений WMI на C++

Примеры приложений WMI в этом разделе написаны на C++. Они демонстрируют ряд задач, которые можно выполнить с помощью компонентов WMI, и предложить альтернативный вариант использования сценариев Visual Basic. Каждое приложение разделяется на последовательность шагов таким же образом, чтобы разделы кода из разных примеров можно было легко комбинировать для формирования настраиваемых приложений.
