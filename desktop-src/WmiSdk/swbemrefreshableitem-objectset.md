# <a name="swbemrefreshableitemobjectset-property"></a>Свбемрефрешаблеитем. Objecting, свойство

Свойство **свбемрефрешаблеитем. objecting** представляет набор объектов для обновления. Свойство "набор **объектов** " является либо перечислителем, либо элементом сбора, содержащимся в объекте [**свбемрефрешер**](swbemrefresher.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemRefreshableItem.ObjectSet As Object
```



## <a name="property-value"></a>Значение свойства

## <a name="remarks"></a>Комментарии

Это свойство имеет **значение NULL** , если только [**свбемрефрешаблеитем. Иссет**](swbemrefreshableitem-isset.md) имеет **значение true**.
