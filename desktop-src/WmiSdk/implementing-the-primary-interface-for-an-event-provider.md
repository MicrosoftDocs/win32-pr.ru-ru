<span data-ttu-id="90ee2-107">Поставщик событий должен располагать уведомления, созданные с помощью [**ивбемевентпровидер**](/windows/desktop/api/Wbemprov/nn-wbemprov-iwbemeventprovider) , в объектах     <span data-ttu-id="90ee2-116">Если не обработать запрос, то повторное срабатывание события может заблокировать событие после того, как оно будет принято.</span><span class="sxs-lookup"><span data-stu-id="90ee2-116">If you do not process the request, then refiring the event might block the event from ever being accepted.</span></span>
