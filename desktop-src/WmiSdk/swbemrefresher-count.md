# <a name="swbemrefreshercount-property"></a>Свбемрефрешер. Count, свойство

Свойство **Count** объекта [**свбемрефрешер**](swbemrefresher.md) содержит количество элементов в коллекции [**свбемрефрешер**](swbemrefresher.md) . Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemRefresher.Count As Integer
```
