# <a name="swbemprivilegesetcount-property"></a>Свбемпривилежесет. Count, свойство

Свойство **Count** объекта [**свбемпривилеже**](swbemprivilege.md) содержит количество элементов в коллекции [**свбемпривилежесет**](swbemprivilegeset.md) . Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemPrivilegeSet.Count As Integer
```
