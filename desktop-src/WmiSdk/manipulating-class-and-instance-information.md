Methods \_**](swbemobject-methods-.md) Возвращает коллекцию [**свбеммесодсет**](swbemmethodset.md) методов [**\_ процесса Win32**](/windows/desktop/CIMWin32Prov/win32-process) .</span><span class="sxs-lookup"><span data-stu-id="bb2a8-135">The [**SWbemObject.Methods\_**](swbemobject-methods-.md) property returns an [**SWbemMethodSet**](swbemmethodset.md) collection of the [**Win32\_Process**](/windows/desktop/CIMWin32Prov/win32-process) methods.</span></span> <span data-ttu-id="bb2a8-136">Членами набора методов являются объекты [**свбеммесод**](swbemmethod.md) и [**свбеммесод. outs**](swbemmethod-inparameters.md) Возвращает входные параметры для метода [**CREATE**](/windows/desktop/CIMWin32Prov/create-method-in-class-win32-process) .</span><span class="sxs-lookup"><span data-stu-id="bb2a8-136">The members of the method set are [**SWbemMethod**](swbemmethod.md) objects and [**SWbemMethod.InParameters**](swbemmethod-inparameters.md) returns the input parameters for the [**Create**](/windows/desktop/CIMWin32Prov/create-method-in-class-win32-process) method.</span></span> <span data-ttu-id="bb2a8-137">Требуемый входной параметр *командной строки* имеет значение "calc.exe".</span><span class="sxs-lookup"><span data-stu-id="bb2a8-137">The required *CommandLine* input parameter is set to "calc.exe".</span></span> <span data-ttu-id="bb2a8-138">Затем метод выполняется [**SWbemServices.Exeкмесод**](swbemservices-execmethod.md), что приводит к запуску calc.exe процесса.</span><span class="sxs-lookup"><span data-stu-id="bb2a8-138">The method is then executed by [**SWbemServices.ExecMethod**](swbemservices-execmethod.md), resulting in the launch of a calc.exe process.</span></span>
