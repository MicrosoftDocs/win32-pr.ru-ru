# <a name="swbemnamedvaluesetcount-property"></a>Свбемнамедвалуесет. Count, свойство

Используйте свойство **Count** объекта [**свбемнамедвалуесет**](swbemnamedvalueset.md) , чтобы определить количество элементов в коллекции. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemNamedValueSet.Count As Integer
```
