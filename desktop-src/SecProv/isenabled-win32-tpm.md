# <a name="isenabled-method-of-the-win32_tpm-class"></a>Включенный метод \_ класса TPM Win32

Метод **Enable** класса [**\_ TPM Win32**](win32-tpm.md) указывает, включено ли устройство. Это значение можно изменить с помощью методов [**Enable**](enable-win32-tpm.md) и [**Disable**](disable-win32-tpm.md) .

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsEnabled(
  [out] boolean IsEnabled
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Включено* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если задано **значение true**, устройство включено.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
