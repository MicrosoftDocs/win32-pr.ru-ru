# <a name="changeownerauth-method-of-the-win32_tpm-class"></a>Метод Чанжеовнераус \_ класса TPM Win32

Метод **чанжеовнераус** класса [**\_ TPM Win32**](win32-tpm.md) изменяет значение авторизации владельца доверенного платформенного модуля.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ChangeOwnerAuth(
  [in, optional] string OldOwnerAuth,
  [in, optional] string NewOwnerAuth
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Олдовнераус* \[ в необязательное\]
</dt> <dd>

Тип: **строка**

Строка, которая присваивает имя текущему значению авторизации владельца доверенного платформенного модуля для устройства. Используйте метод [**конверттувнераус**](converttoownerauth-win32-tpm.md) , чтобы преобразовать пароль в это значение авторизации. Параметр *олдовнераус* не указан, или указана пустая строка, этот метод получает значение из реестра, если оно есть.

</dd> <dt>

*Невовнераус* \[ в необязательное\]
</dt> <dd>

Тип: **строка**

Строка, которая присваивает имя новому значению авторизации владельца доверенного платформенного модуля. Используйте метод [**конверттувнераус**](converttoownerauth-win32-tpm.md) , чтобы преобразовать пароль в это значение авторизации. Параметр *невовнераус* не может быть пустым или иметь **значение null.**

</dd> </dl>

## <a name="remarks"></a>Remarks

Метод **чанжеовнераус** создает резервную копию новой авторизации владельца доверенного платформенного модуля для домен Active Directory служб, если настроены соответствующие параметры групповая политика.

Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
