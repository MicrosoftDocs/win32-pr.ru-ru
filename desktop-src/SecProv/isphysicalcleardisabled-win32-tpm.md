# <a name="isphysicalcleardisabled-method-of-the-win32_tpm-class"></a>Метод Исфисикалклеардисаблед \_ класса TPM Win32

Метод **исфисикалклеардисаблед** класса [**\_ TPM Win32**](win32-tpm.md) указывает, может ли только владелец устройства очистить устройство.

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsPhysicalClearDisabled(
  [out] boolean IsPhysicalClearDisabled
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Исфисикалклеардисаблед* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если **значение — true**, устройство может быть очищено только владельцем устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |



 

## <a name="remarks"></a>Remarks
