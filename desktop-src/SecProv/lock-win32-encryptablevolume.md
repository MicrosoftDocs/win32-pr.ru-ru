# <a name="lock-method-of-the-win32_encryptablevolume-class"></a>Метод Lock \_ класса Енкриптаблеволуме Win32

Метод **Lock** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) отключает том и удаляет ключ шифрования тома из системной памяти. Содержимое тома остается недоступным до тех пор, пока не будет разблокировано методом [**унлокквисекстерналкэй**](unlockwithexternalkey-win32-encryptablevolume.md) или методом [**унлокквиснумерикалпассворд**](unlockwithnumericalpassword-win32-encryptablevolume.md) . Этот метод не может быть выполнен для текущего тома операционной системы.

> [!Note]  
> Если диск поддерживает аппаратное шифрование, для полосы задается значение "заблокировано".

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 Lock(
  [in, optional] boolean ForceDismount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Форцедисмаунт* \[ в необязательное\]
</dt> <dd>

Тип: **Boolean**

Если задано **значение true** , диск принудительно отключается.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                           | Описание                                                                                                                                                               |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                           | Метод успешно выполнен.<br/>                                                                                                                                     |
| <dl> <dt>**Д \_ \_Отказано в доступе**</dt> <dt>2147942405 (0x80070005)</dt> </dl>               | Приложения обращаются к этому тому. Если доступ имеет неисключительный результат, при указании значения true для параметра *форцедисмаунт* можно успешно запустить метод.<br/> |
| <dl> <dt>**ФВЕ \_ E \_ не \_ зашифровано**</dt> <dt>2150694913 (0x80310001)</dt> </dl>          | Том полностью расшифровывается и не может быть заблокирован.<br/>                                                                                                            |
| <dl> <dt>**ФВЕ \_ \_Защита E Protection \_ отключена**</dt> <dt>2150694945 (0x80310021)</dt> </dl>    | Ключ шифрования тома доступен на диске в открытом виде, что предотвращает блокировку тома.<br/>                                                    |
| <dl> <dt>**ФВЕ \_ 2150694946 \_ \_ \_ требуется ключ восстановления E**</dt> <dt>(0x80310022)</dt> </dl> | Том не имеет предохранителей ключей типа "числовой пароль" или "внешний ключ", необходимых для разблокировки тома.<br/>                            |
