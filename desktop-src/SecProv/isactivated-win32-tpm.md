# <a name="isactivated-method-of-the-win32_tpm-class"></a>Метод активации \_ класса TPM Win32

Метод **Activate** класса [**\_ TPM Win32**](win32-tpm.md) указывает, активировано ли устройство.

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsActivated(
  [out] boolean IsActivated
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Активируется* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если задано **значение true**, устройство активируется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
