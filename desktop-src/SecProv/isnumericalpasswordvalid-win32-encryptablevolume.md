# <a name="isnumericalpasswordvalid-method-of-the-win32_encryptablevolume-class"></a>Метод Иснумерикалпассвордвалид \_ класса Win32 енкриптаблеволуме

Метод **иснумерикалпассвордвалид** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) указывает, соответствует ли числовой пароль специальным требованиям к формату для этого значения проверки подлинности.

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsNumericalPasswordValid(
  [in]  string  NumericalPassword,
  [out] boolean IsNumericalPasswordValid
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Нумерикалпассворд* \[ окне\]
</dt> <dd>

Тип: **строка**

Строка, указывающая числовой пароль.

Числовой пароль должен содержать 48 цифр. Эти цифры можно разделить на 8 групп из 6 цифр, с последней цифрой в каждой группе, указывающей на значение контрольной суммы для группы. Каждая группа из 6 цифр должна быть кратной 11 и должна быть меньше 720896. При условии, что группа из шести цифр помечена как x1, x2, X3, X4, x5 и X6, контрольная сумма X6 вычисляется как – x1 + x2 – X3 + X4 – x5 mod 11.

Группы цифр можно разделять пробелами или дефисами. Таким образом, "XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX" или "XXXXXX XXXXXX XXXXXX XXXXXX (xxxxxx) XXXXXX XXXXXX" может также содержать допустимые числовые пароли.

</dd> <dt>

*Иснумерикалпассвордвалид* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Логическое значение, равное true, если числовой пароль соответствует требованиям специального формата для этого значения проверки подлинности, в противном случае — значение false.

</dd> </dl>
