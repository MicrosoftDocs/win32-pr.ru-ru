# <a name="isownershipallowed-method-of-the-win32_tpm-class"></a>Метод Исовнершипалловед \_ класса TPM Win32

Метод **исовнершипалловед** класса [**\_ TPM Win32**](win32-tpm.md) указывает, разрешена ли возможность установки владельца для устройства.

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsOwnershipAllowed(
  [out] boolean IsOwnershipAllowed
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Исовнершипалловед* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если задано **значение true**, то разрешена возможность установки владельца устройства. Если задано **значение false**, методу [**такеовнершип**](takeownership-win32-tpm.md) не удастся установить владельца для устройства.

Это значение можно изменить с помощью метода [**сетфисикалпресенцерекуест**](setphysicalpresencerequest-win32-tpm.md) . После выполнения метода [**clear**](clear-win32-tpm.md) значение сбрасывается в **true** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
