# <a name="device-management-functions"></a>Функции управления устройствами

В управлении устройствами используются следующие функции.



| Функция                                                             | Описание                                                                           |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**DeviceIoControl**](/windows/win32/api/ioapiset/nf-ioapiset-deviceiocontrol)                           | Отправляет управляющий код непосредственно в указанный драйвер устройства.                           |
| [**инсталлневдевице**](installnewdevice.md)                         | Устанавливает новое устройство. Пользователю предлагается выбрать устройство.                     |
| [**регистердевиценотификатион**](/windows/desktop/api/Winuser/nf-winuser-registerdevicenotificationa)     | Регистрация устройства или типа устройства, для которых окно будет получать уведомления. |
| [**унрегистердевиценотификатион**](/windows/desktop/api/Winuser/nf-winuser-unregisterdevicenotification) | Закрывает указанный маркер уведомления устройства.                                      |

| [**кдроменабледигиталплайбакк**](/windows/desktop/api/StorProp/nf-storprop-cdromenabledigitalplayback)       | Включает цифровое воспроизведение для указанного дисковода компакт-дисков или DVD-дисков.                                     |
| [**кдромисдигиталплайбаккенаблед**](/windows/desktop/api/StorProp/nf-storprop-cdromisdigitalplaybackenabled) | Определяет, включено ли цифровое воспроизведение для указанного дисковода компакт-дисков или DVD-дисков.               |
| [**кдромкновнгуддигиталплайбакк**](/windows/desktop/api/Storprop/nf-storprop-cdromknowngooddigitalplayback) | Определяет, имеет ли указанный дисковод компакт-дисков или DVD-дисков цифровое воспроизведение, которое известно как хорошее. |
| [**двдлаунчер**](dvdlauncher.md)                                     | Проверяет, соответствует ли регион носителя в DVD-дисководе региону DVD-дисковода.                       |
