# <a name="stubfunction-element"></a>Стубфунктион, элемент

Указывает, должны ли ссылки на функции-заглушки включаться в структуры операций в определениях типов портов для односторонних и двусторонних операций.

## <a name="usage"></a>Использование

``` syntax
<stubFunction/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                       | Описание                                                  |
|---------------------------------------------------------------|--------------------------------------------------------------|
| [**порттипедефинитионс**](porttypedefinitions.md)<br/> | Создает константы C для типов портов.<br/> <br/> |



## <a name="remarks"></a>Примечания

Ссылки на функции-заглушки используются в сценариях размещения для односторонних и двусторонних операций.

Допустимые значения для этого элемента: 1 (включены ссылки на функции-заглушки) и 0 (ложь/нет ссылок на функции-заглушки).

|-------------------------------------|---------------|

| Может быть пустым                        | Да           |
