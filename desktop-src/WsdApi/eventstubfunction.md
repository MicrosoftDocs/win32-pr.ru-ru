# <a name="eventstubfunction-element"></a>Евентстубфунктион, элемент

Указывает, должны ли ссылки на функции-заглушки включаться в структуры операций в определениях типов портов для операций уведомления.

## <a name="usage"></a>Использование

``` syntax
<eventStubFunction/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                       | Описание                                                  |
|---------------------------------------------------------------|--------------------------------------------------------------|
| [**порттипедефинитионс**](porttypedefinitions.md)<br/> | Создает константы C для типов портов.<br/> <br/> |



## <a name="remarks"></a>Примечания

Ссылки на функции-заглушки находятся в клиентских сценариях для операций уведомления (событий).

Допустимые значения для этого элемента: 1 (включены ссылки на функции-заглушки) и 0 (ложь/нет ссылок на функции-заглушки).

|-------------------------------------|---------------|

| Может быть пустым                        | Да           |
