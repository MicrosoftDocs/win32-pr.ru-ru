# <a name="stubdeclarations-element"></a>Стубдекларатионс, элемент

Создает объявления для функций-заглушек для операций с типом порта.

## <a name="usage"></a>Использование

``` syntax
<stubDeclarations>
  child elements
</stubDeclarations>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                   | Описание                                                                                      |
|-------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**событиях**](events.md)<br/>       | Указывает, включаются ли в создаваемые функции связанные события.<br/> <br/> |
| [**операцию**](operation.md)<br/> | Указывает операцию, для которой создается код.<br/> <br/>                 |
| [**Порта**](porttype.md)<br/>   | Указывает тип порта, для которого создается код.<br/> <br/>                |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  portType?, 
  operation*, 
  events?
)
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
