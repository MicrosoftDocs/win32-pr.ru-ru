# <a name="thismodelmetadata-element"></a>Сисмоделметадата, элемент

Определяет производителя и метаданные модели для реализуемого устройства. Этот элемент используется только для реализаций устройств (узлов).

## <a name="usage"></a>Использование

``` syntax
<thisModelMetadata>
  child elements
</thisModelMetadata>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                                     | Описание                                                                                                                                                                        |
|-------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**производителя**](manufacturer.md)<br/>             | Имя производителя. В метаданных должен присутствовать по крайней мере один из [**производителей**](manufacturer.md) или [**мануфактурерлс**](manufacturerls.md) .<br/> <br/> |
| [**мануфактурерлс**](manufacturerls.md)<br/>         | Локализованная версия имени производителя.<br/> <br/>                                                                                                                 |
| [**мануфактурерурл**](manufacturerurl.md)<br/>       | URL-адрес изготовителя.<br/> <br/>                                                                                                                                   |
| [**modelName**](modelname.md)<br/>                   | Имя устройства. В метаданных должен присутствовать хотя бы один из [**modelName**](modelname.md) или [**моделнамелс**](modelnamels.md) .<br/> <br/>                   |
| [**моделнамелс**](modelnamels.md)<br/>               | Локализованная версия имени устройства.<br/> <br/>                                                                                                                       |
| [**моделнумбер**](modelnumber.md)<br/>               | Номер модели устройства.<br/> <br/>                                                                                                                                 |
| [**моделурл**](modelurl.md)<br/>                     | URL-адрес модели устройства.<br/> <br/>                                                                                                                           |
| [**пнпксдевицекатегори**](pnpxdevicecategory.md)<br/> | Категория PnP-X, к которой принадлежит устройство. <br/> <br/>                                                                                                                |
| [**пресентатионурл**](presentationurl.md)<br/>       | URL-адрес данных представления модели устройства.<br/> <br/>                                                                                                        |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  manufacturer?, 
  manufacturerLS*, 
  manufacturerURL?, 
  modelName?, 
  modelNameLS*, 
  modelNumber, 
  modelURL?, 
  PnPXDeviceCategory?, 
  presentationURL?
)
```

## <a name="remarks"></a>Примечания

Метаданные производителя соответствуют разделу метаданных производителя, описанному в профиле устройства (Дополнительные сведения см. в профиле устройства). Необходимо указать имя производителя или хотя бы одну локализованную версию имени производителя. Необходимо указать имя модели или хотя бы одну локализованную версию имени модели.

Элемент [**сисмоделметадатадефинитион**](thismodelmetadatadefinition.md) затем используется для создания константы C, содержащей эти сведения.

Если имеется элемент [**пнпксдевицекатегори**](pnpxdevicecategory.md) , то по крайней мере один [**размещенный**](hosted.md) элемент должен содержать как элементы [**пнпксхардвареид**](pnpxhardwareid.md) , так и [**пнпкскомпатиблеид**](pnpxcompatibleid.md) . Аналогично, если элементы **пнпксхардвареид** и **пнпкскомпатиблеид** представлены в **размещенном** элементе, элемент **пнпксдевицекатегори** должен присутствовать в элементе **сисмоделметадата** .
