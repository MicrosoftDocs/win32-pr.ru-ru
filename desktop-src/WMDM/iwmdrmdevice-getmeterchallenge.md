# <a name="iwmdrmdevicegetmeterchallenge-method"></a>Метод Ивмдрмдевице:: Жетметерчалленже

Метод **жетметерчалленже** извлекает запрос контроля использования.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetMeterChallenge(
  [in]  BSTR  bstrMeterCert,
  [out] BYTE  **ppbMeterChallenge,
  [out] DWORD *pcbMeterChallenge
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстрметерцерт* \[ окне\]
</dt> <dd>

Сертификат контроля использования, который владелец содержимого отправляет на главный компьютер для сбора данных об измерении, связанных с измерением на устройстве

</dd> <dt>

*ппбметерчалленже* \[ заполняет\]
</dt> <dd>

Получен результат запроса контроля использования.

</dd> <dt>

*пкбметерчалленже* \[ заполняет\]
</dt> <dd>

Размер задачи отслеживания в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Remarks

Данные контроля использования собираются и сохраняются в хранилище данных DRM на устройстве для содержимого с включенным измерением. Будут записаны такие действия, как воспроизведение. При вызове этой функции устройство собирает данные об измерениях в хранилище данных DRM в виде XML-документа и отправляет их в хосткомпутер. Если объем данных слишком большой, он отправляется поэтапно.

Когда главный компьютер получает данные отслеживания использования, он отправляет данные через Интернет по URL-адресу, указанному в сертификате измерения.
