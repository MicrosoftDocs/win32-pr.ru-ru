# <a name="iwmdrmdevicegetsecureclock-method"></a>Метод Ивмдрмдевице:: Жетсекуреклокк

Метод **жетсекуреклокк** извлекает безопасное время, поэтому лицензии на основе времени можно применить.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSecureClock(
  [out] BYTE  **ppbSecureClock,
  [out] DWORD *pcbSecureClock,
  [out] DWORD *pdwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппбсекуреклокк* \[ заполняет\]
</dt> <dd>

Получены безопасные часы.

</dd> <dt>

*пкбсекуреклокк* \[ заполняет\]
</dt> <dd>

Размер безопасного времени в байтах.

</dd> <dt>

*пдвфлагс* \[ заполняет\]
</dt> <dd>

Флаги состояния устройства. Это значение должно быть одним из следующих флагов.



| Flag                     | Описание                            |
|--------------------------|----------------------------------------|
| \_исвмдрм устройства \_ WMDRM   | Устройство поддерживает Windows Media DRM. |
| \_нидклокк устройства \_ WMDRM | Устройству требуется таймер.                |
| \_устройство WMDRM \_ отозвано   | Устройство было отозвано.           |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
