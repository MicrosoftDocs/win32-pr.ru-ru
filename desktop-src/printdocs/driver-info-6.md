# <a name="driver_info_6-structure"></a>\_Сведения о драйвере \_ 6 структура

Структура **драйвера \_ \_ 6** содержит сведения о драйвере принтера.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _DRIVER_INFO_6 {
  DWORD     cVersion;
  LPTSTR    pName;
  LPTSTR    pEnvironment;
  LPTSTR    pDriverPath;
  LPTSTR    pDataFile;
  LPTSTR    pConfigFile;
  LPTSTR    pHelpFile;
  LPTSTR    pDependentFiles;
  LPTSTR    pMonitorName;
  LPTSTR    pDefaultDataType;
  LPTSTR    pszzPreviousNames;
  FILETIME  ftDriverDate;
  DWORDLONG dwlDriverVersion;
  LPTSTR    pszMfgName;
  LPTSTR    pszOEMUrl;
  LPTSTR    pszHardwareID;
  LPTSTR    pszProvider;
} DRIVER_INFO_6, *PDRIVER_INFO_6, *LPDRIVER_INFO_6;
```



## <a name="members"></a>Члены

<dl> <dt>

**кверсион**
</dt> <dd>

Версия операционной системы, для которой был записан драйвер. Поддерживаемое значение равно 3.

</dd> <dt>

**pName**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя драйвера (например, QMS 810).

</dd> <dt>

**пенвиронмент**
</dt> <dd>

указатель на строку, завершающуюся нулем, которая указывает среду, для которой был записан драйвер (например, Windows NT x86, Windows IA64 и Windows x64).

</dd> <dt>

**пдриверпас**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего драйвер устройства (например, C: \\ drivers \\Pscript.dll).

</dd> <dt>

**пдатафиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего данные драйвера (например, C: \\ Drivers \\ Qms810. PPD).

</dd> <dt>

**пконфигфиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для библиотеки динамической компоновки конфигурации драйвера устройства (например, C: \\ drivers \\Pscrptui.dll).

</dd> <dt>

**фелпфиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла справки драйвера устройства (например, C: \\ Drivers \\ пскрптуи. hlp).

</dd> <dt>

**пдепендентфилес**
</dt> <dd>

Указатель на буфер Мултисз, содержащий последовательность строк, заканчивающихся нулем. Каждая строка в буфере, завершающаяся нулем, содержит имя файла, от которого зависит драйвер. Последовательность строк завершается пустой строкой нулевой длины. Если **пдепендентфилес** не имеет **значение NULL** и не содержит имен файлов, он будет указывать на буфер, содержащий две пустые строки.

</dd> <dt>

**пмониторнаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает монитор языка (например, "монитор PJL"). Этот элемент может иметь **значение NULL** и указываться только для принтеров, поддерживающих двунаправленный обмен данными.

</dd> <dt>

**пдефаултдататипе**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает тип данных по умолчанию для задания печати (например, "EMF").

</dd> <dt>

**псззпревиауснамес**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает предыдущие имена драйверов принтеров, совместимые с этим драйвером. Например, OldName1 \\ 0OldName2 \\ 0 \\ 0.

</dd> <dt>

**фтдривердате**
</dt> <dd>

Дата пакета драйверов, закодированная в файлах драйверов.

</dd> <dt>

**двлдриверверсион**
</dt> <dd>

Номер версии драйвера. Она выходит из структуры версии драйвера.

</dd> <dt>

**псзмфгнаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя производителя.

</dd> <dt>

**псзоемурл**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает URL-адрес производителя.

</dd> <dt>

**псзардвареид**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает идентификатор оборудования для драйвера принтера.

</dd> <dt>

**псзпровидер**
</dt> <dd>

указатель на строку, завершающуюся нулем, которая указывает поставщика драйвера принтера (например, "Microsoft Windows 2000").

</dd> </dl>

## <a name="remarks"></a>Remarks

Строки для этих элементов содержатся в INF-файле, который используется для добавления драйвера.

Если вы вызываете [**аддпринтердривер**](addprinterdriver.md) или [**аддпринтердриверекс**](addprinterdriverex.md) с *уровнем* не равным 6, а затем вызываете [**жетпринтердривер**](getprinterdriver.md) или [**Енумпринтердриверс**](enumprinterdrivers.md) с *уровнем* , равным 6, структура **драйвера \_ \_ 6** возвращается с **псзмфгнаме**, **pszOEMUrl**, **pszHardwareID**, и **pszProvider** устанавливается в **null**, **dwlDriverVersion** устанавливается в 0, а **ftDriverDate** — в значение (0, 0).
