# <a name="extensions"></a>Модули

Расширения набора свойств, как определено в исходном формате набора свойств COM, были удалены и не поддерживаются, за исключением тех, которые указаны в [наборах свойств документсуммаринформатион и пользователяопределенные](the-documentsummaryinformation-and-userdefined-property-sets.md).
