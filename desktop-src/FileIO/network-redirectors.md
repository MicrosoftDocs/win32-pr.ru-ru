# <a name="network-redirectors"></a>Сетевые перенаправителя

Сетевой перенаправитель — это драйвер файловой системы (или ФСД), который работает следующим образом:

-   В качестве клиента в сетевой операции ввода-вывода путем отправки запросов ввода-вывода на серверы и обработки ответов от серверов.
-   Как сервер в сетевой операции ввода-вывода, получая запросы ввода-вывода от серверов и обрабатывая запросы.

Он выполняет все низкоуровневые действия с сервером при разрешении имени файла, предоставленного приложением, с расположением ресурса на удаленном сервере. Таким образом перенаправитель позволяет приложению получать доступ к ресурсам на удаленных серверах и управлять ими, как если бы они находились на локальном компьютере.

Перенаправления работают полностью в режиме ядра. Это обеспечивает следующие преимущества производительности по сравнению с альтернативами пользовательского режима:

-   Он может взаимодействовать с ФСДС режима ядра, работающим на сервере, например ФСД сервера, без необходимости переключения контекста пользователя на ядро и режима ядра в пользовательском режиме.
-   Он может взаимодействовать в режиме ядра с диспетчером кэша на сервере для кэширования данных ввода-вывода, которые диспетчер кэша сервера отправляет клиенту.
-   Функции API, настраиваемые для удаленных запросов ввода-вывода, и изменения стандартных функций файлового ввода-вывода для предоставления этой функции не требуются.
