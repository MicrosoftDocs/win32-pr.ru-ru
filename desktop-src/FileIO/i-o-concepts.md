В этом разделе описаны следующие основные понятия ввода-вывода.

| Раздел                                                                               | Описание                                                                                                                                                         |
|-------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Буферизация файла](file-buffering.md)<br/>                                     | Описывает вопросы управления приложениями для буферизации файлов, которые также называются небуферизованными файлами ввода-вывода.<br/>                                    |
| [Кэширование файлов](file-caching.md)<br/>                                         | Windows кэширует файловые данные, которые считываются с дисков и записываются на диски.<br/>                                                                                   |

| [Ввод-вывод с оповещением](alertable-i-o.md)<br/>                                       | С помощью предупреждений I/O потоки приложений обрабатывают асинхронные запросы ввода-вывода только в том случае, если они находятся в состоянии оповещения.<br/>                     |
| [Порты завершения ввода-вывода](i-o-completion-ports.md)<br/>                         | Порты завершения ввода-вывода предоставляют эффективную потоковую модель для обработки нескольких асинхронных запросов ввода-вывода в многопроцессорной системе.<br/>                  |
