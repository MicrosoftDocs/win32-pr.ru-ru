<span data-ttu-id="96ea7-104">**Формат строки дескриптора безопасности** — это текстовый формат для хранения или передачи сведений в дескрипторе безопасности.</span><span class="sxs-lookup"><span data-stu-id="96ea7-104">The **Security Descriptor String Format** is a text format for storing or transporting information in a security descriptor.</span></span> <span data-ttu-id="96ea7-105">Функции [**конвертсекуритидескриптортострингсекуритидескриптор**](/windows/desktop/api/Sddl/nf-sddl-convertsecuritydescriptortostringsecuritydescriptora) и [**конвертстрингсекуритидескриптортосекуритидескриптор**](/windows/desktop/api/Sddl/nf-sddl-convertstringsecuritydescriptortosecuritydescriptora) используют этот формат.</span><span class="sxs-lookup"><span data-stu-id="96ea7-105">The [**ConvertSecurityDescriptorToStringSecurityDescriptor**](/windows/desktop/api/Sddl/nf-sddl-convertsecuritydescriptortostringsecuritydescriptora) and [**ConvertStringSecurityDescriptorToSecurityDescriptor**](/windows/desktop/api/Sddl/nf-sddl-convertstringsecuritydescriptortosecuritydescriptora) functions use this format.</span></span>

<span data-ttu-id="96ea7-106">Формат является строкой, завершающейся **нулем**, с токенами для указания каждого из четырех основных компонентов дескриптора безопасности: владелец (O:), основная группа (G:), DACL (D:) и SACL (S:).</span><span class="sxs-lookup"><span data-stu-id="96ea7-106">The format is a **null**-terminated string with tokens to indicate each of the four main components of a security descriptor: owner (O:), primary group (G:), DACL (D:), and SACL (S:).</span></span>

> [!Note]  
> <span data-ttu-id="96ea7-107">[*Записи контроля доступа*](/windows/desktop/SecGloss/a-gly) (ACE) и условные элементы ACE имеют разные форматы.</span><span class="sxs-lookup"><span data-stu-id="96ea7-107">[*Access control entries*](/windows/desktop/SecGloss/a-gly) (ACEs) and conditional ACEs have differing formats.</span></span> <span data-ttu-id="96ea7-108">Сведения об элементах ACE см. в разделе [строки ACE](ace-strings.md).</span><span class="sxs-lookup"><span data-stu-id="96ea7-108">For ACEs, see [ACE Strings](ace-strings.md).</span></span> <span data-ttu-id="96ea7-109">Сведения об условных ACE см. в разделе [язык определения дескрипторов безопасности для условных элементов ACE](security-descriptor-definition-language-for-conditional-aces-.md).</span><span class="sxs-lookup"><span data-stu-id="96ea7-109">For conditional ACEs, see [Security Descriptor Definition Language for Conditional ACEs](security-descriptor-definition-language-for-conditional-aces-.md).</span></span>
