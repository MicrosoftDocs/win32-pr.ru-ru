# <a name="basic-security-property-page"></a>Страница свойств «базовая безопасность»

Страница свойств основная безопасность — это начальная страница страницы свойств, отображаемая функцией [**едитсекурити**](/windows/desktop/api/Aclui/nf-aclui-editsecurity) . Можно также использовать функцию [**креатесекуритипаже**](/windows/desktop/api/Aclui/nf-aclui-createsecuritypage) для создания базовой страницы свойств безопасности для вставки на собственную страницу свойств.

На странице свойств отображается список [доверенных лиц](trustees.md) , указанных в [*записях управления доступом*](/windows/desktop/SecGloss/a-gly) (ACE) [*списка управления доступом*](/windows/desktop/SecGloss/d-gly) (DACL) объекта. На странице также содержится список прав доступа, поддерживаемых объектом. Когда пользователь выбирает имя из списка доверенных лиц, флажки рядом с каждым правом доступа указывают на права, которые разрешены или запрещены для этого доверенного лица. Затем пользователь может установить или снять флажки, чтобы изменить права доступа доверенного лица. Пользователь также может добавлять или удалять доверенные лица из списка.

На странице "основные свойства безопасности" не могут отображаться сложные элементы управления доступом, такие как [записи ACE для конкретного объекта](object-specific-aces.md)или сведения о наследовании ACE. Чтобы предоставить пользователю возможность просматривать или изменять такие сведения, можно добавить кнопку **Дополнительно** на странице основная безопасность. Пользователь может нажать кнопку " **Дополнительно** ", чтобы отобразить [страницу свойств повышенной безопасности](advanced-security-property-sheet.md). На этой странице свойств есть страницы свойств, позволяющие пользователю изменять [*системный список управления доступом*](/windows/desktop/SecGloss/s-gly) (SACL) объекта, изменять владельца объекта или выполнять Расширенное редактирование списка DACL объекта. Чтобы отобразить кнопку " **Дополнительно** ", установите флаг "Advanced Si" \_ в структуре [**\_ \_ сведений об объекте Si**](/windows/desktop/api/Aclui/ns-aclui-si_object_info) , возвращаемой реализацией [**исекуритинформатион:: жетобжектинформатион**](/windows/win32/api/aclui/nf-aclui-isecurityinformation-getobjectinformation) .

Для указания заголовка базовой страницы свойств безопасности можно использовать элемент **псзпажетитле** в структуре [**\_ \_ сведений об объекте Si**](/windows/desktop/api/Aclui/ns-aclui-si_object_info) . Заголовок по умолчанию — **Security**.
