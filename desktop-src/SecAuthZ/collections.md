# <a name="collections-authorization"></a>Коллекции (авторизация)

API диспетчера авторизации предоставляет интерфейсы, представляющие коллекции других объектов. Например, интерфейс [**иазролес**](/windows/desktop/api/Azroles/nn-azroles-iazroles) представляет коллекцию объектов [**иазроле**](/windows/desktop/api/Azroles/nn-azroles-iazrole) .

Другие интерфейсы, представляющие коллекции, — это [**иазаппликатионс**](/windows/desktop/api/Azroles/nn-azroles-iazapplications), [**иазаппликатионграупс**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroups), [**иазоператионс**](/windows/desktop/api/Azroles/nn-azroles-iazoperations), [**иазскопес**](/windows/desktop/api/Azroles/nn-azroles-iazscopes)и [**иазтаскс**](/windows/desktop/api/Azroles/nn-azroles-iaztasks). Каждый из этих интерфейсов предоставляет свойства, упрощающие перечисление объектов в коллекции.
