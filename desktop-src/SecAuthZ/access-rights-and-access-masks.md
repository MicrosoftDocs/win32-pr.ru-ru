<span data-ttu-id="71ce2-104">*Право доступа* — это битовый флаг, соответствующий определенному набору операций, которые поток может выполнять над защищаемым объектом.</span><span class="sxs-lookup"><span data-stu-id="71ce2-104">An *access right* is a bit flag that corresponds to a particular set of operations that a thread can perform on a securable object.</span></span> <span data-ttu-id="71ce2-105">Например, раздел реестра имеет \_ \_ право доступа к набору значений ключа, которое соответствует возможности потока для установки значения в ключе.</span><span class="sxs-lookup"><span data-stu-id="71ce2-105">For example, a registry key has the KEY\_SET\_VALUE access right, which corresponds to the ability of a thread to set a value under the key.</span></span> <span data-ttu-id="71ce2-106">Если поток пытается выполнить операцию над объектом, но не имеет необходимого права доступа к объекту, система не выполняет эту операцию.</span><span class="sxs-lookup"><span data-stu-id="71ce2-106">If a thread tries to perform an operation on an object, but does not have the necessary access right to the object, the system does not carry out the operation.</span></span>

-   [<span data-ttu-id="71ce2-109">Универсальные права доступа</span><span class="sxs-lookup"><span data-stu-id="71ce2-109">Generic access rights</span></span>](generic-access-rights.md)
-   [<span data-ttu-id="71ce2-110">Стандартные права доступа</span><span class="sxs-lookup"><span data-stu-id="71ce2-110">Standard access rights</span></span>](standard-access-rights.md)

<span data-ttu-id="71ce2-113">Когда поток пытается открыть обработчик для объекта, поток обычно задает маску доступа для [запроса набора прав доступа](requesting-access-rights-to-an-object.md).</span><span class="sxs-lookup"><span data-stu-id="71ce2-113">When a thread tries to open a handle to an object, the thread typically specifies an access mask to [request a set of access rights](requesting-access-rights-to-an-object.md).</span></span> <span data-ttu-id="71ce2-114">Например, приложение, которое должно устанавливать и запрашивать значения раздела реестра, может открыть ключ, используя маску доступа, чтобы запросить \_ \_ права доступа значения набора ключей и ключевых \_ запросов \_ .</span><span class="sxs-lookup"><span data-stu-id="71ce2-114">For example, an application that needs to set and query the values of a registry key can open the key by using an access mask to request the KEY\_SET\_VALUE and KEY\_QUERY\_VALUE access rights.</span></span>
