# <a name="user-interface---user-account-control-dialog-updates"></a>Пользовательский интерфейс — обновления диалогового окна управления учетными записями пользователей

## <a name="affected-platforms"></a>Затронутые платформы

**клиенты** — Windows 7  
**серверы** — Windows Server 2008 R2  









## <a name="feature-impact"></a>Воздействие на функции

**Серьезность** — низкая  
**Частота** — средний  











## <a name="description"></a>Описание

в Windows 7 мы создали стандартизированные параметры диалогового окна UAC. Ранее пользователям пришлось выбирать из нескольких вариантов, например "продолжить", "Отмена" и т. д. Теперь все диалоговые окна предоставляют пользователям простые «Yes» и «No». Теперь в макете диалогового окна также ясно отображается имя программы, издатель и источник.

## <a name="leveraging-feature-capabilities"></a>Использование возможностей функций

требования к разработке приложений в Windows 7 для обеспечения совместимости UAC одинаковы, как и в Windows Vista. Windows приложения, совместимые с Vista, будут взаимодействовать с UAC в Windows 7 без каких-либо изменений. сведения о том, как обеспечить правильную работу приложений Windows XP на Windows 7, см. в разделе управление учетными записями пользователей статьи *Cookbook Windows Vista Compatibility* . хотя улучшения UAC для Windows 7 влияют на взаимодействие с пользователем, они не влияют на интерфейс приложения. Однако при наличии содержимого справки, связанного с диалоговыми окнами UAC, может потребоваться обновить снимки экрана.
