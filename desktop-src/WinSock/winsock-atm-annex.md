# <a name="winsock-atm-annex"></a>Приложение Winsock ATM

ATM применяется как к окружениям локальной сети, так и к глобальной сети. ATM-сеть одновременно транспортают широкий спектр сетевых трафика — голоса, данных, изображений и видео. Она предоставляет пользователям гарантированное качество обслуживания для каждого виртуального канала (VC).



| Элемент          | Описание                                                                                                                                                 |
|------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Имена протоколов | АТМПРОТО \_ AAL5, атмпрото \_ аалусер                                                                                                                           |
| Описание      | ATM AAL5 предоставляет транспортную службу, ориентированную на соединение, сохранение границ сообщений и гарантированное качество обслуживания. АТМПРОТО \_ аалусер — это определяемый пользователем AAL. |
| Семейство адресов   | AF \_ ATM                                                                                                                                                     |
| Файл заголовка      | Ws2atm. h                                                                                                                                                    |



 

В этом разделе описываются расширения асинхронной пересылки (ATM), необходимые для поддержки собственных служб ATM, как предоставленные и указанные в спецификации сетевого интерфейса пользователя (UNI) ATM Forum версии 3. x (3,0 и 3,1). Этот документ поддерживает тип AAL 5 (режим сообщения) и определяемый пользователем AAL. Будущие версии этого документа будут поддерживать другие типы AAL, а также UNI 4,0.
