# <a name="error-constants"></a>Константы ошибок

Методы портативных устройств Windows могут возвращать различные коды ошибок **HRESULT** . Некоторые из них относятся к портативным устройствам Windows, а другие — по другим пакетам SDK или пакетам переноса, используемым для создания устройства. В этом разделе описаны различные коды ошибок, которые можно вернуть, упорядоченные по источнику.

WPD определяет следующие константы ошибок.
