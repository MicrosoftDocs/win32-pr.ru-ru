Значение 1 (представляющее 10) означает отсутствие цифрового масштабирования. значение 2 (представляющее 20) обозначает двукратное масштаб (1/4 изображения, отображаемого при изменении масштаба 1x).                  |

| **\_неподвижный \_ \_ \_ уровень \_ компенсации для перераспределения рисков**                                                                                                            | **VT \_ UI4**    | Число, указывающее единицы ВЕРШИНЕ для корректировки экспозиции. Значение 0 обозначает завод по умолчанию.                                                                                                                                                                                       |

| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**\_по-прежнему образ WPD — \_ \_ \_ режим программы экспозиции \_** | **VT \_ UI4**    | Перечислитель по [**\_ \_ \_ режимам программы**](wpd-exposure-program-modes.md) , который указывает программу экспозиции, используемую при записи изображения.                                                                                                                                                                         |

| **\_по- \_ прежнему \_ ФНУМБЕРные образы WPD**                                                                                                                                 | **VT \_ UI4**    | Число, указывающее апертуру линзы при записи изображения. Это свойство обычно допустимо только в том случае, если для параметра [ \_ \_ \_ \_ \_ режим установки программы WPD](/windows) установлено значение вручную или с приоритетом диафрагмы.<br/> |

Например, 35mm будет 3500.                                                                                                                                                                                                                 |
| **у WPD \_ все еще есть \_ изображение на \_ расстоянии фокуса \_**                                                                                                                         | **VT \_ UI4**    | Расстояние фокуса, которое устройство должно использовать, в миллиметрах. | **\_Коэффициент отобразов WPD по-прежнему равен \_ \_ RGB \_**                                                                                                                               | **VT \_ LPWSTR** | Строка, завершающаяся нулем и определяющая коэффициент RGB, используемый при записи изображения.
