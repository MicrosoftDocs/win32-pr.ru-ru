# <a name="image-properties"></a>Свойства изображения

Портативные устройства Windows поддерживают следующие свойства образа.



| Свойство                                                                                                                                       | VarType     | Описание                                                                                                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_битдепс образа \_ WPD**                                                                                                                       | **VT \_ UI4** | Битовая глубина изображения.                                                                                                                                                                                                                                                                                                                     |
| <span id="wpd_image_color_corrected_status"></span><span id="WPD_IMAGE_COLOR_CORRECTED_STATUS"></span>**\_ \_ \_ Исправлено состояние цвета образа WPD \_** | **VT \_ UI4** | Перечисление [**\_ \_ \_ \_ значений состояния WPD Color**](wpd-color-corrected-status-values.md) , указывающее, был ли файл исправлен цветом. Это предотвращает автоматическое выделение цветом изображения во время последующей обработки на нескольких устройствах.<br/>                                                                       |
| **\_ \_ состояние кадрированного изображения \_ WPD**                                                                                                                | **VT \_ UI4** | Перечисление [**\_ \_ \_ значений состояния WPD**](wpd-cropped-status-values.md) , которое указывает, обрезан ли файл. Это предотвращает автоматическую обрезку изображения на нескольких устройствах во время последующей обработки.<br/>                                                                                                        |
| **\_ \_ индекс экспозиции образа WPD \_**                                                                                                                | **VT \_ UI4** | Значение, определяющее настройки скорости пленки при записи этого образа. Параметры соответствуют стандартам ISO для ASA/DIN.<br/> Как правило, устройство поддерживает дискретные перечислимые значения, но возможно непрерывное управление диапазоном.<br/> Значение 0xFFFFFFFF соответствует автоматическому параметру ISO.<br/> |
| **\_ \_ время экспозиции образа WPD \_**                                                                                                                 | **VT \_ UI4** | Определяет скорость затвора устройства при записи этого образа. Единицы находятся в секундах, масштабируемых на 10000.<br/>                                                                                                                                                                                                                        |
| **\_фнумбер образа \_ WPD**                                                                                                                        | **VT \_ UI4** | Определяет параметр апертуры линзы при записи этого изображения. Единицы измерения равны f-номеру, масштабированному на 100.<br/>                                                                                                                                                                                                              |
| **\_разрешение изображения WPD \_ по горизонтали \_**                                                                                                         | **VT \_ R8**  | Указывает разрешение изображения по горизонтали в точках на дюйм (DPI).                                                                                                                                                                                                                                                                       |
| **\_разрешение изображения WPD \_ по вертикали \_**                                                                                                           | **VT \_ R8**  | Обозначает вертикальное разрешение изображения в точках на дюйм (DPI).                                                                                                                                                                                                                                                                         |
