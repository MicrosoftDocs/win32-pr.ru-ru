# <a name="command-parameters"></a>Параметры команд

Портативные устройства Windows (WPD) поддерживают следующие свойства параметров команды.




| **Свойство**                                            | **VarType**     | **Описание**                                                                                                                                                              |
|---------------------------------------------------------|-----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_Свойства WPD \_ Общие \_ \_ сведения о клиенте**          | **VT \_ Unknown** | Интерфейс [**ипортабледевицевалуес**](iportabledevicevalues.md) , используемый драйвером для обнаружения клиента.                                                             |
| **\_ \_ \_ \_ контекст сведений об общем КЛИЕНТе для свойства WPD \_** | **VT \_ LPWSTR**  | Заданный драйвером контекст, который определяет клиента для всех последующих операций.                                                                                          |
| **\_ \_ \_ Категория общей команды \_ Свойства WPD**            | **VT \_ CLSID**   | Часть **идентификатора GUID** , относящаяся к значению **PROPERTYKEY** команды.                                                                                                            |
| **\_ \_ \_ идентификатор общей команды \_ Свойства WPD**                  | **VT \_ UI4**     | Значение постоянного уникального идентификатора (PID) значения **PROPERTYKEY** команды.                                                                                          |
| **\_ \_ \_ Цель общей команды \_ для свойства WPD**              | **VT \_ LPWSTR**  | Идентификатор целевого объекта.                                                                                                                                                |
| **\_ \_ \_ код ошибки общего драйвера \_ для свойства WPD \_**          | **VT \_ UI4**     | Код ошибки, относящийся к драйверу, возвращенный драйвером WPD.                                                                                                                       |
| **Свойство WPD, \_ \_ Общее \_ значение HRESULT**                      | **\_Ошибка VT**   | Значение **HRESULT** , возвращаемое драйвером WPD для определенной операции.                                                                                                   |
| **\_ \_ \_ идентификаторы общих объектов свойств WPD \_**                  | **VT \_ Unknown** | Интерфейс [**ипортабледевицепропвариантколлектион**](iportabledevicepropvariantcollection.md) типа Variant **VT \_ LPWSTR** , содержащий список идентификаторов объектов. |
| **свойства WPD. \_ \_ Общие \_ постоянные \_ уникальные \_ идентификаторы**      | **VT \_ Unknown** | Интерфейс [**ипортабледевицепропвариантколлектион**](iportabledevicepropvariantcollection.md) типа Variant **VT \_ LPWSTR** , содержащий список идентификаторов PID.               |
