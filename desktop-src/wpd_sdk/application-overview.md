# <a name="application-overview"></a><span data-ttu-id="05816-103">Обзор приложений</span><span class="sxs-lookup"><span data-stu-id="05816-103">Application Overview</span></span>

<span data-ttu-id="05816-104">Портативные устройства Windows (WPD) предоставляют инфраструктуру, которая обеспечивает доступ ПК к многофункциональным устройствам, в которых хранятся различные мультимедийные и мультимедийные данные.</span><span class="sxs-lookup"><span data-stu-id="05816-104">Windows Portable Devices (WPD) provides an infrastructure that enables PC access to multifunction devices which store various media and non-media content.</span></span> <span data-ttu-id="05816-105">Эта инфраструктура обеспечивает расхождение устройств и изначально сосредоточена на интеллектуальных запоминающих устройствах, таких как портативные проигрыватели мультимедиа, цифровые камеры и мобильные телефоны.</span><span class="sxs-lookup"><span data-stu-id="05816-105">This infrastructure enables convergent devices and is initially focused on smart storage devices such as portable media players, digital still cameras, and mobile phones.</span></span>

<span data-ttu-id="05816-106">WPD состоит из интерфейса драйвера устройства (DDI) и набора прикладных программных интерфейсов (API).</span><span class="sxs-lookup"><span data-stu-id="05816-106">WPD consists of a Device Driver Interface (DDI) and a set of application programming interfaces (APIs).</span></span> <span data-ttu-id="05816-107">В этом обзоре описывается, как можно воспользоваться преимуществами этих API-интерфейсов в приложении Windows.</span><span class="sxs-lookup"><span data-stu-id="05816-107">This overview describes how you can take advantage of these APIs in your Windows application.</span></span>

<span data-ttu-id="05816-108">Инфраструктура WPD предлагает высоко расширяемые механизмы представления и управления устройствами.</span><span class="sxs-lookup"><span data-stu-id="05816-108">The WPD infrastructure offers highly extensible device representation and control mechanisms.</span></span> <span data-ttu-id="05816-109">Это позволяет поставщикам решений по вертикали использовать API-интерфейс WPD для создания улучшенных пользовательских интерфейсов, которые выходят за пределы стандартизированного набора WPD.</span><span class="sxs-lookup"><span data-stu-id="05816-109">This enables vertical solution providers to use the WPD API to create enhanced user experiences that are outside of the WPD standardized set.</span></span> <span data-ttu-id="05816-110">К таким примерам относится предоставляемое поставщиком программное обеспечение, которое выполняет обновления встроенного по или диагностику устройств (для удаленной поддержки).</span><span class="sxs-lookup"><span data-stu-id="05816-110">Examples of this include vendor-provided software that performs firmware updates or device diagnostics (for remote support).</span></span>
