# <a name="mtp-extension-properties"></a>Свойства расширения MTP

Свойства описывают объекты, свойства, ресурсы и устройства.

## <a name="vendor-extended-object-properties"></a>Поставщик — свойства расширенного объекта

Когда производитель устройства создает свойство, расширенное поставщиком для объекта устройства, он объединяет свойства WPD для идентификатора GUID **\_ \_ \_ \_ расширенного \_ объекта \_ PROPS поставщика MTP** с кодом свойства объекта для создания структуры **PROPERTYKEY** .

Например, если код свойства объекта — 0xD801, результирующий **PROPERTYKEY** будет выглядеть так, как показано в следующем примере:


```C++
{4D545058-4FCE-4578-95C8-8698A9BC0F49}\D801
```



## <a name="vendor-extended-device-properties"></a>Поставщик — расширенные свойства устройства

Когда производитель устройства создает для устройства свойство, расширенное поставщиком, он объединяет **\_ Свойства WPD \_ \_ поставщика MTP \_ расширенных свойств \_ \_ PROPS** класса "идентификатор GUID" с кодом свойства объекта для создания структуры **PROPERTYKEY** .

Например, если код свойства объекта — 0xD001, результирующий **PROPERTYKEY** будет выглядеть так, как показано в следующем примере:


```C++
{4D545058-8900-40b3-8F1D-DC246E1E8370}\D001
```
