# <a name="iwiadevmgr2-interface"></a>Интерфейс IWiaDevMgr2

Интерфейс **IWiaDevMgr2** используется для создания устройств получения изображений и управления ими, а также для регистрации событий устройств.

## <a name="members"></a>Элементы

Интерфейс **IWiaDevMgr2** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **IWiaDevMgr2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IWiaDevMgr2** содержит следующие методы.



| Метод                                                                                    | Описание                                                                                                                                                                                                                                                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**креатедевице**](-wia-iwiadevmgr2-createdevice.md)                                     | Создает иерархическое дерево объектов [**IWiaItem2**](-wia-iwiaitem2.md) для устройства WIA 2,0. <br/>                                                                                                                                                                                                                                                                                                 |
| [**енумдевицеинфо**](-wia-iwiadevmgr2-enumdeviceinfo.md)                                 | Создает перечислитель сведений о свойстве для каждого доступного устройства WIA 2,0. <br/>                                                                                                                                                                                                                                                                                                                 |
| [**жетимажедлг**](-wia-iwiadevmgr2-getimagedlg.md)                                       | Метод [**IWiaDevMgr2:: жетимажедлг**](-wia-iwiadevmgr2-getimagedlg.md) отображает одно или несколько диалоговых окон, которые позволяют пользователю получить изображение с устройства WIA 2,0 и записать образ в указанный файл. Этот метод расширяет функциональные возможности [**IWiaDevMgr2:: селектдевицедлг**](-wia-iwiadevmgr2-selectdevicedlg.md) для инкапсуляции получения образа в рамках одного вызова API. <br/> |
| [**регистеревенткаллбаккклсид**](-wia-iwiadevmgr2-registereventcallbackclsid.md)         | Метод [**IWiaDevMgr2:: регистеревенткаллбаккклсид**](-wia-iwiadevmgr2-registereventcallbackclsid.md) регистрирует приложение для получения событий, даже если приложение не работает.<br/>                                                                                                                                                                                                      |
| [**регистеревенткаллбаккинтерфаце**](-wia-iwiadevmgr2-registereventcallbackinterface.md) | Регистрирует выполняющееся приложение для уведомления о событии WIA 2,0. <br/>                                                                                                                                                                                                                                                                                                                                  |
| [**регистеревенткаллбаккпрограм**](-wia-iwiadevmgr2-registereventcallbackprogram.md)     | Метод [**IWiaDevMgr2:: регистеревенткаллбаккпрограм**](-wia-iwiadevmgr2-registereventcallbackprogram.md) регистрирует приложение для получения событий устройства. В основном он предоставляется для обеспечения обратной совместимости с приложениями, которые не были написаны для WIA 2,0.<br/>                                                                                                                         |
| [**селектдевицедлг**](-wia-iwiadevmgr2-selectdevicedlg.md)                               | Отображает диалоговое окно, позволяющее пользователю выбрать аппаратное устройство для получения образа. <br/>                                                                                                                                                                                                                                                                                                   |
| [**селектдевицедлгид**](-wia-iwiadevmgr2-selectdevicedlgid.md)                           | Отображает диалоговое окно, позволяющее пользователю выбрать аппаратное устройство для получения образа. <br/>                                                                                                                                                                                                                                                                                                   |

| Методы IUnknown                                        | Описание                               |
|---------------------------------------------------------|-------------------------------------------|
| [IUnknown:: QueryInterface](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) | Возвращает указатели на поддерживаемые интерфейсы. |
| [IUnknown:: AddRef](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref)                 | Увеличивает значение счетчика ссылок.               |
| [IUnknown:: Release](/windows/win32/api/unknwn/nf-unknwn-iunknown-release)               | Уменьшает значение счетчика ссылок.               |
