# <a name="sti-compatibility"></a>Совместимость сти

Архитектура Windows для получения изображений (WIA) основана на фундаменте, установленном с помощью архитектуры образов по-прежнему (сти). Драйверы WIA-устройств будут соответствовать модели драйвера пользовательского режима (USD) сти. Хотя первоначальный USD предназначен для поддержки источников данных TWAIN и других интерфейсов API, драйверы WIA поддерживают новый интерфейс приложения WIA.
