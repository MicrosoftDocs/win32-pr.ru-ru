# <a name="ip-helper"></a>Вспомогательная служба IP

## <a name="purpose"></a>Назначение

API вспомогательного приложения протокола Интернета (IP-адрес) обеспечивает извлечение и изменение параметров конфигурации сети для локального компьютера.

## <a name="where-applicable"></a>Где применимо

API вспомогательной функции IP применим в любой вычислительной среде, где программно манипулировать сетью и конфигурацией TCP/IP очень удобно. Типичные приложения включают протоколы IP-маршрутизации и агенты SNMP.

## <a name="developer-audience"></a>Аудитория разработчиков

API вспомогательной функции IP предназначен для использования программистами C/C++. программисты также должны быть знакомы с Windows сетями и понятиями сети TCP/IP.

## <a name="run-time-requirements"></a>Требования к среде выполнения

API вспомогательной функции IP-адресов можно использовать на всех платформах Windows. Не все операционные системы поддерживают все функции. если существуют определенные реализации или возможности Windows сокеты 2 ограничения платформы, они четко указаны в документации. Если вспомогательная функция IP вызывается на платформе, которая не поддерживает эту функцию, возвращается ошибка \_ не \_ поддерживается. Дополнительные сведения о том, какие операционные системы поддерживают определенную функцию, см. в разделах о требованиях в документации.
