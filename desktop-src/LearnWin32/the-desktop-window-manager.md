<span data-ttu-id="f2db3-112">Windows Vista фундаментально изменила внешний вид Windows, представляя диспетчер окон рабочего стола (DWM).</span><span class="sxs-lookup"><span data-stu-id="f2db3-112">Windows Vista fundamentally changed how windows are drawn, by introducing the Desktop Window Manager (DWM).</span></span> <span data-ttu-id="f2db3-113">Когда DWM включен, окно больше не рисуется непосредственно в буфере экрана.</span><span class="sxs-lookup"><span data-stu-id="f2db3-113">When the DWM is enabled, a window no longer draws directly to the display buffer.</span></span> <span data-ttu-id="f2db3-114">Вместо этого каждое окно рисуется в буфере памяти экрана, также называемом заданной *областью*.</span><span class="sxs-lookup"><span data-stu-id="f2db3-114">Instead, each window draws to an offscreen memory buffer, also called an *offscreen surface*.</span></span> <span data-ttu-id="f2db3-115">Затем DWM перекомбинирует эти поверхности на экран.</span><span class="sxs-lookup"><span data-stu-id="f2db3-115">The DWM then composites these surfaces to the screen.</span></span>
