# <a name="iwmpcdromburnburnplaylist-property"></a>Свойство Ивмпкдромбурн:: Бурнплайлист

Свойство **бурнплайлист** получает текущий список воспроизведения для записи на компакт-диск.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPPlaylist burnPlaylist {get;}
```


```VB

Public ReadOnly Property burnPlaylist As IWMPPlaylist
```





## <a name="property-value"></a>Значение свойства

Интерфейс **вмплиб. ивмпплайлист** списка воспроизведения для записи.
