# <a name="buttontiled"></a>BUTTON. мозаичное заполнение

Атрибут **мозаичного заполнения** указывает или получает значение, указывающее, будет ли изображение **кнопки** мозаичным.

``` syntax
        elementID.tiled
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                       |
|-------|-----------------------------------|
| true  | Изображение будет мозаичным.              |
| false | По умолчанию. Изображение не будет мозаичным. |



 

## <a name="remarks"></a>Комментарии

Если изображение меньше фактической области элемента управления, мозаичное изображение будет повторяться до тех пор, пока не заполнит весь регион. Если образ не указан или не может быть получен, для **мозаичного заполнения** будет установлено значение false.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
