# <a name="iwmpnetworkmaxbitrate-property"></a>Свойство Ивмпнетворк:: Максбитрате

Свойство **максбитрате** Возвращает максимальную возможную скорость видео.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.Int32 maxBitRate {get; set;}
```


```VB

Public ReadOnly Property maxBitRate As System.Int32
```





## <a name="property-value"></a>Значение свойства

Значение **System. Int32** , которое является максимальной скоростью.
