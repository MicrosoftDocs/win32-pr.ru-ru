<span data-ttu-id="c23c6-110">Для пользователя подключаемые модули DSP — это надстройки, которые по каким-либо причинам изменяют способ работы пользователя с аудио или видео, который воспроизводит проигрыватель Windows Media.</span><span class="sxs-lookup"><span data-stu-id="c23c6-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="c23c6-111">Результат использования подключаемого модуля DSP может быть очевидным, как, например, цвета в видеоизображении отменяются, так что он выглядит как изображение негативно или что-то очень незаметно, например, при светлой отспаЦиауснессе звука.</span><span class="sxs-lookup"><span data-stu-id="c23c6-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>

<span data-ttu-id="c23c6-112">Пользователи могут устанавливать и удалять подключаемые модули DSP с помощью вкладки " **подключаемые модули** " диалогового окна "Параметры" проигрывателя Windows Media.</span><span class="sxs-lookup"><span data-stu-id="c23c6-112">Users can install and uninstall DSP plug-ins by using the **Plug-ins** tab of the Options dialog box in Windows Media Player.</span></span> <span data-ttu-id="c23c6-113">После этого пользователи могут включать и отключать установленные подключаемые модули с помощью флажков на вкладке " **подключаемые модули** ". подключаемые модули DSP могут реализовывать страницу свойств, к которой пользователи также могут обращаться из вкладки " **подключаемые модули** ". Страница свойств предоставляет пользователю способ изменения параметров, влияющих на способ работы подключаемого модуля DSP.</span><span class="sxs-lookup"><span data-stu-id="c23c6-113">Users can then enable and disable installed plug-ins using the checkboxes provided on the **Plug-ins** tab. DSP plug-ins may implement a property page, which users can also access from the **Plug-ins** tab. A property page provides the user with a way to change settings that affect the way a DSP plug-in works.</span></span> <span data-ttu-id="c23c6-114">Например, страница свойств может включать элемент управления "ползунок", который позволяет пользователю изменить время Decay для измененного результата.</span><span class="sxs-lookup"><span data-stu-id="c23c6-114">For example, a property page might include a slider control that allows the user to change the decay time of a reverb effect.</span></span>
