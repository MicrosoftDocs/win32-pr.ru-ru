# <a name="about-sami-urls"></a>Об URL-адресах SAMI

Файлы SAMI можно связать с цифровыми файлами мультимедиа с помощью одного URL-адреса. Это достигается с помощью параметра URL-адреса *Sami* . Параметру URL предшествует базовый URL-адрес и? . URL-адрес с параметром *Sami* следует за следующим синтаксисом:

*URL-адрес*? *саамский* = *каптионсурл*.

Значение параметра URL-адреса соответствует имени параметра и знаку равенства, как показано в следующем примере:


```C++
https://proseware.com/samitest.wma?sami=https://acc.proseware.com/test.smi

```



Этот синтаксис URL-адреса обычно используется в гиперссылке или метафайле Windows Media для прямой связи с расположением файла мультимедиа и файла SAMI. Когда пользователь щелкает гиперссылку, проигрыватель Windows Media запускается в полноэкранном режиме и воспроизводит мультимедийное содержимое.

Если параметр URL-адреса *Sami* не указан, проигрыватель Windows Media выполнит поиск файла Sami, расположенного в том же расположении, что и файл цифрового мультимедиа, с тем же именем, кроме расширения имени файла, которое должно быть. SMI. Если такой файл существует, он будет открыт автоматически, если в проигрывателе был включен режим отображения субтитров.

Скрытые субтитры включаются в проигрывателе Windows Media. для этого в меню **Воспроизведение** выберите пункт **субтитры и субтитры**, а затем щелкните **элемент.** Если включены субтитры, то заголовки, содержащиеся в файле SAMI, будут отображаться во время воспроизведения элемента мультимедиа.
