# <a name="using-borders-in-windows-media-download-packages-deprecated"></a>использование границ в Windows скачать пакеты мультимедиа (устарело)

эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрыватель Windows Media и пакета SDK для проигрыватель Windows Media.

С помощью границ можно создать настраиваемый графический пользовательский интерфейс для упакованного содержимого. Эта граница может включать такие элементы, как изображения, интерактивные элементы управления и ссылки на веб-сайты. Границы можно использовать в тех случаях, когда требуется добавить дополнительное значение в упакованное содержимое, например для фирменной символики или рекламы. после того как пользователи загрузили и откроет пакет скачивания Windows Media, проигрыватель Windows Media автоматически отобразит пользовательскую границу при воспроизведении упакованного содержимого.

в отличие от обложки, которая позволяет пользователям полностью заменить проигрыватель Windows Media пользовательский интерфейс, граница отображается только на панели **воспроизведение** в проигрывателе полного режима. Тем не менее те же средства и технологии, которые используются для создания обложек, также используются для создания границ. На следующем рисунке показана граница.

![отображаемая граница Windows Media Player 10](images/border-v10.png)

Важно понимать основные методы создания обложки, прежде чем пытаться создать границу. Программирование границ осуществляется с помощью двух языков программирования: язык XML (XML) и Microsoft JScript. XML используется для определения элементов интерфейса, таких как кнопки, ползунки и текстовые поля. Вам не нужно понимать все детали XML, так как вам не нужно писать новые элементы кода XML. вы можете просто использовать те, которые предоставлены проигрыватель Windows Media. хотя JScript не требуется для создания границ, ее можно использовать для предоставления дополнительных функциональных возможностей.

Сжатый файл границ с расширением WMZ включает файл определения границы с расширением WMS и все файлы изображений, используемые в границах.

чтобы включить границу в пакет скачивания Windowsного носителя, просто создайте границу и укажите ссылку на эту границу в Windows списке воспроизведения метафайла мультимедиа. файл border загружается в проигрыватель Windows Media после того, как проигрыватель анализирует метафайл и интерпретирует элемент **SKIN** , который ссылается на эту границу. Элемент **Skin** используется только для границ, а атрибут href элемента **Skin** может ссылаться только на одну обложку для каждого пакета.
