<span data-ttu-id="d4127-111">Изображения сопоставления используются для конкретных элементов управления, чтобы указать, какие регионы будут отвечать на нажатия клавиш мыши, и определить, какие элементы управления получают какие события.</span><span class="sxs-lookup"><span data-stu-id="d4127-111">Mapping images are used for specific controls to specify which regions will respond to mouse clicks and to determine which controls receive which events.</span></span> <span data-ttu-id="d4127-112">Изображение сопоставления должно иметь ту же высоту и ширину, что и первичный образ, и должно быть создано таким образом, чтобы изображения в обоих файлах выстроились точно.</span><span class="sxs-lookup"><span data-stu-id="d4127-112">The mapping image must be the same height and width as the primary image and should be created so that the images in both files line up exactly.</span></span> <span data-ttu-id="d4127-113">По этой причине рекомендуется использовать программу редактирования графики, поддерживающую слои, так как можно быть уверенным, что изображения будут выровнены по точности.</span><span class="sxs-lookup"><span data-stu-id="d4127-113">For this reason, a graphics editing program that supports layers is recommended because you can be certain your images are perfectly aligned.</span></span>
