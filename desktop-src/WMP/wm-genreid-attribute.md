# <a name="wmgenreid-attribute"></a>Атрибут WM/GenreID

Атрибут **WM/GenreID** является идентификатором жанра, совместимым с тегом Ткон в ID3v2.

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)
-   [Часто используемые атрибуты файлов Windows Media](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Комментарии

Этот атрибут хранится только в цифровом файле мультимедиа.

ID3 версии 2 — это соглашение о разметке тегов, изначально разработанное для MP3. Дополнительные сведения см. на [веб-сайте Организации ID3](https://id3.org/).

Константа Windows Media Format SDK для этого атрибута — g \_ всзженреид.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
