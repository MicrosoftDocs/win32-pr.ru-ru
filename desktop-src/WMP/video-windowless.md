# <a name="videowindowless"></a>ВИДЕО. без окон

Атрибут без **окон** указывает или получает значение, указывающее, будет ли элемент управления видео отображаться в окне или без окон; то есть независимо от того, будет ли весь прямоугольник элемента управления видимым в любое время или же его можно обрезать. Можно задать только во время разработки.

``` syntax
        elementID.windowless
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** , заданным во время разработки, и затем доступен только для чтения.



| Значение | Описание                              |
|-------|------------------------------------------|
| true  | Элемент управления видео будет безоконным.        |
| false | По умолчанию. Элемент управления видео будет отображаться в окне. |



 

## <a name="remarks"></a>Комментарии

Если требуется Непрямоугольное видео-окно или вы хотите охватить любую часть видеоокна с изображением, этот атрибут должен иметь значение true. Это потребует некоторой производительности для выполнения необходимой обрезки.

Воспроизведение видео оптимизировано для необрезанного воспроизведения. В этом случае атрибут без **окон** имеет значение false, и всегда отображается весь прямоугольник видео. Все изображения, охватывающие окно видео, игнорируются, и в окне видео отображается z-порядок самого высокого уровня.
