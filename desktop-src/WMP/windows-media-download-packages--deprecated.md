<span data-ttu-id="c112f-109">Эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрывателя Windows Media и Windows Media Player SDK.</span><span class="sxs-lookup"><span data-stu-id="c112f-109">This page documents a feature that may be unavailable in future versions of Windows Media Player and the Windows Media Player SDK.</span></span>

-   <span data-ttu-id="c112f-121">Упаковка поддерживается для нескольких типов звуковых и видеофайлов.</span><span class="sxs-lookup"><span data-stu-id="c112f-121">Multiple audio and video file types are supported for packaging.</span></span>

<span data-ttu-id="c112f-122">Все эти преимущества можно использовать вместе.</span><span class="sxs-lookup"><span data-stu-id="c112f-122">All of these benefits can be used together.</span></span> <span data-ttu-id="c112f-123">Например, один пакет загрузки Windows Media может предоставить пользователям возможность просматривать песни песен по мере их воспроизведения, отображать видео, сопровождающее песни, просматривать рекламные сведения о распространителе записей, просматривать обложку альбома, посещать веб-сайт вентилятора и каталогизировать содержимое в библиотеке.</span><span class="sxs-lookup"><span data-stu-id="c112f-123">For example, a single Windows Media Download package could give users the opportunity to view the lyrics to songs as they play, display a video that accompanies the songs, view advertising information about the record distributor, view album cover art, visit a fan website, and catalog the content in the library.</span></span>
