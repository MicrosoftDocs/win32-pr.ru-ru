# <a name="media-object"></a>Объект мультимедиа

Объект **мультимедиа** предоставляет способ задания или извлечения свойств элемента мультимедиа с помощью следующих свойств и методов.

Объект **мультимедиа** поддерживает следующие свойства.



| Свойство                                         | Описание                                                                                        |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [аттрибутекаунт](media-attributecount.md)       | Возвращает количество атрибутов, которые могут быть запрошены или заданы для элемента мультимедиа.              |
| [duration](media-duration.md)                   | Возвращает длительность текущего элемента мультимедиа в секундах.                                       |
| [дуратионстринг](media-durationstring.md)       | Извлекает **строковое** значение, указывающее длительность текущего элемента мультимедиа в формате чч: мм: СС. |
| [error](media-error.md)                         | Извлекает объект [ерроритем](erroritem-object.md) , если у элемента мультимедиа есть условие ошибки.    |
| [имажесаурцехеигхт](media-imagesourceheight.md) | Извлекает высоту текущего элемента мультимедиа в пикселях.                                          |
| [имажесаурцевидс](media-imagesourcewidth.md)   | Получает ширину текущего элемента мультимедиа в пикселях.                                           |
| [маркеркаунт](media-markercount.md)             | Возвращает количество маркеров в элементе мультимедиа.                                                 |
| [name](media-name.md)                           | Указывает или получает имя элемента мультимедиа.                                                 |
| [саурцеурл](media-sourceurl.md)                 | Извлекает URL-адрес элемента мультимедиа.                                                               |



 

Объект **мультимедиа** поддерживает следующие методы.



| Метод                                                       | Описание                                                                                              |
|--------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [жетаттрибутекаунтбитипе](media-getattributecountbytype.md) | Извлекает количество атрибутов, связанных с указанным именем атрибута и языком.            |
| [жетаттрибутенаме](media-getattributename.md)               | Возвращает имя атрибута, соответствующего указанному индексу.                                |
| [getItemInfo](media-getiteminfo.md)                         | Возвращает значение указанного атрибута для элемента мультимедиа.                                       |
| [жетитеминфобятом](media-getiteminfobyatom.md)             | Извлекает значение атрибута с указанным номером индекса.                                    |
| [жетитеминфобитипе](media-getiteminfobytype.md)             | Извлекает значение атрибута, соответствующего указанному имени атрибута, языку и индексу. |
| [жетмаркернаме](media-getmarkername.md)                     | Возвращает имя маркера по указанному индексу.                                                 |
| [жетмаркертиме](media-getmarkertime.md)                     | Возвращает время маркера по указанному индексу.                                                 |
| [Идентично](media-isidentical.md)                         | Получает значение, указывающее, совпадает ли указанный объект с текущим объектом.                 |
| [isMemberOf](media-ismemberof.md)                           | Получает значение, указывающее, является ли указанный элемент мультимедиа членом указанного списка воспроизведения.     |
| [исреадонлитем](media-isreadonlyitem.md)                   | Извлекает значение, указывающее, можно ли изменять атрибуты указанного элемента мультимедиа.           |
| [сетитеминфо](media-setiteminfo.md)                         | Задает значение указанного атрибута для элемента мультимедиа.                                            |



 

Доступ к объекту **мультимедиа** осуществляется с помощью следующих свойств и методов.



| Объект                          | Свойство или метод                                                       |
|---------------------------------|--------------------------------------------------------------------------|
| [Элементы управления](controls-object.md) | [currentItem](controls-currentitem.md)                                  |
| [Игрок](player-object.md)     | [куррентмедиа](player-currentmedia.md), [невмедиа](player-newmedia.md) |
| [Списком](playlist-object.md) | [Item](playlist-item.md)                                                |



 

Так как это наиболее распространенные средства доступа, *Player*. **куррентмедиа** используется в целях иллюстрации в разделах синтаксиса Reference.
