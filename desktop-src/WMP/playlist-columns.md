# <a name="playlistcolumns"></a>Список воспроизведения. столбцы

Атрибут **Columns** определяет столбцы, которые отображаются в элементе **списка воспроизведения** .

``` syntax
        elementID.columns
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи в следующем формате:

имя базы данных \_ = понятное \_ имя; имя базы данных \_ = понятное \_ имя;

ПОНЯТНОе \_ имя — это значение, отображаемое в заголовке столбца элемента списка воспроизведения, а имя базы данных \_ — значение из следующей таблицы. Обратите внимание, что элемент списка воспроизведения может быть элементом мультимедиа из библиотеки или с компакт-диска или другим **списком воспроизведения** , созданным пользователем или полученным с компакт-диска. Некоторые столбцы действительны только с определенными элементами списка воспроизведения, как указано в столбце Описание.

## <a name="remarks"></a>Комментарии

Если один из столбцов не существует в библиотеке, он остается пустым.

Может быть указано не более 31 столбцов.

При указании повторяющегося столбца данные в первом столбце будут выставляться по левому краю, но все остальные дублирующиеся столбцы будут выставляться по правому краю. Например, если имеется два столбца длительности, то данные будут выставляться по левому краю в первом и по правому краю во втором.
