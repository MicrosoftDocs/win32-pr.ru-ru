# <a name="viewresizable"></a>Просмотр. изменяемый размер

Атрибут с **изменяемым размером** извлекает значение, указывающее, можно ли изменить размер **представления** .

``` syntax
        elementID.resizable
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** , доступным только для чтения, со значением по умолчанию, равным атрибуту **заголовка** окна.



| Значения | Описание             |
|--------|-------------------------|
| true   | Можно изменить размер представления.    |
| false  | Невозможно изменить размер представления. |



 

## <a name="remarks"></a>Комментарии

Если **строка заголовка** отсутствует, и, следовательно, нет окна или границы, необходимо использовать метод **size** для изменения размера элемента **View** .

| Требование | Значение |
|--------------------|------------------------------------------------------|
