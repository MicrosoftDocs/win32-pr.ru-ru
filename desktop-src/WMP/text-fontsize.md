# <a name="textfontsize"></a>TEXT. fontSize

Атрибут **fontSize** указывает или получает размер шрифта для элемента управления "текст".

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное целое**), определяющим размер шрифта в пунктах. Он имеет значение по умолчанию 10.

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
