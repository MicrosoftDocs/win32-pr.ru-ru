# <a name="how-windows-media-download-packages-work-deprecated"></a>как работают пакеты загрузки мультимедиа Windows (не рекомендуется)

эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрыватель Windows Media и пакета SDK для проигрыватель Windows Media.

пакет загрузки Windows носителя запускается с веб-сайта, когда пользователь щелкает ссылку в веб-браузере, например Microsoft Internet Explorer. это действие открывает проигрыватель Windows Media, а затем скачивает и отменяет упаковку пакета скачивания Windows носителя на жестком диске пользователя в папке по умолчанию.

после извлечения файлов из пакета скачивания Windows носителя проигрыватель Windows Media находит файл списка воспроизведения Windows media metafile с расширением. asx в упакованных файлах. Если такой объект найден, проигрыватель создает список воспроизведения на основе включенного метафайла. Файлы, содержащие мультимедийное содержимое, затем добавляются в библиотеку.

проигрыватель Windows Media также выполняет поиск элемента **SKIN** в метафайле. Если элемент **Skin** содержит ссылку на файл границы с расширением WMZ, проигрыватель загружает границу **в область текущий** . Затем проигрыватель начинает воспроизводить содержимое, указанное в пакете.

на следующей схеме показано, как содержимое упаковано в файл скачивания Windowsного носителя, помещен на веб-сайт, скачан и воспроизведен на клиентском компьютере с помощью проигрыватель Windows Media.

![получение и воспроизведение файла загрузки Windows Media.](images/wmd-image.png) Windows Загрузка носителя

|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------|

Методы, используемые для создания границ, похожи на те, которые используются для создания обложек. | .wmz                                     |
