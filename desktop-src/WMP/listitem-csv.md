# <a name="listitemcsv"></a>listitem.csv

Этот файл содержит записи для каждого элемента, включенного в список. Каждая запись представляет собой строку, состоящую из полей, разделенных табуляцией, описанных в таблице ниже. Поля должны присутствовать в указанном порядке.

Все поля обязательные.

В столбце Format в следующей таблице описывается способ форматирования каждого текстового поля в Юникоде. Он не ссылается на тип данных содержимого. Например, если в столбце Format указано целое число, это означает, что поле содержит строку в Юникоде, представляющую целочисленное значение, а не фактическое целое число.



| Поле              | Формат                                          | Описание                                                                                                            |
|--------------------|-------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| Связанный \_ ListID     | Неотрицательное целое число. Пример: 465<br/>    | Идентификатор списка, частью которого является этот элемент.                                                                               |
| Связанный \_ ListItem   | Неотрицательное целое число. Пример: 684793<br/> | Идентификатор элемента. Может быть ИДЕНТИФИКАТОРом записи, альбома, исполнителя, жанра, субженре, канала радио или другого списка. |
| итемпоситионинлист | Неотрицательное целое число. Пример: 5<br/>      | Позиция элемента в списке. Первый элемент в списке находится в позиции 0.                                   |
