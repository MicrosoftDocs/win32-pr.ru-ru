# <a name="iwmpstringcollection2-vb-and-c-interface"></a>Интерфейс IWMPStringCollection2 (VB и C#)

Предоставляет методы, дополняют интерфейс **ивмпстрингколлектион** .

## <a name="members"></a>Элементы

Интерфейс **IWMPStringCollection2 (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IWMPStringCollection2 (VB и C#)** содержит следующие методы.



| Метод                                                                                                                 | Описание                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------|
| [**жетаттрибутекаунтбитипе**](wmplibiwmpstringcollection2-iwmpstringcollection2-getattributecountbytype--vb-and-c.md) | Возвращает число атрибутов, связанных с указанным индексом элемента коллекции строк, именем атрибута и языком.<br/> |
| [**getItemInfo**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfo--vb-and-c.md)                         | Возвращает строку, соответствующую указанному индексу элемента коллекции строк и имени.<br/>                                   |
| [**жетитеминфобитипе**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfobytype--vb-and-c.md)             | Возвращает значение, соответствующее указанному индексу элемента коллекции строк, имени, языку и индексу атрибута.<br/>        |
| [**Идентично**](wmplibiwmpstringcollection2-iwmpstringcollection2-isidentical--vb-and-c.md)                         | Возвращает значение, указывающее, совпадает ли указанный объект с текущим объектом.<br/>                                     |



 

Получите интерфейс **IWMPStringCollection2** путем приведения значения, возвращаемого методом [**ивмпмедиаколлектион. жетаттрибутестрингколлектион**](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection-getattributestringcollection) .
