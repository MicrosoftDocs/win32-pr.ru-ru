# <a name="building-a-visualization"></a>Создание визуализации

Вы можете создать пользовательскую визуализацию с помощью Visual Studio вместе с мастером подключаемых модулей проигрывателя Windows Media. Сведения о настройке мастера и создании подключаемого модуля см. в разделе [Создание подключаемого модуля](building-a-plug-in.md).

При запуске мастера вам будет предоставлено несколько различных типов подключаемых модулей. Выберите визуализация.

После завершения работы мастера и создания подключаемого модуля запустите песню в проигрывателе Windows Media и щелкните визуализации. Предустановки, которые автоматически создаются для вас, называются панелью *имя_проекта* и *имя_проекта* Wave, где *ProjectName* — это имя проекта.
