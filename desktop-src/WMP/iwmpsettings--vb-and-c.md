# <a name="iwmpsettings-vb-and-c-interface"></a>Интерфейс Ивмпсеттингс (VB и C#)

Предоставляет свойства и методы, которые получают или устанавливают значения параметров проигрывателя Windows Media.

Интерфейс **ивмпсеттингс** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **ивмпсеттингс (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмпсеттингс (VB и C#)** содержит следующие методы.



| Метод                                                               | Описание                                                                        |
|:---------------------------------------------------------------------|:-----------------------------------------------------------------------------------|
| [**Режим**](wmplibiwmpsettings-iwmpsettings-getmode--vb-and-c.md) | Возвращает значение, указывающее, активен ли режим цикла или случайный режим.<br/> |
| [**setMode**](wmplibiwmpsettings-iwmpsettings-setmode--vb-and-c.md) | Устанавливает режим цикла или случайный режим на активный или неактивный.<br/>               |
