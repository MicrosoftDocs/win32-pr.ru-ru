# <a name="library-integration"></a><span data-ttu-id="7f93e-112">Интеграция библиотеки</span><span class="sxs-lookup"><span data-stu-id="7f93e-112">Library Integration</span></span>

<span data-ttu-id="7f93e-113">Пользовательский интерфейс проигрывателя Windows Media организован в области функций, называемые *областями задач*, которые инкапсулируют различные функции высокого уровня программы.</span><span class="sxs-lookup"><span data-stu-id="7f93e-113">The Windows Media Player user interface is organized into feature areas, called *task panes*, which encapsulate the various high-level features of the program.</span></span> <span data-ttu-id="7f93e-114">К ним относятся **библиотеки**, **Синхронизация** и **запись** панелей задач (среди прочих).</span><span class="sxs-lookup"><span data-stu-id="7f93e-114">These include the **Library**, **Sync**, and **Burn** task panes (among others).</span></span> <span data-ttu-id="7f93e-115">Область задач **Библиотека** позволяет пользователям работать с библиотекой; область задач **Синхронизация** позволяет пользователям синхронизировать цифровые файлы мультимедиа с портативным устройством. и область задач **запись** позволяет пользователям записывать цифровые файлы мультимедиа на компакт-диск или DVD-диск.</span><span class="sxs-lookup"><span data-stu-id="7f93e-115">The **Library** task pane enables users to work with the library; the **Sync** task pane enables users to synchronize digital media files to a portable device; and the **Burn** task pane enables users to burn digital media files to a CD or DVD.</span></span>

> [!Note]  
> <span data-ttu-id="7f93e-116">Область задач **Библиотека** иногда называется область задач **Обзор** .</span><span class="sxs-lookup"><span data-stu-id="7f93e-116">The **Library** task pane is sometimes called the **Browse** task pane.</span></span>

 

<span data-ttu-id="7f93e-117">Каждая из этих областей задач имеет определенный уровень интеграции с библиотекой.</span><span class="sxs-lookup"><span data-stu-id="7f93e-117">Each of these task panes has some level of integration with the library.</span></span> <span data-ttu-id="7f93e-118">Например, если пользователь хочет записать музыку на компакт-диск, имеет смысл позволить пользователю выбрать музыку для записи, просмотрев библиотеку и просто перетащив мультимедийные элементы в список.</span><span class="sxs-lookup"><span data-stu-id="7f93e-118">For example, if the user wants to burn music to a CD, it makes sense to let the user choose the music to burn by browsing the library and simply dragging and dropping media items into a list.</span></span> <span data-ttu-id="7f93e-119">Это означает, что пользователи могут просматривать и использовать каталог интерактивного магазина, который полностью интегрирован в библиотеку при работе в областях задач **Библиотека**, **Синхронизация** и **запись** .</span><span class="sxs-lookup"><span data-stu-id="7f93e-119">This means that users can view and use an online store catalog that is fully integrated into the library when working in the **Library**, **Sync**, and **Burn** task panes.</span></span> <span data-ttu-id="7f93e-120">Перечисление [вмптасктипе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype) содержит значения, представляющие эти три области задач, чтобы их можно было идентифицировать программным способом.</span><span class="sxs-lookup"><span data-stu-id="7f93e-120">The [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype) enumeration contains values that represent these three task panes so they can be identified programmatically.</span></span>

<span data-ttu-id="7f93e-121">Каждая из этих трех областей задач состоит из трех основных частей.</span><span class="sxs-lookup"><span data-stu-id="7f93e-121">Each of these three task panes is organized into three major parts.</span></span> <span data-ttu-id="7f93e-122">Первая часть — это элемент управления древовидного представления библиотеки.</span><span class="sxs-lookup"><span data-stu-id="7f93e-122">The first part is the library tree-view control.</span></span> <span data-ttu-id="7f93e-123">Этот элемент управления предоставляет пользователю иерархическое представление библиотеки проигрывателя Windows Media, включая функции классификации по песням, исполнителям, альбомам и т. д.</span><span class="sxs-lookup"><span data-stu-id="7f93e-123">This control provides the user with a hierarchical view of the Windows Media Player library, including categorization features by song, artist, album, and so forth.</span></span> <span data-ttu-id="7f93e-124">Вторая часть области задач — это область сведений.</span><span class="sxs-lookup"><span data-stu-id="7f93e-124">The second task pane part is the details pane.</span></span> <span data-ttu-id="7f93e-125">Эта панель содержит подробные сведения, организованные в соответствии с категорией, выбранной в элементе управления деревом библиотеки.</span><span class="sxs-lookup"><span data-stu-id="7f93e-125">This pane provides detailed information organized according to the category currently selected in the library tree-view control.</span></span> <span data-ttu-id="7f93e-126">Например, если пользователь щелкнул **песни** в представлении в виде дерева, в области сведений отобразятся названия песен, находящиеся в библиотеке, а также другие сведения, такие как Длина и название альбома.</span><span class="sxs-lookup"><span data-stu-id="7f93e-126">For example, if the user has clicked **Songs** in the tree view, the details pane will show the song titles currently in the library, along with other information such as length and album title.</span></span> <span data-ttu-id="7f93e-127">Третья часть — панель списка или *Корзина*.</span><span class="sxs-lookup"><span data-stu-id="7f93e-127">The third part is the list pane, or *basket*.</span></span> <span data-ttu-id="7f93e-128">Пользователи могут перетаскивать элементы мультимедиа на корзину для создания списков, например списков воспроизведения, списков синхронизации и записи.</span><span class="sxs-lookup"><span data-stu-id="7f93e-128">Users can drag and drop media items onto the basket to build lists, such as playlists, sync lists, and burn lists.</span></span>

<span data-ttu-id="7f93e-129">Когда каталог Интернет-магазина интегрирован с библиотекой, Интернет-магазин отображается в виде категории верхнего уровня (или *узла*) в элементе управления "дерево библиотеки — представление".</span><span class="sxs-lookup"><span data-stu-id="7f93e-129">When an online store catalog is integrated with the library, the online store appears as a top-level category, or *node*, in the library tree-view control.</span></span> <span data-ttu-id="7f93e-130">(В каждый момент времени для пользователя отображается только один каталог Интернет-магазина.) Когда пользователь выбирает просмотр каталога Интернет-магазина, щелкнув узел, в области сведений отображаются сведения о музыке в каталоге интернет-магазина.</span><span class="sxs-lookup"><span data-stu-id="7f93e-130">(Only one online store catalog is visible to the user at a time.) <span data-ttu-id="7f93e-132">Узел Интернет-магазина верхнего уровня имеет набор дочерних узлов, предоставляемых проигрывателем Windows Media.</span><span class="sxs-lookup"><span data-stu-id="7f93e-132">The top-level online store node has a set of child nodes provided by Windows Media Player.</span></span> <span data-ttu-id="7f93e-133">Например, в узле Интернет-магазина верхнего уровня есть дочерние узлы Радио, исполнителей и альбомов, а также другие.</span><span class="sxs-lookup"><span data-stu-id="7f93e-133">For example, the top-level online store node has Radio, Artist, and Album child nodes, among others.</span></span> <span data-ttu-id="7f93e-134">Узел Интернет-магазина верхнего уровня также может иметь до восьми настраиваемых дочерних узлов, предоставляемых Интернет-магазином.</span><span class="sxs-lookup"><span data-stu-id="7f93e-134">The top-level online store node can also have up to eight custom child nodes provided by the online store.</span></span> <span data-ttu-id="7f93e-135">Проигрыватель Windows Media создает пользовательский дочерний узел для любого списка, имеющего идентификатор списка в диапазоне от 0 до 7.</span><span class="sxs-lookup"><span data-stu-id="7f93e-135">Windows Media Player creates a custom child node for any list that has a list identifier in the range 0 through 7.</span></span> <span data-ttu-id="7f93e-136">В Интернет-магазине указывается идентификатор списка в [list.csv](list-csv.md) файле, который является частью каталога магазина.</span><span class="sxs-lookup"><span data-stu-id="7f93e-136">The online store specifies the identifier of a list in the [list.csv](list-csv.md) file that is part of the store's catalog.</span></span>

<span data-ttu-id="7f93e-138">При переходе пользователя по каталогу проигрыватель Windows Media выполняет вызовы **ивмпконтентпартнер:: GetItemInfo** для получения метаданных из подключаемого модуля партнера по содержимому о музыкальных элементах, выбранных пользователем.</span><span class="sxs-lookup"><span data-stu-id="7f93e-138">As the user navigates through the catalog, Windows Media Player makes calls to **IWMPContentPartner::GetItemInfo** to retrieve metadata from the content partner plug-in about the music items the user selects.</span></span> <span data-ttu-id="7f93e-139">Эти метаданные предоставляют проигрывателю сведения, чтобы проигрыватель мог отображать сведения об элементах каталога.</span><span class="sxs-lookup"><span data-stu-id="7f93e-139">This metadata provides information to the Player so that the Player can display details about the catalog items.</span></span> <span data-ttu-id="7f93e-140">Например, если пользователь выбирает альбом, проигрыватель Windows Media извлекает URL-адрес обложки альбома, чтобы пользователь мог видеть обложку альбома.</span><span class="sxs-lookup"><span data-stu-id="7f93e-140">For example, if the user selects an album, Windows Media Player retrieves the album art URL so that the user can see the album cover art.</span></span>
