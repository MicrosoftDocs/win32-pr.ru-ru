# <a name="object-model-reference-for-c"></a>Справочник по объектной модели для C++

> [!Note]  
> использование элемента управления ActiveX проигрыватель Windows Media с C++ поддерживается для серии проигрыватель Windows Media 9 или более поздних версий.

 

справочник по объектной модели для C++ содержит документацию для проигрыватель Windows Media и проигрыватель Windows Media мобильных интерфейсов и типов перечислений.

если вы хотите использовать удаленное взаимодействие или обложки с элементом управления проигрыватель Windows Media, вам также потребуется изучить интерфейсы **ивмпплайерсервицес** и **ивмпремотемедиасервицес** , предназначенные для использования с C++.

демонстрацию внедрения элемента управления проигрыватель Windows Media в программу на c++ см. в примерах c++, входящих в состав этого пакета SDK.

в следующих разделах описываются интерфейсы и перечисления C++, предоставляемые объектной моделью проигрыватель Windows Media.



| Section                                    | Описание                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Интерфейсы](interfaces.md)               | список интерфейсов и методов, доступных для программистов C++, использующих элемент управления проигрыватель Windows Media ActiveX. |
| [Типы перечислений](enumeration-types.md) | Перечисляет типы перечисления, доступные в коде C++.                                                              |
