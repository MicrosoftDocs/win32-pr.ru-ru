# <a name="infocenter-element"></a>Элемент справочного центра

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

элемент " **информационный центр** " указывает URL-адрес веб-страницы, которая проигрыватель Windows Media отображается в представлении центра сведений в **воспроизведении** , когда интернет-магазин активен.

``` syntax
<InfoCenter
    URL = "URL"
/>
```

## <a name="remarks"></a>Remarks

Пользователь управляет, когда представление центра сведений активно.

чтобы получить сведения о воспроизводимом в данный момент элементе мультимедиа, необходимо внедрить экземпляр элемента управления проигрыватель Windows Media на веб-странице и использовать объектную модель проигрывателя.

В следующей таблице приведены параметры, отправленные с помощью запроса URL-адреса. Другие могут присутствовать в целях совместимости с прежними версиями.



| Имя         | Значение                                                                                                                                                               |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *графическом*      | идентификатор географического расположения Windows. Идентификатор расположения задается пользователем в области **Расположение** в разделе региональные и языковые параметры панели управления. |
| *locale*     | проигрыватель Windows Media код локали.                                                                                                                                     |
| *UserLocale* | Windows код локали. Языковой стандарт задается пользователем в области **стандарты и форматы** языковых и региональных параметров на панели управления.        |
| *version*    | проигрыватель Windows Media номер версии в следующем формате: 10.0. x. xxxx или 11.0. x. xxxx.                                                                         |
