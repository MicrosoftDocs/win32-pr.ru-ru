# <a name="listboxdeleteitem"></a>LISTBOX. deleteItem

Метод **deleteItem** удаляет элемент управления "список" по указанному индексу.

``` syntax
        elementID.deleteItem(index)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число** (**Long**), содержащее индекс удаляемого элемента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Строки под удаленной строкой будут перемещаться вверх на одно значение индекса.
