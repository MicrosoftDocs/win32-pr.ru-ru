<span data-ttu-id="1d761-109">Образец подключаемого модуля DSP, созданный с помощью мастера подключаемых модулей проигрывателя Windows Media, не требует дополнительных буферов потоковой передачи.</span><span class="sxs-lookup"><span data-stu-id="1d761-109">The sample DSP plug-in generated by the Windows Media Player Plug-in Wizard does not require any additional streaming buffers.</span></span> <span data-ttu-id="1d761-110">Однако может потребоваться выделить ресурсы памяти для подключаемого модуля DSP.</span><span class="sxs-lookup"><span data-stu-id="1d761-110">However, you might want to allocate memory resources for your DSP plug-in.</span></span> <span data-ttu-id="1d761-111">Например, подключаемый модуль, который создает эхо-результат, потребует дополнительного буфера для создания необходимой временной задержки.</span><span class="sxs-lookup"><span data-stu-id="1d761-111">For example, a plug-in that produces an echo effect would require a secondary buffer to create the necessary time delay.</span></span>
