# <a name="about-the-metadatatext-object"></a>Сведения об объекте Метадататекст

Некоторые атрибуты метаданных, такие как **WM/песен \_ синхронизированы**, содержат текст, сопровождающий описание текста. Объект **метадататекст** предоставляет способ извлечения текста и описания отдельно. При использовании *носителя*. **жетитеминфобитипе** для получения сведений о сложных текстовых атрибутах извлекается объект **метадататекст** со свойствами, которые содержат отдельные данные.
