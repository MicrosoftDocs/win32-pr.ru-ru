# <a name="input-settings"></a>Входные параметры

Следующие глобальные константы используются для задания входных параметров для модуля записи.



| Глобальная константа                        | \_ \_ тип данных ВМТ attr                                                                                                                       | Описание параметра *pValue*                                                                                                                                                                                                                                                 |
|----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ всздеинтерлацемоде                  | **ВМТ \_ Введите \_** в разделе DWORD значение одного из значений в таблице Mode раздела [, чтобы выполнить дечередование видео](to-deinterlace-video.md).            | Если задано значение, задает тип перечередования содержимого входных данных. Дополнительные сведения см. в статье о [деразвертке видео](to-deinterlace-video.md).                                                                                                                            |
| g \_ всзфикседфрамерате                   | **\_bool типа \_ ВМТ**                                                                                                                       | Если задано значение true, то кодек не удаляет какие бы то ни было кадры во время кодирования. Это приведет к тому, что [*Частота кадров*](wmformat-glossary.md) выходного видеопотока будет постоянной. Частота кадров входного потока не должна быть постоянной. |
| g \_ всзинитиалпаттернфоринверсетелеЦине | **ВМТ \_ Введите \_ DWORD** в качестве значения для одного из значений в исходной таблице шаблона в разделе, [чтобы выполнить дечередование видео](to-deinterlace-video.md). | Если для режима чередования задано значение "инверсетелеЦине" в формате WM \_ DM \_ UNчересстрочную развертка \_ , это можно задать для указания шаблона входных данных [*чересстрочности*](wmformat-glossary.md) . Дополнительные сведения см. в статье о [деразвертке видео](to-deinterlace-video.md).        |
| g \_ всзинтерлацедкодинг                 | **\_bool типа \_ ВМТ**                                                                                                                       | При установке значения true указывает, что кодек должен кодировать поток как содержимое с чередованием. Дополнительные сведения см. [в разделе Использование чередования видео](to-use-interlaced-video.md).                                                                                       |
| g \_ всзжпегкомпрессионкуалити           | **ВМТ, \_ тип \_ DWORD**                                                                                                                      | Задает уровень качества JPEG (от 1 до 100), используемый для входных данных.                                                                                                                                                                                               |
| g \_ всзватермаркклсид                   | **\_GUID типа \_ ВМТ**                                                                                                                       | В качестве значения задается идентификатор GUID водяного знака.                                                                                                                                                                                                                                 |
| g \_ всзватермаркконфиг                  | **\_Строка типа \_ ВМТ**                                                                                                                     | В качестве значения задается конфигурация водяного знака. Это значение будет зависеть от подложки DMO. Дополнительные сведения см. в документации по подложке.                                                                                   |



 

> [!Note]  
> Входные параметры, настроенные для потока, не сохраняются в записанном файле. Если вы хотите, чтобы пользовательское средство чтения имело доступ к этим параметрам кодирования, необходимо создать настраиваемые атрибуты, чтобы сохранить их в заголовке файла.
