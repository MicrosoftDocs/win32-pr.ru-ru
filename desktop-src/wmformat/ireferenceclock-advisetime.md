# <a name="ireferenceclockadvisetime-method"></a>Метод Иреференцеклокк:: Адвисетиме

Метод **адвисетиме** запрашивает асинхронное уведомление о том, что прошло время.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AdviseTime(
  [in]  REFERENCE_TIME rtBaseTime,
  [in]  REFERENCE_TIME rtStreamTime,
  [in]  HEVENT         hEvent,
  [out] DWORD          *pdwAdviseCookie
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ртбасетиме* \[ окне\]
</dt> <dd>

Базовое время ссылки в единицах измерения 100-наносекундных.

</dd> <dt>

*ртстреамтиме* \[ окне\]
</dt> <dd>

Время смещения потока (в 100-наносекундных единицах).

</dd> <dt>

*Хевент* \[ окне\]
</dt> <dd>

Обработчик события, созданный вызывающим объектом. Это событие будет сигнальным по истечении указанного времени.

</dd> <dt>

*пдвадвисекукие* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает идентификатор для запроса. Он используется для обнаружения этого вызова в **адвисетиме** в будущем, например для отмены запроса.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                               | Описание                                             |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>      | Метод выполнен успешно.<br/>                        |
| <dl> <dt>**\_указатель E**</dt> </dl> | Параметр *пдвадвисекукие* имеет **значение NULL**.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>    | Неопределенный сбой.<br/>                         |
