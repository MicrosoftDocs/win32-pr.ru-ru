# <a name="output-settings"></a>Параметры вывода

Следующие глобальные константы используются для задания параметров вывода для модуля чтения и синхронного объекта чтения.



| Глобальная константа                | \_ \_ тип данных ВМТ attr  | Описание параметра *pValue*                                                                                                                                                                                                                                                                                                    |
|--------------------------------|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ всзалловинтерлацедаутпут    | **\_bool типа \_ ВМТ**  | Если значение — true, модуль чтения будет доставлять кадры с чередованием, если они поддерживаются выходными данными.                                                                                                                                                                                                                                            |
| g \_ всздедикатедделиверисреад  | **\_bool типа \_ ВМТ**  | Если значение — true, этот выход будет иметь выделенный поток, созданный для доставки своих выборок. Не поддерживается в синхронном модуле чтения.                                                                                                                                                                                            |
| g \_ всзделиверонрецеиве         | **\_bool типа \_ ВМТ**  | Если значение — true, выборки для этого выхода будут доставляться сразу после того, как они будут доступны из средства чтения. Это может привести к тому, что образцы из этого выхода будут доставлены не поочередно и до соответствующих образцов из других выходных данных.                                                                                            |
| g \_ всздинамикранжеконтрол      | **ВМТ, \_ тип \_ DWORD** | Задает уровень управления динамическим диапазоном, используемый для выходных данных. Задайте значение от 0 до 2, где 0 означает отсутствие динамического управления диапазоном (по умолчанию), а 2 — максимальный уровень динамического управления диапазоном (самый маленький динамический диапазон).                                                                                |
| g \_ всзеарлидатаделивери        | **ВМТ, \_ тип \_ DWORD** | Время в миллисекундах, которое указывает, сколько времени раньше было доставлено выборок. Если значение больше нуля, образцы из этого результата будут получены и декодированы, чтобы образцы были доставлены раньше, чем примеры для других выходных данных. Обычно читатель предоставляет образцы в порядке времени презентации.         |
| g \_ всзенабледискретеаутпут     | **\_bool типа \_ ВМТ**  | Если значение — true, модуль чтения включит многоканальные звуковые выходные данные с высоким уровнем четкости. Этот параметр действителен только для звуковых потоков, закодированных с помощью кодека Windows Media Audio 9 Professional. Если этот параметр имеет значение true, необходимо также указать конфигурацию динамика клиентского компьютера, задав параметр g \_ всзспеакерконфиг. |
| g \_ всзенаблефрамеинтерполатион | **\_bool типа \_ ВМТ**  | Если значение равно true, кодек будет доставлять видеопоток с более высокой [*частотой кадров*](wmformat-glossary.md), что приводит к интерполяции кадров алгоритмически.                                                                                                                                                          |
| g \_ всзжустинтимедекоде         | **\_bool типа \_ ВМТ**  | Если значение равно true, данные должны быть декодированы как можно позднее. Не поддерживается в синхронном модуле чтения.                                                                                                                                                                                                                            |
| g \_ всзнидспревиауссампле      | **\_bool типа \_ ВМТ**  | Если значение — true, пример требует распаковки предыдущего примера. Этот параметр применяется только к разностным кадрам в сжатом видео и доступен только для чтения.                                                                                                                                                                       |
| g \_ всзскрамбледаудио           | **\_bool типа \_ ВМТ**  | Если задано значение true, то в выходных данных будет использоваться схема скрытия перешифрованного звука. Это допустимый параметр только для звуковых выходов.                                                                                                                                                                                                |
| g \_ всзсинглеаутпутбуффер       | **\_bool типа \_ ВМТ**  | Если значение — true, необходимо использовать один выходной буфер (например, видеобуферы DirectDraw®). Не поддерживается в синхронном модуле чтения.                                                                                                                                                                                           |
| g \_ всзсофтварескалинг          | **\_bool типа \_ ВМТ**  | Если значение равно false, видео не масштабируется. (В решении не должно быть изменений.)                                                                                                                                                                                                                                                |
| g \_ всзспеакерконфиг            | **ВМТ, \_ тип \_ DWORD** | Если многоканальное декодирование звука включено с помощью параметра g \_ всзенабледискретеаутпут, этот параметр указывает конфигурацию динамика клиентского компьютера. Задайте одну из констант конфигурации динамика DirectSound.                                                                                                   |
| g \_ всзстреамлангуаже           | **\_слово типа \_ ВМТ**  | Индекс в списке языков того языка, который должен быть доставлен для этого выхода. Используется для выходов, представляющих потоки, взаимоисключающих по языку.                                                                                                                                                                      |
| g \_ всзвидеосампледуратионс     | **\_bool типа \_ ВМТ**  | Если значение равно true, средство чтения будет доставлять точную длительность выборки.                                                                                                                                                                                                                                                                |
| g \_ всзенаблевмапроспдифаутпут  | **\_bool типа \_ ВМТ**  | Если значение — true, модуль чтения будет включать формат цифрового интерфейса Sony/Филипс (S/PDIF) в перечислимые типы вывода.                                                                                                                                                                                                       |
