# <a name="wmaudiosourceurl"></a>WM/Аудиосаурцеурл

Атрибут **WM/аудиосаурцеурл** содержит адрес официальной веб-страницы исходного носителя. Например, песня из звуковой дорожки может иметь ссылку на официальную страницу фильма.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмаудиосаурцеурл

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**
