# <a name="dialog-box-styles"></a>Стили диалоговых окон

В следующей таблице перечислены стили диалоговых окон, которые можно указать при создании диалогового окна. Эти стили можно использовать в вызовах функций [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) и [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) в элементе **Style** структур [**длгтемплате**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) и [**длгтемплатикс**](dlgtemplateex.md) , а также в инструкции определения диалогового окна в файле ресурсов.



| Константа                                                                                                                                                               | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**\_3DLOOK DS**</dt> </dl>                      | Является устаревшей. Система автоматически применяет трехмерный вид для диалоговых окон, созданных приложениями.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**\_АБСАЛИГН DS**</dt> </dl>                | Указывает, что координаты диалогового окна являются экранными координатами. Если этот стиль не указан, координаты являются клиентскими координатами.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**\_центр DS**</dt> </dl>                      | Центрирование диалогового окна в рабочей области монитора, содержащей окно «владелец». Если окно владелец не указано, диалоговое окно центрируется в рабочей области монитора, определенной системой. Рабочая область — это область, которая не скрывается панелью задач или AppBar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**\_ЦЕНТЕРМАУСЕ DS**</dt> </dl>       | Центрирование диалогового окна в курсоре мыши.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**\_КОНТЕКССЕЛП DS**</dt> </dl>       | Содержит вопросительный знак в заголовке диалогового окна. Когда пользователь нажимает на вопросительный знак, курсор превращается в вопросительный знак с указателем. Если пользователь щелкает элемент управления в диалоговом окне, элемент управления получает сообщение [**\_ справки WM**](../shell/wm-help.md) . Элемент управления должен передать сообщение процедуре диалогового окна, которая должна вызывать функцию с помощью команды **Help \_ WM \_ Help** . В приложении "Справка" отображается всплывающее окно, которое обычно содержит справку для элемента управления.<br/> Обратите внимание, что **\_ контексселп DS** — это только заполнитель. При создании диалогового окна система проверяет наличие **DS \_ контексселп** и, если она есть, добавляет **WS \_ ex \_ контексселп** в расширенный стиль диалогового окна. **Служба WS \_ EX \_ контексселп** нельзя использовать с стилями **WS \_ максимизебокс** и **WS \_ минимизебокс** .<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**\_элемент управления DS**</dt> </dl>                   | Создает диалоговое окно, которое хорошо работает как дочернее окно другого диалогового окна, во многом похоже на страницу в окне свойств. Этот стиль позволяет пользователю переходить между окнами управления дочернего диалогового окна, использовать его сочетания клавиш и т. д.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**\_ФИКСЕДСИС DS**</dt> </dl>                | Заставляет диалоговое окно использовать **\_ фиксированный системный \_ Шрифт** вместо **системного \_ шрифта** по умолчанию. Это очень пробельный шрифт, совместимый с системным шрифтом в 16-разрядных версиях Windows более 3,0.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**\_ЛОКАЛЕДИТ DS**</dt> </dl>             | Применяется только к 16-разрядным приложениям. Этот стиль направляет элементы управления "поле ввода" в диалоговом окне для выделения памяти из сегмента данных приложения. В противном случае элементы управления "Изменить" выделяют хранилище из глобального объекта памяти.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**\_МОДАЛФРАМЕ DS**</dt> </dl>          | Создает диалоговое окно с модальным фреймом диалогового окна, которое можно объединить с заголовком окна и меню окон путем указания стилей **WS \_ Caption** и **WS \_ сисмену** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**\_НОФАИЛКРЕАТЕ DS**</dt> </dl>    | Создает диалоговое окно, даже если возникают ошибки, например, если дочернее окно не может быть создано или система не может создать специальный сегмент данных для элемента управления "поле ввода".<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**\_НОИДЛЕМСГ DS**</dt> </dl>             | Подавляет сообщения [**WM \_ ентеридле**](wm-enteridle.md) , которые система в противном случае отправляет владельцу диалогового окна во время отображения диалогового окна.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**\_СЕТФОНТ DS**</dt> </dl>                   | Указывает, что заголовок шаблона диалогового окна (стандартный или расширенный) содержит дополнительные данные, указывающие шрифт, используемый для текста в клиентской области и элементах управления диалогового окна. По возможности система выбирает шрифт в соответствии с указанными данными шрифта. Система передает указатель на шрифт в диалоговое окно и каждый элемент управления, отправляя им сообщение [**\_ сетфонт WM**](/windows/desktop/winmsg/wm-setfont) . Описание формата данных этого шрифта см. в разделе [**длгтемплате**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) and [**длгтемплатикс**](dlgtemplateex.md).<br/> Если не указаны ни **DS \_ сетфонт** , ни **DS \_ шеллфонт** , шаблон диалогового окна не включает данные шрифта.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**\_СЕТФОРЕГРАУНД DS**</dt> </dl> | Заставляет систему использовать функцию [**сетфореграундвиндов**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) для переноса диалогового окна на передний план. Этот стиль удобен для модальных диалоговых окон, которые требуют немедленного вмешательства пользователя независимо от того, является ли окно владельца передним.<br/> Система позволяет определить, какие процессы могут устанавливать окно переднего плана. Дополнительные сведения см. в разделе [окна переднего плана и фона](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**\_ШЕЛЛФОНТ DS**</dt> </dl>             | Указывает, что диалоговое окно должно использовать системный шрифт. Элемент **гарнитуры** шаблона расширенного диалогового окна должен иметь значение MS Shell Dlg. В противном случае этот стиль не оказывает никакого влияния. Также рекомендуется использовать [ресурс диаложекс](../menurc/dialogex-resource.md), а не [ресурс диалогового окна](../menurc/dialog-resource.md). Дополнительные сведения см. в разделе [Шрифты диалогового окна](about-dialog-boxes.md).<br/> Система выбирает шрифт, используя данные шрифта, указанные в элементах **PointSize**, **Weight** и **Italic** . Система передает указатель на шрифт в диалоговое окно и каждый элемент управления, отправляя им сообщение [**\_ сетфонт WM**](/windows/desktop/winmsg/wm-setfont) . Описание формата данных этого шрифта см. в разделе [**длгтемплатикс**](dlgtemplateex.md). <br/> Если не указаны ни **DS \_ шеллфонт** , ни **DS \_ сетфонт** , расширенный шаблон диалогового окна не включает данные шрифта.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**\_СИСМОДАЛ DS**</dt> </dl>                | Этот стиль устарел и включен для обеспечения совместимости с 16-разрядными версиями Windows. Если задан этот стиль, система создает диалоговое окно с **\_ \_ верхним стилем WS ex** . Этот стиль не запрещает пользователю доступ к другим окнам на рабочем столе.<br/> Не следует объединять этот стиль с стилем **\_ элемента управления DS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
