# <a name="iamtimelinesrcsplicewithnext-method"></a>Метод Иамтимелинесрк:: Сплицевиснекст

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`SpliceWithNext`Метод присоединяет исходный объект к другому исходному объекту.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SpliceWithNext(
   IAMTimelineObj *pNext
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пнекст* 
</dt> <dd>

Указатель на интерфейс [**иамтимелинеобж**](iamtimelineobj.md) исходного объекта, который необходимо присоединить к текущему источнику.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные возвращаемые значения включают следующее.



| Код возврата                                                                                   | Описание                                                              |
|-----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Успешно.<br/>                                                      |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Недопустимый аргумент.<br/>                                             |
| <dl> <dt>**\_интерфейс E**</dt> </dl> | Объект, указанный параметром *пнекст* , не является исходным объектом.<br/> |
| <dl> <dt>**\_указатель E**</dt> </dl>     | **Пустой** аргумент указателя.<br/>                                    |



 

## <a name="remarks"></a>Комментарии

Как уже реализовано, этот метод отклоняет любые эффекты в *пнекст*.

Чтобы этот метод был выполнен, *пнекст* должен быть кадром соответствия текущего исходного объекта, который определяется следующим образом:

-   Он должен использовать один и тот же исходный файл.
-   Время запуска носителя должно совпадать с временем окончания текущего источника.
-   Скорость воспроизведения должна быть одинаковой. Скорость воспроизведения — длительность носителя, деленная на длительность временной шкалы.
