# <a name="cdeferredcommandgetresult-method"></a>Кдеферредкомманд. метод Result

`GetResult`Метод получает список результирующих аргументов, если он существует.

## <a name="syntax"></a>Синтаксис


```C++
VARIANT* GetResult();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает указатель на **значение типа Variant** , содержащее список аргументов метода, если он существует.
