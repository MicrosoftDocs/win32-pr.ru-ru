# <a name="overlay-mixer-filter"></a><span data-ttu-id="e4181-103">Фильтр смешивания наложений</span><span class="sxs-lookup"><span data-stu-id="e4181-103">Overlay Mixer Filter</span></span>

<span data-ttu-id="e4181-104">Фильтр микшера наложения — это модуль подготовки видео, разработанный специально для воспроизведения DVD-дисков и широковещательных видеопотоков с закрытым субтитром Line-21.</span><span class="sxs-lookup"><span data-stu-id="e4181-104">The Overlay Mixer filter is a video renderer designed specifically for DVD playback and broadcast video streams with line-21 closed captioning.</span></span> <span data-ttu-id="e4181-105">Микшер наложения также поддерживает расширения видеопортов (Впес), позволяя работать с аппаратными декодерами MPEG-2 или аналоговыми ТЕЛЕВИЗИОНными тюнерами, которые отправляют видео непосредственно на графическую карту, а не через шину PCI.</span><span class="sxs-lookup"><span data-stu-id="e4181-105">The Overlay Mixer also supports Video Port Extensions (VPEs), enabling it to work with hardware MPEG-2 decoders or analog TV tuners that send video directly to the graphics card, rather than over the PCI bus.</span></span>

> [!Note]  
> <span data-ttu-id="e4181-106">Модуль [подготовки видео 9](video-mixing-renderer-filter-9.md) теперь является предпочтительным для фильтра микшера наложения, за исключением сценариев впе.</span><span class="sxs-lookup"><span data-stu-id="e4181-106">The [Video Mixing Renderer 9](video-mixing-renderer-filter-9.md) is now preferred over the Overlay Mixer filter, except in VPE scenarios.</span></span>

### <a name="remarks"></a><span data-ttu-id="e4181-147">Комментарии</span><span class="sxs-lookup"><span data-stu-id="e4181-147">Remarks</span></span>

<span data-ttu-id="e4181-148">Микшер наложения использует сочетание цветов назначения для смешивания поверхностей видео с наложенными слоями.</span><span class="sxs-lookup"><span data-stu-id="e4181-148">The Overlay Mixer uses destination color keying to mix video surfaces with overlays.</span></span> <span data-ttu-id="e4181-149">Он блитс цветовой ключ и вторичное видео на основную поверхность и отправляет основное видео в область наложения.</span><span class="sxs-lookup"><span data-stu-id="e4181-149">It blits the color key and the secondary video to the primary surface, and sends the primary video to the overlay surface.</span></span> <span data-ttu-id="e4181-150">Затем графическая карта комбинирует две поверхности в буфер фрейма.</span><span class="sxs-lookup"><span data-stu-id="e4181-150">The graphics card then composites the two surfaces into its frame buffer.</span></span>
