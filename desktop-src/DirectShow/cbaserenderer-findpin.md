# <a name="cbaserendererfindpin-method"></a>Кбасерендерер. Финдпин, метод

`FindPin`Метод извлекает ПИН-код с указанным идентификатором.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindPin(
   LPCWSTR Id,
   IPin    **ppPin
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Id* 
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой расширенных символов, определяющую ПИН-код. Должен быть L "in".

</dd> <dt>

*пппин* 
</dt> <dd>

Адрес переменной, которая получает указатель на интерфейс [**Ипин**](/windows/desktop/api/Strmif/nn-strmif-ipin) ПИН-кода. Если метод завершается с ошибкой, *\* пппин* имеет значение **null**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                       | Описание                           |
|---------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>              | Успешно.<br/>                   |
| <dl> <dt>**\_указатель E**</dt> </dl>         | **Пустой** аргумент указателя.<br/> |
| <dl> <dt>**VFW \_ E \_ не \_ найден**</dt> </dl> | Не найдено.<br/>                 |



 

## <a name="remarks"></a>Комментарии

Этот метод переопределяет метод [**кбасефилтер:: финдпин**](cbasefilter-findpin.md) . ПИН-код фильтра (входной ПИН-код) называется "in".
