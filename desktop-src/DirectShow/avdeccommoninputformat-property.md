# <a name="avdeccommoninputformat-property"></a>Авдеккоммонинпутформат, свойство

Задает текущий формат входных данных для декодера.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдеккоммонинпутформат**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является строкой **BSTR** , содержащей строковое представление идентификатора GUID. Определены следующие идентификаторы GUID.



| **GUID**                                            | Описание                                    |
|-----------------------------------------------------|------------------------------------------------|
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутаак**              | Расширенное аудио кодирование (AAC)                    |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутдолбидигиталплус** | Dolby Digital Plus Audio                       |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутдолби**            | Dolby Audio                                    |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутдтс**              | Аудио службы DTS                                      |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпусеаак**            | High-Efficiency расширенное аудио кодирование (HE-AAC) |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутмпег**             | Аудио MPEG                                     |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутпкм**              | Звук PCM                                      |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутвма**              | Windows Media Audio                            |
| **КОДЕКАПИ \_ GUID \_ авдекаудиоинпутвмапро**           | Windows Professional Media Audio 9 (WMA Pro)   |
