# <a name="directshow-error-and-success-codes"></a>Коды ошибок и успешности DirectShow

В этом разделе перечислены коды возврата **HRESULT** , определенные для DirectShow. Обратите внимание, что API-интерфейсы DirectShow могут возвращать другие значения **HRESULT** , не показанные здесь. Дополнительные значения **HRESULT** см. в разделе [коды ошибок COM](/windows/desktop/com/com-error-codes).

В следующей таблице приведены коды ошибок и успешности выполнения DirectShow в числовом порядке. Коды возврата, начинающиеся с VFW \_ S, \_ указывают на успешное выполнение. Операция может быть частично выполнена, и по-прежнему будет возвращено успешное возвращаемое значение. Например, в **VFW \_ S \_ некоторые \_ данные \_ игнорируются** , что означает, что файл был визуализирован путем игнорирования некоторых параметров свойств. Коды возврата, начинающиеся с VFW \_ E, \_ указывают на ошибку.

Эти значения определены в файле заголовка Вфвмсгс. h.
