# <a name="playperiodintitleautostop-method"></a>Метод Плайпериодинтитлеаутостоп

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`PlayPeriodInTitleAutoStop`Метод запускает воспроизведение в указанное время в заданном заголовке до указанного времени окончания.

``` syntax
MSWebDVD.PlayPeriodInTitleAutoStop(iTitle, sStartTime, sEndTime)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*ититле*
</dt> <dd>

Задает заголовок в виде целого числа.

</dd> <dt>

<span id="sStartTime"></span><span id="sstarttime"></span><span id="SSTARTTIME"></span>*сстарттиме*
</dt> <dd>

Указывает время начала в виде строки в формате "чч: мм: СС: FF" (с указанием часов, минут, секунд, кадров).

</dd> <dt>

<span id="sEndTime"></span><span id="sendtime"></span><span id="SENDTIME"></span>*сендтиме*
</dt> <dd>

Задает время окончания в виде строки в формате "чч: мм: СС: FF".

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
