# <a name="llmuldiv-function"></a>Функция Ллмулдив

`llMulDiv`Функция реализует формулу, `((a*b)+rnd)/c` в которой каждый термин является 64-битным значением.

Метки времени и время поиска представляют собой 64-битные значения, поэтому эта функция полезна для выполнения преобразований в 32-разрядных системах. Например, формула для байт в секунду —


```C++
(Number of Bytes * Reference Time) / 10,000,000
```



который можно вычислить как `llMulDiv(nBytes, rtTime, 10000000, 0)` . Используйте параметр *Rnd* в качестве фактора округления.

## <a name="syntax"></a>Синтаксис


```C++
LONGLONG WINAPI Int64x32Div32(
   LONGLONG a,
   LONGLONG b,
   LONGLONG c,
   LONGLONG rnd
);
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает либо `(a * b + rnd)/c` вычисление, либо одно из следующих значений.



| Код возврата                                                                                       | Описание                                                              |
|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| <dl> <dt>**0x7FFFFFFFFFFFFFFF**</dt> </dl> | Произошло переполнение, так как результат слишком велик (положительный).<br/> |
| <dl> <dt>**0x8000000000000000**</dt> </dl> | Произошло переполнение, так как результат слишком большой (отрицательный).<br/> |
