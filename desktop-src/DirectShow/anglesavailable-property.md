# <a name="anglesavailable-property"></a>Англесаваилабле, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`AnglesAvailable`Свойство получает количество доступных в данный момент углов.

``` syntax
[ iAngles = ] MSWebDVD.AnglesAvailable
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение от 1 до 9, представляющее число доступных в данный момент углов. If


```
iAngles
```



= 1, в текущем расположении отсутствует блок Angle.

## <a name="remarks"></a>Комментарии

Это свойство доступно только для чтения и не имеет значения по умолчанию. *Блок угла* — это сегмент видео, который был снимком более чем одного угла камеры. В блоке углов может содержаться до девяти углов, пронумерованных от 1 до 9. Когда Навигатор DVD впервые входит в блок угла, он отправляет приложению уведомление о том, \_ что \_ \_ число углов в *lParam1*. Можно создать диск, чтобы автоматически отображать меню для доступных углов при входе в блок угла, но обычно приложение должно определить число доступных углов и предложить пользователю выбрать один из них.
