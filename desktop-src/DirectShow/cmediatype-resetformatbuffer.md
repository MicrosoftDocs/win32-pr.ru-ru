# <a name="cmediatyperesetformatbuffer-method"></a>Кмедиатипе. Ресетформатбуффер, метод

`ResetFormatBuffer`Метод удаляет блок формата. Метод задает для элементов **кбформат** и **пбформат** структуры **\_ \_ типа мультимедиа AM** **значение NULL**.

## <a name="syntax"></a>Синтаксис


```C++
void ResetFormatBuffer();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
