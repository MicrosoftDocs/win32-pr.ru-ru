# <a name="directshow-system-overview"></a>DirectShow Обзор системы

**Задача мультимедиа**

Работа с мультимедиа представляет ряд основных проблем:

-   Мультимедийные потоки содержат большие объемы данных, которые должны обрабатываться очень быстро.
-   Необходимо синхронизировать аудио и видео, чтобы они были запущены и остановлены одновременно и воспроизводились с одинаковой скоростью.
-   Данные могут поступать из многих источников, включая локальные файлы, компьютерные сети, телевизионные трансляции и видеокамеры.
-   Данные поступают в различных форматах, таких как Audio-Video с чередованием (AVI), расширенный потоковый формат (ASF), Группа экспертов по воспроизведению изображений (MPEG) и цифровое видео (DV).
-   Программист не знает заранее, какие аппаратные устройства будут присутствовать в системе конечного пользователя.

**DirectShow решение**

DirectShow предназначены для решения каждой из этих задач. основная цель проектирования — упростить задачу создания мультимедийных приложений на Windowsной платформе путем изоляции приложений от сложных транспортов данных, аппаратных различий и синхронизации.

для обеспечения пропускной способности, необходимой для потоковой передачи видео и аудио, DirectShow использует Direct3D и DirectSound, когда это возможно. Эти технологии эффективно отображают данные для звуковых и графических плат пользователя. DirectShow синхронизирует воспроизведение, инкапсулируются данные мультимедиа в образцах с отметкой времени. для поддержки различных источников, форматов и устройств, которые можно использовать, DirectShow использует модульную архитектуру, в которой приложение смешивается и соответствует различным программным компонентам, называемым *фильтрами*.

DirectShow предоставляет фильтры, поддерживающие запись и настройку устройств на основе WDM (WDM), а также фильтры, поддерживающие более старое видео для карт записи Windows (VfW), и кодеки, написанные для интерфейсов диспетчера аудиосжатия (ACM) и диспетчера сжатия видео (вкм).

как показано здесь, DirectShow фильтры взаимодействуют с самыми разнообразными устройствами, в том числе с помощью локальной файловой системы, тв-тюнера и видеозаписей, кодеков VfW, экрана видео (с помощью DirectDraw или GDI) и звуковой карты (с помощью DirectSound). таким же DirectShow изолирует приложение от многих сложностей этих устройств. DirectShow также предоставляет фильтры машинного сжатия и распаковки для определенных форматов файлов.
