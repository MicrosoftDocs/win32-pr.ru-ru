# <a name="avencmpaemphasistype-property"></a>Авенкмпаемфасистипе, свойство

Указывает тип фильтра отмены выделения, который должен использоваться при декодировании. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпаемфасистипе**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпаемфасистипе**](/windows/win32/api/codecapi/ne-codecapi-eavencmpaemphasistype) .

## <a name="remarks"></a>Комментарии

Это свойство задает биты выделения в заголовке фрейма.
