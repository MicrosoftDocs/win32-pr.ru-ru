# <a name="getkaraokechannelcontent-method"></a>Метод Жеткараокечаннелконтент

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetKaraokeChannelContent`Метод получает значение, указывающее тип содержимого в указанном канале караоке в указанном потоке.

``` syntax
[ iContent = ] MSWebDVD.GetKaraokeChannelContent(iStream, iChannel)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iStream"></span><span id="istream"></span><span id="ISTREAM"></span>*iStream*
</dt> <dd>

Задает аудиопоток в виде целого числа.

</dd> <dt>

<span id="iChannel"></span><span id="ichannel"></span><span id="ICHANNEL"></span>*iChannel*
</dt> <dd>

Указывает канал в виде целого числа. Возможные значения для каждого канала:



| Значение  | Описание    |
|--------|----------------|
| 0x0001 | Guide вокал 1  |
| 0x0002 | Guide вокал 2  |
| 0x0004 | Guide Мелодия 1 |
| 0x0008 | Guide Мелодия 2 |
| 0x0010 | Мелодия |
| 0x0020 | Guide Мелодия B |
| 0x0040 | Звуковой результат A |
| 0x0080 | Звуковой результат б |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, отдельные биты которого задают содержимое канала караоке.

## <a name="remarks"></a>Remarks

Нумерация видеоканалов DVD отсчитывается от нуля, поэтому каналы 2, 3 и 4 являются вспомогательными каналами караоке. После возврата метода выполните операцию побитового и для *иконтент* , чтобы определить содержимое каждого канала. Так как на одном канале может быть записано несколько типов содержимого, следует проверить все возможные значения даже после обнаружения совпадения.

После того как пользователь знает содержимое каждого канала, он должен иметь возможность настраивать том или включать или отключать отдельные каналы по мере необходимости. Реализуйте эту функцию в приложении с помощью свойства [**караокеаудиопресентатионмоде**](karaokeaudiopresentationmode-property.md) .

> [!Note]  
> для воспроизведения дисков караоке аудио декодер в системе пользователя должен быть совместим с реализацией DirectShow 8 караоке.
