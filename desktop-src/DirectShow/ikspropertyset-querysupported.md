# <a name="ikspropertysetquerysupported-method"></a>Метод Икспропертисет:: Куерисуппортед

`QuerySupported`Метод определяет, поддерживает ли объект заданный набор свойств.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT QuerySupported(
  [in]  REFGUID guidPropSet,
  [in]  DWORD   dwPropID,
  [out] DWORD   *pTypeSupport
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*гуидпропсет* \[ окне\]
</dt> <dd>

Идентификатор GUID набора свойств.

</dd> <dt>

*dwPropID* \[ окне\]
</dt> <dd>

Идентификатор свойства в наборе свойств.

</dd> <dt>

*птипесуппорт* \[ заполняет\]
</dt> <dd>

Указатель на значение, в котором сохраняются флаги, указывающие на поддержку, предоставляемую драйвером. Поддерживаются следующие флаги:



| Значение                    | Описание                                                                                            |
|--------------------------|--------------------------------------------------------------------------------------------------------|
| \_Поддержка кспроперти \_ Get | Свойство можно получить, вызвав метод [**икспропертисет:: Get**](ikspropertyset-get.md) . |
| \_набор поддержки \_ кспроперти | Свойство можно изменить, вызвав [**икспропертисет:: Set**](ikspropertyset-set.md).              |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Ниже приведены возможные значения.



| Код возврата                                                                                              | Описание                                                                     |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                     | Поддерживается сочетание указанного набора свойств и идентификатора свойства.<br/> |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>                | Набор свойств не поддерживается.<br/>                                       |
| <dl> <dt>**идентификатор "E \_ prop" \_ \_ не поддерживается**</dt> </dl>  | Идентификатор свойства не поддерживается для указанного набора свойств.<br/>         |
| <dl> <dt>**\_набор свойств \_ E \_ не поддерживается**</dt> </dl> | Набор свойств не поддерживается.<br/>                                       |



 

## <a name="remarks"></a>Комментарии

> [!Note]  
> Другой интерфейс с таким именем существует в файле заголовка дсаунд. h. Два интерфейса несовместимы. Интерфейс **иксконтрол** , документированный в пакете DDK по DirectShow, теперь является рекомендуемым интерфейсом для передачи наборов свойств между драйверами WDM и компонентами пользовательского режима.

 

Перед Кспрокси. h необходимо включить KS. h.
