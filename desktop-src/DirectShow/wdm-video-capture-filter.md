# <a name="wdm-video-capture-filter"></a>Фильтр записи видео WDM

Фильтр записи видео WDM контролирует устройства аналогового захвата, использующие драйверы WDM (WDM).

Этот фильтр на самом деле является подключаемым модулем Кспрокси в режиме ядра. Он предоставляет страницы свойств и интерфейсы COM для драйверов WDM, которые управляют устройствами аналогового захвата (также называемыми аналоговыми видеодекодерами). Приложение может обрабатывать его просто как фильтр. Чтобы добавить этот фильтр к графу фильтра, используйте [перечислитель системных устройств](system-device-enumerator.md). Он возвращает уникальный моникер для каждого устройства, использующего этот подключаемый модуль. Дополнительные сведения см. в разделе [Перечисление устройств и фильтров](enumerating-devices-and-filters.md) и [как устройства участвуют в графе фильтра](how-hardware-devices-participate-in-the-filter-graph.md). Как и в случае с любым фильтром, основанным на Кспрокси, понятное имя фильтра будет зависеть от базового драйвера.

Не все устройства, доступные в фильтре записи видео WDM, реализуют все перечисленные ниже интерфейсы. Приложение может использовать **QueryInterface** для определения интерфейсов, поддерживаемых определенным устройством.
