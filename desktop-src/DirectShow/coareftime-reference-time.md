# <a name="coareftimeoperator-reference_time-method"></a>Метод времени ссылки Коарефтиме. operator \_

Этот оператор получает время ссылки в виде значения **\_ времени ссылки** .

## <a name="syntax"></a>Синтаксис


```C++
REFERENCE_TIME operator REFERENCE_TIME();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает время ссылки в единицах измерения 100-наносекундных.
