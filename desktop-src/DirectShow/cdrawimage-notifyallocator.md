# <a name="cdrawimagenotifyallocator-method"></a>Кдравимаже. Нотифяллокатор, метод

`NotifyAllocator`Метод информирует объект **кдравимаже** о том, является ли распределитель для соединения объектом [**Цимажеаллокатор**](cimageallocator.md) .

## <a name="syntax"></a>Синтаксис


```C++
void NotifyAllocator(
   BOOL bUsingImageAllocator
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бусингимажеаллокатор* 
</dt> <dd>

Логическое значение, указывающее, какой тип распределителя используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Фильтр-владелец должен вызвать этот метод после того, как ПИН-коды согласны с распределителем. Если фильтр знает, что распределитель является объектом **Цимажеаллокатор** , он должен вызвать этот метод со значением **true**. (Как правило, фильтр будет узнавать это только в том случае, если он владеет этим распределителем.) В противном случае он должен вызвать этот метод со значением **false**.
