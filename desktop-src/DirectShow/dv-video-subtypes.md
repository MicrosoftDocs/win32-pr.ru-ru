# <a name="dv-video-subtypes"></a>Подтипы видео DV

Для видео DV определено несколько подтипов. У каждого есть код FOURCC и соответствующее значение идентификатора GUID. Поддерживаются не все форматы. Дополнительные сведения см. в разделе "Примечания".

## <a name="consumer-formats"></a>Форматы потребителей



| FOURCC | GUID               | Скорость передачи данных | Описание                  |
|--------|--------------------|-----------|------------------------------|
| "двсл" | МЕДИАСУБТИПЕ \_ двсл | 12,5 Мбит/с | SD-ДВКР (525-60 или 625-50)   |
| "двсд" | МЕДИАСУБТИПЕ \_ двсд | 25 Мбит/с   | SDL-ДВКР (525-60 или 625-50)  |
| "двхд" | МЕДИАСУБТИПЕ \_ двхд | 50 Мбит/с   | HD-ДВКР (1125-60 или 1250-50) |



 

Дополнительные сведения об этих форматах см. в статье IEC-61834.

## <a name="professional-formats"></a>Professional Форматирование



| FOURCC | GUID               | Скорость передачи данных | Описание                                 |
|--------|--------------------|-----------|---------------------------------------------|
| 'dv25' | МЕДИАСУБТИПЕ \_ DV25 | 25 Мбит/с   | DVCPRO 25 (525-60 или 625-50).               |
| 'dv50' | МЕДИАСУБТИПЕ \_ DV50 | 50 Мбит/с   | DVCPRO 50 (525-60 или 625-50)                |
| 'dvh1' | МЕДИАСУБТИПЕ \_ dvh1 | 100 Мбит/с  | DVCPRO 100 (1080/60i, 1080/50i или 720/60P) |



 

Дополнительные сведения о DV25 и DV50 и SMPTE 370M для получения дополнительных сведений о dvh1 см. в разделе SMPTE 314M.

## <a name="miscellaneous"></a>Разное

Два дополнительных подтипа DV определены в файле заголовков UUID. h. Они соответствуют кодам FOURCC, формируемым определенными кодеками DV; они не соответствуют ни одному из заданных стандартов DV. Эти подтипы являются устаревшими и не должны использоваться.



| FOURCC | GUID               |
|--------|--------------------|
| DVCS | МЕДИАСУБТИПЕ \_ DVCS |
| "ДВСД" | МЕДИАСУБТИПЕ \_ двсд |
