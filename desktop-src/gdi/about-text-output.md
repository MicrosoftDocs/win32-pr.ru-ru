<span data-ttu-id="d83c2-104">Текстовый вывод является наиболее распространенным типом графических данных, найденных в клиентской области. они используются приложениями различными способами.</span><span class="sxs-lookup"><span data-stu-id="d83c2-104">Text output is the most common type of graphic output found within the client area; it is used by applications in different ways.</span></span> <span data-ttu-id="d83c2-105">Приложения обработки и настольной публикации создают документы с форматированным текстом; приложения электронных таблиц используют текст, числа и символы для указания формул, столбцов меток и значений списка; приложения базы данных создают записи и отображают запросы с текстом, а приложения САПР используют текст для меток объектов и отображают размеры.</span><span class="sxs-lookup"><span data-stu-id="d83c2-105">Word processing and desktop publishing applications create documents with formatted text; spreadsheet applications use text, numbers, and symbols to specify formulas, label columns, and list values; database applications create records and display queries with text, and CAD applications use text to label objects and display dimensions.</span></span>
