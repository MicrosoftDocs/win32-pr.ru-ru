# <a name="window-coordinate-system"></a><span data-ttu-id="8e52b-103">Система координат окна</span><span class="sxs-lookup"><span data-stu-id="8e52b-103">Window Coordinate System</span></span>

<span data-ttu-id="8e52b-104">Система координат для окна основана на системе координат устройства дисплея.</span><span class="sxs-lookup"><span data-stu-id="8e52b-104">The coordinate system for a window is based on the coordinate system of the display device.</span></span> <span data-ttu-id="8e52b-105">Базовая единица измерения — это единица устройства (как правило, пиксель).</span><span class="sxs-lookup"><span data-stu-id="8e52b-105">The basic unit of measure is the device unit (typically, the pixel).</span></span> <span data-ttu-id="8e52b-106">Точки на экране описаны парами координат x и y.</span><span class="sxs-lookup"><span data-stu-id="8e52b-106">Points on the screen are described by x- and y-coordinate pairs.</span></span> <span data-ttu-id="8e52b-107">Координаты x увеличиваются вправо; координаты y увеличиваются сверху вниз.</span><span class="sxs-lookup"><span data-stu-id="8e52b-107">The x-coordinates increase to the right; y-coordinates increase from top to bottom.</span></span> <span data-ttu-id="8e52b-108">Источник (0, 0) для системы зависит от типа используемых координат.</span><span class="sxs-lookup"><span data-stu-id="8e52b-108">The origin (0,0) for the system depends on the type of coordinates being used.</span></span>

<span data-ttu-id="8e52b-109">Система и приложения определяют расположение окна на экране в *экранных координатах*.</span><span class="sxs-lookup"><span data-stu-id="8e52b-109">The system and applications specify the position of a window on the screen in *screen coordinates*.</span></span> <span data-ttu-id="8e52b-110">Для экранных координат источник — это верхний левый угол экрана.</span><span class="sxs-lookup"><span data-stu-id="8e52b-110">For screen coordinates, the origin is the upper-left corner of the screen.</span></span> <span data-ttu-id="8e52b-111">Полное расположение окна часто описывается структурой [**Rect**](/previous-versions//dd162897(v=vs.85)) , содержащей координаты экрана двух точек, определяющих верхний левый и правый нижний угол окна.</span><span class="sxs-lookup"><span data-stu-id="8e52b-111">The full position of a window is often described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure containing the screen coordinates of two points that define the upper-left and lower-right corners of the window.</span></span>

<span data-ttu-id="8e52b-112">Система и приложения определяют расположение точек в окне с помощью *клиентских координат*.</span><span class="sxs-lookup"><span data-stu-id="8e52b-112">The system and applications specify the position of points in a window by using *client coordinates*.</span></span> <span data-ttu-id="8e52b-113">Источником в этом случае является левый верхний угол окна или клиентской области.</span><span class="sxs-lookup"><span data-stu-id="8e52b-113">The origin in this case is the upper-left corner of the window or client area.</span></span> <span data-ttu-id="8e52b-114">Клиентские координаты гарантируют, что приложение может использовать согласованные значения координат во время рисования в окне, независимо от положения окна на экране.</span><span class="sxs-lookup"><span data-stu-id="8e52b-114">Client coordinates ensure that an application can use consistent coordinate values while drawing in the window, regardless of the position of the window on the screen.</span></span>

<span data-ttu-id="8e52b-115">Размеры клиентской области также описываются структурой [**Rect**](/previous-versions//dd162897(v=vs.85)) , которая содержит клиентские координаты области.</span><span class="sxs-lookup"><span data-stu-id="8e52b-115">The dimensions of the client area are also described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure that contains client coordinates for the area.</span></span> <span data-ttu-id="8e52b-116">Во всех случаях верхняя левая координата прямоугольника включается в окно или клиентскую область, а координата с нижней правой стороны исключается.</span><span class="sxs-lookup"><span data-stu-id="8e52b-116">In all cases, the upper-left coordinate of the rectangle is included in the window or client area, while the lower-right coordinate is excluded.</span></span> <span data-ttu-id="8e52b-117">Графические операции в окне или клиентской области исключаются от правых и нижних границ охватывающего прямоугольника.</span><span class="sxs-lookup"><span data-stu-id="8e52b-117">Graphics operations in a window or client area are excluded from the right and lower edges of the enclosing rectangle.</span></span>

<span data-ttu-id="8e52b-118">Иногда может потребоваться, чтобы приложения сопоставляли координаты в одном окне с другими окнами.</span><span class="sxs-lookup"><span data-stu-id="8e52b-118">Occasionally, applications may be required to map coordinates in one window to those of another window.</span></span> <span data-ttu-id="8e52b-119">Приложение может сопоставлять координаты с помощью функции [**мапвиндовпоинтс**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) .</span><span class="sxs-lookup"><span data-stu-id="8e52b-119">An application can map coordinates by using the [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) function.</span></span> <span data-ttu-id="8e52b-120">Если одно из окон является окном рабочего стола, функция фактически преобразует экранные координаты в клиентские координаты и наоборот; окно рабочего стола всегда указывается в экранных координатах.</span><span class="sxs-lookup"><span data-stu-id="8e52b-120">If one of the windows is the desktop window, the function effectively converts screen coordinates to client coordinates and vice versa; the desktop window is always specified in screen coordinates.</span></span>
