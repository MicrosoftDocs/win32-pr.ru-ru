<span data-ttu-id="8567a-104">Когда несколько мониторов являются частью рабочего стола, объекты могут беспрепятственно перемещаться между мониторами.</span><span class="sxs-lookup"><span data-stu-id="8567a-104">When multiple monitors are part of the desktop, objects can travel seamlessly between monitors.</span></span> <span data-ttu-id="8567a-105">То есть можно перетаскивать окна или ярлыки с одного монитора на другой, а размеры окон можно закрывать более чем на одном мониторе.</span><span class="sxs-lookup"><span data-stu-id="8567a-105">That is, you can drag windows or shortcuts from one monitor to another, and you can size windows to cover more than one monitor.</span></span> <span data-ttu-id="8567a-106">Кроме того, если один монитор установлен над другим, то в верхней части нижнего монитора появится курсор, который оставляет нижнюю часть верхнего монитора.</span><span class="sxs-lookup"><span data-stu-id="8567a-106">Also, if one monitor is installed above another, a cursor that leaves the bottom of the upper monitor appears at the top of the lower monitor.</span></span>

<span data-ttu-id="8567a-107">Как правило, пользователь упорядочивает мониторы в системе в соответствии с расположением физических единиц отображения. Например, параллельно или один верхний колонтитул — другой...</span><span class="sxs-lookup"><span data-stu-id="8567a-107">Typically, a user arranges the monitors in the system to reflect the arrangement of the physical display units; for example, side-by-side or one-on-top-of-the-other.</span></span> <span data-ttu-id="8567a-108">Пользователь делает это на вкладке мониторы, которая заменяет вкладку **Параметры** в диалоговом окне **Свойства экрана** с помощью панели управления.</span><span class="sxs-lookup"><span data-stu-id="8567a-108">The user does this with the Monitors tab, which replaces the **Settings** tab in the **Display Properties** dialog box through Control Panel.</span></span> <span data-ttu-id="8567a-109">Мониторы должны формировать один непрерывный регион, т. е. <span data-ttu-id="8567a-110">При перемещении окна или изменении его размера часть заголовка всегда отображается, поэтому пользователь может перемещать окно и изменять его размер с помощью мыши.</span><span class="sxs-lookup"><span data-stu-id="8567a-110">When a window is moved or resized, some part of the caption is always visible so the user can move and resize the window using the mouse.</span></span> <span data-ttu-id="8567a-111">Перемещение курсора ограничено областью мониторов, поэтому оно всегда отображается.</span><span class="sxs-lookup"><span data-stu-id="8567a-111">Cursor movement is restricted to the area of the monitors, so it is always visible.</span></span> <span data-ttu-id="8567a-112">Значки оболочки расположены на том же мониторе, что и панель задач, и панель задач может находиться на любом мониторе, см. <span data-ttu-id="8567a-113">Несколько систем мониторинга влияют на определенные сочетания клавиш.</span><span class="sxs-lookup"><span data-stu-id="8567a-113">A multiple monitor system affects certain key combinations.</span></span> <span data-ttu-id="8567a-114">Сочетание клавиш ALT + ПРИНТСКРН создает моментальный снимок окна переднего плана, как всегда.</span><span class="sxs-lookup"><span data-stu-id="8567a-114">The ALT+PRINTSCRN key combination takes a snapshot of the foreground window, as always.</span></span> <span data-ttu-id="8567a-115">Однако ключ ПРИНТСКРН создает моментальный снимок монитора, на котором находится мышь.</span><span class="sxs-lookup"><span data-stu-id="8567a-115">However, the PRINTSCRN key takes a snapshot of the monitor that has the mouse.</span></span> <span data-ttu-id="8567a-116">Сочетание клавиш CTRL + ПРИНТСКРН создает моментальный снимок всего виртуального экрана, см. [виртуальный экран](the-virtual-screen.md).</span><span class="sxs-lookup"><span data-stu-id="8567a-116">The CTRL+PRINTSCRN key combination takes a snapshot of the entire virtual screen, see [The Virtual Screen](the-virtual-screen.md).</span></span>

<span data-ttu-id="8567a-117">Поддержка нескольких мониторов не влияет на производительность приложений при работе в одной среде экрана.</span><span class="sxs-lookup"><span data-stu-id="8567a-117">The support for multiple monitors does not affect the performance of applications when running in a single display environment.</span></span> <span data-ttu-id="8567a-118">То есть при работе в одной системе отображения дополнительные издержки в коде графических операций с высокой производительностью не будут представлены.</span><span class="sxs-lookup"><span data-stu-id="8567a-118">That is, when running on a single display system, no additional overhead will be present in the high-performance graphics operations code.</span></span> <span data-ttu-id="8567a-119">Однако в системе с несколькими мониторами производительность немного меняется, если приложение выполняется только на одном из графических устройств.</span><span class="sxs-lookup"><span data-stu-id="8567a-119">However, in a multiple monitor system, performance is slightly affected if an application runs only on one of the graphics devices.</span></span> <span data-ttu-id="8567a-120">Кроме того, производительность может сильно повлиять, если приложение охватывает несколько экранов, особенно для ресурсоемких графических операций.</span><span class="sxs-lookup"><span data-stu-id="8567a-120">Also, performance may be greatly affected if an application spans multiple displays, especially for graphics-intensive operations.</span></span>

<span data-ttu-id="8567a-121">*Полноэкранный* режим — это функция, предоставляемая операционной системой, которая позволяет пользователю переключать приложение в особое состояние, когда приложение может напрямую обращаться к графическому устройству VGA.</span><span class="sxs-lookup"><span data-stu-id="8567a-121">*Full-screen* is a feature provided by the operating system that allows a user to toggle an application into a special state where the application can access VGA graphics hardware directly.</span></span> <span data-ttu-id="8567a-122">Это ключевая функция для игр и других ориентированных на графические приложения, требующих высокой производительности.</span><span class="sxs-lookup"><span data-stu-id="8567a-122">This is a key feature for games and other graphics-focused applications that require high performance.</span></span> <span data-ttu-id="8567a-123">Кроме того, он часто используется разработчиками для редактирования текста, так как он обеспечивает очень быструю прокрутку текста.</span><span class="sxs-lookup"><span data-stu-id="8567a-123">Also, it is often used by developers for text editing since it enables very fast text scrolling.</span></span>

<span data-ttu-id="8567a-124">В среде с несколькими мониторами только одно графическое устройство может быть совместимо с VGA.</span><span class="sxs-lookup"><span data-stu-id="8567a-124">In a multiple monitor environment, only one graphics device can be VGA compatible.</span></span> <span data-ttu-id="8567a-125">Это ограничение оборудования компьютера, которое требует, чтобы только одно устройство отвечало на любой аппаратный адрес.</span><span class="sxs-lookup"><span data-stu-id="8567a-125">This is a limitation of computer hardware which requires that only one device respond to any hardware address.</span></span> <span data-ttu-id="8567a-126">Поскольку в стандарте совместимости оборудования VGA требуются определенные аппаратные адреса, на компьютере может присутствовать только одно графическое устройство VGA, и только это устройство может физически отвечать на адреса VGA.</span><span class="sxs-lookup"><span data-stu-id="8567a-126">Because the VGA hardware compatibility standard requires specific hardware addresses, only one VGA graphics device can be present in a machine and only this device can physically respond to VGA addresses.</span></span> <span data-ttu-id="8567a-127">Поэтому приложения, требующие полноэкранного режима, будут работать только на конкретном устройстве, поддерживающем аппаратную совместимость с VGA.</span><span class="sxs-lookup"><span data-stu-id="8567a-127">Thus, applications that require going full screen will only run on the particular device that supports VGA hardware compatibility.</span></span>
