# <a name="character-widths"></a><span data-ttu-id="dc4aa-103">Ширина символов</span><span class="sxs-lookup"><span data-stu-id="dc4aa-103">Character Widths</span></span>

<span data-ttu-id="dc4aa-104">Приложениям необходимо получить данные ширины символов, когда они выполняют такие задачи, как подгонка строк текста к ширине страницы или столбца.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-104">Applications need to retrieve character-width data when they perform such tasks as fitting strings of text to page or column widths.</span></span> <span data-ttu-id="dc4aa-105">Существует четыре функции, которые приложение может использовать для извлечения данных ширины символов.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-105">There are four functions that an application can use to retrieve character-width data.</span></span> <span data-ttu-id="dc4aa-106">Две из этих функций извлекают ширину символов и две из этих функций для получения фактических данных по ширине символов.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-106">Two of these functions retrieve the character-advance width and two of these functions retrieve actual character-width data.</span></span>

<span data-ttu-id="dc4aa-107">Приложение может использовать функции [GetCharWidth32](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidth32a) и [жетчарвидсфлоат](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidthfloata) для получения предварительной ширины отдельных символов или символов в текстовой строке.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-107">An application can use the [GetCharWidth32](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidth32a) and [GetCharWidthFloat](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidthfloata) functions to retrieve the advance width for individual characters or symbols in a string of text.</span></span> <span data-ttu-id="dc4aa-108">Ширина — это расстояние, на которое курсор на дисплее или печатающем головке на принтере должен быть перемещен перед печатью следующего символа в текстовой строке.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-108">The advance width is the distance that the cursor on a video display or the print-head on a printer must advance before printing the next character in a string of text.</span></span> <span data-ttu-id="dc4aa-109">Функция [**GetCharWidth32**](/windows/win32/api/wingdi/nf-wingdi-getcharwidth32a) Возвращает ширину в виде целочисленного значения.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-109">The [**GetCharWidth32**](/windows/win32/api/wingdi/nf-wingdi-getcharwidth32a) function returns the advance width as an integer value.</span></span> <span data-ttu-id="dc4aa-110">Если требуется большая точность, приложение может использовать функцию [**жетчарвидсфлоат**](/windows/win32/api/wingdi/nf-wingdi-getcharwidthfloata) для получения дробных значений по расширенной ширине.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-110">If greater precision is required, an application can use the [**GetCharWidthFloat**](/windows/win32/api/wingdi/nf-wingdi-getcharwidthfloata) function to retrieve fractional advance-width values.</span></span>

<span data-ttu-id="dc4aa-111">Приложение может получать фактические данные по ширине символов с помощью функций [жетчарабквидсс](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsa) и [**жетчарабквидссфлоат**](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsfloata) .</span><span class="sxs-lookup"><span data-stu-id="dc4aa-111">An application can retrieve actual character-width data by using the [GetCharABCWidths](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsa) and [**GetCharABCWidthsFloat**](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsfloata) functions.</span></span> <span data-ttu-id="dc4aa-112">Функция **жетчарабквидссфлоат** работает со всеми шрифтами.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-112">The **GetCharABCWidthsFloat** function works with all fonts.</span></span> <span data-ttu-id="dc4aa-113">Функция [**жетчарабквидсс**](/windows/win32/api/wingdi/nf-wingdi-getcharabcwidthsa) работает только с шрифтами TrueType и OpenType.</span><span class="sxs-lookup"><span data-stu-id="dc4aa-113">The [**GetCharABCWidths**](/windows/win32/api/wingdi/nf-wingdi-getcharabcwidthsa) function only works with TrueType and OpenType fonts.</span></span> <span data-ttu-id="dc4aa-114">Дополнительные сведения о шрифтах TrueType и OpenType см. в разделе [растровые, векторные, шрифты TrueType и OpenType](raster--vector--truetype--and-opentype-fonts.md).</span><span class="sxs-lookup"><span data-stu-id="dc4aa-114">For more information about TrueType and OpenType fonts, see [Raster, Vector, TrueType, and OpenType Fonts](raster--vector--truetype--and-opentype-fonts.md).</span></span>
