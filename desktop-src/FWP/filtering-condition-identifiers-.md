<span data-ttu-id="29f2a-104">Идентификаторы условий фильтрации платформы фильтрации Windows (WFP) представлены с помощью идентификатора **GUID**.</span><span class="sxs-lookup"><span data-stu-id="29f2a-104">The Windows Filtering Platform (WFP) filtering condition identifiers are each represented by a **GUID**.</span></span> <span data-ttu-id="29f2a-105">Тип данных для значения условия для каждого условия фильтрации указывается как [**\_ \_ тип данных FWP**](/windows/desktop/api/Fwptypes/ne-fwptypes-fwp_data_type).</span><span class="sxs-lookup"><span data-stu-id="29f2a-105">The data type for the condition value for each filtering condition is specified as an [**FWP\_DATA\_TYPE**](/windows/desktop/api/Fwptypes/ne-fwptypes-fwp_data_type).</span></span> <span data-ttu-id="29f2a-106">Эти идентификаторы и их типы данных определяются здесь.</span><span class="sxs-lookup"><span data-stu-id="29f2a-106">These identifiers and their data types are defined here.</span></span>
