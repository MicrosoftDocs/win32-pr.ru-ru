# <a name="filtering-condition-identifiers"></a>Фильтрация идентификаторов условий

идентификаторы условий фильтрации платформы фильтрации Windows (WFP) представляются с помощью идентификатора **GUID**. Тип данных для значения условия для каждого условия фильтрации указывается как [**\_ \_ тип данных FWP**](/windows/desktop/api/Fwptypes/ne-fwptypes-fwp_data_type). Эти идентификаторы и их типы данных определяются здесь.

Стандартные условия перечислены первыми, за которыми следуют условия, относящиеся к пользовательскому режиму. Условия группируются по поддерживаемой операционной системе, что позволяет легко определить, какие условия поддерживаются для конкретной ОС.

> [!Note]  
> Каждое из следующих условий фильтрации доступно только в подмножестве уровней фильтрации WFP. Дополнительные сведения о доступности каждого условия на любом конкретном уровне см. в разделе [**Фильтрация условий, доступных на каждом слое фильтрации**](filtering-conditions-available-at-each-filtering-layer.md).
