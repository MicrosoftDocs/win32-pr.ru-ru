# <a name="netconfigset"></a>нетконфигсет

Функция **нетконфигсет** устарела. Он включен для обеспечения совместимости с 16-разрядными версиями Windows. Другие приложения должны использовать [Реестр](/windows/desktop/SysInfo/registry).
