# <a name="parameter-error-reporting"></a>Отчеты об ошибках параметров

Функции управления сетью Add и Сетинфо возвращают индекс для параметра, который находится в ошибке. Вызывающий объект может передать **пустой** указатель для параметра *ParM \_ Err* , указывающий, что поле не должно быть задано функцией.
