# <a name="itparticipantcontrolget_participants-method"></a>Метод ИтпартиЦипантконтрол:: Get \_ участников

\[**получить \_ Участники** не могут использоваться в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ Участники** создает коллекцию участников, связанных с текущей Конференцией. Этот метод предоставляется для клиентских приложений автоматизации, например, написанных на Visual Basic. Приложения C и C++ должны использовать метод [**енумератепартиЦипантс**](itparticipantcontrol-enumerateparticipants.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_Participants(
  [out] VARIANT *pVariant
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвариант* \[ заполняет\]
</dt> <dd>

Указатель на **вариант** , содержащий [**итколлектион**](/windows/desktop/api/tapi3if/nn-tapi3if-itcollection) указателей интерфейса [**итпартиЦипант**](itparticipant.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                         | Значение                                                         |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *пвариант* не является допустимым указателем.<br/>     |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |



 

## <a name="remarks"></a>Комментарии

TAPI вызывает метод **AddRef** в интерфейсе [**итпартиЦипант**](itparticipant.md) , возвращенном **итпартиЦипантконтрол:: Get \_ Участники**. Приложение должно вызвать **выпуск** в интерфейсе **итпартиЦипант** , чтобы освободить ресурсы, связанные с ним.
