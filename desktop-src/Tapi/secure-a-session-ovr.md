# <a name="secure-a-session"></a>Обеспечение безопасности сеанса

Если приложение не хочет никаких помех извне за пределами событий для сеанса от TAPI или поставщика услуг, он должен обеспечить защиту вызова. Например, в аналоговой среде в режиме ожидания вызовов может уничтожиться сеанс факса или модема при первоначальном вызове.

Приложение может защитить вызов во время вызова или после того, как вызов уже существует.

Не все поставщики служб поддерживают использование этой операции.

**Интерфейс TAPI 2. x:** См. [**линесекурекалл**](/windows/win32/api/tapi/nf-tapi-linesecurecall).

**Интерфейс TAPI 3. x:** См. раздел [**итаддресс:: Forward**](/windows/desktop/api/tapi3if/nf-tapi3if-itaddress-forward).
