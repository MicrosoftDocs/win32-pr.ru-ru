# <a name="stringformat_-constants"></a>\_Константы STRINGFORMAT

**\_ Константы STRINGFORMAT** описывают различные форматы строк.

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**STRINGFORMAT \_ ASCII**
</dt> <dd> <dl> <dt>



Задает стандартный формат символов ASCII, использующий один байт на символ.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**\_двоичный файл STRINGFORMAT**
</dt> <dd> <dl> <dt>



Это массив беззнаковых символов; может использоваться для числовых значений.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**STRINGFORMAT \_ DBCS**
</dt> <dd> <dl> <dt>



Указывает стандартный формат символов DBCS, использующий один или два байта на символ.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**STRINGFORMAT \_ Юникод**
</dt> <dd> <dl> <dt>



Задает стандартный символьный формат Юникода с использованием двух байтов на символ.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.
