# <a name="line_linedevstate-message"></a>Строка \_ сообщения линедевстате

Сообщение **\_ ЛИНЕДЕВСТАТЕ линии** TAPI отправляется при изменении состояния линейного устройства. Приложение может вызвать [**линежетлинедевстатус**](/windows/desktop/api/Tapi/nf-tapi-linegetlinedevstatus) , чтобы определить новое состояние строки.


```C++
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хдевице* 
</dt> <dd>

Маркер устройства линии. Этот параметр имеет **значение NULL** , если *dwParam1* — линедевстате \_ reinit.

</dd> <dt>

*двкаллбаккинстанце* 
</dt> <dd>

Экземпляр обратного вызова, указанный при открытии строки. Если параметр *dwParam1* имеет значение ЛИНЕДЕВСТАТЕ \_ reinit, то параметр *двкаллбаккинстанце* является недопустимым и имеет значение 0.

</dd> <dt>

*dwParam1* 
</dt> <dd>

Элемент состояния линейного устройства, который был изменен. Параметр может быть одной или несколькими [**\_ константами линедевстате**](linedevstate--constants.md).

</dd> <dt>

*dwParam2* 
</dt> <dd>

Интерпретация этого параметра зависит от значения *dwParam1*. Если *dwParam1* — линедевстате \_ Ring, *dwParam2* содержит режим Ring, с помощью которого параметр указывает линии на звонок. Допустимые режимы звонка — это числа в диапазоне от 1 до **двнумрингмодес**, где **двнумрингмодес** является возможностью линейного устройства.

Если *dwParam1* имеет значение ЛИНЕДЕВСТАТЕ \_ reinit, и сообщение было выдано TAPI в результате перевода нового сообщения API в сообщение о повторной инициализации, *dwParam2* содержит параметр *Двмсг* исходного сообщения (например, [**строку \_ CREATE**](line-create.md) или Line \_ линедевстате). Если значение *dwParam2* равно нулю, это означает, что сообщение о переинициализации является "реальным" сообщением о переинициализации, которое требует, чтобы приложение вызывало [**линешутдовн**](/windows/desktop/api/Tapi/nf-tapi-lineshutdown) на самом раннем удобство.

</dd> <dt>

*dwParam3* 
</dt> <dd>

Интерпретация этого параметра зависит от значения *dwParam1*. Если *dwParam1* — линедевстате \_ Звонок, *dwParam3* содержит число колец для этого события кольца. Число звонков начинается с нуля.

Если *dwParam1* имеет значение ЛИНЕДЕВСТАТЕ \_ reinit, и сообщение было выдано TAPI в результате перевода нового сообщения API в сообщение о повторной инициализации, *dwParam3* содержит параметр *DWPARAM1* исходного сообщения (например, линедевстате \_ транслатечанже или другое \_ значение линедевстате, если *dwParam2* — Line \_ Линедевстате, или новый идентификатор устройства, если *dwParam2* — это [**\_ Создание строки**](line-create.md)).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Отправкой сообщения **Line \_ линедевстате** можно управлять с помощью [**линесетстатусмессажес**](/windows/desktop/api/Tapi/nf-tapi-linesetstatusmessages). Приложение может указывать на изменения элементов состояния, о которых требуется получать уведомления. По умолчанию все отчеты о состоянии отключены, за исключением ЛИНЕДЕВСТАТЕ \_ reinit, которые не могут быть отключены. Это сообщение отправляется всем приложениям, имеющим в строке маркер, включая те, которые вызвали [**линеопен**](/windows/desktop/api/Tapi/nf-tapi-lineopen) с параметром *двпривилежес* , для которого задано значение линекаллпривилеже \_ None, линекаллпривилеже \_ owner, линекаллпривилеже \_ Monitor или разрешенные сочетания.
