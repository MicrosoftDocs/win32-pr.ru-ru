# <a name="itsdp-interface"></a>Интерфейс Итсдп

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итсдп** предоставляет методы для обработки протокола дескриптора сеанса (см. статью RFC 2327) о компоненте большого двоичного объекта для конференций. Решение предоставляет следующие возможности:

-   Предоставляет доступ к некоторым свойствам, которые являются общими для всех носителей. К ним относятся атрибуты, относящиеся к личным сведениям создателя, описанию сеанса и сведениям о типе адреса.
-   Предоставляет отправную точку для доступа к коллекциям времени и носителей через свойства.

Интерфейс **итсдп** создается путем вызова **QueryInterface** в [**итконференцеблоб**](itconferenceblob.md).

## <a name="members"></a>Элементы

Интерфейс **итсдп** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Итсдп** также имеет следующие типы членов:
