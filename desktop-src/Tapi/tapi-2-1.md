# <a name="tapi-21"></a>TAPI 2,1

TAPI 2,1 предоставляет одно дополнение к предыдущей спецификации TAPI 2,0: функциональность клиента/сервера. С помощью TAPI 2,1 и более поздних версий оборудование телефонии может быть установлено на сервере телефонии, к которому можно получить доступ с любого компьютера в сети. Новый набор функций позволяет записывать библиотеки DLL администрирования для управления доступом к удаленным ресурсам.

**Windows Me/98/95:** TAPI 2,1 обеспечивает поддержку 32-разрядной специалистами. Системные двоичные файлы TAPI 2,1, если установленная поддержка:

-   32-разрядные приложения для TAPI 1,3, TAPI 1,4 и TAPI 2. x
-   32-разрядный интерфейс TAPI 2. x специалистами
-   16-разрядные приложения для TAPI 1,3 и TAPI 1,4
-   16-разрядный интерфейс TAPI 1,3 и TAPI 1,4 специалистами
