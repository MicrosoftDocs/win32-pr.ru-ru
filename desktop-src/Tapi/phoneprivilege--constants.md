# <a name="phoneprivilege_-constants"></a>\_Константы фонепривилеже

Константы побитового флага **фонепривилеже \_** описывают различные способы открытия телефонного устройства.

<dl> <dt>

<span id="PHONEPRIVILEGE_MONITOR"></span><span id="phoneprivilege_monitor"></span>**\_монитор фонепривилеже**
</dt> <dd> <dl> <dt>



Приложение, открывающее телефонное устройство с правами монитора, уведомляет о событиях и изменениях состояния, происходящих на телефоне. Приложение не может вызывать никаких операций на телефонном устройстве, которые могли бы изменить свое состояние, поэтому могут быть вызваны только операции состояния. Несколько приложений могут отслеживать телефонное устройство в любое заданное время.


</dt> </dl> </dd> <dt>

<span id="PHONEPRIVILEGE_OWNER"></span><span id="phoneprivilege_owner"></span>**\_владелец фонепривилеже**
</dt> <dd> <dl> <dt>



Приложение, открывающее телефонное устройство с правами владельца, может изменять состояние блоков ламп, колец, дисплея, хуксвитч и данных телефона. Открытие телефонного устройства в режиме владельца также обеспечивает мониторинг телефонного устройства. Только одно приложение может быть владельцем телефонного устройства в любое заданное время.


</dt> </dl> </dd> </dl>
