<dl> <dt>

<span id="tapi2.integrated_services_digital_network_isdn__tapgloss"></span><span id="TAPI2.INTEGRATED_SERVICES_DIGITAL_NETWORK_ISDN__TAPGLOSS"></span>**Цифровая сеть интегрированных служб (ISDN)**
</dt> <dd>

Полностью цифровая телефонная или телекоммуникационная сеть для хранения голоса, данных, изображений и видео с высокой скоростью, отправляя сигналы с цифровой кодировкой на скорости до 144 000 бит/с в линии с интерфейсом Basic Rate (Анд-ISDN) и значительно выше в линиях интерфейса основной частоты (PRI-ISDN). ISDN состоит по крайней мере из трех каналов и столько же, сколько 32 каналов, для одновременной передачи голоса и данных. См. раздел [*базовый интерфейс Rate (Анд-ISDN)*](b-tapgloss.md), [*интерфейс основной частоты (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.interactive_voice_response_ivr_application_tapgloss"></span><span id="TAPI2.INTERACTIVE_VOICE_RESPONSE_IVR_APPLICATION_TAPGLOSS"></span>**Приложение для интерактивного речевого ответа (IVR)**
</dt> <dd>

Приложение телефонии, которое вызывает телефонный вызов через иерархию меню, предоставляет голосовой ответ, собирает входные данные голоса и данных и выполняет другие операции от имени вызывающего или спонсора программы. См. раздел [*веб-телефония*](w-tapgloss.md).

</dd> <dt>

<span id="tapi2.inverse_multiplexing_tapgloss"></span><span id="TAPI2.INVERSE_MULTIPLEXING_TAPGLOSS"></span>**Обратная мультиплексирование**
</dt> <dd>

Процесс, в ходе которого пропускная способность позволяет передавать больше информации одновременно в тех же временных интервалах, чем данные из одного сигнала, переданного по одному каналу. Это достигается путем разбиения одного высокоскоростного канала на несколько сигналов, передающих несколько сигналов по нескольким средствам, которые работают с более низкой скоростью, чем исходный сигнал, а затем переобъединять отдельные передаваемые части в исходный сигнал на основе исходной ставки. См. раздел [*пропускная способность*](b-tapgloss.md).

</dd> <dt>

<span id="tapi2.ip_multicast_tapgloss"></span><span id="TAPI2.IP_MULTICAST_TAPGLOSS"></span>**Многоадресная рассылка IP**
</dt> <dd>

Многоадресная рассылка IP — это схема для одновременной отправки одних и тех же данных на несколько компьютеров. Сеть с включенными маршрутизаторами, поддерживающими многоадресную рассылку, может передавать такую информацию гораздо быстрее, отправляя пакеты только один раз между маршрутизаторами. Дополнительные сведения см. в статье [об Конференц-телефонии для встреч](./about-rendezvous-ip-telephony-conferencing.md).

</dd> <dt>

</dt> <dd>

См. раздел *интегрированная служба Digital Network (ISDN)*.

</dd> </dl>
