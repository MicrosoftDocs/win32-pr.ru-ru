# <a name="audclnt_sessionflags_xxx-constants"></a>\_ \_ Константы аудклнт сессионфлагс XXX

\_ \_ Константы аудклнт сессионфлагс XXX указывают характеристики сеанса звука, связанного с потоком. Клиент может указать эти параметры во время инициализации потока с помощью параметра *стреамфлагс* метода [**Иаудиоклиент:: Initialize**](/windows/desktop/api/Audioclient/nf-audioclient-iaudioclient-initialize) .



| Константа/значение                                                                                                                                                                                                                                                                                                                   | Описание                                                                                                                                                                                                                                                                                                      |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="AUDCLNT_SESSIONFLAGS_EXPIREWHENUNOWNED"></span><span id="audclnt_sessionflags_expirewhenunowned"></span><dl> <dt>**Аудклнт \_ СЕССИОНФЛАГС \_ експиревхенуновнед**</dt> <dt>0x10000000</dt> </dl>                       | Срок действия сеанса истекает, если нет связанных потоков и владеющих объектами управления сеансами ссылки.<br/>                                                                                                                                                                                       |
| <span id="AUDCLNT_SESSIONFLAGS_DISPLAY_HIDE"></span><span id="audclnt_sessionflags_display_hide"></span><dl> <dt>**Аудклнт \_ СЕССИОНФЛАГС \_ отобразить \_ Скрыть**</dt> <dt>0x20000000</dt> </dl>                                     | Элемент управления "Громкость" скрывается в пользовательском интерфейсе микшера громкости при создании сеанса звука. Если сеанс, связанный с потоком, уже существует до того, как [**иаудиоклиент:: Initialize**](/windows/desktop/api/Audioclient/nf-audioclient-iaudioclient-initialize) откроет поток, в микшере тома появится регулятор громкости.<br/> |
| <span id="_AUDCLNT_SESSIONFLAGS_DISPLAY_HIDEWHENEXPIRED"></span><span id="_audclnt_sessionflags_display_hidewhenexpired"></span><dl> <dt> **Аудклнт \_ сессионфлагс \_ Отображение \_ хидевхенекспиред**</dt> <dt>0x40000000</dt> </dl> | После истечения срока действия сеанса элемент управления громкостью скрывается в пользовательском интерфейсе микшера тома. <br/>                                                                                                                                                                                                           |
