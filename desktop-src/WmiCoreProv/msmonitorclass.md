# <a name="msmonitorclass-class"></a>Класс Мсмониторкласс

**Мсмониторкласс** является абстрактным базовым классом WMI. Классы, описывающие мониторы экрана видео, наследуют от этого **мсмониторкласс**.

Классы монитора отображения видео, которые наследуют от **мсмониторкласс** , включают:

-   [**вмимонитораналогвидеоинпутпарамс**](wmimonitoranalogvideoinputparams.md)
-   [**вмимониторбасикдисплайпарамс**](wmimonitorbasicdisplayparams.md)
-   [**вмимониторбригхтнесс**](wmimonitorbrightness.md)
-   [**вмимониторбригхтнессевент**](wmimonitorbrightnessevent.md)
-   [**вмимониторбригхтнессмесодс**](wmimonitorbrightnessmethods.md)
-   [**вмимониторколорчарактеристикс**](wmimonitorcolorcharacteristics.md)
-   [**вмимониторконнектионпарамс**](wmimonitorconnectionparams.md)
-   [**вмимонитордескриптормесодс**](wmimonitordescriptormethods.md)
-   [**вмимонитордигиталвидеоинпутпарамс**](wmimonitordigitalvideoinputparams.md)
-   [**вмимониторид**](wmimonitorid.md)
-   [**вмимониторлистедфрекуенциранжес**](wmimonitorlistedfrequencyranges.md)
-   [**вмимониторлистедсуппортедсаурцемодес**](wmimonitorlistedsupportedsourcemodes.md)
