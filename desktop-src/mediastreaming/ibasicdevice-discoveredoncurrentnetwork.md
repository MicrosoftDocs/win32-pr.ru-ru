# <a name="ibasicdevicediscoveredoncurrentnetwork-method"></a>Ибасикдевице: метод:D Исковередонкуррентнетворк

Возвращает значение, указывающее, находится ли устройство в текущей сети.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DiscoveredOnCurrentNetwork(
  [out] boolean *value
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*значение* \[ заполняет\]
</dt> <dd>

Получает указатель на логическое значение, равное **true** , если устройство находится в текущей сети.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
