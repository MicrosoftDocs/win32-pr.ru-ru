# <a name="eaphost-supplicant-callbacks"></a>Обратные вызовы с незапрашивающими сторонами EAPHost

| Обратный вызов                                           | Описание                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**нотификатионхандлер**](/previous-versions/windows/desktop/api) | Прототип обратного вызова, уведомляющий о выпуске, что имеется изменение в состоянии работоспособности (SoH), и требуется повторная проверка подлинности подключения к системе [защиты доступа к сети](/windows/desktop/NAP/network-access-protection-start-page) (NAP). |
