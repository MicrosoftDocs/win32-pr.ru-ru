# <a name="eapmethod-eaphostusercredentials-element"></a>Еапмесод (Еафостусеркредентиалс), элемент

Элемент **еапмесод (еафостусеркредентиалс)** определяет метод, на который указывает ссылка.

``` syntax
<xs:element name="EapMethod"
    type="EapMethodType"
 />
```

Элемент **еапмесод** определяется элементом [**еафостусеркредентиалс**](eaphostusercredentialsschema-eaphostusercredentials-element.md) .

## <a name="requirements"></a>Требования



| Роль | Минимальная поддерживаемая версия ОС |
|------|------------------------------|
| Клиент<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Сервер<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
