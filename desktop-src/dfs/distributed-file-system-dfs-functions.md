# <a name="distributed-file-system-dfs-functions"></a><span data-ttu-id="72bd2-104">Функции распределенная файловая система (DFS)</span><span class="sxs-lookup"><span data-stu-id="72bd2-104">Distributed File System (DFS) Functions</span></span>

<span data-ttu-id="72bd2-105">Функции распределенная файловая система (DFS) предоставляют возможность логически группировать общие ресурсы на нескольких серверах и прозрачно связывать общие ресурсы в одно иерархическое пространство имен.</span><span class="sxs-lookup"><span data-stu-id="72bd2-105">The Distributed File System (DFS) functions provide the ability to logically group shares on multiple servers and to transparently link shares into a single hierarchical namespace.</span></span> <span data-ttu-id="72bd2-106">DFS организует общие ресурсы в сети в структуре древовидная.</span><span class="sxs-lookup"><span data-stu-id="72bd2-106">DFS organizes shared resources on a network in a treelike structure.</span></span>

<span data-ttu-id="72bd2-107">DFS поддерживает *изолированные* пространства имен DFS, те, которые имеют один сервер узла, и пространства имен *на основе домена* с несколькими серверами узлов и высокой доступностью.</span><span class="sxs-lookup"><span data-stu-id="72bd2-107">DFS supports *stand-alone* DFS namespaces, those with one host server, and *domain-based* namespaces that have multiple host servers and high availability.</span></span> <span data-ttu-id="72bd2-108">*Данные ТОПОЛОГИИ* DFS для пространств имен на основе домена хранятся в Active Directory.</span><span class="sxs-lookup"><span data-stu-id="72bd2-108">The DFS *topology data* for domain-based namespaces is stored in Active Directory.</span></span> <span data-ttu-id="72bd2-109">Данные включают корень DFS, ссылки DFS и целевые объекты DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-109">The data includes the DFS root, DFS links, and DFS targets.</span></span>

<span data-ttu-id="72bd2-110">Каждая древовидная структура DFS имеет одну или несколько *корневых целевых объектов*.</span><span class="sxs-lookup"><span data-stu-id="72bd2-110">Each DFS tree structure has one or more *root targets*.</span></span> <span data-ttu-id="72bd2-111">Корневой целевой объект — это сервер узла, на котором выполняется служба DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-111">The root target is a host server that runs the DFS service.</span></span> <span data-ttu-id="72bd2-112">Древовидная структура DFS может содержать одну или несколько *ссылок DFS*.</span><span class="sxs-lookup"><span data-stu-id="72bd2-112">A DFS tree structure can contain one or more *DFS links*.</span></span> <span data-ttu-id="72bd2-113">Каждая ссылка DFS указывает на одну или несколько общих папок в сети.</span><span class="sxs-lookup"><span data-stu-id="72bd2-113">Each DFS link points to one or more shared folders on the network.</span></span> <span data-ttu-id="72bd2-114">Можно добавлять, изменять и удалять ссылки DFS из пространства имен DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-114">You can add, modify and delete DFS links from a DFS namespace.</span></span> <span data-ttu-id="72bd2-115">При удалении последнего целевого объекта, связанного с ссылкой DFS, DFS удаляет ссылку DFS в пространстве имен DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-115">When you remove the last target associated with a DFS link, DFS deletes the DFS link in the DFS namespace.</span></span> <span data-ttu-id="72bd2-116">(В более ранней документации ссылки DFS назывались точками соединения.)</span><span class="sxs-lookup"><span data-stu-id="72bd2-116">(In earlier documentation, DFS links were called junction points.)</span></span>

<span data-ttu-id="72bd2-117">Ссылка DFS может указывать на одну или несколько общих папок. папки называются *целевыми объектами*.</span><span class="sxs-lookup"><span data-stu-id="72bd2-117">A DFS link can point to one or more shared folders; the folders are called *targets*.</span></span> <span data-ttu-id="72bd2-118">Когда пользователи обращаются к ссылке DFS, сервер DFS выбирает набор целевых объектов на основе данных сайта клиента.</span><span class="sxs-lookup"><span data-stu-id="72bd2-118">When users access a DFS link, the DFS server selects a set of targets based on a client's site information.</span></span> <span data-ttu-id="72bd2-119">Клиент получает доступ к первой доступной цели в наборе.</span><span class="sxs-lookup"><span data-stu-id="72bd2-119">The client accesses the first available target in the set.</span></span> <span data-ttu-id="72bd2-120">Это позволяет распределять клиентские запросы между возможными целевыми объектами и обеспечивать доступ пользователей даже в случае сбоя некоторых серверов.</span><span class="sxs-lookup"><span data-stu-id="72bd2-120">This helps to distribute client requests across the possible targets and can provide continued accessibility for users even when some servers fail.</span></span>

<span data-ttu-id="72bd2-121">Приложение может использовать функции DFS для:</span><span class="sxs-lookup"><span data-stu-id="72bd2-121">An application can use the DFS functions to:</span></span>

- <span data-ttu-id="72bd2-124">Добавьте целевые объекты к существующей ссылке DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-124">Add targets to an existing DFS link.</span></span>
- <span data-ttu-id="72bd2-125">Удалите ссылку DFS из корня DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-125">Remove a DFS link from a DFS root.</span></span>
- <span data-ttu-id="72bd2-126">Удалите целевой объект из ссылки DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-126">Remove a target from a DFS link.</span></span>
- <span data-ttu-id="72bd2-127">Просмотр и Настройка сведений о корнях и ссылках DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-127">View and configure information about DFS roots and links.</span></span>

<span data-ttu-id="72bd2-130">Целевые объекты на компьютерах, работающих под управлением Microsoft Windows, могут быть опубликованы в пространстве имен DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-130">Targets on computers that are running Microsoft Windows can be published in a DFS namespace.</span></span> <span data-ttu-id="72bd2-131">Можно также опубликовать любые общие ресурсы сторонних производителей, для которых в пространстве имен DFS доступны перенаправления клиентов.</span><span class="sxs-lookup"><span data-stu-id="72bd2-131">You can also publish any non-Microsoft shares for which client redirectors are available in a DFS namespace.</span></span> <span data-ttu-id="72bd2-132">Однако в отличие от общего ресурса, опубликованного на сервере под Windows Server, он не может разместить корень DFS или предоставить ссылки другим целевым объектам DFS.</span><span class="sxs-lookup"><span data-stu-id="72bd2-132">However, unlike a share that is published on a server that is running Windows Server, they cannot host a DFS root or provide referrals to other DFS targets.</span></span>
