# <a name="omt-indx-guid-attribute"></a>Атрибут ОМТ-Индкс-GUID

Идентификатор индекса для записи таблицы Link-Track-Object-Move.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | ОМТ-Индкс-GUID                                         |
| LDAP-отображаемое имя | омтиндксгуид                                           |
| Размер              | 16 байт                                              |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При каждом изменении ссылки на файл.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID — GUID    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
