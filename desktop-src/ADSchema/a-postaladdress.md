# <a name="postal-address-attribute"></a>Атрибут Postal-Address

Почтовый адрес для объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Postal-Address                                                              |
| LDAP-отображаемое имя | посталаддресс                                                               |
| Размер              | \-                                                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                      |
| Частота обновления  | При создании записи пользователя и при необходимости изменения адреса. |
| Attribute-Id      | 2.5.4.16                                                                    |
| System-ID — GUID    | bf9679fc-0de6-11d0-a285-00aa003049e2                                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                 |
