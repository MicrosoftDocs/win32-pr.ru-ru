# <a name="mssfu-30-max-uid-number-attribute"></a>атрибут Мссфу-30-Max-UID-Number

Содержит максимальное число пользователей, которые могут быть перенесены в домен NIS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Мссфу-30-Max-UID-Number              |
| LDAP-отображаемое имя | msSFU30MaxUidNumber                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.343          |
| System-ID — GUID    | ec998437-d944-4a28-8500-217588adfc75 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
