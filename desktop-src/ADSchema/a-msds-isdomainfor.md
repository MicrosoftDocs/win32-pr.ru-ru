# <a name="ms-ds-is-domain-for-attribute"></a>Для атрибута ms-DS---domain-

Обратная ссылка для [**MS-DS-with-Domain-NC**](a-msds-hasdomainncs.md). Определяет, какие контроллеры домена содержат этот раздел в качестве основного домена.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS--домен — для                     |
| LDAP-отображаемое имя | msDS-Исдомаинфор                        |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1933                 |
| System-ID — GUID    | ff155a2a-44e5-4de0-8318-13a58988de4f    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
