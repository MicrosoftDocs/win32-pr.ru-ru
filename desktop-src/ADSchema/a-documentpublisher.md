|-------------------|---------------------------------------------|
| CN                | документпублишер                           |
| LDAP-отображаемое имя | документпублишер                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.56                  |
| System-ID — GUID    | 170f09d7-eb69-448a-9a30-f1afecfd32d7        |
