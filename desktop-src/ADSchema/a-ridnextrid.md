# <a name="rid-next-rid-attribute"></a>Атрибут RID-Next-RID

Следующий свободный относительный идентификатор в текущем пуле.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | RID-Next-RID                         |
| LDAP-отображаемое имя | риднекстрид                           |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.374               |
| System-ID — GUID    | 6617188c-8f3c-11d0-afda-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
