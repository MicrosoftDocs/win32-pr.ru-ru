# <a name="membernisnetgroup-attribute"></a>атрибут Мемберниснетграуп

Содержит список сетевых групп, входящих в эту сетевую группу.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мемберниснетграуп                    |
| LDAP-отображаемое имя | мемберниснетграуп                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.13                     |
| System-ID — GUID    | 0f6a17dc-53e5-4be8-9442-8f3ce2f9012a |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
