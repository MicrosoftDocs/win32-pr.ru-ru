# <a name="company-attribute"></a>Атрибут Организации

Имя компании пользователя.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Company                                                                          |
| LDAP-отображаемое имя | company                                                                          |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.2.146                                                           |
| System-ID — GUID    | f0f8ff88-1191-11d0-a060-00aa006c33ed                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
