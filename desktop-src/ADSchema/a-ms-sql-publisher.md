# <a name="ms-sql-publisher-attribute"></a>Атрибут MS-SQL-Publisher

Имя базы данных издателя для репликации.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Издатель MS-SQL-Publisher                            |
| LDAP-отображаемое имя | Издатель mS-SQL-Publisher                            |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке репликации.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID — GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
