# <a name="super-scopes-attribute"></a>Атрибут Super-Scopes

Этот атрибут используется для объединения всех областей, используемых в классе DHCP, в одну сущность.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Super-Scopes                         |
| LDAP-отображаемое имя | Суперобласти                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.710               |
| System-ID — GUID    | 963d274b-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
