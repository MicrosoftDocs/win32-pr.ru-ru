# <a name="acs-permission-bits-attribute"></a>ACS-разрешения — атрибут BITS

Разрешает исходящий трафик многоадресной рассылки для данного пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-разрешения — биты                  |
| LDAP-отображаемое имя | акспермиссионбитс                    |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| System-ID — GUID    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
