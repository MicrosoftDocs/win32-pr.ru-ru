# <a name="ipprotocolnumber-attribute"></a>атрибут Иппротоколнумбер

Содержит раздел с картой Protocols, в котором хранится уникальный номер, идентифицирующий протокол.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | иппротоколнумбер                     |
| LDAP-отображаемое имя | иппротоколнумбер                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.17                     |
| System-ID — GUID    | ebf5c6eb-0e2d-4415-9670-1081993b4211 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
