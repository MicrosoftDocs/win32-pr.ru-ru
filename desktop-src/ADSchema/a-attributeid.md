# <a name="attribute-id-attribute"></a>Атрибут Attribute-ID

Уникальный OID X. 500 для идентификации атрибута.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Attribute-ID                                                    |
| LDAP-отображаемое имя | attributeID                                                     |
| Размер              | \-                                                              |
| Привилегия обновления  | Администратор схемы                                            |
| Частота обновления  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-ID — GUID    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
