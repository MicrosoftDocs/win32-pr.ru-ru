# <a name="netboot-guid-attribute"></a>Атрибут нетбут-GUID

Бездисковая Загрузка: встроенный GUID компьютера. Соответствует MAC-адресу сетевого адаптера компьютера.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Нетбут — GUID                                          |
| LDAP-отображаемое имя | нетбутгуид                                           |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| System-ID — GUID    | 3e978921-8c01-11d0-afda-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
