# <a name="ms-wmi-int8max-attribute"></a>атрибут MS-WMI-int8Max

Максимальное значение для объекта целочисленного параметра WMI 64-bit.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Max                       |
| LDAP-отображаемое имя | Мсвми — Int8Max                        |
| Размер              | 8 байт                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1633              |
| System-ID — GUID    | e3d8b547-003D-4946-a32b-dc7cedc96b74 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
