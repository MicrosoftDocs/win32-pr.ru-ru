# <a name="change-rid-master-extended-right"></a>Change-RID-главное расширенное право

Расширенное право, необходимое для изменения владельца роли FSMO "хозяин относительных идентификаторов".



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Change-RID — master                    |
| Display-Name | Изменить хозяина RID                    |
| Rights-GUID  | d58d5f36-0a98-11d1-adbb-00c04fd8d5cd |
