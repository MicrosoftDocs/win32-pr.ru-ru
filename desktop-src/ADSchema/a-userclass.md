# <a name="userclass-attribute"></a>userClass, атрибут

Тип атрибута userClass указывает категорию пользователя компьютера.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| LDAP-отображаемое имя | userClass                                   |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID — GUID    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
