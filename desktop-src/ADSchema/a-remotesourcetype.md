# <a name="remote-source-type-attribute"></a>Атрибут Remote-Source-Type

Тип указателя на внешний объект.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Удаленный источник-тип                   |
| LDAP-отображаемое имя | ремотесаурцетипе                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.108               |
| System-ID — GUID    | bf967a15-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
