# <a name="uidnumber-attribute"></a>атрибут Уиднумбер

Содержит целое число, уникально идентифицирующее пользователя в домене администратора.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | уиднумбер                            |
| LDAP-отображаемое имя | уиднумбер                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.0                      |
| System-ID — GUID    | 850fcc8f-9c6b-47e1-b671-7c654be4d5b3 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
