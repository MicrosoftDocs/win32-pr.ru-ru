# <a name="is-ephemeral-attribute"></a>Атрибут Is-Ephemeral

Атрибут с **временным** использованием в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Is-Ephemeral                         |
| LDAP-отображаемое имя | Эфемерный                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1212              |
| System-ID — GUID    | f4c453f0-c5f1-11d1-bbcb-0080c76670c0 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
