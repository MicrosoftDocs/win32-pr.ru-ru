# <a name="is-single-valued-attribute"></a>Атрибут имеет значение с одним значением

Если **значение — true**, этот атрибут может хранить только одно значение.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Является однозначным                     |
| LDAP-отображаемое имя | isSingleValued                       |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор схемы                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.33                |
| System-ID — GUID    | bf967992-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
