# <a name="send-as-extended-right"></a>Send-As расширенное право

Exchange right: разрешает отправку почты в качестве почтового ящика.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Send-As                              |
| Display-Name | Отправить как                              |
| Rights-GUID  | ab721a54-1e2f-11d0-9819-00aa0040529b |
