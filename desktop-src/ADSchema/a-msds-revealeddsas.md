# <a name="ms-ds-revealed-dsas-attribute"></a>Атрибут ms-DS-выявил-DSA

Обратная ссылка для [**MS-DS-раскрытых пользователей**](a-msds-revealedusers.md). Определяет, какой RODC содержит секрет пользователя.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-выявил-DSA                     |
| LDAP-отображаемое имя | msDS-Ревеаледдсас                       |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1930                 |
| System-ID — GUID    | 94f6f2ac-c76d-4b5e-b71f-f332c3e93c22    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
