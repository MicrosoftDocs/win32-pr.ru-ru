# <a name="texture3dsamplegrad-methods"></a>Методы Texture3D:: Самплеград

Выбор текстуры с помощью градиента, чтобы повлиять на способ вычисления расположения образца.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                       | Описание                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплеград (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения образца.<br/>                                                                                                             |
| [**Самплеград (S, float, float, float, int, float)**](t3d-samplegrad-s-float-float-float-int-float-.md)           | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод).<br/>                                     |
| [**Самплеград (S, float, float, float, int, float, uint)**](t3d-samplegrad-s-float-float-float-int-float-uint-.md) | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
