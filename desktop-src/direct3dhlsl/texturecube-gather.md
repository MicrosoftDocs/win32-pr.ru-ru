# <a name="texturecubegather-methods"></a>Методы Текстурекубе:: собрать

Возвращает четыре значения шаг текселя, которые будут использоваться в операции линейной фильтрации.

Дополнительные сведения, описывающие базовую инструкцию DXBC, см. в документации по [gather4](./gather4--sm5---asm-.md) .

### <a name="overload-list"></a>Список перегрузок



| Метод                                                     | Описание                                                                                                              |
|:-----------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------|
| [**Сбор данных (S, float)**](dx-graphics-hlsl-to-gather.md)      | Выбирает текстуру и возвращает четыре образца (только красный компонент), которые используются для интерполяции билинейной.<br/> |
| [**Сбор данных (S, float, uint)**](tcube-gather-s-float-uint-.md) | Выбирает текстуру и возвращает все четыре компонента вместе с состоянием операции.<br/>                      |
