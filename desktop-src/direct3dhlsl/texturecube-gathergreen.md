# <a name="texturecubegathergreen-methods"></a>Методы Текстурекубе:: Гасергрин

Возвращает зеленые компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                               | Описание                                                                                                                                       |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасергрин (S, float, uint)**](tcube-gathergreen-s-float-uint-.md)  | Возвращает зеленые компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
