# <a name="quadreadacrossdiagonal-function"></a>Функция Куадреадакроссдиагонал

Возвращает заданное локальное значение, которое считывается с диагонального противоположного полосы в данном Quad.

## <a name="syntax"></a>Синтаксис


``` syntax
<type> QuadReadAcrossDiagonal(
   <type> localValue
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*локалвалуе* 
</dt> <dd>

Запрошенный тип.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Заданное локальное значение, которое считывается с диагонального противоположного полосы в данном Quad.
