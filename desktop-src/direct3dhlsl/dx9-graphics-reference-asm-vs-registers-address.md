# <a name="address-register"></a>Регистр адресов

Регистр a0 — это регистр адресов. Один регистр доступен в версии VS 1 \_ \_ . Регистр адресов, обозначенный как a0. x в VS \_ 1 \_ 1, может использоваться как смещение целого числа со знаком для относительной адресации в файле регистра констант. Для версий VS \_ 2 \_ и выше все четыре компонента (. x,. y,. z,. w) доступны для относительной адресации.


```
c[a0.x + n]
```



Невозможно прочитать регистр адресов с помощью шейдера вершин, он может использоваться только для относительной адресации регистра констант. При чтении значений за пределами допустимого диапазона будут возвращены значения (0,0, 0,0, 0,0, 0,0). Регистр адресов может быть назначен только для инструкций [MOV-VS](mov---vs.md) . Если число с плавающей запятой перемещается в целочисленный регистр, происходит преобразование «цикл-к-ближайшей».

Все шейдеры должны инициализировать регистр адресов, прежде чем использовать его. В сравнении с версиями \_ \_ и 2 0 и выше инструкция [МОВА-VS](mova---vs.md) может переместить значение с плавающей запятой в регистр адресов.

|------------------------|------|------|-------|------|------|-------|
| Регистр адресов       | x    | x    | x     | x    | x    | x     |
