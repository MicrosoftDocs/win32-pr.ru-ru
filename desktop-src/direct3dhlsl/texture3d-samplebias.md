# <a name="texture3dsamplebias-methods"></a>Методы Texture3D:: Самплебиас

Выбор текстуры после применения значения смещения к уровню mipmap.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                           | Описание                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплебиас (S, float, float, int)**](dx-graphics-hlsl-to-samplebias.md)                          | Выбор текстуры после применения значения смещения к уровню mipmap.<br/>                                                                                                             |
| [**Самплебиас (S, float, float, int, float)**](t3d-samplebias-s-float-float-int-float-.md)           | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод).<br/>                                     |
| [**Самплебиас (S, float, float, int, float, uint)**](t3d-samplebias-s-float-float-int-float-uint-.md) | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
