# <a name="rcp-sm5---asm"></a>rcp (SM5-ASM)

Обратная на уровне компонентов.



| rcp No \[ \_ \] dest \[ . Mask \] , \[ - \] src0 \[ \_ ABS \] \[ . свиззле\] |
|------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                                         |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результатов<br/> *конечный адрес*  =  **1.0 f**  /  *src0*.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[\]число, которое нужно взять за обратную величину.<br/>                             |



 

## <a name="remarks"></a>Примечания

Используйте эту инструкцию для обратной величины с низкой точностью, независимо от строгих требований деления.

Максимальная относительная ошибка — 2-21. (Погрешность ошибки только соответствует РСК)

В следующей таблице показаны результаты, полученные при выполнении инструкции с различными классами чисел.



|        |          |        |             |        |        |             |        |          |         |
|--------|----------|--------|-------------|--------|--------|-------------|--------|----------|---------|
| *src*  | **-INF** | **-F** | **— денорма** | **-0** | **+0** | **+ денорма** | **+ F** | **+ INF** | **Не число** |
| *dest* | -0       | -F     | -inf        | -inf   | +inf   | +inf        | + F     | +0       | Не число     |



 

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |
