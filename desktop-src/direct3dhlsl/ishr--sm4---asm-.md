# <a name="ishr-sm4---asm"></a>ишр (SM4-ASM)

Арифметический сдвиг вправо (расширение знака).



| ишр dest \[ . Mask \] , src0 \[ . свиззле \] , src1. Выбор \_ компонента |
|--------------------------------------------------------------|



 



| Элемент                                                            | Описание                                             |
|-----------------------------------------------------------------|---------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] содержит результат операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] содержит значение для сдвига.<br/>     |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] содержит сумму сдвига.<br/>            |



 

## <a name="remarks"></a>Примечания

Эта инструкция выполняет арифметический сдвиг для каждого 32-разрядного значения в *src0* прямо за пределами целого числа без знака, предоставленного числом битов ЛСБ 5 (диапазон 0-31) в *src1. Select \_ ,* реплицирует значение бита 31. Результат 32-бит на компонент помещается в *dest*. Счетчик — это скалярное значение, применяемое ко всем компонентам.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
