## <a name="remarks"></a><span data-ttu-id="2baf1-130">Примечания</span><span class="sxs-lookup"><span data-stu-id="2baf1-130">Remarks</span></span>


```
[branch] switch(a)
{
    case 0:
        return 0; 
    case 1:
        return 1; 
    case 2:
        return 3; 
    default:
        return 6; 
}
```



<span data-ttu-id="2baf1-131">Эквивалентно следующему:</span><span class="sxs-lookup"><span data-stu-id="2baf1-131">Is equivalent to:</span></span>


```
[branch] if( a == 2 )
    return 3;
else if( a == 1 )
    return 1;
else if( a == 0 )
    return 0;
else
    return 6;
```



<span data-ttu-id="2baf1-132">Ниже приведены примеры использования атрибутов управления потоком форцекасе и Call.</span><span class="sxs-lookup"><span data-stu-id="2baf1-132">Here are example usages of forcecase and call flow control attributes:</span></span>

[forcecase] switch(a)
{
    case 0:
        return 0; 
    case 1:
        return 1; 
    case 2:
        return 3; 
    default:
        return 6; 
}

[call] switch(a)
{
    case 0:
        return 0; 
    case 1:
        return 1;

        return 3; 
    default:
        return 6; 
}
