# <a name="itof-sm4---asm"></a>итоф (SM4-ASM)

Преобразование целого числа со знаком в плавающую точку.



| итоф dest \[ . Mask \] , \[ - \] src0 \[ . свиззле\] |
|-------------------------------------------|



 



| Элемент                                                            | Описание                                             |
|-----------------------------------------------------------------|---------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] содержит результат операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] содержит значение для преобразования.<br/>        |



 

## <a name="remarks"></a>Примечания

Эта инструкция по преобразованию целого числа со знаком в число с плавающей запятой предполагает, что *src0* содержит 32-разрядное целое число с подписью 4. После выполнения инструкции *dest* будет содержать кортеж из четырех точек с плавающей запятой.

Преобразование выполняется для каждого компонента.

Если значение целочисленного ввода слишком велико для точного представления в формате с плавающей запятой, округление до ближайшего четного режима настоятельно рекомендуется, но не является обязательным.

Необязательный модификатор отрицания для исходного операнда перед выполнением арифметической операции принимает 2 дополнения.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
