# <a name="ine-sm4---asm"></a>строка (SM4-ASM)

Сравнение неравенства целочисленных значений в объекте на уровне компонента.



| Строка dest \[ . Mask \] , src0 \[ . свиззле \] , src1 \[ . свиззле\] |
|-------------------------------------------------------|



 



| Элемент                                                            | Описание                                                   |
|-----------------------------------------------------------------|---------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результата операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] содержит значение, сравниваемое с *src1*.<br/>    |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] содержит значение, сравниваемое с *src0*.<br/>    |



 

## <a name="remarks"></a>Примечания

Выполняет сравнение целых чисел (*src0* ! = *src1*) для каждого компонента и записывает результат в *dest*.

Если сравнение истинно, то для этого компонента возвращается значение 0xFFFFFFFF. В противном случае возвращается 0x0000000

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
