# <a name="texture2darraygatheralpha-methods"></a>Методы Texture2DArray:: Гасералфа

Возвращает альфа-компоненты четырех шаг текселя значений [**Texture2DArray**](sm5-object-texture2darray.md), которые будут использоваться в операции линейной фильтрации.

| Метод                                                                                                          | Описание                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасералфа (S, float, int)**](sm5-object-texture2darray-gatheralpha.md)                                        | Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.<br/>                                 |
| [**Гасералфа (S, float, int, uint)**](t2darray-gatheralpha-s-float-int-uint-.md)                                  | Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, а также состояние сопоставления плиток.<br/> |
| [**Гасералфа (S, float, int2, int2, int2, int2)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-.md)            | Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.<br/>                                 |
| [**Гасералфа (S, float, int2, int2, int2, int2, uint)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-uint-.md)  | Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, а также состояние сопоставления плиток.<br/> |
