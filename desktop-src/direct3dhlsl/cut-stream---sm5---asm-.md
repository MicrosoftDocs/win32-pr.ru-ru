# <a name="cut_stream-sm5---asm"></a>вырезать \_ поток (SM5-ASM)

Инструкция шейдера геометрии, которая завершает текущую топологию примитива в указанном потоке, если в нее были переданы вершины, и запускает новую топологию типа, объявленного шейдером геометрии в этом потоке.



| вырезать \_ поток стреаминдекс |
|-------------------------|



 



| Элемент                                                                                                               | Описание                         |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------|
| <span id="streamIndex"></span><span id="streamindex"></span><span id="STREAMINDEX"></span>*стреаминдекс*<br/> | \[в \] индексе потока.<br/> |



 

## <a name="remarks"></a>Remarks

При выполнении этой инструкции все ранее выпущенные топологии по вызову шейдера Geometry завершаются. Если для предыдущей простой топологии не хватает вершин, они отбрасываются. Поскольку единственными доступными выходными топологиями шейдера Geometry являются поинтлист, линестрип и трианглестрип, нет ни одного из оставшихся вершин.

*стреаминдекс* должно быть непосредственным значением \[ 0.. 3 \] для объявленного потока.

После завершения предыдущей топологии, если таковая имеется, эта инструкция приводит к началу новой топологии с использованием топологии, объявленной в качестве выходных данных для шейдера Geometry.

### <a name="restrictions"></a>Ограничения

-   Эта инструкция применяется только к шейдеру Geometry.
-   **Вырезать \_ поток** может отображаться любое количество раз в шейдере Geometry, в том числе в элементе управления потоком.
-   Если шейдер геометрии завершается и порождены вершины, топология, которую они создают, завершается, как если бы инструкция **Вырезать \_ поток** выполнялась в качестве последней инструкции.
-   Если потоки не объявлены, необходимо использовать функцию [Cut](cut--sm4---asm-.md) вместо **Вырезать \_ поток**.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        |      |        | X        |       |         |
