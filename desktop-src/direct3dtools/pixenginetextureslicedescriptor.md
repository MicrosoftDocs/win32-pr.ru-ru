# <a name="span-idvspixenginepixenginetextureslicedescriptorspanpixenginetextureslicedescriptor-structure"></a><span id="vspixengine.pixenginetextureslicedescriptor"></span>Структура Пиксенгинетекстуреслицедескриптор

Представляет описание среза текстуры.

## <a name="syntax"></a>Синтаксис


```C++
} PixEngineTextureSliceDescriptor;
```

## <a name="members"></a>Члены

**width**  
Ширина (число выборок на оси X) среза.

**height**  
Высота (число выборок на оси Y) среза.

**блоккровбитес**  
Число байтов в каждой строке в каждом блоке.

**offset**  
Смещение среза в пределах всей текстуры.
