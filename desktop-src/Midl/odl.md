# <a name="odl-attribute"></a>odl - атрибут

MKTYPLIB требовал атрибут **\[ ODL \]** для интерфейсов ODL. Компилятор MIDL не требует наличия атрибута **\[ \] ODL** ; он распознается только для совместимости с более старыми **ODL** -файлами.

> [!Note]  
> Средство Mktyplib.exe устарело. Вместо этого используйте компилятор MIDL.
