# <a name="local-attribute"></a><span data-ttu-id="be563-104">local - атрибут</span><span class="sxs-lookup"><span data-stu-id="be563-104">local attribute</span></span>

<span data-ttu-id="be563-105">**\[ Локальный \]** атрибут указывает компилятору MIDL, что интерфейс или функция не являются удаленными.</span><span class="sxs-lookup"><span data-stu-id="be563-105">The **\[local\]** attribute specifies to the MIDL compiler that an interface or function is not remote.</span></span>

``` syntax
[ 
    local 
    [, interface-attribute-list] 
] 
interface interface-name
{
}

[ 
    object, 
    uuid(string-uuid), 
    local [, interface-attribute-list] 
]
interface interface-name
{
}

[ local [, function-attribute-list] ] function-declarator ;
```

## <a name="remarks"></a><span data-ttu-id="be563-123">Комментарии</span><span class="sxs-lookup"><span data-stu-id="be563-123">Remarks</span></span>

<span data-ttu-id="be563-124">**\[ Локальный \]** атрибут может применяться к отдельным функциям или к интерфейсу в целом.</span><span class="sxs-lookup"><span data-stu-id="be563-124">The **\[local\]** attribute can be applied to individual functions or to the interface as a whole.</span></span>

<span data-ttu-id="be563-125">При использовании в заголовке интерфейса **\[ локальный \]** атрибут позволяет использовать компилятор MIDL в качестве генератора заголовков.</span><span class="sxs-lookup"><span data-stu-id="be563-125">When used in the interface header, the **\[local\]** attribute allows you to use the MIDL compiler as a header generator.</span></span> <span data-ttu-id="be563-126">Компилятор не создает заглушки для всех функций и не гарантирует, что заголовок может быть передан.</span><span class="sxs-lookup"><span data-stu-id="be563-126">The compiler does not generate stubs for any functions and does not ensure that the header can be transmitted.</span></span>

<span data-ttu-id="be563-127">Для интерфейса RPC **\[ локальный \]** атрибут нельзя использовать одновременно с **\[** [](uuid.md) **\]** атрибутом UUID.</span><span class="sxs-lookup"><span data-stu-id="be563-127">For an RPC interface, the **\[local\]** attribute cannot be used at the same time as the **\[**[**uuid**](uuid.md)**\]** attribute.</span></span> <span data-ttu-id="be563-128">В заголовке интерфейса должен присутствовать либо **\[ UUID \]** , либо **\[ Local \]** . выбранный вариант должен быть ровно один раз.</span><span class="sxs-lookup"><span data-stu-id="be563-128">Either **\[uuid\]** or **\[local\]** must be present in the interface header, and the one you choose must occur exactly once.</span></span>

<span data-ttu-id="be563-130">При использовании в отдельной функции **\[ локальный \]** атрибут обозначает локальную процедуру, для которой заглушки не создаются.</span><span class="sxs-lookup"><span data-stu-id="be563-130">When used in an individual function, the **\[local\]** attribute designates a local procedure for which no stubs are generated.</span></span> <span data-ttu-id="be563-131">Использование **\[ Local \]** в качестве атрибута функции — это расширение Майкрософт для устройства DCE IDL.</span><span class="sxs-lookup"><span data-stu-id="be563-131">Using **\[local\]** as a function attribute is a Microsoft extension to DCE IDL.</span></span> <span data-ttu-id="be563-132">Поэтому этот атрибут недоступен при компиляции с помощью параметра MIDL [**/ОСФ**](-osf.md) .</span><span class="sxs-lookup"><span data-stu-id="be563-132">Therefore this attribute is not available when you compile using the MIDL [**/osf**](-osf.md) switch.</span></span>

<span data-ttu-id="be563-133">Обратите внимание, что интерфейс без атрибутов можно импортировать в базовый IDL-файл.</span><span class="sxs-lookup"><span data-stu-id="be563-133">Note that an interface without attributes can be imported into a base IDL file.</span></span> <span data-ttu-id="be563-134">Однако интерфейс должен содержать только типы данными без процедур.</span><span class="sxs-lookup"><span data-stu-id="be563-134">However, the interface must contain only datatypes with no procedures.</span></span> <span data-ttu-id="be563-135">Если в интерфейсе содержится даже одна процедура, необходимо указать локальный или UUID.</span><span class="sxs-lookup"><span data-stu-id="be563-135">If even one procedure is contained in the interface, a local or UUID attribute must be specified.</span></span>

## <a name="examples"></a><span data-ttu-id="be563-136">Примеры</span><span class="sxs-lookup"><span data-stu-id="be563-136">Examples</span></span>

``` syntax
/* IDL file #1 */ 
[
    local
] 
interface local_procs 
{ 
    void MyLocalProc(void);
} 
 
/* IDL file #2 */ 
[
    object, 
    uuid(12345678-1234-1234-123456789ABC), 
    local
] 
interface local_object_procs : IUnknown
{ 
    void MyLocalObjectProc(void);
} 
 
/* IDL file #3 */ 
[
    uuid(87654321-1234-1234-123456789ABC)
] 
interface mixed_procs 
{ 
    [local] void MyLocalProc(void); 
    HRESULT MyRemoteProc([in] short sParam); 
}
```
