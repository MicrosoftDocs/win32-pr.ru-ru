# <a name="amd64-switch"></a>/AMD64, параметр

Параметр **/AMD64** направляет компилятор MIDL для создания файлов заглушек или файла библиотеки типов для расширенной 64-разрядной среды Micro Devices.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Комментарии

> [!Note]  
> Параметр **/x64** , описанный в справке командной строки MIDL.exe, является устаревшим. Вместо этого следует использовать **/AMD64** .
