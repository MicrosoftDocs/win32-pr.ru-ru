# <a name="readermodeinfo-structure"></a>Структура РЕАДЕРМОДЕИНФО

\[**реадермодеинфо** поддерживается в Windows XP с пакетом обновления 2 (SP2). В последующих версиях он может не поддерживаться.\]

Содержит сведения, необходимые для инициализации функции [**дореадермоде**](doreadermode.md) .

## <a name="syntax"></a>Синтаксис


```C++
typedef struct tagReaderModeInfo {
  UINT                       cbSize;
  HWND                       hwnd;
  DWORD                      fFlags;
  LPRECT                     prc;
  PFNREADERSCROLL            pfnScroll;
  PFNREADERTRANSLATEDISPATCH fFlags;
  LPARAM                     lParam;
} READERMODEINFO, *PREADERMODEINFO;
```
