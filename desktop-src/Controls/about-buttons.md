# <a name="about-buttons"></a>О кнопках

Кнопка — это элемент управления, который пользователь может щелкнуть, чтобы предоставить входные данные приложению.

Существует несколько типов кнопок и один или несколько стилей кнопок для различения кнопок одного типа. Пользователь нажимает кнопку с помощью мыши или клавиатуры. При нажатии кнопки обычно изменяется внешний вид и состояние отображения (например, значение снято с очистки). Система, кнопка и приложение взаимодействуют при изменении внешнего вида и состояния кнопки. Кнопка может отправить сообщения в свое родительское окно, а родительское окно может посылать сообщения кнопке. Некоторые кнопки закрашены системой, а некоторые — приложением. Кнопки могут использоваться отдельно или в группах и могут отображаться с текстом или без него (метка). Они принадлежат классу окна кнопки.
