# <a name="tvm_setautoscrollinfo-message"></a>\_Сообщение TVM сетаутоскроллинфо

Задает сведения, используемые для определения характеристик автоматической прокрутки. Это сообщение можно отправить явно или с помощью макроса [**\_ сетаутоскроллинфо TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_setautoscrollinfo) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Указывает количество пикселей в секунду. Смещение для прокрутки делится на *wParam* , чтобы определить общую длительность автоматической прокрутки.

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

Задает интервал времени перерисовки. Перерисовка при каждом еласпед интервале, пока элемент не будет прокручивается в представление. Заданному параметром *wParam*, выполняется вычисление расположения элемента и происходит перерисовка. Задайте это значение, чтобы создать плавную прокрутку.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true**.

## <a name="remarks"></a>Remarks

Сведения о прокрутке используются для прокрутки невидимого элемента на представление. Элемент управления должен иметь расширенный стиль [**телевизоры \_ \_ аутохскролл**](tree-view-control-window-extended-styles.md) . Дополнительные сведения о расширенных стилях см. в разделе Tree-View Control Extended Styles.
