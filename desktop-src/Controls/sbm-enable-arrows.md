# <a name="sbm_enable_arrows-message"></a>СБМ \_ включить \_ сообщение со стрелками

Приложение отправляет сообщение **\_ \_ со стрелками СБМ** , чтобы включить или отключить одну или обе стрелки элемента управления "полоса прокрутки".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает, включены ли стрелки полосы прокрутки или отключены, и указывает, какие стрелки включены или отключены. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                   | Значение                                                                                                    |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| <span id="ESB_DISABLE_BOTH"></span><span id="esb_disable_both"></span><dl> <dt>**\_отключение \_ обеих**</dt> </dl> | Отключает обе стрелки на полосе прокрутки.<br/>                                                           |
| <span id="ESB_DISABLE_DOWN"></span><span id="esb_disable_down"></span><dl> <dt>**\_Отключение ESB \_**</dt> </dl> | Отключает стрелку вниз на вертикальной полосе прокрутки.<br/>                                               |
| <span id="ESB_DISABLE_LTUP"></span><span id="esb_disable_ltup"></span><dl> <dt>**\_лтуп отключение \_ ESB**</dt> </dl> | Отключает стрелку влево на горизонтальной полосе прокрутки или на стрелке вверх на вертикальной полосе прокрутки.<br/>    |
| <span id="ESB_DISABLE_LEFT"></span><span id="esb_disable_left"></span><dl> <dt>**Отключение ESB, \_ \_ слева**</dt> </dl> | Отключает стрелку влево на горизонтальной полосе прокрутки.<br/>                                             |
| <span id="ESB_DISABLE_RTDN"></span><span id="esb_disable_rtdn"></span><dl> <dt>**\_ртдн отключение \_ ESB**</dt> </dl> | Отключает стрелку вправо на горизонтальной полосе прокрутки или на стрелке вниз на вертикальной полосе прокрутки.<br/> |
| <span id="ESB_DISABLE_UP"></span><span id="esb_disable_up"></span><dl> <dt>**\_Отключение ESB \_**</dt> </dl>       | Отключает стрелку вверх на вертикальной полосе прокрутки.<br/>                                                 |
| <span id="ESB_ENABLE_BOTH"></span><span id="esb_enable_both"></span><dl> <dt>**\_Активация ESB \_**</dt> </dl>    | Включает обе стрелки на полосе прокрутки.<br/>                                                            |



 

</dd> <dt>

*lParam*

## <a name="return-value"></a>Возвращаемое значение

Если сообщение прошло удачно, возвращается значение **true**. в противном случае — **false**.
