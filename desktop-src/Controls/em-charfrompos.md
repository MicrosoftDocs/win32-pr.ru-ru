# <a name="em_charfrompos-message"></a>\_Сообщение ЧАРФРОМПОС EM

Получает сведения о символе, ближайшее к заданной точке в клиентской области элемента управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Координаты точки в клиентской области элемента управления. Координаты находятся в единицах экрана и задаются относительно левого верхнего угла клиентской области элемента управления.

**Элементы управления Rich Edit:** Указатель на [**точку**](/previous-versions//dd162807(v=vs.85)) , которая содержит координаты горизонтальной и вертикальной структуры.

**Изменить элементы управления:** [**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) содержит горизонтальную координату. [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) содержит вертикальную координату.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

**Элементы управления Rich Edit:** Возвращаемое значение указывает Отсчитываемый от нуля индекс символа, расположенный ближе к указанной точке. Возвращаемое значение указывает последний символ в элементе управления "поле ввода", если указанная точка находится за последним символом в элементе управления.

**Изменить элементы управления:** [**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) Указывает отсчитываемый от нуля индекс символа, ближайшего к указанной точке. Этот индекс задается относительно начала элемента управления, а не от начала строки. Если указанная точка находится за последним символом в элементе управления "поле ввода", возвращаемое значение обозначает последний символ в элементе управления. [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) Указывает отсчитываемый от нуля индекс строки, содержащей символ. Для однострочных элементов управления редактирования это значение равно нулю. Индекс указывает разделитель строк, если указанная точка выходит за пределы последнего видимого символа в строке.

## <a name="remarks"></a>Комментарии

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).

Если точка передается в **EM \_ Чарфромпос** как *lParam* , а точка находится вне границ элемента управления "поле ввода", то *lResult* имеет значение (65535, 65535).
