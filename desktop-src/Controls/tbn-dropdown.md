# <a name="tbn_dropdown-notification-code"></a>\_Код уведомления раскрывающегося списка ТБН

Посылается элементом управления ToolBar, когда пользователь нажимает кнопку раскрывающегося списка. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TBN_DROPDOWN

    lpnmtb = (LPNMTOOLBAR) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтулбар**](/windows/win32/api/commctrl/ns-commctrl-nmtoolbara) , содержащую сведения об этом коде уведомления. Для этого кода уведомления допустимы только элементы **HDR** и **Член iItem** этой структуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений:



| Код возврата                                                                                          | Описание                                                                       |
|------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| <dl> <dt>**ТБДДРЕТ \_ по умолчанию**</dt> </dl>      | Раскрывающийся список был обработан.<br/>                                             |
| <dl> <dt>**ТБДДРЕТ \_ по умолчанию**</dt> </dl>    | Раскрывающийся список не был обработан.<br/>                                         |
| <dl> <dt>**ТБДДРЕТ \_ треатпрессед**</dt> </dl> | Раскрывающийся список был обработан, но рассматривает кнопку как обычную кнопку.<br/> |



 

## <a name="remarks"></a>Remarks

> [!Note]  
> Кнопки раскрывающегося списка могут быть простыми ([**\_ раскрывающийся список бтнс**](toolbar-control-and-button-styles.md) ), отображать стрелку рядом с изображением кнопки (стиль [**бтнс \_ вхоледропдовн**](toolbar-control-and-button-styles.md) ) или отображать стрелку, отделенную от изображения ([**тбстиле \_ \_ дравддарровс**](toolbar-extended-styles.md) Style). Если используется разделенная стрелка, \_ раскрывающийся список ТБН отправляется только в том случае, если пользователь нажимает на кнопку стрелку. Если пользователь нажимает на главную часть кнопки, отправляется сообщение [**\_ команды WM**](/windows/desktop/menurc/wm-command) с идентификатором кнопки, точно так же, как и стандартная кнопка. Для двух других стилей кнопки dropdown \_ раскрывающийся список ТБН отправляется, когда пользователь щелкает любую часть кнопки.
