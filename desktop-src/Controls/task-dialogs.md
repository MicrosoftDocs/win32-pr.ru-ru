# <a name="task-dialog"></a>Диалоговое окно задачи

В этом разделе содержатся сведения об элементах программирования, используемых в диалоговом окне задачи. *Диалоговое окно задачи* аналогично, а намного более гибкое, чем простое окно сообщения.

### <a name="overviews"></a>Разделы общих сведений



| Раздел                                           | Содержимое                                            |
|-------------------------------------------------|-----------------------------------------------------|
| [О диалоговых окнах задач](task-dialogs-overview.md) | Описывает элементы диалогового окна задачи.<br/> |



 

### <a name="functions"></a>Функции



| Раздел                                                  | Содержимое                                                                                                                                                                                                                                                                                                                                                                                              |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**таскдиалог**](/windows/desktop/api/Commctrl/nf-commctrl-taskdialog)                       | Создает, отображает и работает с диалоговым окном задачи. В диалоговом окне задачи содержатся определяемые приложением текст и заголовок, значки и любое сочетание стандартных кнопок push-уведомлений. Эта функция не поддерживает регистрацию функции обратного вызова для получения уведомлений.<br/>                                                                                                                |
| [*таскдиалогкаллбаккпрок*](/windows/win32/api/commctrl/nc-commctrl-pftaskdialogcallback) | Определяемая приложением функция, используемая с функцией [**таскдиалогиндирект**](/windows/desktop/api/Commctrl/nf-commctrl-taskdialogindirect) . Он получает сообщения из диалогового окна задачи при возникновении различных событий.<br/> Тип **пфтаскдиалогкаллбакк** определяет указатель на эту функцию обратного вызова. [*Таскдиалогкаллбаккпрок*](/windows/win32/api/commctrl/nc-commctrl-pftaskdialogcallback) — это заполнитель для имени функции, определяемой приложением.<br/> |
| [**таскдиалогиндирект**](/windows/desktop/api/Commctrl/nf-commctrl-taskdialogindirect)       | Создает, отображает и работает с диалоговым окном задачи. Диалоговое окно задачи содержит значки, сообщения, заголовок, проверку, флажки, ссылки на команды, кнопки и переключатели, установленные приложением. Эта функция может зарегистрировать функцию обратного вызова для получения сообщений уведомления.<br/>                                                                                                               |
