# <a name="em_setcuebanner-message"></a>\_Сообщение СЕТКУЕБАННЕР EM

Задает текстовую подсказку или подсказку, отображаемую элементом управления "поле ввода" для запроса сведений о пользователе.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

**Значение true** , если баннер подсказки должен отображаться даже в том случае, если элемент управления "поле ввода" имеет фокус; в противном случае — **значение false**. **False** — это поведение по умолчанию, которое закрывается при щелчке элемента управления пользователем.

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

Указатель на строку в Юникоде, содержащую текст, отображаемый как текстовая подсказка.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение завершается с ошибкой, возвращается **значение true**. В противном случае возвращается **значение false**.

## <a name="remarks"></a>Комментарии

В элементе управления "поле ввода", используемом для начала поиска, может отображаться текст "введите Поиск здесь" в виде текстового подсказки. Когда пользователь щелкает текст, текст исчезает, и пользователь может ввести.

Нельзя задать заголовок подсказки в многострочном элементе управления Edit или в элементе управления Rich Edit.

> [!Note]  
> Чтобы использовать этот API, необходимо предоставить манифест, задающий Comclt32.dll версии 6,0. Дополнительные сведения о манифестах см. в разделе [Включение визуальных стилей](cookbook-overview.md).
