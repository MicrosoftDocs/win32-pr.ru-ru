<span data-ttu-id="9b7f4-166">*Переключатель* (также называемый переключателем) состоит из круглой кнопки и определяемой приложением метки, значка или точечного рисунка, указывающего на выбор, который пользователь может сделать, нажав кнопку.</span><span class="sxs-lookup"><span data-stu-id="9b7f4-166">A *radio button* (also called option button) consists of a round button and an application-defined label, icon, or bitmap that indicates a choice the user can make by selecting the button.</span></span> <span data-ttu-id="9b7f4-167">Приложение обычно использует переключатели в поле группы, чтобы пользователь выбрал один из набора взаимосвязанных, но взаимоисключающих параметров.</span><span class="sxs-lookup"><span data-stu-id="9b7f4-167">An application typically uses radio buttons in a group box to enable the user to choose one of a set of related but mutually exclusive options.</span></span>

<span data-ttu-id="9b7f4-175">Переключатели размещаются в группах, и в любое время можно проверить только одну кнопку в группе.</span><span class="sxs-lookup"><span data-stu-id="9b7f4-175">Radio buttons are arranged in groups, and only one button in the group can be checked at any time.</span></span> <span data-ttu-id="9b7f4-176">Если флаг [**WS \_ Group**](/windows/desktop/winmsg/window-styles) установлен для любого переключателя, эта кнопка является первой кнопкой в группе, а все кнопки, которые следуют за ней непосредственно в последовательности табуляции (но не имеют флага **\_ группы WS** ), являются частью своей группы.</span><span class="sxs-lookup"><span data-stu-id="9b7f4-176">If the [**WS\_GROUP**](/windows/desktop/winmsg/window-styles) flag is set for any radio button, that button is the first button in a group, and all buttons that follow it immediately in the tab order (but do not themselves have the **WS\_GROUP** flag) are part of its group.</span></span> <span data-ttu-id="9b7f4-177">Если ни один из переключателей не имеет флага **\_ группы WS** , все переключатели в диалоговом окне обрабатываются как одна группа.</span><span class="sxs-lookup"><span data-stu-id="9b7f4-177">If no radio buttons have the **WS\_GROUP** flag, all the radio buttons in the dialog box are treated as a single group.</span></span>
