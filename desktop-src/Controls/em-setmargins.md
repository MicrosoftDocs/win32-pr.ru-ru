# <a name="em_setmargins-message"></a>\_Сообщение СЕТМАРГИНС EM

Задает ширину левого и правого полей для элемента управления "поле ввода". Сообщение перерисовывает элемент управления для отражения новых полей. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Поля, которые нужно задать. Этот параметр может принимать одно или несколько следующих значений.



| Значение                                                                                                                                                            | Значение                                                                                                                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="EC_LEFTMARGIN"></span><span id="ec_leftmargin"></span><dl> <dt>**EC \_ LEFTMARGIN**</dt> </dl>    | Задает левое поле.<br/>                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="EC_RIGHTMARGIN"></span><span id="ec_rightmargin"></span><dl> <dt>**EC \_ RIGHTMARGIN**</dt> </dl> | Задает правое поле.<br/>                                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="EC_USEFONTINFO"></span><span id="ec_usefontinfo"></span><dl> <dt>**EC \_ усефонтинфо**</dt> </dl> | **Элементы управления Rich Edit:** Задает ширину левого и правого поля, вычисленную с использованием метрик текста текущего шрифта элемента управления. Если для элемента управления не задан шрифт, поля задаются равными нулю. Параметр *lParam* игнорируется. <br/> **Изменить элементы управления:** Значение **EC \_ усефонтинфо** нельзя использовать в параметре *wParam* . Его можно использовать только в параметре *lParam* .<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) задает новую ширину левого поля в пикселях. Это значение игнорируется, если *wParam* не включает **EC \_ LEFTMARGIN**.

**Изменение элементов управления и Расширенное редактирование 3,0 и более поздних версий:** [**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) может указать значение **EC \_ усефонтинфо** , чтобы задать для левого поля меньшую ширину, вычисленную с использованием метрик текста текущего шрифта элемента управления. Если для элемента управления не задан шрифт, поле устанавливается в нулевое значение.

[**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) указывает новую ширину правого поля в пикселях. Это значение игнорируется, если *wParam* не включает **EC \_ RIGHTMARGIN**.

**Изменение элементов управления и Расширенное редактирование 3,0 и более поздних версий:** [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) может указать значение **EC \_ усефонтинфо** , чтобы задать для правого поля меньшую ширину, вычисленную с использованием метрик текста текущего шрифта элемента управления. Если для элемента управления не задан шрифт, поле устанавливается в нулевое значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не возвращает значение.

## <a name="remarks"></a>Remarks

**Изменить элементы управления:** В параметре *wParam* нельзя использовать **EC \_ усефонтинфо** , но его можно использовать в параметре *lParam* .

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Все версии расширенного редактирования поддерживают использование **EC \_ усефонтинфо** в параметре *wParam* . Однако в параметре *lParam* поддерживается использование **EC \_ Усефонтинфо** только в Microsoft Rich Edit 3,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
