# <a name="em_redo-message"></a>\_Сообщение о возврате EM

Отправляет сообщение **о \_ возврате EM** в форматируемый элемент управления, чтобы повторить следующее действие в очереди повторов элемента управления.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если операция **повтора** завершилась удачно, возвращаемое значение будет ненулевым.

Если операция **повтора** завершается неудачно, возвращаемое значение равно нулю.

## <a name="remarks"></a>Remarks

Чтобы определить, есть ли какие либо действия в очереди повторов элемента управления, отправьте сообщение [**EM \_ канредо**](em-canredo.md) .
