# <a name="udm_setrange32-message"></a>\_Сообщение SETRANGE32 UDM

Задает 32-разрядный диапазон элемента управления "вверх/вниз".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Целочисленное значение со знаком, представляющее новую нижнюю границу диапазона управления "вверх/вниз".

</dd> <dt>

*lParam* 
</dt> <dd>

Целочисленное значение со знаком, представляющее новый верхний предел диапазона управления "вверх/вниз".

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение для этого сообщения не используется.
