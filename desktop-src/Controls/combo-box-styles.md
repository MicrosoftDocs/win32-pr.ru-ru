# <a name="combo-box-styles"></a>Стили полей со списками

Чтобы создать поле со списком с помощью функции [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) или [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , укажите класс ComboBox, соответствующие константы стиля окна и сочетание следующих стилей поля со списком.



| Константа                                                                                                                                                                              | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="CBS_AUTOHSCROLL"></span><span id="cbs_autohscroll"></span><dl> <dt>**CBS \_ аутохскролл**</dt> </dl>                   | Автоматически прокручивает текст в элементе управления "поле ввода" вправо, когда пользователь вводит символ в конце строки. Если этот стиль не задан, допускается только текст, который умещается в прямоугольной области.<br/>                                                                                                                                                                                                                                                                                  |
| <span id="CBS_DISABLENOSCROLL"></span><span id="cbs_disablenoscroll"></span><dl> <dt>**CBS \_ дисабленоскролл**</dt> </dl>       | Показывает отключенную вертикальную полосу прокрутки в списке, если поле не содержит достаточно элементов для прокрутки. Если этот стиль не задан, полоса прокрутки будет скрыта при недостаточном количестве элементов в списке.<br/>                                                                                                                                                                                                                                                                                          |
| <span id="CBS_DROPDOWN"></span><span id="cbs_dropdown"></span><dl> <dt>**\_раскрывающийся список CBS**</dt> </dl>                            | Аналогично \_ простому CBS, за исключением того, что список не отображается, если только пользователь не выберет значок рядом с элементом управления редактирования.<br/>                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="CBS_DROPDOWNLIST"></span><span id="cbs_dropdownlist"></span><dl> <dt>**DROPDOWNLIST для CBS \_**</dt> </dl>                | Аналогично \_ раскрывающимся списку CBS, за исключением того, что элемент управления "поле ввода" заменяется статическим текстовым элементом, который отображает текущее выделение в списке.<br/>                                                                                                                                                                                                                                                                                                                                                     |
| <span id="CBS_HASSTRINGS"></span><span id="cbs_hasstrings"></span><dl> <dt>**CBS \_ хасстрингс**</dt> </dl>                      | Указывает, что рисуемое владельцем поле со списком содержит элементы, состоящие из строк. Поле со списком поддерживает память и адрес для строк, поэтому приложение может использовать сообщение [**\_ жетлбтекст CB**](cb-getlbtext.md) для получения текста для определенного элемента.<br/> Сведения о проблемах со специальными возможностями см. в разделе [предоставление Owner-Drawn элементов поля со списком](/windows/desktop/WinAuto/exposing-owner-drawn-combo-box-items)<br/>                                                                                               |
| <span id="CBS_LOWERCASE"></span><span id="cbs_lowercase"></span><dl> <dt>**\_нижний регистр CBS**</dt> </dl>                         | Преобразует в нижний регистр весь текст как в поле выбора, так и в списке. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="CBS_NOINTEGRALHEIGHT"></span><span id="cbs_nointegralheight"></span><dl> <dt>**CBS \_ ноинтегралхеигхт**</dt> </dl>    | Указывает, что размер поля со списком будет точно совпадать с размером, заданным приложением при создании поля со списком. Как правило, система изменяет размер поля со списком таким образом, чтобы не отображались частичные элементы.<br/>                                                                                                                                                                                                                                                                                        |
| <span id="CBS_OEMCONVERT"></span><span id="cbs_oemconvert"></span><dl> <dt>**CBS \_ оемконверт**</dt> </dl>                      | преобразует текст, указанный в поле со списком, с Windows набора символов в набор символов OEM, а затем обратно в Windows кодировку. это обеспечивает правильное преобразование символов, когда приложение вызывает функцию [**чартуем**](/windows/desktop/api/winuser/nf-winuser-chartooema) для преобразования строки Windows в поле со списком в символы OEM. Этот стиль наиболее удобен для полей со списком, содержащих имена файлов, и применяется только к полям со списком, созданным с помощью меню "простой" или "CBS" CBS \_ \_ .<br/> |
| <span id="CBS_OWNERDRAWFIXED"></span><span id="cbs_ownerdrawfixed"></span><dl> <dt>**CBS \_ овнердравфиксед**</dt> </dl>          | Указывает, что владелец списка отвечает за рисование его содержимого и то, что элементы в списке имеют одинаковую высоту. Окно-владелец получает сообщение [**WM \_ меасуреитем**](wm-measureitem.md) при создании поля со списком и сообщение [**WM \_ DRAWITEM**](wm-drawitem.md) при изменении визуального аспекта поля со списком.<br/>                                                                                                                                     |
| <span id="CBS_OWNERDRAWVARIABLE"></span><span id="cbs_ownerdrawvariable"></span><dl> <dt>**CBS \_ овнердраввариабле**</dt> </dl> | Указывает, что владелец списка отвечает за рисование его содержимого и то, что элементы в списке имеют переменную высоту. Окно-владелец получает сообщение [**WM \_ меасуреитем**](wm-measureitem.md) для каждого элемента в поле со списком при создании поля со списком и сообщения [**\_ DRAWITEM WM**](wm-drawitem.md) при изменении визуального аспекта поля со списком.<br/>                                                                                                       |
| <span id="CBS_SIMPLE"></span><span id="cbs_simple"></span><dl> <dt>**\_простой CBS**</dt> </dl>                                  | Отображает список в любое время. Текущее выделение в списке отображается в элементе управления "Поле ввода".<br/>                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="CBS_SORT"></span><span id="cbs_sort"></span><dl> <dt>**\_Сортировка CBS**</dt> </dl>                                        | Автоматически сортирует строки, добавленные в список.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="CBS_UPPERCASE"></span><span id="cbs_uppercase"></span><dl> <dt>**\_нижний регистр CBS**</dt> </dl>                         | Преобразует в верхний регистр весь текст как в поле выбора, так и в списке.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                          |
