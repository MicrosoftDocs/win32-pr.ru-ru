# <a name="em_setrectnp-message"></a>\_Сообщение СЕТРЕКТНП EM

Задает [прямоугольник форматирования](about-edit-controls.md) для многострочного элемента управления "поле ввода". Сообщение **EM \_ сетректнп** идентично сообщению [**\_ SETRECT EM**](em-setrect.md) , за исключением того, что **EM \_ сетректнп** *не перерисовывает* окно элемента управления редактированием.

Прямоугольник форматирования — это ограничивающий прямоугольник, в который элемент управления рисует текст. Ограничивающий прямоугольник не зависит от размера окна элемента управления "поле ввода".

Это сообщение обрабатывается только многострочными элементами управления Edit. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

**Расширенное редактирование 3,0 и более поздних версий:** Указывает, содержит ли новый прямоугольник абсолютные или относительные координаты. Нулевое значение обозначает абсолютные координаты. Значение 1 указывает смещения относительно текущего прямоугольника форматирования. (Смещение может быть положительным или отрицательным.)

**Изменить элементы управления:** Этот параметр не используется и должен быть равен нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , указывающую новые размеры прямоугольника. Если этот параметр имеет **значение NULL**, то для прямоугольника форматирования задаются значения по умолчанию.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не возвращает значение.

## <a name="remarks"></a>Remarks

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 3,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
