# <a name="sb_simple-message"></a>\_Простое сообщение SB

Указывает, отображает ли окно состояния простой текст или отображает все фрагменты окна, заданные предыдущим сообщением [**SB \_ сетпартс**](sb-setparts.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Флаг отображаемого типа. Если этот параметр имеет **значение true**, в окне отображается простой текст. Если значение равно **false**, отображается несколько частей.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение не используется.

## <a name="remarks"></a>Комментарии

Если окно состояния изменяется с непростого на простой или наоборот, окно немедленно перерисовывается.
