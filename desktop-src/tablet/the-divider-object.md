# <a name="the-divider-object"></a>Объект разделителя

Технология планшетных ПК включает анализ макета рукописного ввода, который сочетает рукописный ввод и рисование. Объект- [**Разделитель**](inkdivider-class.md) предоставляет функции анализа макета, которые классифицируют и группируют штрихи в разные структурные элементы.

В [образце раздела "разделитель рукописного ввода](ink-divider-sample.md) " показано, как использовать объект- [**Разделитель**](inkdivider-class.md) с объектами [**рукописного**](inkdisp-class.md) ввода для выполнения анализа макета рукописного текста.
