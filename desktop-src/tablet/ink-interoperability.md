# <a name="ink-interoperability"></a>Взаимодействие рукописного ввода

Сохраняемость и взаимодействие являются двумя важнейши важными функциями, особенно для приложений, использующих рукописный ввод. Чтобы обеспечить сохранение и взаимодействие рукописного ввода, приложение должно предоставить пользователям следующие возможности:

-   Сохраняйте и открывайте свои документы с полной точностью рукописных данных.
-   Перемещение данных в другие приложения на основе Microsoft Windows и из него с помощью буфера обмена.

В следующих разделах подробно описывается совместимость рукописного ввода.

-   [Форматы данных рукописного ввода](ink-data-formats.md)
-   [Важные сценарии взаимодействия](important-interoperability-scenarios.md)
-   [Форматы сохраняемости](persistence-formats.md)
-   [Выбор поддерживаемых форматов](choosing-which-formats-to-support.md)
-   [Стандартные блоки](building-blocks.md)
-   [Сохранение рукописного ввода в формате HTML](storing-ink-in-html.md)

В этом документе предполагается подробное изучение буфера обмена. Дополнительные сведения о буфере обмена см. в [документации по буферу обмена](/windows/desktop/dataxchg/clipboard).
