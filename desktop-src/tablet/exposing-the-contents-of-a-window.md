# <a name="exposing-the-contents-of-a-window"></a>Предоставление содержимого окна

Приложение должно предоставлять доступ к содержимому пользовательских классов окон, использующих Microsoft Active Accessibility. Все окно должно рассматриваться как пользовательский элемент управления, который является контейнером для других стандартных, настраиваемых или пользовательских элементов управления.

Дополнительные сведения о предоставлении содержимого настраиваемых окон с помощью Active Accessibility см. в разделе [Специальные возможности](/windows/desktop/accessibility) .
