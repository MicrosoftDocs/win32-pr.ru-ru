# <a name="about-language-models"></a>О языковых моделях

Распознаватели рукописного ввода используют определенный набор правил для данного языка в качестве основного контекста для распознавания слов. Эти правила называются моделями языка. Языковая модель реализована распознавателем и не может быть непосредственно изменена разработчиком с помощью пакета средств разработки программного обеспечения Microsoft Windows XP Tablet PC Edition (SDK). Однако существуют способы уточнения контекста для конкретных полей в приложении. К этим методам относятся следующие:

-   Области ввода
-   фактоидс
-   Словари
-   Списки фраз
