<span data-ttu-id="44bcd-111">Свойство [**Strokes**](/windows/desktop/api/msinkaut15/nf-msinkaut15-iinkdivisionunit-get_strokes) объекта **дивисионунит** содержит подмножество штрихов в объекте **дивисионресулт** , соответствующем этому элементу.</span><span class="sxs-lookup"><span data-stu-id="44bcd-111">The [**Strokes**](/windows/desktop/api/msinkaut15/nf-msinkaut15-iinkdivisionunit-get_strokes) property of the **DivisionUnit** object contains the subset of the strokes in the **DivisionResult** object that correspond to this element.</span></span> <span data-ttu-id="44bcd-112">Поскольку элементы рукописного ввода существуют для разных уровней детализации, коллекции **штрихов** для различных элементов могут перекрываться.</span><span class="sxs-lookup"><span data-stu-id="44bcd-112">Because handwriting elements exist for different levels of detail, the **Strokes** collections for different elements may overlap.</span></span> <span data-ttu-id="44bcd-113">В частности, сегмент распознавания разделяет штрихи на линию и блокирует его, а линия разделяется на блок, частью которого он является.</span><span class="sxs-lookup"><span data-stu-id="44bcd-113">Specifically, a recognition segment shares strokes with the line and block it is part of, and a line shares strokes with the block it is part of.</span></span>
